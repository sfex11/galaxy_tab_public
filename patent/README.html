핵심 청구항

항유형핵심 내용
1독립항커밋먼트 → Paillier 동형 오퍼 → 암호화 합산 → 수렴 판정 → Groth16 IR 증명 → ε-Pareto 증명 (6단계)
2종속Paillier 암호 + 암호화 상태 비교
3종속Groth16, O(1) 증명 크기·검증
4종속PCN 3계층 분리 (Proof-Communication-Negotiation)
5종속N≥3, ε→0 NP-hard 근사
6종속크로스 플랫폼 신원 이식 — 제1→제2 플랫폼 ZKP 자격 증명
7종속이식 증명 π_transfer 상세 (Groth16, ~200 bytes)
8시스템컴퓨터 기록 매체
