### MathDuels: Evaluating LLMs as Problem Posers and Solvers (2026-04-26)

LLM이 문제 출제자와 해결자를 동시에 수행하는 셀프플레이 벤치마크로, 3단계 적대적 문제 생성 파이프라인(기초→난이도 상향→적대적 프롬프팅)을 통해 정적 벤치마크의 천장 성능 문제를 구조적으로 해결한다.

### MathDuels: Evaluating LLMs as Problem Posers and Solvers (2026-04-27)

### Verifier-Backed Hard Problem Generation for Mathematical Reasoning (2026-05-10)

VHG는 MathDuels의 적대적 프롬프팅 파이프라인에 대한 구조적 비판을 제공한다. MathDuels가 '어떻게 더 어려운 문제를 만들까'에 집중했다면, VHG는 '어떻게 유효하면서 어려운 문제를 만들까'로 문제를 재정의하며, 두 접근의 결합이 천장 성능 문제 해결에 필수적임을 시사한다.
