# 의미론적 C/R 원시 연산

**생성일**: 2026-05-02

## 정의

체크포인트/복원(C/R) 연산을 상태의 의미론적 의존 그래프 구조를 인식하는 실행 환경 원시 연산으로 재설계하는 메커니즘. 전체 스냅샷이 아닌 의미 단위 기반 증분 스냅샷을 생성하고, 복원 시 의미론적 일관성을 원시 연산 수준에서 보장함으로써 상태 전이 검증을 애플리케이션 프로토콜에서 실행 환경 계층으로 위임한다. 기존 checkpoint-restore-efficiency-spectrum이 효율성 스펙트럼을 다룬다면, 이 개념은 C/R 연산 자체의 시맨틱스 재정의—즉 C/R이 '무엇을 잘라내고 붙이는가'를 의미론적으로 이해하는지 여부—를 다룬다.
