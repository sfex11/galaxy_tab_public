연구 배경과 동기

경도인지장애(MCI)의 진단 난제

  • 정상 노화와 치매 사이의 전이 단계
  • 조기 발견이 치료 개입에 결정적
  • 기존 방식: 구조화된 평가 · 신경영상 — 비용 높고 빈도 낮음
  • 일상 행동의 점진적 변화 포착 불가

언어를 인지 바이오마커로

  • 어휘 다양성, 유창성, 정지 패턴이 인지 상태와 상관
  • 자연스러운 대화 데이터가 제약된 임상 과제보다 풍부
  • 비침습적 · 확장 가능한 평가 수단
  • 시간적 변화를 포착하는 데 유리
기존 접근법은 예측에 집중하지만, 디지털 트윈은 개인별 행동 모델링을 통해 지속적이고 전체적인 인지 모니터링을 가능하게 함

스타일로메트릭 증강

개인의 말하기 패턴을 포착하기 위해 전사본에 시간적 말 동적 특징을 주석으로 추가

PAUSE 토큰

NONE SHORT MED LONG VLONG

정지 시간을 5단계로 범주화하여 인지 상태와의 상관관계 모델링

TEMPO 토큰

SLOW MED FAST VFAST

말의 속도와 리듬을 4단계로 인코딩

SFT 데이터 포맷

System: 모방 과제 정의 프롬프트

User: 질문 + 메타데이터 (참가자 ID, 나이, 성별, 면접 날짜, 주제)

Assistant: 참가자 응답 + 스타일로메트릭 토큰

학습 목표 함수

ℒSFT = −𝔼(x,y)[log pθ(y|x)]

x=(q, m): 질문과 메타데이터, y: 응답

베이스 모델: GPT-4.1-mini

cVAE 기반 평가자: 아키텍처

생성된 응답의 충실도와 인지 일관성을 평가하기 위한 다중 헤드 조건부 변분 오토인코더

인코더

qϕ(z|q, m, y) — 질문, 메타데이터, 응답을 잠재 표현 z로 매핑

디코더

pθ(y|q, m, z) — 잠재 공간에서 응답 재구성

MCI 예측 헤드

ŝ = f(z) — 잠재 표현에서 인지 점수 직접 예측

잠재 공간 파라미터화

평균 μ 와 로그 분산 log σ² 로 잠재 공간 정의

챌린저 역할

표준 유사도 메트릭과 달리:

  • 스타일적 충실도 평가
  • 인지 관련성 동시 평가
  • 생성 응답이 실제 언어·인지 패턴과 얼마나 일치하는지 측정

손실 함수 설계

세 가지 손실을 결합하여 스타일 재구성, 잠재 공간 정규화, 인지 점수 예측을 동시 최적화

재구성 손실

ℒrec = 𝔼[‖y − ŷ‖²]

원본 응답과 재구성 응답 간의 MSE — 언어적 충실도 측정

KL 발산 손실

ℒKL = −½𝔼[1+log σ²−μ²−exp(log σ²)]

잠재 공간을 사전 분포에 가깝게 정규화 — 과적합 방지

MCI 예측 손실

ℒMCI = 𝔼[‖s − ŝ‖²]

실제 MoCA 점수와 예측 점수 간 MSE — 인지 예측 정확도

ℒ = ℒrec + ℒKL + λ · ℒMCI

λ: 인지 예측의 중요도 가중치 — MCI 손실에 더 높은 가중치 할당

데이터셋: I-CONECT

데이터셋 특징

  • 고령층(75세 이상) 대상 무작위 임상시험 기반
  • 인지 정상(CN) 및 MCI 참가자 포함
  • 제약된 과제가 아닌 자연스러운 종단적 대화
  • 전사본, 상호작용 동적, 메타데이터 등 다중 모달

선정 기준

약 70명 참가자 중 세션 수가 가장 많은 5명 선정

충분한 종단적 데이터 확보를 위함

MoCA란?

몬트리올 인지 평가 — 기억, 주의력, 언어, 실행 기능 등을 평가하는 30점 만점 선별 도구. 점수가 낮을수록 인지 손상이 큼

결론 및 향후 과제

결론

  • 언어 기반 디지털 트윈 프레임워크 최초 제안
  • 스타일로메트릭 단서(PAUSE/TEMPO)를 통한 개인 말하기 패턴 모델링
  • 다중 헤드 cVAE로 충실도와 인지 일관성 동시 평가
  • 베이스라인 GPT 대비 우수한 개인화 성능 확인
  • 집단 수준에서 개인 수준 언어 중심 디지털 트윈으로 패러다임 전환

향후 과제

  • 더 큰 규모 참가자 집단으로 일반화 가능성 검증
  • 음성 prosody, 비언어적 단서 등 다중 모달 통합
  • 장기적 인지 궤적 예측 모델 개발
  • 실제 임상 환경에서의 배치 및 유효성 평가
  • 실시간 인지 건강 모니터링 시스템 구현
언어 기반 디지털 트윈은 확장 가능하고 비침습적인 개인 맞춤형 인지 건강 모니터링을 위한 유망한 접근법
