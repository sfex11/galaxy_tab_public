Phase 1: 앵커 식별 및 초기화

밀집 앵커 검색

Bi-encoder 아키텍처 (DPR 유사)

  • 모든 엔티티 임베딩 사전 계산
  • FAISS 기반 MIPS로 효율적 검색
  • 정확 문자열 매칭 불필요 → 동의어 처리

서브그래프 구성

T-홉 이웃 추출 후 특징 재초기화

𝐡_v = W_𝒱 · E_Q(text(v))

𝐞_r = W_ℛ · E_Q(text(r))

질의 인코더와 동일한 인코더 사용으로 텍스트-그래프 정렬 보장

Phase 2: 순환 소프트 플로우 (RSF) 모듈

T步의 미분 가능한 추론 수행 — 플로우(Flow)와 의도(Intent)의 동시 진화

초기화: π⁽⁰⁾ = one-hot(v_topic), q₀ = v_Q

2.1

나침반 (Compass): 동적 관계 주의 — 질의 상태와 관계 임베딩 간 어텐션으로 방향 탐지

2.2

전파 (Flow): 관계 유도 소프트 플로우 전파 — 연속적 확률 질량 그래프 상 확산

2.3

읽기 (Read): 문맥 집계 — 이웃 노드 정보의 가중 합산

2.4

쓰기 (Write): 미분 가능 질의 갱신 — GRU 기반 순환 질의 업데이터

동적 게이팅 메커니즘: 의미론적 간극 해소의 핵심

문제 상황

의미 신호가 오도될 때 (중간 노드와 질의 간 어휘 중복 없음)

→ 유사도 기반 전파가 유효 경로를 억제

해결 메커니즘

게이트 g_t가 노드 콘텐츠 vs 그래프 위상의 영향력을 적응적으로 조절

→ 구조적 관계만으로 의미론적으로 단절된 노드 탐색 가능

  • 의미 신호가 오도될 때: 순수 그래프 위상에 의존하여 탐색
  • 동음이의어 등 모호성 해소 필요 시: 콘텐츠 편향 재도입
  • 이는 구조-의미 분리(Structure–Semantics Decoupling)을 실현

플로우 희소성 정규화 (Flow Sparsity Regularization)

소프트 확률 분포를 이산적 추론 경로로 수렴시키는 원리적 메커니즘

엔트로피 기반 정규화

각 스텝의 플로우 분포 π⁽ᵗ⁾에 엔트로피 페널티 부과

  • 미분 가능성 유지 (Gumbel-Softmax 기법)
  • 훈련 중 점진적 희소화 유도

보장되는 효과

  • 해석 가능성: 소프트→이산 경로로의 수렴
  • 충실성: 추출된 경로의 구조적 정확성 보장
  • 이산화 없이도 미분 가능한 학습 가능

Phase 3 & 4: 경로 추출·텍스트화 및 LLM 미세조정

경로 추출 & 텍스트화

RSF 모듈의 희소화된 플로우에서 이산 경로 P 추출

P = (v₀, r₁, v₁, ..., r_T, v_T)

예: [Inception, directed_by, Christopher Nolan, won, Academy Award]

자연어 템플릿으로 변환하여 LLM 입력으로 제공

LLM 미세조정

경로 조건부 생성: P(A | Q, P)

  • 그래프 위상에 엄격히 정Ground
  • 환각(Hallucination) 방지
  • 에이전트 루프 불필요 → 추론 효율성 극대화

이론적 보장

정리 2.1

희소성 수렴

엔트로피 정규화 하에서 플로우 분포가 이산 경로로 수렴함을 수학적으로 증명

정리 2.2

의미론적 간극 해소

동적 게이팅이 의미론적으로 단절된 브리지 노드를 통한 확률 전파를 보장함을 증명

정리 2.3

구조적 충실성 보장

추출된 경로가 지식 그래프 상 유효한 삼중항으로 구성됨을 보장

모든 증명은 부록 A에 상세 수록 (A.1–A.3)

실험 설정

벤치마크 데이터셋

  • WebQSP: 1~2홉 질의 중심
  • CWQ: 2~4홉 복합 질의, 의미론적 간극 다수 포함

평가 지표

  • Hits@1, Hits@5 (정확도)
  • F1 점수
  • 추론 효율성 (지연시간, FLOPs)

비교 대상: GraftNet, PullNet, NSM, R-GCN, SubgraphRAG, RoG, GoT 등 다양한 패러다임의 최신 베이스라인

주요 성능 결과

모델 WebQSP Hits@1 CWQ Hits@1 추론 효율성
NSM 69.6 - 높음
RoG 75.2 53.8 낮음
GoT - ~51.0 매우 낮음
RSF-GLLM 경쟁적 경쟁적 매우 높음

LLM 중심·에이전트 방식 대비 차수 단위의 추론 효율성 향상을 유지하면서도 경쟁적 정확도 달성

절제 연구 (Ablation Study)

핵심 구성요소 제거 효과

  • 동적 게이팅 제거 → 의미론적 간극 질의에서 성능 급락
  • 희소성 정규화 제거 → 경로 해석 가능성 상실, F1 하락
  • GRU 질의 업데이터 제거 → 다홉 추론 성능 저하

검색 성능 & 효율성

RSF 모듈 단독 검색 평가:

  • 이산 검색기 대비 유사 또는 우수한 검색 정확도
  • LLM 에이전트 방식 대비 추론 지연시간 크게 단축
  • 의미론적 간극이 큰 질의에서 강인성(Robustness) 우수

결론 및 향후 과제

핵심 기여 요약

  • 의미론적 간극을 미분 가능 KGQA의 핵심 실패 모드로 공식화
  • 동적 구조-의미 분리를 통한 RSF 모듈 제안
  • 엔트로피 기반 희소성 정규화로 해석 가능·충실한 경로 추출 보장
  • 추론-생성 분리 아키텍처로 효율성과 정확도 동시 달성

향후 과제

더 큰 규모의 지식 그래프로 확장, 다양한 도메인(생물의학, 금융)으로의 일반화, RSF 모듈과 LLM의 엔드투엔드 공동 최적화 탐색
