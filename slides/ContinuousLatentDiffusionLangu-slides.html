Continuous Latent Diffusion Language Model

계층적 연속 잠재 공간에서의 확산 언어 모델링

Hongcan Guo, Qinyu Zhao, Yian Zhao, Shen Nie, Rui Zhu, Qiushan Guo, Feng Wang, Tao Yang, Hengshuang Zhao, Guoqiang Wei, Yan Zeng

ByteDance Seed · The University of Hong Kong · ANU · Peking University · Renmin University of China

배경

자기회귀 패러다임의 성공과 한계

자기회귀 모델의 성과

  • 체인 규칙 기반 명확한 학습 목표
  • 대규모 사전학습·오픈 생성·전이학습 주도
  • 현대 언어 모델링의 지배적 접근법

근본적 한계

  • 생성이 고정된 좌→우 순서에 묶임
  • 추론이 본질적으로 순차적 → 효율성 제한
  • 단일 토큰 순서에 대한 강한 귀납적 편향
고품질 텍스트 생성은 반드시 고정된 좌→우 순서와 결합될 필요가 없다.
문제 정의

기존 대안들의 공동 한계

접근법 생성 효율성 표현 학습 확장성 글로벌 의미 모델링
자기회귀 (AR) 순차 추론 비용 토큰 수준 명시적 확률 고정 순서 편향
이산 확산 (LLaDA 등) 다단계 샘플링 비용 이산 중간 상태의 한계 관측 복원에 집중
연속 확산 (기존) 연속 표현 공간 토큰 정렬 복원에 집중 잠재 사전 모델링 부재

세 가지 목표를 동시에 달성하는 통합 프레임워크가 부재

제안

Cola DLM: 핵심 아이디어

확산을 토큰 수준 관측 복원이 아닌 잠재 사전 전송(Latent Prior Transport)으로 사용한다.

계층적 정보 분해

  • 글로벌 의미 조직: 연속 잠재 공간에서 블록-인과 DiT로 사전 모델링
  • 로컬 텍스트 실현: 조건부 디코더로 텍스트 생성

설계 이점

  • 고정 토큰 순서 편향 완화
  • 연속 공간 기하 구조로 의미 압축 지원
  • 블록 내 병렬 계산 가능
  • 다른 연속 모달리티로 자연 확장
이론

확률적 프레임워크

1

계층적 잠재 변수 모델

텍스트 x → 잠재 z 로 인코딩 후, z에 대한 사전 p(z)를 확산 과정으로 학습

2

ELBO와 사전 학습

외부 ELBO를 통해 VAE의 사후와 DiT 사전을 일치시키며, Flow Matching은 사전 풀이기로 작용

3

확률 추정

조건부 및 무조건부 로그-가능도를 CNF 사전 밀도와 디코더를 통해 엄밀하게 추정

Flow Matching은 모델 정의가 아닌 사전 풀이기(Prior Solver)의 역할을 수행한다는 점이 기존 연속 확산 LM과의 결정적 차이이다.
파이프라인

Cola DLM 3단계 워크플로우

1
Text VAE 사전학습
텍스트를 연속 잠재 공간으로 안정적으로 매핑하는 인코더-디코더 학습
2
블록-인과 DiT 사전 학습
잠재 공간에서 블록 간 인과성을 유지하며 블록 내 병렬로 사전 분포 모델링
3
조건부 디코딩
프리픽스 인코딩 → 블록 단위 잠재 생성 → 조건부 텍스트 디코딩

블록-인과 구조란?

텍스트를 블록 단위로 나누어, 블록 간에는 인과적 순서를 유지하되 블록 내부에서는 병렬 처리가 가능한 구조

조건부 디코딩 전략

첫 블록은 프리픽스 조건으로 Clean Conditioning, 이후 블록은 이전 블록 잠재로 조건부 생성

통일적 관점

통일된 마르코프 경로 관점

생성 모델을 확률적 경로의 관점에서 통일적으로 비교할 수 있음:

AR과의 차이: 왜 확산인가?

AR은 이산 확률 객체를 직접 최적화하지만, 확산은 연속 경로를 통해 더 유연한 사전 학습을 가능하게 함

이산 확산과의 차이: 왜 연속인가?

이산 중간 상태는 글로벌 의미 구조 표현에 부적합하며, 연속 공간은 의미 압축과 부드러운 보간을 지원

Plaid와의 차이: 왜 잠재인가?

토큰 정렬 표현 복원이 아닌 잠재 사전 전송을 수행하여, 관측 공간과 모델링 공간을 분리

단순 잠재 역확산 LM과의 차이

Cola DLM은 CNF 사전 기반의 엄밀한 확률 프레임워크를 제공하며, 단순한 인코더-디코더 확산과는 구조적으로 상이

벤치마크 & 베이스라인

엄밀 매칭 베이스라인:

  • AR: ~2B 파라미터 자기회귀 모델
  • LLaDA: ~2B 파라미터 이산 확산 모델

스케일링: 약 2,000 EFLOPs까지 스케일링 곡선 확인

RQ1

잠재 공간의 글로벌 의미 구조 증거

Shift Scaling Law 발견

VAE 사후의 logSNR이 모델 스케일에 따라 체계적으로 이동하는 현상을 관측. 이는 잠재 공간 내에 학습 가능한 공유 의미 인자(Shared Semantic Factor)가 존재함을 정보이론적 관점에서 시사

의미 구조의 함의

  • 잠재 공간이 단순한 재구성 기반이 아닌 의미적 구조를 포착
  • 이 구조를 사전 모델이 학습함으로써 효과적인 생성 가능
  • Cola DLM의 설계 철학(의미-텍스트 분리)을 실증적 지지
RQ2–RQ3

잠재 공간 설계와 확산 과정 분석

RQ2: 잠재 공간 분석

  • 고정 vs 진화 잠재 공간: 진화 공간이 일관적으로 우수
  • 차원성: 적절한 차원 선택이 성능에 결정적
  • 의미 중요도 & 부드러움: 의미적으로 중요한 차원이 부드러운 변화를 보임

RQ3: 확산 과정 소거

  • DiT 블록 크기: 블록 크기 조정이 성능-효율 트레이드오프에 핵심적 역할
  • 노이즈 스케줄: 최적 스케줄은 유효 의미 정보 체제의 보정으로 해석 가능
  • 추론 단계 & CFG: 적절한 스케일에서 안정적 성능 달성
노이즈 스케줄과 logSNR는 동등한 파라미터화이며, 최적 스케줄은 유효 의미 정보 체제(Effective Semantic Information Regime)의 보정으로 이해할 수 있다.
RQ4

스케일링 성능 비교

스케일링 곡선 결과

  • 약 2,000 EFLOPs까지 강한 스케일링 거동 확인
  • 텍스트 생성 품질에서 AR 및 LLaDA 베이스라인과 경쟁력 있는 성능
  • Cola DLM의 효과적인 전체 구성(Effective Configuration) 식별 완료

가능도 vs 생성 품질

Cola DLM에서는 생성 품질과 스케일링이 가능도보다 모델 능력을 더 잘 반영할 수 있다.

Flow Matching이 벡터장 회귀를 최적화하므로, 사전 평균이 참값에서 멀어도 생성 품질은 유지되지만 조건부 PPL은 저하되는 구조적 격차 존재

핵심 통찰

세 가지 지배 곡선과 적용 경계

Rate-Distortion 곡선

잠재 병목의 압축률과 정보 손실 간 트레이드오프

구조적 생성 체제

Cola DLM이 참 생성 메커니즘에 더 잘 부합하는 영역

적용 가능성 경계

잠재 병목이 효과적인 작업 유형의 한계 규명

멀티모달 통합 방향

Text→Text, Text→Image, Image×Text→Text 실험을 통해 이산 텍스트와 연속 모달리티 간 통합 모델링의 구체적 경로 제시

첫 생성 블록의 조건부 전략

Clean Conditioning > Left Padding > Partial Repaint 순으로 성능 우수하며, 이는 Flow Matching 관점에서 수학적으로 설명 가능

한계 & 전망

제한점과 향후 방향

현재 한계

  • 가능도(PPL)와 생성 품질 간 구조적 격차 존재
  • VAE 재구성의 견고성 의존
  • 잠재 공간 압축 한계에 따른 적용 경계
  • 첫 블록 조건부 전략의 민감도

향후 연구 방향

  • 이산 텍스트와 연속 모달리티의 통일 모델
  • 평가 지표의 전환: 가능도 중심 → 생성 품질 중심
  • 모델-환경 상호작용 관점에서의 통일 이해
  • 상태 공간에서 계층적 텍스트 생성으로의 패러다임 전환
Cola DLM은 계층적 연속 잠재 사전 모델링이 엄격한 토큰 수준 언어 모델링에 대한 원칙적 대안(Principled Alternative)임을 확립한다.
