핵심 통찰

LLM의 공간 위험 추론 원리

"LLM은 복귀 옵션이 제한된 기하학적으로 제약된 영역을 식별하여 공간적 위험을 추론한다."

병목 구간 (Bottleneck)

장애물로 둘러싸인 좁은 통로 — 동적 장애물 출현 시 탈출 불가

개방 통로 (Open Corridor)

주변에 장애물이 적은 넓은 공간 — 위험 발생 시 다양한 회피 경로 존재

이 원리를 국소 장애물 인접도 점수로 정형화 → A*의 가중치 패널티로 삽입

방법론

환경 모델

이산 격자 세계

  • N×N 격자 𝒢 (기본 15×15)
  • 각 셀: 자유(0) 또는 차단(1)
  • 4-연결성 이동 (상하좌우)

장애물 설정

  • 정적 장애물: 밀도 ρ ∈ [0.10, 0.30]
  • 동적 장애물: 에이전트 인접 셀에
    확률적 출현 (pdyn = 0.06)
  • 이동 물체 및 센서 노이즈 시뮬레이션

각 시행마다 새로운 랜덤 격자 생성 (seed = 시행 인덱스), 세 계획법이 동일한 격자와 동적 장애물 이벤트 공유

핵심 제안

의미론적 위험 비용 함수 φ(s)

각 자유 셀 s에 대해 8-이웃 내 차단 셀 수 A(s)를 계산하여 통과 패널티 부여:

2.0

병목 구간
A(s) ≥ 3

0.8

중간 위험
A(s) = 2

0.0

개방 통로
A(s) ≤ 1

A* 총 가중치: c(s, s') = 1 + φ(s')

휴리스틱 가중치: w = 1.2 (가중치 A* — 약간의 최적성 양보로 속도 확보)

재계획 트리거: 다음 계획 단계가 새로 출현한 동적 장애물에 의해 차단된 경우

종료 조건: 목표 도달 (성공) 또는 Tmax = 300 단계 초과 (실패)

베이스라인

비교 대상 계획법

BFS + 재계획

  • 최단 비가중 경로 탐색
  • SRAH와 동일한 재계획 트리거 사용
  • 모든 빈 셀을 동등 비용 취급
  • 의미론적 바이어스 없음

Greedy (재계획 없음)

  • 맨해튼 거리만 사용하는 탐욕 탐색
  • 초기 경로에 확정적 커밋
  • 동적 장애물 출현 시 재계획 불가
  • 온라인 적응 능력이 없는 고정 정책 기준선

Greedy는 D* Lite가 해결하려 했던 고전적 실패 모드를 재현하며, 폐루프 재계획의 필수성을 확인하는 역할

분석

계획 오버헤드 vs 복구 능력

시간 오버헤드

SRAH 평균 계획 시간: 2.61 ms

  • BFS 대비 약 3× 높음 (의미론적 바이어스 + 가중치 A*)
  • 표준 제어 주기(10~50 Hz)에서 무시 가능
  • LLM 추론(500~5000 ms)보다 최소 190× 빠름

재계획 빈도

SRAH 재계획: 1.80회 / BFS: 1.66회

  • 약간 높은 재계획 = 위험 경로에 대한 능동적 회피
  • 위험한 경로에 "머무르지 않는" 전략적 선택
  • Greedy의 0회 재계획 = 근본적 적응 불가

핵심 트레이드오프: 경미한 오버헤드(2.61ms)로 유의미한 복구 능력 향상 확보 — 실시간 시스템에 완전히 실용적

정성 분석

경로 품질 분석

SRAH 경로 특성

  • 의미론적 위험 영역(φ(s) > 0)을 회피
  • 개방 통로를 선호하며 약간 더 긴 경로 수용
  • 경로 길이: 32.3 ± 6.2 vs BFS 31.9 ± 5.3
  • 통계적으로 유의미한 경로 연장 없음

BFS 경로 문제

  • 위험한 좁은 통로를 최단 경로로 선택
  • 동적 장애물 출현 시 회복 옵션 부족
  • 재계획 시에도 동일한 패턴 반복
"이 동작은 순수하게 기하학적 비용 신호에서 발현되며, 명시적 장애물 유형 라벨링이 전혀 필요하지 않다."

논의

논의 및 한계

주요 기여

  • LLM 추론 원리를 경량 비용 함수로 증류하는 원칙적 방법 제시
  • 빠르고 해석 가능하며 자원 제한 하드웨어에 배포 가능
  • 런타임 LLM 추론 완전 제거
  • 다양한 환경 난이도에서 일관된 향상

한계점

  • 이산 4-연결 격자 세계로 평가 제한
  • 실제 로봇은 연속 공간·비홀로노믹 제약에서 동작
  • φ(s)의 임계값(A≥3, A=2)에 대한 체계적 튜닝 미흡
  • 실제 로봇 플랫폼에서의 검증 필요
결론

결론 및 향후 방향

"경량화된 LLM 영감 휴리스틱만으로도 자율 로봇 내비게이션에서 측정 가능한 안전성 및 강건성 향상을 얻을 수 있다."

핵심 성과

  • 성공률 62.0% — BFS 대비 9.7% 상대 향상
  • 2.61ms 계획 시간 — 실시간 배포 실용적
  • 경로 길이 증가 없이 안전성 향상
  • 밀도 절제 연구에서 일관된 우위

향후 방향

  • 연속 공간으로의 확장 (RRT*, 비홀로노믹)
  • φ(s) 임계값 자동 튜닝 메커니즘
  • 의미론적 분할(semantic segmentation)과의 결합
  • 실제 로봇 플랫폼 실험 검증
