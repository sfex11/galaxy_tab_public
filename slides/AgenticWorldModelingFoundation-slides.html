개념 파편화 문제

"월드 모델"이라는 용어가 연구 커뮤니티마다 다른 의미로 사용됨.

커뮤니티 "월드 모델"의 의미 평가 기준
강화학습 전이 구조 학습으로 미래 상상 과업 성능 향상
컴퓨터 비전 비디오/3D 생성기 시각적 충실도
언어모델·에이전트 텍스트 기반 시뮬레이션 계획의 유효성
로봇공학 학습된 동역학 Sim-to-Real 전이
과학 AI 대체 모델 + 가설 실험 발견의 질

→ 서로 다른 해석 하의 결과는 비교 불가하며, 개념적 clarity가 시급함.

"Levels × Laws" 분류 체계

두 축으로 구성된 통합 프레임워크 — 400편 이상의 논문, 100개 이상의 대표 시스템 종합

능력 수준 (Levels)
L1
Predictor — 1단계 국소 예측
L2
Simulator — 다단계 행동조건부 롤아웃
L3
Evolver — 증거 기반 모델 자가 수정
지배 법칙 체제 (Laws)
  • 물리적 세계 — 물리 법칙 준수
  • 디지털 세계 — 소프트웨어/GUI 규칙
  • 사회적 세계 — 다자간 전략 상호작용
  • 과학적 세계 — 가설-실험 주기

본 서베이는 월드 모델의 가치를 "하위 에이전트가 내리는 결정의 질"로 정의

L1 Predictor: 국소 마르코프 예측

패턴에서 한 걸음 앞의 예측으로 — 귀납적 사전지식과 No-Free-Lunch 정리의 철학적 배경

1
상태 추론
관측 → 잠재 상태 z
2
순방향 동역학
zₜ, aₜ → zₜ₊₁ (핵심 L1 연산자)
3
관층 디코딩
zₜ₊₁ → 예측 관측
4
역방향 동역학
zₜ, zₜ₊₁ → aₜ 추정
핵심 특성
  • 단일 전이 연산자 학습
  • 마르코프 가정 의존
  • 국소적 예측 정확도 측정
한계
  • 다단계 롤아웃 시 오류 누적
  • 행동 조건부 시뮬레이션 불가
  • 결정에 직접 활용 어려움

L2 Simulator: 결정-가능한 다단계 시뮬레이션

L1을 구성하여 행동 조건부 롤아웃 생성 — 도메인 법칙을 준수해야 "결정에 쓸 만함"

L1 → L2 승격 요구사항
잔여 프레임 문제 해결
행동이 미치지 않는 영역도 정확히 유지
제약 조건 일관성
물리 법칙, 소프트웨어 규칙 등 위배 불가
장기horizon 일관성
오류 누적 없이 긴 시퀀스 생성
개입 민감도
행동 변화가 결과에 올바르게 반영
L2 실패 모드
환각적 일관성
시각적으로 그럴싸하나 물리적으로 불가능
인과 누락
행동과 결과 간 인과관계 상실
체제 전이 실패
학습 범위 밖 상황에서 붕괴

L3 Evolver: 증거 기반 모델 수정

예측이 새로운 증거와 충돌할 때 모델을 자율적으로 수정 — 반증가능성과 패러다임 전환의 철학적 배경

L2 vs L3 핵심 차이
특성L2L3
모델고정적응형
성장없음지속적
증거수동능동적 수집
오류무시/누적수정 트리거
수정 루프 예시
1
잡기 실패 → 동역학 모델 수정
2
서비스 환경 → 전략 수정
3
설치 실패 → 재사용 스킬 생성
4
폐루프 재료 발견

L3 성숙도는 도메인마다 다름: 물리적·디지털 세계는 비교적 진전, 사회적·과학적 세계는 초기 단계. 거버넌스 과제 수반.

네 가지 지배 법칙 체제

각 체제가 월드 모델에 부과하는 제약과 실패 지점이 다름

  • GUI 에이전트

역사적 발전 궤적

수학적 원리에서 생성 혁명까지 — 월드 모델링의 4가지 파동

1
수학적 원리 (~1956)
미분방정식, 물리 모델링, 결정론적 시뮬레이션 — 인간이 법칙을 명시
2
기호적 지능 (1956–1986)
규칙 기반 시스템, 전문가 시스템, 상태 공간 탐색 — 기호로 세계 표현
3
연결주의 부흥 (1986–2020)
신경망 기반 동역학 학습, MBRL, Dreamer — 데이터에서 전이 구조 학습
4
생성 혁명 (2020–현재)
비디오 생성(Sora 등), 대규모 시뮬레이션, 에이전틱 월드 모델 — 능동적 환경 모델링

미해결 문제

능력 수준별·교차 체제별 핵심 과제

L1: 표현과 국소 예측
  • 다모달 표현 통합
  • 복잡한 전이의 근사 한계
  • 비마르코프 환경 처리
L2: 시뮬레이션 충실도와 개입
  • 장기 일관성 보장
  • 희귀 행동에 대한 민감도
  • 분포 외 일반화
L3: 증거 기반 수정과 자가 진화
  • 수정 트리거의 신뢰성
  • 파국적 망각 방지
  • 능동적 증거 수집 전략
교차 체제 공통 과제
  • 체제 전이 시의 견고성
  • 스케일 압력 하의 법칙 준수
  • 안전성과 거버넌스
