웹페이지 생성을 위한
계층적 멀티모달 웹 에이전트

다수의 AIGC 도구를 계층적으로 조율하여
시각적으로 일관된 멀티모달 웹페이지를 자동 생성하는 프레임워크

← → 키 또는 스와이프로 탐색

해결하려는 문제

현재 AI 웹 생성 방식은 텍스트·이미지·비디오·차트를 독립적으로 생성하여
시각적 일관성과 스타일 통일성이 부족합니다.

기존 방식의 한계

  • ⚠️ 코드만으로 생성 → 플레이스홀더 이미지
  • ⚠️ 낮은 품질의 내장 에셋
  • ⚠️ 멀티모달 요소 간 스타일 불일치
  • ⚠️ AIGC 도구 간 조율 부재

핵심 질문

여러 AI 콘텐츠 생성 도구를
어떻게 체계적으로 조율하여
텍스트·이미지·비디오·차트가
하나의 일관된 디자인으로 보이는
웹페이지를 만들 수 있을까?

핵심 아이디어

MM-WebAgent는 학습 없이(Training-Free) 동작하는 에이전트 프레임워크로,
두 가지 핵심 메커니즘을 통해 문제를 해결합니다.

계층적 계획 (Hierarchical Planning)

글로벌 레이아웃 계획

섹션 계층 구조, 공간 배치, 색상/타이포그래피/간격 등 페이지 수준 스타일 속성 정의. 멀티모달 컴포넌트를 위한 플레이스홀더(크기, 위치, 역할) 설정.

로컬 요소 계획

각 멀티모달 요소(이미지·비디오·차트)에 대해 기능적 역할, 주변 텍스트/시각 컨텍스트, 크기 제약, 글로벌 계획에서 파생된 스타일 가이드를 포함한 세부 계획 생성.

계획 실행

글로벌 계획 → HTML/CSS 코드 변환. 로컬 계획 → 지정된 AIGC 도구 호출. 이미지·비디오·차트 생성을 병렬로 실행하여 지연 시간 최소화.

계층적 자기반성 (Self-Reflection)

3단계 반복 정제 루프 (최대 3회 반복)

🔍

핵심 인사이트

MM-WebAgent는 기존 코드 전용 방식 대비 약 79%의 일대일 비교 승률을 달성하면서도, 실행 시간과 비용 면에서 실용적인 수준을 유지합니다.

한계점 & 향후 연구

한계점

  • ⚠️ 외부 AIGC 도구의 불안정성·편향·안전 필터에 의존
  • ⚠️ 고정된 도구 세트 (동적 확장 불가)
  • ⚠️ 학습 없는 오케스트레이션 (최적화 없음)

향후 방향

  • 🔬 강화학습으로 에이전트 행동 최적화
  • 🔬 장기 상호작용에서의 학습
  • 🔬 동적 도구 확장 가능성

결론

계층적 계획 + 반복적 다수준 자기반성을 통해
다수의 AIGC 도구를 효과적으로 조율하여
시각적으로 일관된 멀티모달 웹페이지를 생성할 수 있음을 입증
