세 가지 핵심 축

🌍 공간 (Space) — 위상적 근거

환경을 리만 다양체로 모델링하여 혼잡 의존적 접근 비용을 포착합니다. API 스팸 및 연쇄 장애를 방지합니다.

Lrouting(γt, ut, t) = c · gμν(γt, t) · utμ · utν

⏱ 시간 (Time) — 시간적 일관성

경과 시간에 따른 의사결정 효용의 지수적 감쇠를 반영합니다. 시간 페널티 β(t)로 무한 숙고를 방지합니다.

🧠 인식 (Epistemic Truth) — 믿음 유지

Kushner-Stratonovich 비선형 필터링으로 가설에 대한 보정된 확률 분포를 유지합니다. 정보 가치(VOI)를 기대 엔트로피 감소로 측정합니다.

수학적 프레임워크

결합 관측 모델

관측 과정은 공간 라우팅 상태에 결합된 이토 확산(Itô diffusion)을 따릅니다:

dYt = h(θ, γt)dt + σ(γt)dWt

h(θ, γt)

네트워크 위치 γt에서 가설 θ 하의 기대 증거

σ(γt)

공간 상태에 의존하는 관측 노이즈

에이전트는 시공간적 마찰을 최소화하면서 기대 정보 이득을 최대화하는 삼중 가치 함수를 최적화합니다.

최적 정지 이론

HJB 변분 부등식

삼중 가치 함수는 다음의 자유 경계 문제로 특성화됩니다:

/* 최적 정지 영역 */
D = { (p, γ, t) : V(p, γ, t) = 0 }

✨ 정리 1: 최적 정지

h(θ,·)와 σ(·)에 대한 Lipschitz 가정 및 σσT의 균일 비퇴화성 하에서, 삼중 제어 문제는 최적 정지 시간을 가집니다:

T* = inf{ s ≥ t : (ps, γs, s) ∈ D }
실용적 구현

이산 시간 순 효용 정지 규칙

연속 HJB 방정식의 계산 가능한 이산 근사:

// 순 효용 함수
U(a; bt, t, Ct) = ΔH(a|bt) - α(λS(Ct + Ω(a)) + β(t + τ(a)))

// 정지 조건
STOP ⇔ maxa∈A U(a; bt, t, Ct) ≤ 0

ΔH(a|bt)

행동 a의 기대 엔트로피 감소
(몬테카를로 추정)

α, λS, β

비용-효용 스케일링, 공간 비용 가중치, 시간 감쇠율

Ω(a)

행동의 구조적 부하 (공간 비용)

τ(a)

행동의 지연 시간 (시간 비용)

관련 연구

기존 프레임워크와의 비교

LLM 기반 에이전트 (ReAct, Tree-of-Thoughts, Reflexion)

이산 휴리스틱과 임의적 정지 토큰에 의존 → TCA는 수학적으로 엄밀한 숙고 경계 제공

제한된 합리성 (Russell & Wefald, Simon)

고전적 제한 최적성 개념 확장 → 프롬프트 기반 도구 사용을 연속 시간 물리적 프로세스로 형식화

능동적 추론 & 자유 에너지 원리 (Friston)

철학적 기반 공유하지만 → LLM을 위한 계산적으로 다루기 쉬운 대안 제공

한계 & 향후 연구

알려진 한계와 확장 방향

⚠ 한계점

  • 근시안적 정지 (η=0): 연속 가치 근사 생략
  • 대규모 도구 생태계에서 VOI 추정 비용 증가
  • EMDG 단일 환경 검증만 수행
  • α, β, λS 하이퍼파라미터 민감도 분석 미완

🚀 향후 방향

  • 다단계 롤아웃 또는 학습된 대리 모델로 연속 가치 근사
  • 학습된 엔트로피 감소 모델을 통한 VOI 상각
  • 이종 다중 도메인 에이전트 과제 평가
  • 연속 가설 공간으로의 확장
