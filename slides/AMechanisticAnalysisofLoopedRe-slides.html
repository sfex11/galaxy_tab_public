배경

왜 루프 언어 모델을 분석해야 하는가?

최근 LLM은 추론 능력이 핵심 역량이 되었고, 레이어를 반복 순환시켜 성능을 높이는 새로운 접근이 등장했습니다. 그러나 이 모델들의 내부 동역학은 아직 충분히 이해되지 않고 있습니다.

🔄

피드포워드 vs 루프

기존 모델은 레이어를 한 번만 통과하지만, 루프 모델은 같은 레이어 블록을 여러 번 반복하여 기능적 깊이를 매개변수 수로부터 분리합니다.

🧠

추론 단계(Stages of Inference)

피드포워드 트랜스포머는 추론 시 여러 계산 단계로 분해됩니다. 루프 모델에서도 동일한 단계가 나타나는지가 핵심 질문입니다.

순환 블록 합성

S_k(X) = B_k(B_{k-1}(...B_1(X)...))

k개의 트랜스포머 블록 B₁~Bₖ를 순차적으로 적용한 결과를 l번 반복합니다.

입력 주입(Input Injection)

S_k(X, Z) = B_k(...B_1([X, Z] · W_I)...)

각 반복마다 원본 입력 Z를 재주입하여 정보 손실을 방지합니다.

각 트랜스포머 블록은 어텐션 → 잔차 연결 → MLP → 잔차 연결 순서로 구성되며, 정규화(normalization) 레이어의 위치에 따라 Pre-Norm, Post-Norm, Sandwich-Norm으로 나뉩니다.
이론

순환 고정점 이론

루프 모델이 반복을 거듭할수록 수렴하는 순환 고정점(Cyclic Fixed Point)의 수학적 토대

명제 4.1: 순환 고정점

(l, k)-순환 블록이 고정점 X' = S_k(X')에 도달하면, 블록의 모든 순환 순열도 각자의 고정점에 도달합니다.

→ 각 레이어가 서로 다른 고유한 고정점에 수렴하여, 잠재 공간에서 일관된 순환 궤적을 형성

명제 4.2: 어텐션 패턴 수렴

‖S_l(X_{l,t}) - S_l(X_{l,t-1})‖ ≤ L_sm · (2B · κ_l / √d) · ‖X_{l,t} - X_{l,t-1}‖

잔차 스트림이 수렴하면 어텐션 패턴도 반드시 수렴합니다. κ_l = ‖W_Q · W_K^T‖이며, L_sm은 소프트맥스의 립시츠 상수입니다.

ColSum 집중도 지표

C = 1 - H_col ∈ [0, 1] C = 1 + (1/log T) · Σ ĉⱼ · log(ĉⱼ)

높은 C = 집중된 어텐션 (어텐션 싱크)
낮은 C = 분산된 어텐션 (혼합/mixing)

Retrofitted 모델이 우수

사전학습된 피드포워드 모델을 순환 구조로 변환한 모델들이 고정점 수렴과 안정적인 추론 단계 모두에서 가장 좋은 결과를 보였습니다.

Ouro의 한계

설계 지침: 순환 고정점 수렴을 위해서는 입력 주입이 필수적이며, Pre-Normalization 구성이 가장 안정적인 결과를 제공합니다. Huginn-0125의 경우 Sandwich Norm이 잔차 스트림의 크기 변화를 억제하여 혼합 단계 형성을 방해합니다.

Huginn-0125의 정규화 문제

Sandwich normalization이 적용되면 레이어 출력이 항상 정규화되어 잔차 스트림의 활성화 크기가 변하지 않습니다. 이로 인해 ColSum 혼합 행동에 필요한 조건이 충족되지 않아, 어텐션 패턴이 수렴하더라도 의미 있는 추론 단계가 형성되지 않습니다.

안정성 (고정점 모델)

Retrofitted Llama의 레이어별 ColSum 집중도는 빠르게 안정화되며, 128회 반복까지도 안정적으로 유지됩니다. 학습 깊이를 훨씬 초과하는 테스트 시에도 추론 단계가 보존됩니다.

불안정성 (비고정점 모델)

Ouro의 레이어들은 반복에 따라 계속 변화하며, 학습 시 반복 횟수를 초과하면 단계가 붕괴되고 성능이 저하됩니다.

시사점

핵심 시사점과 설계 지침

이 연구의 발견들이 루프 언어 모델의 아키텍처 설계에 주는 실질적 가이드라인

01

기능적 깊이와 매개변수의 분리

루프 모델은 적은 매개변수로 깊은 계산을 수행할 수 있으며, 이는 피드포워드 모델의 추론 단계를 각 반복에서 재현함으로써 달성됩니다.

02

입력 주입은 필수

순환 고정점 수렴을 위해 각 반복마다 원본 입력을 재주입하는 것이 핵심입니다. 없으면 고정점 행동이 크게 저하됩니다.

03

정규화 방식이 중요

Pre-normalization이 가장 안정적이며, Sandwich normalization은 혼합 단계 형성을 방해할 수 있습니다.

04

고정점 수렴 = 일반화 능력

고정점에 수렴하는 모델은 학습 시 사용한 반복 횟수를 초과해도 안정적으로 동작하여, 테스트 시 추가 연산 예산 배분이 가능합니다.

메커니즘적 통찰의 전이: 피드포워드 모델에 대한 기존 메커니즘적 이해를 루프 아키텍처 설계에 직접 활용할 수 있습니다. 이는 모델 해석 가능성과 효율적인 아키텍처 탐색 모두에 기여합니다.
결론

한계점과 미래 연구 방향

연구의 한계

  • 단일 순환 블록(cyclic recurrence)만 분석 — 다중 순환 블록(sequential recurrence)은 미검토
  • 고정점 수렴이 왜 일어나는지에 대한 해석적 증명 부재
  • 고정점 수렴이 추론 능력에 유리한지 제약인지 아직 미결정

미래 연구 방향

  • 다중 순환 블록 구조에 대한 확장 분석
  • 고정점 수렴의 이론적 원인 규명
  • 적응적 반복 횟수 결정 메커니즘
  • 더 큰 규모의 모델에서 검증
📌
