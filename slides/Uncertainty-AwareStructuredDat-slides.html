문제 정의와 동기

자유 텍스트 CMR 보고서

심장자기공명(CMR) 보고서는 양심실 기능, 심실 크기, 조직 특성 등의 정량적 측정값을 포함하지만 자유 텍스트로 작성됨

구조화된 데이터의 필요성

코호트 구성, 종단적 연구 큐레이션, 임상 의사결정 지원을 위해 감사 가능한 구조화 데이터 변환이 필수

용어, 단위, 형식, 완전성의 이질성으로 인해 완전 자동 파싱이 어려우며, 수동 큐레이션은 비용이 많이 들고 프라이버시 우려를 야기함

기존 연구의 한계

1

문장 수준 처리

문장 단위로 작동하여 문장 간 의존성과 단위 조화를 해결할 수 없음

2

수동 주석 의존

훈련 말뭉치 생성을 위해 대규모 수동 주석이 필요하여 확장성 저하

3

불확실성 추정 부재

필드별 불확실성 추정이 없어 인간 검토의 원칙적 트리아지 불가

CMR-EXTR은 이 세 가지 간극을 동시에 해결하는 최초의 CMR 전용 추출 시스템

불확실성 평가: 3원리

최종 신뢰도 점수 = (분포 점수 + 안정성 점수 + 일관성 점수) / 3

Sdist

분포 타당성

추출값이 참조 정규분포(μ, σ) 하에서 얼마나 가능한지 평가. 성별 분포 중 높은 점수 채택

Sstab

샘플링 안정성

온도 0.3으로 3회 생성 후 투표 기반 최종값 결정. 생성 간 변동성이 낮을수록 높은 점수

Scons

교차 필드 일관성

22개 생리/대수적 공식으로 필드 간 관계 검증. 예: LVEF = LVSV/LVEDV × 100

신뢰도 < 0.7인 필드만 인간 검토 대상으로 에스컬레이션 → 검토 부하 최소화

분포 점수 상세

Sdist = exp( −½ × ((v − μ) / (ασ))² )

v: 추출값, μ/σ: 참조 분포의 평균/표준편차, α: 6 (기본값)

  • 참조값은 Kawel-Boehm et al. (2025)의 사회 가이드라인에서 획득
  • 대부분의 CMR 보고서에 성별 정보가 없으므로 남/녀 분포 각각 계산 후 최댓값 채택
  • 참조 분포가 없는 변수의 경우 기본값 0.7 부여
  • α=6으로 설정하여 너그러운 허용 범위 적용 — 극단값만 낮은 점수

안정성 점수 상세

si,j = exp( −β × |vi − vj| / (vi + vj) )

β: 2 (기본값), Sstab = (s1,2 + s1,3 + s2,3) / 3

생성 과정

온도 0.3으로 3회 독립 생성 → 투표로 최종값 결정. 온도를 낮게 설정하여 의미 있는 변동만 측정

특수 케이스

둘 다 null → 1.0, 하나만 null → 0.5. 상대적 오차 비율을 사용하여 스케일 불변성 확보

일관성 점수 상세

22개 생리/대수적 공식 활용

예시: LVEF = LVSV / LVEDV × 100, LVSV = LVEDV − LVESV, BSA 보정값 = 원시값 / BSA 등

  • 공식 우변을 계산하여 기대값 도출 → 안정성 점수와 동일한 지수 함수로 일치도 평가
  • 변수가 여러 공식에 관여하면 모든 공식의 점수 평균 채택
  • 어느 공식에도 포함되지 않는 변수는 기본값 0.7 부여
  • 단위 불일치나 보고서 자체의 오류를 교차 필드 검증으로 탐지 가능

세 점수의 상호 보완성: 분포는 극단값 탐지, 안정성은 모델 불확실성 측정, 일관성은 필드 간 논리 오류 발견

데이터 준비 및 프라이버시

데이터셋

  • OSU 병원 1,100건 CMR 보고서
  • 20% 테스트 분할 보존
  • 5개 질환 범주 라벨 부착
  • 데이터 준비 과정에 수동 주석 없음

프라이버시 제어

  • 모든 모델 오프라인 배포
  • 외부 데이터 전송 없음
  • MRN, Name, DOB 등 키워드 포함 행 제거
  • 오픈 가중치 모델의 오용 방지

Teacher가 생성한 값 중 신뢰도 < 0.7인 것만 수동 검토 → 전체 말뭉치의 1% 미만만 인간 개입
