FORMULATION

핵심 수학적 정의

Eq.2 — 중요도 함수 (Importance Function)
I(mi, t) = α · R(mi, t) + β · F(mi) + γ · S(mi, qt)
Eq.3 — 제약 최적화 (Constrained Optimization)
Mt* = argmaxM'⊆Mt Σ I(mi, t)   s.t. |M'| ≤ B
Eq.4 — 시간적 감쇠 (Temporal Decay)
R(mi, t) = exp(-λ(t - ti))

λ 작음 → 장기 보존  |  λ 큼 → 공격적 망각

Eq.5 — 통합 손실 함수 (Joint Loss)
Ltotal = Ltask + η · |Mt| / B

작업 성능 + 정규화된 메모리 풋프린트 페널티

수렴 보장: Δt = |Mt| - B → 0 (t → ∞)  —  제한된 메모리 성장 보장

ALGORITHM

알고리즘 1: 적응형 예산 기반 망각

// 제약 관련성 최대화를 통한 적응형 예산 망각
Input: 메모리 예산 B, 감쇠율 λ, 가중치 (α, β, γ)
Init: M0 = ∅

for t = 1 to T do:
  Mt ← F_store(Mt-1, ot, at) // 메모리 업데이트

  for each mi ∈ Mt do:
    R(mi, t) = exp(-λ(t - ti)) // 최신성 계산
    I(mi, t) = α·R + β·F + γ·S // 중요도 산출

  if |Mt| > B then:
    Mt* = argmax Σ I(mi, t)
      s.t. |M'| ≤ B // 예산 제약 하 최적 부분집합
    Mt ← Mt*

Output: 예산 제약 메모리 Mt

O(n log n)

Δt → 0
