파라메트릭 지식 한계

학습 시점에 압축된 지식만 보유. 최신 정보 반영 불가, 환각(Hallucination) 발생.

RAG의 구조적 부족

벡터 유사도 기반 검색은 의미적 근접성만 측정. 개념 간 계층·관계 구조 파악 불가.

일관성 결여

추론 과정에서 모순된 결론 도출 가능. 지식 간 충돌을 체계적으로 감지·해결할 매커니즘 부재.

자동 온톨로지 구축 파이프라인

1

텍스트 수집 및 전처리

도메인 문서를 수집하고 문장 단위로 분할하여 LLM 입력 형식으로 정규화.

2

개체-관계 추출

LLM이 각 문장에서 핵심 개체(Entity)와 개체 간 관계(Relation)를 식별.

3

온톨로지 통합 및 정제

추출된 삼중항(Triple)을 기존 온톨로지와 병합. 중복 제거, 계층 정렬, 충돌 해결.

4

일관성 검증

온톨로지 제약(공리, 카디널리티, 계층 규칙) 위반 사항을 자동 검출하고 수정.

5

지식 기반 추론 계획

구축된 온톨로지를 기반으로 복잡한 질의를 하위 태스크로 분해하고 추론 경로 계획.

외부 계층의 세 가지 역할

메모리 Memory

LLM의 파라메트릭 기억 외부에 구조화된 장기 기억을 유지. 개념, 속성, 관계를 그래프 형태로 저장하여 LLM이 필요 시 조회·갱신 가능. 시간에 따른 지식 변화도 추적.

지식 그래프 삼중항 저장 증분 갱신

검증 Verification

LLM이 생성한 답변이나 추론 결과를 온톨로지의 공리와 제약 조건에 대해 교차 검증. 모순이 발견되면 피드백 루프를 통해 LLM에 수정 요청. 사실적 정확성 보장.

일관성 검사 환각 탐지 피드백 루프

계획 Planning

복잡한 질의를 온톨로지의 개념 계층과 관계 경로를 따라 하위 질의로 분해. 각 단계에서 필요한 지식을 식별하고, 실행 순서를 결정하여 체계적인 다단계 추론을 가능하게 함.

핵심 시사점

본 논문은 LLM이 단독으로 지식을 처리하는 방식에서 벗어나, 온톨로지라는 형식적 지식 표현을 외부 계층으로 결합함으로써 지능 시스템의 신뢰성과 추론 능력을 근본적으로 향상시키는 새로운 패러다임을 제시합니다. 이 하이브리드 접근법은 의료, 법률, 과학 등 일관성이 중요한 도메인에서 특히 강력한 잠재력을 가집니다.
