POPO: 긍정 전용 정책 최적화

온라인 긍정 롤아웃만으로 학습이 가능한 새로운 RLVR 프레임워크

1
유계 중요도 샘플링
긍정 롤아웃 집합에 대해 정규화된 가중치 계산
2
암시적 음의 기울기
확률 재분배를 통해 자연스럽게 음의 신호 발생
3
안정화 메커니즘
Siamese 네트워크 + EMA + 표현 유사도 페널티
부정 롤아웃에 대한 기울기 안내를 전혀 사용하지 않음 — 학습이 온전히 긍정 응답에서만 발생

유계 중요도 샘플링 (Bounded Importance Sampling)

GRPO의 접근

전체 샘플(G개)에 대해 정규화:
어드밴티지가 긍정/부정 나뉨
→ 부정 샘플이 가중치 분모에 포함

POPO의 접근

긍정 샘플(P개)에 만 정규화:
가중치가 항상 양수이며 유계
→ 부정 샘플이 기울기에 전혀 관여하지 않음

핵심 차이: 분모를 전체 G가 아닌 긍정 집합 P로 제한함으로써, 기울기가 부정 샘플에 의존하지 않으면서도 자연스러운 확률 재분배 효과를 얻음

암시적 음의 기울기의 발생 원리

명시적인 부정 롤아웃 없이도 음의 기울기가 자연스럽게 등장하는 이유

①

Softmax 정규화
긍정 로짓 간의 확률 재분배가 음의 로짓에 간접적 영향

②

NLL 기울기 전파
음의 로짓에 대한 NLL 기울기가 음의 방향으로 작용

③

엔트로피 기여
엔트로피 항이 음의 로짓에 대해 음의 기울기 누적

정리 (Theorem 3.1) 유사도 손실은 로짓 수준에서 기여하지 않으며, NLL과 엔트로피 기울기가 결합하여 음의 로짓에 대해 음의 기울기 방향이 성립함을 수학적으로 증명

안정화 메커니즘: 두 가지 핵심 설계

Siamese 정책 네트워크 + EMA

  • 온라인 정책 π_θ와 타겟 정책 π_ξ를 Siamese 구조로 연결
  • ξ는 EMA로 갱신되어 적응적 정책 앵커 역할
  • BYOL/SimSiam의 비대칭 아키텍처 통찰 활용
Lemma 3.2: EMA 기반 파라미터 갭이 정상 상태에서 유계됨을 증명

표현 공간 유사도 페널티

  • KL 발산을 표현 공간의 코사인 유사도로 대체
  • 소프트 유계 제약으로 의미적 구조 보존
  • 토큰 수준의 불안정성 완화
KL 발산과 달리 [-1, 1] 범위에서 유계되어 안정적인 최적화 가능

POPO 손실 함수 전체 구조

L_POPO = L_policy + β · L_similarity + γ · L_entropy

L_policy

긍정 롤아웃에 대한 유계 중요도 샘플링 가중치 × NLL
→ 암시적 음의 기울기 포함

L_similarity

Siamese 표현 공간에서 코사인 유사도 페널티
→ KL 대체, 유계 제약

L_entropy

엔트로피 보너스로 탐색 촉진
→ 음의 로짓에 추가 음의 기울기

부정 롤아웃 데이터가 손실 함수 어디에도 등장하지 않음

결과 분석: 왜 고난도에서 차이가 벌어지는가?

저난도 (MATH-500)

pass@k가 이미 높아 긍정/부정 롤아웃 모두 정보량이 충분함. GRPO와 POPO의 성능 차이가 미미

~0.4%

고난도 (AIME 2025)

pass@k가 극히 낮아 부정 롤아웃이 노이즈로 작용. POPO의 긍정 전용 접근이 명확한 이점 발휘

~6.67%
부정 롤아웃의 "무의미한 페널티"가 문제 난이도가 높아질수록 더 큰 장애물이 됨. POPO는 이를 근본적으로 회피

추가 실험: 일반 텍스트 LLM 적용

범용 모델에서의 성능

수학 특화 모델이 아닌 일반 텍스트 LLM(Qwen2.5-7B 등)에도 POPO를 적용하여 일관된 개선 확인

  • GRPO 대비 MATH-500에서 유의미한 향상
  • 수학 전용 SFT 없이도 추론 능력 향상
  • 범용성 측면에서 POPO의 강점 입증

하이퍼파라미터 민감도

광범위한 스윕 연구를 통해 POPO의 하이퍼파라미터에 대한 견고성 확인

  • EMA 감쇠율(β): 0.99~0.999에서 안정
  • 유사도 가중치(β_sim): 넓은 범위에서 유효
  • GRPO보다 하이퍼파라미터 선택에 덜 민감
