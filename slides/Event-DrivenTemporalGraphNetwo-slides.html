⚠ 문제 정의: Sim2Real 격차

기존 사이버 RL 시뮬레이터의 세 가지 근본적 결함이 실제 SOC 배포를 가로막고 있습니다.

⏱ 동기식 vs 비동기식 시간

  • 기존: 고정 이산 틱으로 행동이 즉시 해결
  • 현실: 확률적 지속시간, 행동 중 적이 중단 가능
  • 실제 사이버 작전은 본질적으로 비동기적

📄 클린 상태 vs NLP 텔레메트리

  • 기존: 깨끗한 원-핫 인코딩 배열
  • 현실: 노이즈가 많은 비정형 Windows 이벤트 XML
  • 이진 지시자는 프로덕션에 배포 불가

🌐 정적 토폴로지

  • MLP는 고정 입력 차원 요구 → 특정 네트워크 구조 암기
  • 동적 노드 프로비저닝이 있는 기업 네트워크에 부적합
  • 제로샷 스케일링 불가능

⚡ CT-MAPPO 최적화

연속 시간 할인 & GAE

// 지수 시간 감쇠 (평탄한 γ 대체)
γ(Δt) = e−β·Δt      β = 0.05

// 연속 시간 일반화 이점 추정
δt = rt + e−βΔt · V(st+1) − V(st)
Ât = δt + (λ · e−βΔt) · Ât+1

물리적 행동 지속시간에 따라 크레딧 할당을 정렬 → 가변 지속시간 행동의 정확한 가치 평가

CTDE 패러다임

  • 분산 액터: πθ(a|o) — 로컬 관측만 사용
  • 중앙 비평가: Vφ(s) — 노이즈 없는 전역 상태
  • PPO 클리핑 (ε = 0.2) + 엔트로피 정규화 (0.01)

주요 하이퍼파라미터

은닉 차원128 GAT 헤드4 β (시간 감쇠)0.05 ODE 솔버RK4 고정 액터 LR3×10⁻⁴ 비평가 LR1×10⁻³ 배치 크기512 롤아웃2048 스텝

  • 계산 오버헤드: Neural ODE RK4가 이산 LSTM보다 상당히 높은 FLOPs 요구
