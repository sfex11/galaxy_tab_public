MeMo: Memory as a Model

NUS, MIT CSAIL, A*STAR, Liquid AI, SMART

문제 정의: 고정된 지식의 한계

대규모 언어 모델(LLM)은 사전학습 이후 매개변수가 고정되어, 시간이 지남에 따라 지식이 점진적으로 구식화됩니다.

"LLM은 사전학습 후 긴 기간 동안 실질적으로 동결되며, 세상이 변화함에 따라 사전학습 지식은 점점 더 유효하지 않게 된다."

  • 실시간 정보 요구 (날짜 민감 질의)
  • 도메인 특화 지식 필요 (의료, 금융 등)
  • 전체 재학습은 현대 규모에서 비용이 기하급수적
  • 정적 지식에 의존하는 것은 근본적 아키텍처 한계

기존 접근법과 한계점

접근법 대표 방법 장점 한계점
비파라메트릭 RAG, BM25, GraphRAG 파라메터 수정 불필요 문서 간 관계 합성 어려움, 검색 노이즈에 취약
파라메트릭 Continual Pretraining, SFT 지식을 파라메터에 내재화 재학습 비용, 치명적 망각, 분포 기억 한계
잠재 메모리 AutoCompressor, MemGen 압축된 쿼리 가능 표현 표현 결합 — 특정 모델에 종속, 이전 불가

세 가지 패러다임의 장점을 결합하면서 개별 한계를 완화할 수 있는가?

MeMo: 핵심 아이디어

새로운 지식을 전용 Memory 모델에 인코딩하고, Executive 모델은 추론 시 Memory 모델에서 필요한 정보를 검색하여 응답을 생성합니다.

핵심 설계 원칙: Reflections — 미래 질의에 대한 사전 지식 없이 코퍼스에서 파생된 구조이면서, 임의의 질의가 코퍼스에 접근할 수 있는 정확한 인터페이스 역할

학습 방식

5단계 파이프라인에서 생성된 Reflection QA 데이터셋을 통해 지도 미세조정(SFT) 수행

핵심 특징

  • 원시 코퍼스가 아닌 구성적 표현으로 학습
  • 단일 문서 내 사실뿐 아니라 교차 문서 관계 캡처
  • 검색 기반 방법보다 복잡한 구조 내재화
  • Full SFT 또는 LoRA 적용 가능

검색 노이즈 견고성 분석

핵심 발견

코퍼스에 노이즈 문서 비율을 점진적으로 증가시킨 실험에서:

  • RAG 방법들은 노이즈 증가와 함께 성능이 급격히 저하
  • MeMo는 노이즈 수준에 상대적으로 안정적인 성능 유지
  • 이는 지식이 모델 파라메터에 내재화되어 검색 단계의 오류 영향을 받지 않기 때문

왜 MeMo가 강한가?

RAG의 취약점

검색기가 노이즈 문서를 잘못 반환하면 LLM이 오도됨 — 검색 품질에 직접 의존

MeMo의 강점

5단계 파이프라인의 검증 단계에서 노이즈가 사전에 필터링되며, 학습 후에는 검색 없이 직접 질의 응답

추가 절제 실험

Memory 모델 크기

모델 크기를 다양하게 변화시킨 결과:

  • 일정 수준 이상의 크기에서 성능 포화 현상 관찰
  • 작은 모델로도 경쟁력 있는 성능 달성 가능
  • 효율성-성능 트레이드오프에서 유연한 선택 지원

Memory 모델 패밀리

서로 다른 모델 패밀리(Qwen, Llama, Gemma 등)를 Memory 모델로 사용한 결과, 패밀리 간 성능 차이는 크지 않음 — 프레임워크의 모델 독립성 입증

데이터 합성 단계 절제

5단계 파이프라인의 각 단계 기여도 분석:

교차 문서 합성 단계가 다중 홉 추론 성능에 가장 큰 기여

검증 단계가 노이즈 환경에서의 견고성에 핵심적 역할

엔티티 표면화가 연결 질의에서 정확도 향상에 기여

지속적 지식 통합: 모델 병합

문제 상황

실제 환경에서는 새로운 코퍼스가 지속적으로 도착합니다. 매번 새로운 Memory 모델을 학습하는 것만으로는 비효율적입니다.

해결책

새로운 코퍼스에 대해 개별적으로 학습된 Memory 모델들을 모델 병합(model merging) 기법으로 결합하여 단일 통합 Memory 모델 구성

코퍼스 A → Memory 모델 A

코퍼스 B → Memory 모델 B

코퍼스 C → Memory 모델 C

모델 병합 → 통합 Memory 모델

재학습 없이도 새로운 지식을 누적 통합 가능

"MeMo는 지식을 모델 자체로 만드는 패러다임 전환을 제안한다. Memory 모델은 단순한 저장소가 아니라, 질의를 이해하고 구성적으로 응답할 수 있는 능동적 지식 실체다."
