arXiv: 2604.11805 · 2026

물리 시뮬레이터 기반 강화학습으로
물리 올림피아드 풀기

Sim2Reason: 합성 데이터로 훈련하여 실제 국제물리올림피아드(IPhO) 문제에서 5~18%p 성능 향상 달성

강화학습 MuJoCo LLM 추론 Sim-to-Real IPhO

Mihir Prabhudesai, Aryan Satpathy, Yangmin Li 외 · Carnegie Mellon University

01 · 문제 정의

LLM 물리 추론의 병목 현상

⚠
데이터 부족
DeepSeek-R1 등의 성과는 인터넷 QA 데이터에 의존하나, 고품질 물리 문제-답변 쌍은 극히 제한적
📊
편향된 분포
기존 QA 데이터는 특정 주제에 편중되어 있어 올림피아드 수준의 다양한 물리 문제를 커버하지 못함
🧮
수학적 오류
기존 LLM은 물리 공식을 문맥 없이 적용하여 산술 실수와 잘못된 방정식 선택이 빈번
🔍
확장 한계
인간이 직접 만드는 QA 데이터로는 무한한 학습 스케일링이 불가능
💡 핵심 질문

인간이 만든 QA 데이터 없이, 물리 시뮬레이터만으로 LLM의 물리 추론 능력을 향상시킬 수 있을까?

🔑 DSL (Domain-Specific Language) 기반 장면 랜덤화
YAML 형식의 DSL로 시스템 동역학에 영향을 주는 매개변수만 선택적으로 랜덤화합니다. 예를 들어 도르래 줄의 절대 길이 같은 무관한 변수는 제외하여 물리적 타당성과 다양성을 동시에 확보합니다.
03 · 질문 유형

3가지 유형의 합성 질문 생성

🔢
수치형 (Numeric)
"줄 T₁의 장력은 얼마인가?"

물리량의 정확한 수치를 묻는 직접적인 질문

🔄
역추론형 (Reverse)
"장력이 5N이라면 질량은?"

결과에서 원인 매개변수를 역으로 추론

📐
기호형 (Symbolic)
"물리량 간 관계식을 표현하라"

물리량 사이의 수학적 관계를 기호로 표현

🔎 핵심 통찰

06 · 핵심 발견

모델이 실제로 배운 것

✅ 산술 오류 제거
RL 미세조정 후 모델은 기본적인 산술 실수를 크게 줄였으며, 계산 정확도가 현저히 향상
🎯 물리적 맥락 인식
방정식을 기계적으로 적용하는 대신, 물리적 설정에 맞는 방정식을 선택하는 능력 획득
🌐 시뮬레이션 범위 초월
MuJoCo로 시뮬레이션 불가능한 문제에서도 성능이 향상 — 일반화 능력 입증
📈 강한 상관관계
합성 과제 성능과 IPhO 역학 점수 사이 강한 양의 상관관계 — 시뮬레이터의 벤치마크 대리 지표 역할 검증
🚀 스케일링 패턴

합성 과제에서의 성능 향상이 3B~32B 모든 모델 크기에서 일관되게 실세계 성능 향상으로 전이됩니다. 이는 시뮬레이터 기반 접근법의 확장 가능성을 보여줍니다.

08 · 한계 및 미래 방향

현재의 한계와 향후 과제

한계점
  • 시뮬레이터 충실도에 의존 — 시뮬레이션 정확도가 학습 품질에 직접 영향
  • 시뮬레이션 가능한 영역에 한정 — 열역학, 전자기학 등은 추가 연구 필요
  • RL 학습과 시뮬레이션의 계산 비용
미래 방향
  • 더 다양한 물리 도메인으로 시뮬레이터 확장
  • 더 큰 모델에 대한 스케일링 법칙 검증
  • 시뮬레이터와 LLM의 더 깊은 통합
🌍 더 넓은 의미

이 연구는 물리학에만 국한되지 않습니다. 다양한 과학 분야에서 시뮬레이터를 활용한 합성 데이터 생성이 LLM 추론 능력 향상의 새로운 패러다임이 될 수 있습니다.
