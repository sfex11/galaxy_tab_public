연구 배경 및 동기

핵심 문제

  • LLM은 복잡한 과제에서 뛰어나지만, 환각(hallucination)과 해석 가능성 부족 문제 존재
  • 노화 관련 유전자 예측 시 제한된 생물학적 지식이 모델 신뢰성을 저해
  • 유전자 온톨로지(GO) 용어 간 계층적 관계가 충분히 활용되지 않음

제안된 해결책

다중 AI 에이전트가 인간의 학술 토론을 모방하는 가상 연구 그룹(VSG)을 구축하여, 계층적 특징 선택으로 추출한 GO 용어로부터 노화 관련 생물학적 지식을 발견합니다.

"LLM 기반 에이전트 AI 프레임워크로 여러 LLM이 인간의 상호작용을 모방하도록 하여, 과학적 발견 파이프라인을 혁신합니다."

유전자 온톨로지 & 계층적 특징 선택

유전자 온톨로지(GO) 데이터베이스

  • 유전자 기능을 설명하는 보편적 어휘 체계
  • 세 가지 도메인: 생물학적 과정, 분자 기능, 세포 구성요소
  • 계층 구조: 상위 = 일반적 정의, 하위 = 구체적 정의

계층적 특징 선택 방법

HIP

계층적 정보 보존 선택 — 고빈도 GO 용어 선별

MR

가장 관련성 높은 용어 선택

HIP-MR

두 방법의 하이브리드 접근

GO:0050789 (생물학적 과정 조절) → GO:0009894 (이화 과정 조절)
상위 용어가 하위 용어를 포괄하는 계층적 관계를 활용합니다.

핵심 생물학적 발견

범생물(Cross-Organism) 공통 패턴

1. 산화 스트레스의 이중성

활성산소종(ROS)은 손상 유발제이자 적응적 신호 분자로 기능 — 조직별 반응이 결정적

2. 생식-체세포 트레이드오프

특정 맥락에서 생식이 자원을 전용하지만, 보편적 트레이드오프는 의문 — 일회용 체세포(Disposable Soma) 이론과 연결

3. 미토콘드리아 가소성

세포는 UPRmt, 해당작용 전환 등 보상 메커니즘으로 전자전달계 기능 저하를 완화

4. 조직 특이적 조절

항산화 효소 발현이 조직별로 상이 — 일반적 주장이 국소적 역학을 은폐

5. 하류 vs. 1차 원인

산화 스트레스는 종종 구조적 붕괴의 결과이지 노화의 1차 원인이 아닐 수 있음

  • 라민 돌연변이의 핵 무결성 영향

🐭 생쥐

  • 정지기 세포 대사가 시간적 노화 주도

수석 연구자(PI)의 통합 통찰

"무척추동물 기반 지식을 포유류로 일반화하는 것에 대한 일반화 가능성이 가장 가치 있는 연구 방향입니다."

범생물 통합 관점

  • ROS 경로는 모든 생물에서 중요하지만, 진화적 맥락에 따라 다르게 작용
  • 조직 특이적 ROS 조절 장애 → 전신 단백질 항상성 조절 허브
  • 생식-노화 연관성은 진화적 관점으로 확장 필요
  • 무척추동물에서 포유류로의 복잡성 차이가 일반화의 핵심 도전

한계점 & 향후 연구

한계점

  • 다층 구조에도 환각 문제 지속
  • 구체적 GO 용어에 대한 프레임워크 능력 미검증
  • 문헌이 존재하는 일반적 수준의 GO 용어에 검증 제한
  • 기존 생물학에 제한 — 진정한 새로운 지식 발견 불가

향후 연구 방향

  • 에이전트 능력을 미지의 생물학적 지식 발견으로 확장
  • 더 구체적인 GO 용어 주석에 대한 프레임워크 평가
  • 환각 완화를 위한 추가 검증 계층 개발
  • 더 많은 모델 생물과 다양한 GO 도메인 포괄
에이전트 AI 프레임워크가 "계층적 특징 선택에서 도출된 GO 용어로부터 의미 있는 노화 관련 생물학적 지식을 성공적으로 추출"했음을 입증하였습니다.
