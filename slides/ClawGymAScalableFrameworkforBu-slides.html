왜 Claw 스타일 에이전트가 어려운가

01

개인화된 요구사항

다양한 직업과 일상 루틴에 걸쳐 대표성 있는 태스크 설정을 정의하기 어려움

02

장기 수행의 검증

파일 조작, 툴 호출, 워크스페이스 업데이트의 긴 시퀀스에 대한 자동 평가 난제

03

로컬 워크스페이스 의존

의미 있는 실행 컨텍스트를 제공할 현실적인 모의 워크스페이스 필요

기존 연구들은 대규모 Claw 스타일 태스크 데이터의 부재로 인해 체계적인 프레임워크 개발이 미탐구 상태입니다.

Claw 스타일 태스크 정의

τ = ⟨ p, s₀, 𝒜, ℱ, 𝒱τ ⟩

입력 (Input)

  • p — 사용자 지시 (파일 정리, 문서 편집, 보고서 생성 등)
  • s₀ — 초기 환경 상태 (로컬 파일, 폴더, 설정 파일 등)
  • 𝒜 — 가용 행동 공간 (파일 읽기/쓰기, 스크립트 실행, 웹 상호작용)

출력 (Output)

  • ξ — 행동-관찰 트레이젝토리: (A₁, O₁, A₂, O₂, ...)
  • ℱ — 도구 실행이 환경 상태를 업데이트하는 방식
  • 𝒱τ — 태스크별 검증자 (성공/실패 판정)

ClawGym-SynData: 듀얼 라우트 합성 전략

Top-Down

페르소나 기반 합성

1
시드 정식화

직업, 역할, 시나리오 의도로 구성된 시드 프롬프트 설계

2
시드 안내 태스크 생성

페르소나와 시나리오를 기반으로 구체적 사용자 지시 생성

→ 높은 수준의 사용자 의도에서 출발하는 다양성 확보

Bottom-Up

스킬 기반 합성

1
스킬 주석 및 필터링

구체적 실행 가능 능력을 스킬 단위로 주석화 후 필터링

2
스킬 구성 태스크 생성

복수 스킬을 조합하여 다단계 워크플로우 태스크 구성

→ 실행 가능성과 검증 가능성을 보장하는 실용적 접근

리소스 준비와 하이브리드 검증

리소스 준비

태스크별로 현실적인 모의 워크스페이스를 자동 생성합니다. 로컬 파일, 폴더 구조, 설정 파일 등 태스크 수행에 필요한 모든 아티팩트를 포함합니다.

검증 설계

결정적 코드 기반 검사기와 정성적 루브릭 기반 검증자를 결합한 하이브리드 프로토콜로 각 태스크에 대한 강건한 판정을 제공합니다.

점수 집계 방식

Code

코드 기반 결정적 검증
(파일 존재, 내용 일치 등)

Rubric

루브릭 기반 정성적 검증
(LLM 판단, 기준 충족 여부)

자동화된 품질 평가

태스크 품질 평가

  • 신규성 (Novelty) — 기존 태스크와의 중복 최소화
  • 타당성 (Plausibility) — 현실적이고 실행 가능한 태스크인지
  • 난이도 (Difficulty) — 적절한 복잡도 보장

검증 품질 평가

  • 코드 평가 — 검증 스크립트의 논리적 정확성 검사
  • 루브릭 평가 — 평가 기준의 명확성과 공정성 검사
13.5K

필터링된 실행 가능 Claw 스타일 태스크

ClawGym-Agents: 학습 파이프라인

1
블랙박스 롤아웃
OpenClaw 하네스에서 대규모 상호작용 트레이젝토리 수집
2
트레이젝토리 집계·선택
고충실도 트레이젝토리 필터링 및 집계
3
지도 미세조정 (SFT)
선택된 트레이젝토리로 에이전트 모델 학습
4
강화학습 (RL)
샌드박스 병렬 롤아웃 기반 경량 RL 파이프라인

SFT 핵심

합성 태스크 풀에서 수집한 고품질 트레이젝토리를 활용하여 OpenClaw 프레임워크 내에서 LLM 성능을 향상

RL 핵심

태스크별 샌드박스를 병렬화하여 롤아웃 효율을 극대화하는 경량 강화학습 파이프라인으로 일관된 성능 향상 달성

ClawGym-Bench: 신뢰성 있는 벤치마크

200

벤치마크 인스턴스 · 6개 카테고리

난이도 인지 필터링

  • 롤아웃 기반 난이도 보정
  • 지나치게 쉽거나 어려운 태스크 제외
  • 판별력 있는 태스크만 선별

LLM 보조 인간 검토

  • 자동 평가 + 인간 검토 이중 통과
  • 학습셋과의 중복 철저히 배제
  • 평가 안정성과 검증 가능성 보장

Claw 에이전트 간의 명확한 능력 격차를 모델 스케일별로 식별할 수 있는 진단 플랫폼

소형 모델

합성 데이터 학습만으로도 대형 모델의 기준 성능을 상회하는 극적인 향상

대형 모델

이미 높은 기준에서도 명확한 성능 향상을 달성, 데이터 품질의 유효성 입증

Claw 에이전트 행동 분석

도구 사용 적절성

상황에 맞는 올바른 도구 선택과 호출 방식 분석. 학습 전후 비교를 통해 불필요한 도구 호출 감소와 적절한 도구-태스크 매칭 개선 확인

장기 수행 견고성

다단계 워크플로우에서의 오류 복구 능력과 세션 간 의존성 처리 능력 평가. 긴 호라이즌에서의 안정성 향상을 정량화

세부 지시 준수

복잡한 다단계 지시에서 각 세부 요구사항을 정확히 수행하는 능력 분석. 부분적 수행 문제 진단 및 개선 방향 제시

단순한 집계 성능을 넘어, 미세한 행동 패턴까지 진단하여 향후 환경 기반 에이전트 연구에 방향성 제공

핵심 기여 요약

I

ClawGym-SynData

Claw 에이전트를 위한 최초의 대규모 합성 데이터셋 (13.5K 태스크). 페르소나 기반 + 스킬 기반 듀얼 라우트로 다양하고 검증 가능한 데이터 확장 생성

II

ClawGym-Agents

블랙박스 롤아웃 기반 SFT로 강력한 에이전트 패밀리 확보. 샌드박스 병렬화 경량 RL 파이프라인으로 일관된 이득 달성

III

ClawGym-Bench

6개 카테고리 200개 벤치마크 인스턴스. 난이도 인지 필터링과 LLM 보조 인간 검토로 신뢰성 보장
