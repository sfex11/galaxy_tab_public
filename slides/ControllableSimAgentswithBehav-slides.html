제어 가능한 시뮬레이션 에이전트

Controllable Sim Agents via Behavior Latents

Juanwu Lu (Purdue University) · Junyu Zhu (University of Tokyo) · Ziran Wang (Purdue University)

arXiv:2607.02496v1 · 2026년 7월

문제 정의

왜 제어 가능한 시뮬레이션이 필요한가?

자율주행 검증의 과제

  • 실세계 테스트는 비용이 막대함
  • 기존 시뮬레이터는 충실도와 확장성 부족
  • 특정 엣지 케이스 재현이 어려움

제어 가능성의 필요성

  • 변수 격리를 통한 체계적 테스트
  • 해석 가능한 축을 따른 에이전트 조향
  • 실제 위험 없는 자율주행 시스템 평가

현실적인 교통 시뮬레이션은 로그된 행동을 모방하면서도, 해석 가능한 축을 따라 조향 가능한 에이전트를 요구한다.

기존 방법의 한계

선택적 접근법들과 그 부족한 점

접근법 장점 한계
모방 학습 다양하고 현실적인 행동 생성 세밀한 제어 불가, 사용자 목적 적응 어려움
VAE 기반 잠재 공간 재샘플링으로 변형 생성 특정 선호도로 의도적 조향 불가
확산 모델 가이던스 주입으로 제어 개선 과도한 가이던스 시 비현실적 궤적 발생
자율 플레이 RL 보상 함수를 통한 행동 유도 온라인 훈련 비용 큼, 보상 변경 시 재훈련 필요

에이전트 집단 간 선호도 프로필의 비교 가능한 표현도 부재하다.

계층적 그래픽 모델

표준 HMM (기존)

히스토리 → 잠재 세계 상태 → 에이전트 관측 + 지도

  • 로그 행동 재현 가능
  • 행동 조향 토글 없음

확장 PGM (제안)

에이전트별 행동 프로필 λ_n ∈ ℝᴷ 추가

  • 각 에이전트가 보상 채널을 가중합
  • λ_n을 조건부 생성 변수로 활용

핵심 인사이트: 행동 이질성 = 각 에이전트가 공유 보상 신호 세트를 어떻게 가중하는가

수식적 전개

보상 경사 인자와 결합 정규 사후 확률

1

단계별 선호 인자: ψ_t = exp(γ^(t−1) · λ_n^⊤ · r(o_{t,n}))

할인 인자 γ와 보상 벡터 r의 내적으로 시간 가중 선호도 누적

2

궤적 수준 인자 축약: ψ(τ_n, λ_n) = exp(λ_n^⊤ · G_n)

신뢰 전파로 궤적 곱을 단일 내적으로 축소

3

채널별 할인 보상: G_n = Σ γ^(t−1) · r(o_{t,n}) ∈ ℝᴷ

K개 보상 채널 각각에 대해 할인된 누적 보상 계산

폐쇄형 결합 정규 사후 확률

정규 사전 확률 p(λ)와 지수족 인자 ψ의 결합으로 해석적 KL 최소화 가능 → 추가 네트워크 없이 각 에이전트의 (μ_n, Σ_n) 직접 계산

핵심 기술 1

소프트 자격 게이트

Soft Eligibility Gates — 임계값 부근 에이전트의 기울기 신호 보존

하드 자격 (기존)

이진 임계값: 임계값 미달 → 보상 신호 완전 차단

  • 임계값 부근 에이전트 기울기 소실
  • 안전성 제어력 저하
  • 보상 해킹 유발 가능

소프트 자격 (제안)

지수 감쇠: 임계값에서 부드러운 전환

  • 근접 에이전트도 기울기 수신
  • 안전성 제어력 현저히 향상
  • 속도 충실도 유지

보상 신호 희소성 문제를 해결하며, 안전성 조향이 단조롭고 실질적임을 실험으로 확인

핵심 기술 2

혼합 채널 마스크 커리큘럼과 CFG

혼합 채널 마스킹

훈련 시 임의의 채널 부분집합을 마스킹하여, 단일 모델이 다양한 조건부/무조건부 조합을 학습

  • 전체 채널 조건부 (완전 제어)
  • 일부 채널 조건부 (부분 제어)
  • 전체 무조건부 (모집단 한계)

Classifier-Free Guidance

추론 시 조건부/무조건부 속도장 선형 보간

  • 가이던스 스케일 α로 조향 강도 조절
  • 사용자가 일부 축만 고정 가능
  • 오일러 ODE 솔버로 궤적 샘플링

컨트래스티브 조건부 학습

동일 장면 내 다른 에이전트의 λ를 네거티브 샘플로 사용하여 조건부 분별력 강화

추가 기법

컨텍스트-잔여 보상 라벨링

Context-Residual Return Labeling — 지도 준수 제어를 위한 보상 재설계

1

컨텍스트 베이스라인

장면 평균 보상으로 정규화된 기준선 계산

2

잔여 보상 추출

개별 에이전트 보상 − 컨텍스트 베이스라인

3

지도 준수 제어

잔여 보상으로 차선 중심선 준수 조향 달성

절대적 지도 보상은 장면 간 분산이 커서 제어 불가 → 장면 내 상대적 편차를 사용하여 해결

실험 결과

Waymo Open Motion Dataset 벤치마크

현실성 벤치마크

  • 경쟁력 있는 현실성 달성
  • 순위가 높은 모방 학습 모델들과 대등
  • 추가 제어 가능성 확보 — 기존 모델들이 결여된 기능

제어 가능성 검증

  • 속도/가속도 기반 조향: 단조 응답
  • 정체 유발 보상 해킹 없음
  • 소프트 자격 게이트: 안전성 제어 단조적·실질적 향상

요약

기여 요약과 시사점
