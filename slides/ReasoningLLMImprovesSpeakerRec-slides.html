문제 정의와 동기

왜 장편 드라마가 어려운가?

  • 수십 시간의 영상, 100명 이상의 등장인물
  • 복잡한 줄거리와 인물 관계망
  • 짧은 대사, 겹치는 말소리, 화면 밖 대사
  • 시즌 간 캐릭터 외모 변화

화자 인식(Speaker Recognition)이란?

각 대사(utterance)를 정확한 등장인물에게 귀속시키는 과제

기존 연구는 화자 분리(diarization)·검증(verification)에 집중했으나, 드라마 특유의 복잡한 서사 맥락은 거의 다루지 않음

DramaSR-LRM 방법론 개요

핵심 아이디어

Large Reasoning Model(LRM)이 다중모달 도구를 자율적으로 호출하여 맥락 증거를 수집하고, 이를 종합하여 화자를 추론

단순한 음성 생체특징 매칭을 넘어, 시각·청각·언어적 단서를 추론 기반으로 통합

백본 모델

Qwen3-8B 기반 LRM

  • SFT (Supervised Fine-Tuning)
  • RL (Reinforcement Learning) — GRPO 기반
  • 추론 일관성 최적화

SFT 데이터: Gemini-3-Pro로 50K 대사에 대해 고품질 추론 궤적 생성

다중모달 도구 세트

voice_sim

음성 지문 간 코사인 유사도 계산. 목표 음성과 알려진 캐릭터 임베딩 간의 음향적 유사성 측정

video_cap

MLLM 기반 비디오 캡셔닝. 대사 발화 시점의 시각적 맥락(누가 화면에 있는지, 무엇을 하는지) 추출

char_relation

관계형 데이터베이스. 사회적 계층과 호칭('엄마', '자기야' 등)을 파싱하여 대화 맥락 보조

LRM은 이 세 가지 도구를 자율적으로 선택·조합하여 단서를 수집하고 최종 화자를 추론

학습 파이프라인

Phase 1 SFT (지도 미세조정)

  • 1개 드라마(50K 대사) 사용
  • Gemini-3-Pro가 고품질 추론 궤적 생성
  • 도구 호출 순서, 증거 수집, 결론 도출 과정 학습

Phase 2 RL (강화학습)

  • GRPO(Group Relative Policy Optimization) 알고리즘
  • 추론 일관성(reasoning consistency) 최적화
  • SFT만으로 부족한 복잡 사례에서의 성능 향상
학습에 사용된 드라마 2편은 평가에서 완전히 제외 — 11편 드라마에서 엄격한 평가 수행

다운스트림 활용: 비디오 이해

화자 인식이 왜 필수적인가?

  • 밀집 비디오 캡셔닝의 정확도 향상
  • 음성 구동 비디오 생성의 입력 품질 보장
  • 의미적 비디오 편집에서 캐릭터 일관성 유지
  • 드라마 줄거리 이해의 기초 전제

예비 평가 결과

정확한 화자 귀속은 드라마 전체 이해를 위한 중요한 전제 조건임이 실험적으로 확인됨

화자 레이블 품질이 직접적으로 다운스트림 비디오 QA 및 캡셔닝 성능에 영향을 미침

결론 및 향후 과제

주요 기여

  • DramaSR-532K: 장편 드라마 화자 인식을 위한 최대 규모 벤치마크 (532K 대사, 900+ 캐릭터)
  • DramaSR-LRM: 다중모달 도구 사용 기반 추론 모델로 SOTA 달성
  • 짧은 대사에서 음성 생체특징의 한계를 추론으로 극복하는 방법론 제시

향후 방향

  • 오픈월드 화자 기술 (Open-world Speaker Description)
  • 엔드-투-엔드 화자 인식 파이프라인
  • 더 큰 백본 모델 적용을 통한 스케일업
  • 다국어 드라마로의 확장
