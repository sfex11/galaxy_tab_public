연구 배경과 동기

LLM의 이중적 모습
  • 고난도 수학 경시대회에서 인상적인 성능
  • 연구 수준의 정리 증명에도 기여
  • 하지만 진정한 추론자인지는 불분명
확률의 특수성
확률 이론은 형식적 엄밀성과 직관 사이의 중간 지대에 위치한다. 정답을 형식적으로 검증할 수 있으면서도, 인지 편향의 함정을 고스란히 간직하고 있다.
핵심 질문: LLM은 확률 문제에서 체계적인 추론 오류를 저지르는가?

관련 연구

인지 편향 연구
  • GPT-3.5에서 닻 효과, 손실 회피 등 인간 유사 편향 확인
  • 반면, 비인간적 오류 패턴도 보고
  • 모델 발전에 따라 인지 테스트 성능 향상
토큰 편향
  • Jiang et al.: 정형 문제 성공이 토큰 패턴 매칭일 가능성
  • 진정한 추론이 아닌 학습 데이터 기억에 의존
아첨 현상
  • 사용자 의견에 맞춰 답변을 변형하는 경향
  • 수학적 증명에서도 오류 정리를 수용
기존 연구의 한계: 인지 편향, 토큰 편향, 아첨을 통합적으로 분리하지 못함

연구 방법 — 두 가지 데이터셋

표준 데이터셋
대학 교재에서 선별한 기본 이산 확률 문제
  • 직관적 해결 경로가 명확
  • 휴리스틱 함정 최소화
  • 기본 확률 도구 적용 평가
직관 위배 데이터셋
휴리스틱 추론이 수학적 정답에서 벗어나는 문제
  • 몬티홀, 생일 문제 등 잘 알려진 역설
  • 덜 알려진 직관 함정 문제 포함
  • 저자 직접 제작한 문제도 포함

연구 방법 — 모델과 실행

최신 상용 LLM 16개 모델 × 2가지 설정
평가 차원 설명 목적
CoT 여부 각 모델을 Chain-of-Thought 유무로 각각 테스트 단계적 추론의 효과 측정
토큰 편향 정형 공식 ↔ 구조 동일 but 위장 변형 비교 학습 데이터 패턴 의존도 확인
아첨 현상 프롬프트에 오답 제시 삽입 오류 제시에 대한 저항력 평가
아첨 실험의 특징: 인위적 오답뿐만 아니라 다른 LLM이 생성한 오답도 프롬프트에 삽입

인기 벤치마크의 복잡한 작업에 비해 상대적으로 단순한 문제임에도 불구하고, 직관 위배 문제에서의 성능 저하는 현저하다. 이는 일반 수학 벤치마크가 확률적 추론의 특정 측면을 포착하지 못함을 시사한다.

핵심 결과 2 — 토큰 편향의 증거

정형 공식을 구조적으로 동일한 위장 변형으로 교체했을 때
>20%
성능 하락폭
  • 문제 구조는 동일 — 수식적 본질 불변
  • 표현 방식만 학습 데이터에서 덜 익숙한 형태로 변경
  • 정답률 급락은 토큰 수준 패턴 매칭에 의존함을 강력히 시사
즉, 모델이 "문제를 이해해서 푸는 것"이 아니라 "학습 데이터에서 본 패턴을 인식하는 것"에 가깝다

사례 연구 — 가면을 쓴 몬티홀 문제

정형 버전
"3개의 문 중 하나에 자동차가 있습니다. 참가자가 하나를 선택하면, 진행자가 자동차가 없는 문을 엽니다. 바꾸는 것이 유리한가?"
→ 대부분의 모델이 정답 도출
위장 버전
동일한 확률 구조이지만, 문/자동차/진행자라는 단어를 다른 맥락의 용어로 치환하여 재구성
→ 정답률 급락
구조적으로 완전히 동일한 문제임에도, 학습 데이터에 빈번히 등장하는 "몬티홀"이라는 토큰 패턴이 사라지면 추론 능력이 붕괴한다.

결론

현재의 LLM은 아직 진정한 확률적 추론자가 아니다.
  • 고도의 수학 문제를 풀 수 있어도, 직관 함정 앞에서는 취약하다
  • 성공이 추론 능력이 아니라 학습 데이터의 패턴 기억에 기인할 가능성이 높다
  • 토큰 편향과 아첨 현상에 의한 체계적 오류가 존재하며, 어떤 모델도 면역되지 않는다
  • 기존 수학 벤치마크는 확률적 추론의 약점을 포착하지 못한다
시사점: 평가 체계를 단순한 정확도에서 추론의 견고성(robustness)으로 전환해야 한다
