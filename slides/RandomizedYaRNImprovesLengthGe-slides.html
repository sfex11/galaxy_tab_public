Randomized YaRN을 활용한
장문맥 추론의 길이 일반화 개선

Manas Mehta · Fangcong Yin · Greg Durrett

New York University

Long-Context Positional Encoding Length Generalization

문제 제기: 길이 일반화의 한계

기존 접근: 데이터 확장

  • 긴 문맥 코퍼스로 추가 학습
  • 특정 길이를 분포 내로 가져옴
  • 진정한 길이 일반화는 달성하지 못함

또 다른 접근: 위치 인코딩 수정

  • YaRN, NTK 보간 등 후처리 기법
  • Needle-in-a-Haystack에서는 효과적
  • 복잡한 추론 과제에서는 성과 불안정

"데이터 엔지니어링과 연산량으로 특정 긴 시퀀스를 분포 내로 가져오는 것은, 진정한 길이 일반화를 부여하지는 않는다."

핵심 아이디어

짧은 문맥 데이터로 학습하면서도, 위치 인코딩을 통해 분포 외(OOD) 위치 표현에 모델을 노출시키자.

랜덤 위치 샘플링

짧은 시퀀스의 각 토큰에 대해, 더 긴 길이 분포에서 무작위로 샘플링된 위치의 YaRN 인코딩을 할당

토큰 순서는 유지되지만, 절대 위치 인덱스는 훨씬 더 넓은 범위를 커버

길이 일반화 커리큘럼

샘플링 길이 분포의 최댓값을 에포크별로 점진적으로 증가시키는 학습 스케줄

예: 8K → 16K → 24K → ... → 128K

핵심 통찰: 표준 학습에서는 결코 볼 수 없는 OOD 위치 회전을 학습 중에 관찰하게 함

배경 지식: YaRN

YaRN은 RoPE 기반의 문맥 확장 방법으로, 스케일 팩터 s를 통해 이론적으로 문맥 길이를 s × Lpre까지 확장 가능

NTK-by-parts 보간

저주파 RoPE 차원에 대해 보간을 적용하여 문맥 길이 확장

길이 의존적 어텐션 온도 스케일링

긴 문맥에서 어텐션 분포를 부드럽게 조정

YaRN은 추가 학습 가능한 파라미터를 도입하지 않으며, 학습 및 추론 시 원본 RoPE 인코딩을 대체하여 사용 가능한 유연성을 제공한다.

단, YaRN이 스케일링 범위 내의 회전을 생성할 수 있어도, 모델이 실제로 그러한 회전을 학습 중에 관찰하지 않으면 일반화가 어렵다.

방법론: 랜덤 위치 인코딩 (RPE)

Ruoss et al. (2023)에서 처음 제안된 전략으로, 훨씬 더 긴 길이 분포에서 위치 인덱스를 무작위 할당하여 모델을 OOD 위치에 노출

I ~ Uniform(PLLt)

PLLt = { S ⊆ {1, ..., Lt} | |S| = L }

PẼ(j) := YaRN(ij; s)

1

무작위 샘플링

{1, ..., Lt}에서 길이 L인 부분집합을 무복원 추출

2

오름차순 정렬

I = {i1 < i2 < ... < iL}로 정렬하여 토큰 순서 보존

3

YaRN 인코딩 할당

j번째 토큰에 YaRN(ij; s) 인코딩을 적용

방법론: 길이 일반화 커리큘럼

샘플링 길이 최댓값 Lt를 에포크별로 점진적으로 증가시켜, 모델이 점진적으로 더 넓은 OOD 위치 범위에 적응하도록 유도

목표 길이 도달

커리큘럼의 직관

  • 먼저 적절한 범위에서 외삽된 YaRN 인코딩 사용법 학습
  • 점진적으로 최종 목표 길이가 요구하는 더 큰 OOD 인덱스로 일반화
  • 초기부터 목표 길이를 설정하는 것보다 안정적이고 강건한 학습

추론 시에는 랜덤 샘플링 없이 표준 YaRN 인코딩 사용 (s' ≥ s 설정 가능)

실험 설정

BABILong

긴 문단 내 다중 needle에 대한 다단계 추론 평가

  • 3-hop reasoning 서브셋 사용
  • 복잡한 장문맥 추론 능력 측정

MRCR

다자간 대화에서 추론 기반 검색이 필요한 벤치마크

  • Multi-Round Coreference Resolution
  • 대화 맥락에서의 공지 참조 해결

제거 연구: 커리큘럼의 중요성

길이 커리큘럼을 제거하고 처음부터 최대 샘플링 길이를 고정하면 성능이 크게 저하

핵심 발견

  • RPE에서 커리큘럼 제거 시 OOD 평균 -18.3%p 급락
  • Randomized YaRN에서도 커리큘럼 제거 시 -7.9%p 저하
  • 점진적 OOD 노출이 강건한 일반화에 필수적

핵심 발견 요약

멀리 벗어난 OOD에서 최대 효과

학습 길이에서 가장 먼 128K에서 가장 큰 성능 격차를 보임. LoRA 대비 BABILong에서 +20.9%p, MRCR에서 +37.1%p

ID 성능 보존

분포 내 길이에서의 성능은 거의 동일하게 유지. 위치 인코딩 수정이 기존 능력을 훼손하지 않음

커리큘럼이 필수

점진적 길이 증가 없이는 OOD 일반화가 크게 저하. 단순 랜덤 샘플링만으로는 불충분

모델 불변성

Qwen2.5와 Olmo3 두 가지 다른 아키텍처에서 일관된 향상을 확인

결론 및 시사점

주요 기여

  • 커리큘럼 기반 학습 방법을 제안하여 LLM의 길이 일반화 능력 향상
  • 장문맥 추론 과제를 위한 효과적인 문맥 확장 레시피 제시
  • 짧은 문맥(<8K) 데이터만으로 16K~128K 범위의 일반화 달성

실용적 시사점

  • 대규모 장문맥 코퍼스 없이도 길이 일반화 가능
  • 추론 시 추가 비용 없음 (학습 시에만 적용)
  • 기존 YaRN 지원 모델과 호환 가능

"모델을 점진적으로 OOD 위치 분포에 노출시키는 것은 일반화 가능한 장문맥 추론을 위한 효과적인 레시피이다."
