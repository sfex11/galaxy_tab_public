배경

장문맥 추론의 새로운 전개

사고 모델의 부상

Qwen3, Claude, DeepSeek 등 단계별 추론 흔적을 생성하는 모델들이 복잡한 작업에서 강력한 성능을 달성

평가의 전환

장문맥 LLM 평가가 단순 검색에서 복잡한 추론으로 이동하며, 사고 모델이 자연스럽게 적용되는 추세

하지만...

사고 모델을 장문맥 추론에 적용할 때 반복적 복사(Repetitive Copying)라는 치명적인 실패 패턴이 발견됨

분석 방법

반복적 복사의 정량화

겹침률(Overlap Rate): 모델 출력 중 입력 프롬프트에서 직접 복사된 부분의 비율

슬라이딩 윈도우로 n-gram을 스캔하여 프롬프트 내 존재 여부를 판별

n = 3 3-gram

전체 복사 양을 포괄적으로 측정. 짧은 구절 포함

n = 10 10-gram

긴 연속 구간만 포착. 의도적 전사를 명확히 식별

대상 모델: Claude-Opus-4.5, DeepSeek-V3.2, Qwen-3.5-Plus, GLM-4.7, QwQ-32B, Qwen3.5-35B-A3B, Qwen3.5-9B

实证 발견

반복적 복사의 부정적 영향

정확도와의 음의 상관관계

  • 복사율이 높은 샘플일수록 정답률이 현저히 낮음
  • 모델이 토큰 예산을 문제 해결이 아닌 프롬프트 재생산에 소모

추론 길이 팽창

  • 불필요하게 긴 사고 흔적 생성
  • 컴퓨팅 비용 증가와 실용성 저하

핵심 발견: 복사 자체가 해로운 것이 아니다. 무차별적 복사가 문제다.

근원 원인 분석

불충분한 그라운딩이 원인

프롬프트를 핵심 증거(Key Evidence)와 무관한 방해 요소(Distractor)로 분리하여 분석

정답 샘플의 복사 패턴

핵심 증거에서 선택적으로 복사. 무관한 컨텍스트는 거의 복사하지 않음

오답 샘플의 복사 패턴

증거와 방해 요소를 구분 없이 무차별적으로 복사

모델이 작업 관련 증거를 무관한 컨텍스트에서 분별하지 못하는 것이 반복적 복사의 근원 원인

제안 방법

GEAR: Grounding Evidence-Aware Reward

정확도 신호를 증거 기반 보상과 방해 요소 페널티로 보강하는 보상 형성(Reward Shaping) 방법

정확도 보상

기존 RL과 동일한
정답 여부 기반 보상

그라운딩 보상

추론 흔적과 핵심 증거 간
n-gram 겹침 측정

방해 요소 페널티

무관한 컨텍스트와의
겹침에 대해 페널티 부여

보상 설계 상세

GEAR 보상의 세 가지 구성 요소

Racc — 정확도 보상

최종 답변이 정확하면 +1, 아니면 0

Rground — 그라운딩 보상

추론 흔적 y와 주석이 달린 지원 구간(support spans) e 간의 n-gram 겹침률로 측정

Rdistract — 방해 요소 페널티

추론 흔적 y와 무관한 컨텍스트 d 간의 n-gram 겹침률을 음수로 반영

RGEAR = Racc + λg · Rground + λd · Rdistract

λg, λd는 균형 하이퍼파라미터

데이터 구성

증거 주석 자동 구성 파이프라인

합성 벤치마크를 넘어 자연어 문서에서도 GEAR를 적용하기 위한 3단계 자동화 파이프라인

1
문서 분할
긴 문서를 청크 단위로 분할
2
QA 생성
LLM으로 청크 기반 QA 쌍 자동 생성
3
증거 주석
정답을 뒷받침하는 핵심 구간 자동 태깅

핵심 장점 임의의 문서 코퍼스에서 증거 주석 장문맥 QA 데이터를 자동으로 구축 가능 → GEAR의 확장성 확보

그라운딩 보상만으로는 부족: 모델이 증거를 더 참조하도록 유도하지만, 동시에 방해 요소도 함께 복사할 수 있음

결론

핵심 시사점

  • 반복적 복사는 장문맥 사고 모델의 만연한 실패 모드이며, 문맥 길이에 따라 악화됨
  • 근원 원인은 불충분한 증거 그라운딩 — 모델이 증거와 방해 요소를 구분하지 못함
  • GEAR는 간단하면서도 효과적인 보상 형성으로 선택적 증거 참여를 장려
  • 자동화 파이프라인으로 합성 벤치마크를 넘어 자연어 데이터로 확장 가능
  • 복잡한 추론으로 평가가 전환되어도, 정확한 그라운딩은 여전히 필수적 역량

"적게 복사하고, 더 많이 그라운딩하라" — 장문맥 추론에서 증거에 정확히 닻을 내리는 것은 여전히 상당한 개선 여지가 있는 능력
