언제 확률이 높은 답이 맞을까?
LLM에서 시퀀스 확률과 정답 여부의 관계에 대한 연구
Johannes Zenn, Jonas Geiping · Max Planck Institute for Intelligent Systems
arXiv:2606.27359v1 · 2026
동기: 디코딩의 근본적 질문
대부분의 디코딩 방법은 더 높은 확률을 갖는 출력으로 확률 질량을 이동시킨다. 그렇다면 — 높은 확률이 항상 정답을 의미하는가?
자연스러운 직관
LLM은 주로 올바른 데이터로 학습되므로, 올바른 연속이 더 높은 확률을 받을 것이라 기대
현실의 반례
Beam search는 가장 확률이 높은 시퀀스를 찾지만, 종종 성능을 저하시킴
디코딩 방법의 두 가지 클래스
Local 로컬 디코딩
매 토큰 단위에서 다음 토큰 분포를 조작
  • Low-temperature sampling (LTS)
  • Top-k sampling
  • Top-p (nucleus) sampling
  • ε-sampling
Global 글로벌 디코딩
전체 시퀀스 확률을 직접 최적화
  • Best-of-N (BoN) sampling
  • Power sampling
  • Scalable Power Sampling (SPS)
  • Mode-seeking (변분 목적)
로컬 vs 글로벌: 핵심 차이
시퀀스 확률: p(s|s̄) = ∏ p(sₜ | s̄, s<t)
로컬 방법의 한계
각 단계에서 국소적으로만 확률을 조정하므로, 전체 시퀀스의 글로벌 확률이 최대가 아닐 수 있음. 토큰별 최적 ≠ 시퀀스별 최적
글로벌 방법의 접근
Power distribution p(s)ᵝ / Z_β 를 사용하여 고확률 시퀀스에 가중치를 집중. BoN은 N→∞에서 글로벌 모드를 근사
두 클래스 모두 더 높은 시퀀스 확률을 추구하지만, 그것이 정답과 연결되는지가 핵심 질문
연구 설계: 4단계 상관관계 분석

발견 1: Within-Dataset 상관관계
고정된 방법으로 데이터셋 내 샘플 간에 로그 확률과 정답 여부의 상관관계 측정
데이터셋 상관관계 해석
MATH500 강한 양의 상관 가장 뚜렷한 확률-정답 연결
GPQA 약한 양의 상관 긍정적이지만 크지 않음
HumanEval, MedQA, MMLU 약한 양의 상관 작지만 일관되게 양수
IFEval 음의 상관 유일한 예외 — 형식 제약 과제
상관관계는 데이터셋과 모델 패밀리에 의해 결정되며, 디코딩 방법에는 크게 의존하지 않음
발견 2: Within-Method 상관관계
같은 방법 내에서 하이퍼파라미터를 조정하면 시퀀스 확률은 올라가지만…
핵심 발견
하이퍼파라미터를 조정하여 시퀀스 확률을 높여도, 더 정확한 시퀀스가 생성되지 않는다.
온도 ↓
로그 확률 상승 ↔ 정확도 불변
Top-k ↓
로그 확률 상승 ↔ 정확도 불변
BoN: N ↑
로그 확률 상승 ↔ 정확도 불변
Power β ↑
로그 확률 상승 ↔ 정확도 불변
발견 3: Across-Method 상관관계
더 높은 확률을 생성하는 방법이 더 정확한가?
대답: 일관되지 않다
방법 간에 확률-정확도 상관관계는 정확도 수준에 비례하여 나타나지만, 높은 확률을 내는 방법이 항상 더 정확하지는 않음.
주요 관찰:
  • 상관관계의 크기는 대략 모델의 기준 정확도(baseline accuracy)에 비례
  • 정확도가 높은 모델·데이터셋 조합에서만 양의 상관관계 관찰
  • 디코딩 방법 자체가 정확도 개선을 보장하지 않음
발견 4: Within-Sample 상관관계
같은 프롬프트에 대해 여러 응답을 생성했을 때, 확률이 높은 응답이 정답일까?
일반적 샘플
상관관계가 약하거나 0 주변에서 대칭적. 확률만으로 동일 프롬프트의 여러 응답 중 정답을 구별하기 어려움
더 많이 맞추는 샘플
정답 비율이 높은 샘플일수록 within-sample 상관관계도 더 크게 나타남
Self-Consistency에서 확률 가중 투표의 신뢰성은 이 약한 within-sample 상관관계에 의해 근본적으로 제한됨
Self-Consistency에 대한 시사점
Self-Consistency(Wang et al., 2023): 다수의 응답을 생성하고 다수결로 정답 선택
일반 다수결
각 응답에 동일 가중치. 확률 정보 미활용
확률 가중 다수결
시퀀스 확률로 가중치 부여. within-sample 상관이 약해 효과 제한적
Verifier 기반
외부 검증기 사용. 확률 의존하지 않아 더 신뢰 가능
확률 기반 자가 개선(Verifier-free)은 모델이 충분한 정확도를 가질 때만 정보로움. 정확도가 낮으면 확률 신호 자체가 신뢰할 수 없음
핵심 요약: 4가지 상관관계의 패턴
분석 수준 상관관계 존재? 디코딩에 활용 가능?
Within-Dataset 대체로 Yes 간접적 — 모델이 정답을 식별할 수 있음을 보여줌
Within-Method 아니오 하이퍼파라미터 튜닝으로 정확도 개선 불가
Across-Method 제한적 정확도가 높을 때만 약한 상관, 일관되지 않음
Within-Sample 약함/없음 확률 가중 집계의 신뢰성 제한
핵심: 데이터셋 내에서는 확률-정답 관계가 존재하지만, 디코딩 결정에 직접 전이되지 않는다
실무적 지침
권장
  • 높은 정확도 모델에서는 확률 정보가 어느 정도 유용
  • Self-Consistency는 다수결 기반으로 사용
  • 형식 제약 과제(IFEval 등)에서는 확률 기반 디코딩 주의
비권장
  • 하이퍼파라미터 튜닝만으로 정확도 개선 기대
  • 낮은 정확도 모델에서 확률 가중 자가 개선
  • 단일 샘플 내 확률만으로 정답 판별
디코딩 방법 선택 시 "확률을 높인다 = 정답이 된다"는 등식이 성립하지 않음을 인지해야 함
결론
LLM에서 시퀀스 확률과 정답 여부 사이에는 단일하고 균일한 관계가 존재하지 않는다. 관계의 존재 여부와 강도는 분석 수준, 모델 정확도, 데이터셋 특성에 따라 달라진다.
기여 요약:
  • 디코딩 방법을 시퀀스 확률 최대화 프레임워크로 통일하여 정식화
  • 4단계 상관관계 분석 체계를 통한 정밀한 진단
  • 디코딩, Self-Consistency, Verifier-free 자가 개선의 한계와 조건 명시
