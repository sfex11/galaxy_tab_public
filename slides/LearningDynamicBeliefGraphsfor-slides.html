마음이론(ToM) 추론

LLM 기반 마음이론 추론에서 사람들의 잠재적·진화하는 신념을
에너지 기반 팩터 그래프로 구조화하여 행동을 예측하는 프레임워크
Ruxiao Chen · Xilei Zhao · Thomas J. Cova · Frank A. Drews · Susu Xu
arXiv: 2603.20170v1 · 2026.03

🎯문제: 왜 마음이론이 어려운가?

"신념은 상호의존적이고 시간에 따라 축적되지만, 행동은 희소하고 지연된다."

마음이론 (Theory of Mind) 추론의 핵심 과제

  • 사람들의 암묵적 신념이 불확실성 하에서 어떻게 행동을 결정하는지 추론
  • 재난 대응, 응급의료, 인간-자율시스템 협업 등 고위험 상황에서 필수
  • 신념은 비선형적으로 업데이트되며, 여러 신념이 서로 영향

❌ 기존 접근법의 한계

  • LLM 직접 프롬프팅 → 환각 문제
  • 잠재 상태 모델 → 신념을 정적·독립적으로 취급
  • 비일관적 정신 모델 생성

✅ 본 논문의 목표

  • 신념 간 상호의존성 모델링
  • 시간적 진화 추적
  • 신념 → 행동 연결의 구조화
  • 신념 레벨 감독 불필요

💡3가지 핵심 기여

1️⃣ 의미-포텐셜 투영 (Semantic-to-Potential Projection)

LLM의 텍스트 확률적 진술을 그래픽 모델 업데이트로 매핑

h_{t,i} = p(b_{t-1,i}=1) · h^Yes + (1-p(b_{t-1,i}=1)) · h^No

2️⃣ 에너지 기반 팩터 그래프 (Energy-Based Factor Graph)

신념 간 상호의존성을 깁스 분포의 에너지 함수로 표현

E(b_t) = Σᵢ log φᵢ(b_{t,i}) + Σᵢ<ⱼ log φᵢⱼ(b_{t,i}, b_{t,j})

3️⃣ ELBO 기반 학습 목표

신념 축적과 지연된 의사결정을 포착하는 변분 학습

ℒ = -Σₜ (ℒ^act_t - ℒ^KL_t)

ℒ^act 행동 우도   ℒ^KL 추론-전이 일관성

포텐셜 함수의 구성

단항 포텐셜 (Unary)

φᵢ = τ(cos(h, h^Yes) - cos(h, h^No)) + wᵤᵀReLU(h) + βᵤ

대조적 정렬 앵커링 + 학습 가능 보정

쌍 포텐셜 (Pairwise)

φᵢⱼ = wₚᵀReLU(hᵢⱼ) + βₚ

🧩 추론 모델 (비대칭 설계)

추론 시: q(bₜ | oₜ, aₜ) — 행동 정보를 조건으로 포함
