다목적 다중 외판원 문제
MOMTSP란 무엇인가?

🤖

문제 설정

  • 완전 그래프 G(Ξ, E)
  • na개 에이전트(로봇)
  • nξ개 목표 노드 + 1개 기지(depot)
  • 모든 노드를 정확히 한 번 방문
⚖️

상충하는 두 목적

  • f₁ : 총 이동 거리 최소화
  • f₂ : 최대 개별 경로(makespan) 최소화
  • 파레토 프론트 근사 필요

🔢 조합 폭발의 규모

100개 도시 + 4개 에이전트 → 약 10163개 후보 솔루션, 파레토 최적 검증에 O(10326) 비교 필요

분해 스칼라화 (Tchebycheff):   minπ gtc(π|λ, z*) = maxj { λj · |fj(π) − zj*| }

인코더 상세

조건부 어텐션 인코더

선호도 벡터 λ를 노드 임베딩에 직접 융합하여, 서로 다른 목적 간 균형을 명시적으로 제어합니다.

  1. 노드 임베딩 및 선호도 벡터에 인스턴스 정규화 적용
  2. Feature-wise Affine (FWA) 변환으로 선호도를 노드 표현에 주입: γ, β 학습 파라미터
  3. 8-head 멀티헤드 어텐션(MHA)으로 노드 간 관계 학습
  4. 게이트 집계(GA) 모듈이 학습 가능한 게이트로 정보 흐름 제어
  5. 2-layer MLP (ReLU) 피드포워드 서브레이어 + 전용 GA 모듈
  6. 6개 레이어를 거쳐 향상된 노드 임베딩 HξL 출력
핵심 혁신

기존의 단순 잔차 연결 대신 게이트 집계(GA)를 도입 — 학습 가능한 시그모이드 게이트가 어텐션 정보의 유입량을 동적으로 조절합니다.

협력 디코더의 이중 선택 메커니즘

🅰 에이전트 선택 모듈

  • max pooling으로 에이전트별 집계 임베딩 생성
  • 전역 표현: g̃ᵗ = 평균(Ĥᵢᵗ) + g
  • 상태 벡터: Cᵢ = (x, y, 누적거리)
  • 어텐션 스코어: κ·tanh(…/√dk)
  • 기지에 있는 에이전트 마스킹

🅱 노드 선택 모듈

  • 선택된 에이전트 맥락 + 누적 거리 연결
  • M=8 멀티헤드 어텐션
  • 방문 노드에 −∞ 마스킹
  • 단일 헤드 어텐션(SHA)으로 최종 노드 선택
  • 자기회귀 확률 ∏ pθ(aₜ,ξₜ)
정책 확률: pθ(π) = ∏t=0T pθ((aₜ, ξₜ) | ℐ, λ, na, nξ, 과거 선택들)

핵심 기여

CAMO의 4대 기여

1. 이중 복잡성 통합 해결

다중 에이전트 조율 + 다목적 트레이드오프를 단일 신경망으로 동시 해결하는 최초의 조건부 솔버

2. 명시적 선호도 제어

조건부 인코더를 통해 선호도 벡터 λ로 목적 간 트레이드오프를 직접 제어 — 재학습 불필요

3. 우수한 파레토 프론트 근사

신경망 및 전통적 휴리스틱 모두를 상회하는 하이퍼볼륨 달성 (최대 +7.55% 개선)

4. 실용적 검증

실제 모바일 로봇 플랫폼에서의 현실 세계 검증 — 시뮬레이션에서 현실로의 전이 가능성 입증
