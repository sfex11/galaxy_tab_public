01 — 연구 배경

왜 P(y)를 최적화하는가?

기존 RLVR의 한계
검증 가능한 보상을 활용한 강화학습(RLVR)은 조건부 분포 P(y|x)를 최적화하여 LLM 추론 능력을 향상시킵니다. 그러나 이 접근법은 기저 모델의 기존 출력 분포에 의해 근본적으로 제한됩니다.
🔒
Post-train 공간 (기존)
조건부 정책 πθ(y|x) 최적화
입력 쿼리 x에 항상 종속
기저 모델 능력의 천장 존재
🔓
Pre-train 공간 (제안)
주변 정책 πθ(y) 최적화
입력 조건 없이 추론 궤적 직접 최적화
출력 분포의 경계를 확장
Post-train: max Ex~X[R(y) · log πθ(y|x)]  →  Pre-train: max E[R(y) · log πθ(y)]
02 — 이론적 기반

그래디언트 정렬 검증

핵심 통찰: log P(y)와 log P(y|x)의 그래디언트가 정렬된다
log πθ'(y|x) ≈ log πθ(y|x) + η · R(y) · [∇ log πθ(y)]T ∇ log πθ(y|x) + O(η²)
1차 테일러 전개로 증명: P(y) 최적화가 P(y|x)를 그래디언트 내적이 양수일 때 함께 개선합니다.
