LLM 학습 효율을 위한
데이터 조직화의 체계적 탐구

Demystifying Data Organization for Enhanced LLM Training

Yalun Dai, Yangyu Huang, Tongshen Yang 등
NTU · Microsoft Research · HKUST

동기와 핵심 질문

"어떤 데이터를 쓸 것인가"뿐만 아니라
"데이터를 어떤 순서로 제공할 것인가"가 중요하다

잘 연구된 영역

  • 데이터 수집 · 필터링
  • 중복 제거 · 데이터 선택
  • 데이터 믹싱 · 합성

미탐구 영역

  • 학습 데이터의 순서 조직화
  • 1~2 epoch 제한 하의 전략적 배치
  • 체계적인 가이드라인 정립

LLM은 대규모 코퍼스를 단 1~2회만 학습하므로, 데이터 제공 순서가 학습 궤적을 직접적으로 좌우함

문제 정의: 3단계 파이프라인

1
Data Scoring
각 샘플에 점수 γ 부여
(품질, 난이도 등)
2
Data Selection
상위 K개 샘플 선별
(데이터셋 크기 축소)
3
Data Organization
순열 π에 따라 재배열
← 본 논문의 핵심

핵심 아이디어

이미 계산된 샘플 점수 γ를 재활용하여 데이터 조직화 수행 → 추가 연산 비용 거의 없음

Curriculum Learning(CL)은 단순히 점수 오름차순 정렬에 해당하는 특수한 경우일 뿐임

G1: Boundary Sharpening

학습의 시작과 끝 경계에서 데이터 특성을 명확히 제어

시작 경계

저점수 데이터로 시작 → 조기 학습 안정화

부적절한 초기 데이터는 최적화 불안정 및 수렴 방해

종료 경계

고점수 데이터로 마무리 → 성능 극대화

복잡한 추론 능력을 효과적으로 학습

구현: Segment Ordering (SEG)

정렬된 데이터를 L개의 백분위 구간으로 분할 → 각 구간 내 셔플 → 순서대로 연결
→ 점수가 점진적으로 상승하는 구조화된 커리큘럼 생성

G2: Cyclic Scheduling

엄격한 저→고 점수 순서는 치명적인 망각 문제 유발

문제점

후반부에 고난도 샘플만 집중하면, 초기에 학습한 기초 지식을 망각
→ 1-pass 학습 체제에서는 되돌릴 수 없음

해결책

주기적 스케줄링: 일정 주기마다 다양한 점수 분포의 데이터를 재도입
→ 기초 지식 유지 + 망각 방지

단순한 Baby Step 방식보다 더 체계적인 점수 분포 재방문 설계 필요

G3: Curriculum Continuity

점수 분포 간의 부드러운 전환으로 학습 안정성 유지

급격한 전환

점수 분포가 갑자기 변하면 학습 충격 발생
→ 최적화 궤적 붕괴, 모멘텀 상실

연속적 전환

인접 세그먼트 간 점수 분포가 점진적으로 변화
→ 모델이 자연스럽게 적응

핵심 원리

연속성이 보장된 데이터 순서는 학습의 안정성과 점진적 적응을 동시에 달성하여
훈련 중 모멘텀을 잃지 않고 일관된 개선 궤적 유지

G4: Local Diversity

미니배치 내 다양한 샘플 혼합으로 과적합 방지

왜 필요한가?

점수 기반 정렬 시, 인접 샘플 간 유사도가 높아짐
→ 동질적인 미니배치 → 국소 과적합 → 일반화 성능 저하

해결 전략

세그먼트 내 무작위 셔플을 통해 지역적 다양성 확보
→ 다양한 패턴 학습 촉진 → 일반화 능력 향상

G1(경계) + G3(연속성)이 전역적 구조를 설계한다면, G4(다양성)은 지역적 품질을 보장하는 상보적 역할

네 가지 가이드 통합 관점

각 가이드는 상호 보완적으로 작동하여 최적의 학습 순서 설계

G1
Boundary Sharpening
시작/끝 경계 제어
G2
Cyclic Scheduling
주기적 재방문으로 망각 방지
G3
Curriculum Continuity
부드러운 전환으로 안정성
G4
Local Diversity
지역적 다양성으로 일반화
가이드 적용 범위 목표
G1 전역 (시작/끝) 안정화 + 성능 극대화
G2 전역 (주기적) 망각 방지
G3 세그먼트 간 학습 안정성
G4 세그먼트 내 일반화 향상

방법론: STR과 SAW

네 가지 가이드를 체계적으로 통합한 두 가지 데이터 순서화 전략

STR Segment-based Ordering

  • G1: 경계 샤프닝 (저→고 점수)
  • G3: 연속적 커리큘럼
  • G4: 세그먼트 내 셔플

직관적이고 구현이 간단한 선형 구조

SAW Sawtooth Ordering

  • G1: 경계 샤프닝 유지
  • G2: 톱니파형 주기 스케줄링
  • G3: 연속성 + G4: 다양성

주기적 재방문으로 망각 방지 강화

SAW는 점수-인덱스 분포에서 톱니파형(sawtooth) 패턴을 형성하여,
전체적으로 상승하면서도 주기적으로 다양한 점수 분포를 재방문

학습 데이터 규모

구성 요소 상세 내용
기본 모델 LLaMA 구조 (160M, 410M, 1B, 1.7B)
베이스라인 Random, Curriculum Learning (CL), DELT
점수 함수 DSIR (사전학습), Llama-3-8B (SFT)
평가 벤치마크 MMLU, ARC, HellaSwag, WinoGrande 등

가이드별 분석 결과

각 가이드가 학습에 미치는 영향을 개별적으로 검증

G1 Boundary Sharpening

고점수로 시작 시 초기 loss 불안정
→ 저점수 시작이 유의미하게 안정적

G2 Cyclic Scheduling

순방향 CL 대비 주기적 재방문이
→ 기초 벤치마크에서 망각 방지 효과 확인

G3 Curriculum Continuity

급격한 점수 점프 시 학습 충격 관찰
→ 연속적 전환이 loss 안정화에 기여

G4 Local Diversity

셔플 없이 정렬만 시 과적합 증가
→ 로컬 다양성이 일반화에 필수적

네 가지 가이드 모두 개별적으로 유의미한 효과를 확인하며, 통합 시 시너지 효과 발현

주요 실험 결과

SAW가 모든 모델 규모에서 일관되게 우수한 성능 달성

규모 확장성

모델이 클수록 SAW의 이득이 더 커짐
→ 대규모 모델에서 더 효과적

학습 안정성

Loss 곡선의 변동성 감소
→ 더 안정적인 수렴 패턴

SFT 단계 및 추가 검증

사전학습뿐 아니라 지도 미세조정(SFT)에서도 유효성 확인

SFT에서의 효과

SFT 데이터에도 SAW 적용 시
지시어 따르기 능력 향상
→ 데이터 조직화가 SFT에서도 유효

데이터 크기 변화

다양한 데이터 크기(2B~15B 토큰)에서
일관된 개선 효과 확인
→ 규모에 구애받지 않는 robustness

Scaling Law 검증

Test loss 외삽 결과, SAW로 조직화된 데이터가 더 낮은 최종 loss로 수렴할 것으로 예측
→ 데이터 순서가 스케일링 법칙 자체에 긍정적 영향

결론 및 핵심 기여

이론적 기여

  • LLM 데이터 조직화의 최초 체계적 탐구
  • 4가지 핵심 가이드라인 공식화
  • 기존 CL의 한계점 식별

실용적 기여

  • STR, SAW 방법론 제안
  • 추가 연산 비용 거의 없음
  • Pre-train + SFT 양쪽에서 검증

"데이터 선택에 쓰인 점수를 재활용하는 것만으로도,
데이터 조직화는 무료로 성능을 끌어올리는 강력한 레버리지다."

향후: 더 다양한 점수 함수, 더 큰 규모(7B+)에서의 검증, 다른 아키텍처로의 일반화 가능성
