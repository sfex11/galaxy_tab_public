Yucheng Ruan, Ling Huang, Qika Lin, Kai He, Mengling Feng
National University of Singapore · Imperial College London

배경과 동기

정신 건강의 중요성

  • 전 세계 주요 공중보건 문제
  • 자동화 예측에 대한 수요 증가
  • SNS, 온라인 포럼 등 텍스트 데이터 활용

딥러닝의 진보

  • BERT 등 인코더 모델의 의미 표현 학습
  • GPT, LLaMA 등 생성형 LLM의 등장
  • Zero-shot / Few-shot 정신 건강 예측 시도
고위험 실제 환경에 배포하기 위해서는 단순한 예측 정확도를 넘어 신뢰성과 불확실성 추정이 필수적이다.

기존 방법의 한계

01

의미 표현 의존

인코더 모델(BERT)에만 의존하여 높은 차원의 추론 신호(인과관계, 감정 역학, 내적 불일치)를 포착하지 못함

02

과신(Overconfidence)

증거가 약하거나 모순될 때도 확신 있는 예측을 출력하여 고위험 환경에서 신뢰성 저하

03

불확실성 부재

베이지안 NN, 앙상블 등은 보정이 부족하고 실제 오분류 위험과 약한 상관관계를 보임

핵심 인사이트: 의미 vs 추론 관점

의미 관점 (Semantic)

"괜찮아", "잘 되고 있어", "아주 좋아" 등
표면적으로 긍정적인 표현이 지배적

→ 인코더 모델은 정상으로 분류할 가능성이 높음

추론 관점 (Reasoning)

논리적 불일치, 부적응적 인과 추론,
기본 생리 규범 위반 등 숨겨진 갈등

→ 디코더 모델이 고차원 추론으로 포착 가능

인간의 정신 건강 평가는 단일 의미 단서가 아닌, 여러 증거에 대한 추론과 사고·감정의 일관성 평가를 기반으로 수행된다.

제안 프레임워크 개요

정신 건강 예측을 다중 뷰 학습 문제로 공식화하고, 주관적 논리(Subjective Logic) 기반 증거적 융합으로 신뢰 가능한 예측 달성

1
다중 뷰 표현 생성
인코더 + 디코더 모델로 의미·추론 뷰 구성
2
증거 모델링
Dirichlet 분포로 신뢰도와 불확실성 명시적 표현
3
증거적 융합
불확실성을 고려한 보완적 뷰 통합
4
신뢰 가능한 예측
예측 + 불확실성 추정 + 해석 가능성

다중 뷰 표현 생성

의미 뷰 (Semantic View)

인코더 전용 모델(BERT 계열) 활용

  • 전체 의미 내용 포착
  • 언어 스타일 표현
  • [CLS] 토큰 임베딩 사용

추론 뷰 (Reasoning Views)

디코더 전용 모델(생성형 LLM) 활용

  • 증상 뷰: 정신 건강 증상 식별
  • 감정 뷰: 감정 패턴 분석
  • 인지 뷰: 인지 상태 추론

디코더 모델은 프롬프트 기반으로 각 추론 차원에 대한 텍스트를 생성한 후 임베딩하여 뷰를 구성

증거 모델링: 주관적 논리

Dempster-Shafer 이론

유한 판별 프레임 Ω = {ω₁, ω₂, ..., ωₖ}에 대해
기본 확률 할당(BPA) m: 2ᴼ → [0, 1]

  • m({ωₖ}): 클래스 ωₖ에 대한 신뢰 질량
  • m(Ω): 증거 불충분으로 인한 잔여 불확실성
  • 무지(ignorance)를 명시적으로 표현

Dirichlet 분포 매개변수화

신경망 출력을 Dirichlet 매개변수 α로 변환하여
클래스별 신뢰도와 전체 불확실성 동시 추정

  • 추가 샘플링 불필요 (MC Dropout 대비)
  • 단일 추론으로 불확실성 획득
  • 예측 신뢰도와 실제 오류의 정렬

다중 뷰 증거 융합 전략

이종 표현의 직접 융합은 예측 신뢰성을 해칠 수 있으므로, 각 뷰의 불확실성을 고려하여 가중 융합 수행

불확실성 기반 가중치 조정

불확실성이 높은 뷰의 증거를 자동으로 할인(discounting)

보완적 뷰 통합

의미 뷰와 추론 뷰의 상호 보완 정보 활용

신뢰 가능한 최종 예측

융합된 증거로부터 클래스 예측과 불확실성 동시 출력

실험 설정

데이터셋 과제 특징

SDCNL 자살 생각 감지 SNS 데이터, 이진 분류

베이스라인: BERT, RoBERTa, MentalBERT, GPT-4 Zero-shot, MC Dropout, Deep Ensemble 등 다양한 모델 비교

SOTA 달성

불확실성 추정 성능

예측 불확실성 vs 실제 오류

  • 본 프레임워크: 불확실성과 오류 간 강한 양의 상관관계
  • MC Dropout: 상관관계 약함, 보정 부족
  • Deep Ensemble: 추가 연산 필요, 개선 한계

고위험 사례 식별

  • 높은 불확실성 출력 → 인간 전문가 검토 대상 자동 선별
  • 위험 민감적 의사결정 지원 가능
  • 단일 추론으로 불확실성 획득 (효율성)

노이즈에 대한 강건성

실제 환경의 오탈자 및 텍스트 오염을 시뮬레이션한 탈자(noise) 실험 수행

실험 조건

  • 문자 수준 무작위 탈자 (5%~30%)
  • 단어 수준 교란 적용
  • 세 데이터셋에서 일관되게 평가

결과

  • 본 프레임워크: 성능 저하 최소화
  • 기존 모델: 노이즈 증가 시 급격한 성능 하락
  • 불확실성 기반 가중 할인이 강건성에 기여

해석 가능성 분석

사례 연구를 통해 프레임워크가 인간이 이해 가능한 추론 신호를 제공함을 확인

의미 뷰 분석

표면적 긍정 표현 → 정상 분류 경향

추론 뷰 분석

논리적 모순, 부적응적 인과 추론, 생리 규범 위반 탐지

융합 결과

추론 뷰의 신뢰도가 높을 경우 의미 뷰의 판단을 수정하여 올바른 예측 도달

불확실성 신호

뷰 간 모순이 큰 경우 높은 불확실성 출력 → 전문가 개입 권장

결론 및 주요 기여

의미를 넘어선 추론 뷰 도입

인코더 + 디코더 모델 결합으로 의미와 고차원 추론 정보 통합

증거적 불확실성 추정

주관적 논리 기반 명시적 불확실성 모델링으로 신뢰 가능한 예측

불확실성 인식 증거 융합

신뢰할 수 없는 뷰를 자동 할인하여 보완적 정보 효과적으로 통합

종합 성과

  • 3개 데이터셋 SOTA 달성
  • 우수한 불확실성 보정 성능
  • 노이즈 환경에서 강건성 입증
  • 인간 이해 가능한 해석 가능성
  • 위험 민감적 실제 응용 적합
