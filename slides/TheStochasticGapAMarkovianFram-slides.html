1

2
상태-행동 사각지대 질량 BSAπ,n(τ)

3

4
기대 감독 비용 항등식

신뢰성과 감독 비용을 동일 통계량으로 연결

"자율성 범위를 결정하는 동일한 통계량이 기대 감독 부담도 결정한다."

핵심 지표 #1: 사각지대 질량

상태 사각지대 질량

Bn(τ) = Σs d̂π(s) · 1{Nn(s) < τ}

지지 임계값 τ 미만인 상태의 배포 점유율 합계

상태-행동 사각지대 질량 (핵심!)

BSAπ,n(τ) = Σs,a d̂π(s,a) · 1{Nn(s,a) < τ}

에이전트가 실제로 내리는 결정(상태-행동 쌍)의 지지도

커버리지 상한 분해

E[Ct] = (1-BSA) · E[Ct|지지] + BSA · E[Ct|비지지]

"지지된 전이에서의 강한 성능만으로는 사각지대 전이의 성능을 보장할 수 없다"

핵심 지표 #2-3: 엔트로피 & 에스컬레이션

섀넌 엔트로피 (분기 모호성)

H(π̂(·|s)) = -Σa π̂(a|s) log2 π̂(a|s)

높은 엔트로피 = 다음 행동이 불확실 → 인간 개입 필요

위험 가중 전이 사각지대

wSA(s,a) = 0.6 · 거래가치 + 0.4 · 예외활동여부

거래 규모(60%)와 예외 민감 활동(40%)의 가중 결합

에스컬레이션 게이트 (인간 개입 기준)

G(s) = 1  if  {지지부족 ∨ 높은엔트로피 ∨ 높은위험}

세 조건 중 하나라도 충족되면 인간에게 에스컬레이션

자율성 비율: 이벤트 수준 Aevent vs. 케이스 수준 Acase
차이 = 경로 복합 효과 (한 단계라도 에스컬레이션되면 전체 케이스가 영향)

핵심 지표 #4: 감독 비용

기대 운영 비용 (케이스당)

C(π;G) = cA · E[T] + (cH - cA) · Σs Dπ(s) · G(s)

cA = 자율 결정 비용  |  cH = 인간 에스컬레이션 비용

완전 자율 완료

C̃0(G) = E[Πt (1-G(st)) · m(st)]

에스컬레이션 없이 모든 단계 정확히 수행

안전 완료 (폴백 포함)

R̃safe = E[Πt (G(st) + (1-G(st))m(st))]

인간 개입이 에스컬레이션된 단계를 복구

핵심 통찰: 자율성 범위를 한정하는 동일한 통계량(지지도, 엔트로피, 위험)이 기대 감독 부담도 결정한다 — 신뢰성과 비용은 분리 불가능

실무 시사점

🔍

배포 전 감사가 먼저

프롬프트 튜닝이나 파인튜닝 전에, 이벤트 로그가 이미 경험적 신뢰성-비용 경계선을 결정한다. 어디서 완전 자율이 정당화되지 않는지 먼저 파악해야 한다.

📊

상태가 아닌 상태-행동을 측정

상태 커버리지만으로는 불충분. 에이전트의 실제 결정 공간(상태-행동 쌍)에서의 지지도가 핵심이다.

⚠

경로 효과를 고려

단계별 자율성 72%도 케이스 전체로는 50% 미만으로 하락할 수 있다. 경로 복합 효과를 반드시 고려해야 한다.

💰

관문 상태에 집중 투자

33개 관문 상태가 케이스의 69.1%에 영향. 이 허브들의 예측력 개선이 전체 자율성을 7배 향상시킬 수 있다.

한계점

1. 관찰 데이터 제약

BPI 로그는 관찰적이며 에이전트가 생성한 것이 아님. 임의 행동의 반사실적 평가 불가. 사전 배포에 적합하나 온라인 평가를 대체하지 못함.

2. 상태 표현 가정

1차 마르코프 근사 사용. 더 풍부한 상태(도구 상태, 대화 이력 등)는 지지 문제를 악화시킬 수 있음. 보고된 사각지대는 하한.

3. 위험 프록시 제한

공개 BPI 필드 기반의 재현 가능한 가중치 설계. 실제 배포에서는 정책 중요도, 규정 준수 부담, 하류 오류 비용 등 도메인 특화 필요.

4. 산업 증거 범위

동기부여 사례들은 포괄적이지 않음. 핵심은 실제 배포가 신뢰성 실패와 감독 비용이라는 두 압력을 이미 겪고 있다는 것.

결론

"기업 환경의 에이전틱 AI는 프롬프트 문제가 아닌
확률적 제어 문제로 분석해야 한다"

주요 발견

  • 상태 커버리지 ≠ 행동 지지도 — 대규모 프로세스도 다음 단계 결정에 상당한 사각지대 유지
  • 이론적 대리지표가 실제 에이전트 행동 예측 — 홀드아웃 검증에서 3.4pp 이내 추적
  • 신뢰성-비용 경계선이 비분리적 — 동일 통계량이 두 차원을 동시에 결정
  • 경로 복합 효과 — 단계별 자율성이 케이스 전체에서 급격히 감소
