핵심 아이디어

기존 MLP 블록을 재활용한 Drop-in 방식

기존 Gated MLP

O = (φ(H·WgateT) ⊙ H·WupT) · WdownT
  • Wup, Wgate → 고정 (느린 가중치)
  • Wdown → 적응형 빠른 가중치

핵심 장점

  • Drop-in 방식: 사전학습된 체크포인트에서 바로 시작 가능
  • Warm-start: 처음부터 재학습 불필요
  • 매 6번째 레이어에만 적용
  • 추가 파라미터 극소 (Conv1D + Wtarget)

MLP의 최종 프로젝션 행렬(Wdown)을
추론 시점에 동적으로 업데이트되는 "빠른 가중치"로 재활용

청크 단위 처리

1단계: 적용 (Apply)

O[i] = Z[i] · (Wdown(i))T

현재 빠른 가중치로 출력 생성

2단계: 업데이트 (Update)

Wdown(i+1) = Wdown(i) + η·ΔW[i]

다음 청크를 위해 가중치 갱신

각 청크에서 Z[i], V̂[i],
ΔW[i] 동시 계산

유효 가중치로
출력 병렬 계산

LM 정렬 목적함수

다음 토큰 예측에 정렬된 학습 목표

기존 TTT 문제점

키와 값이 같은 입력 토큰의 프로젝션 → 미래 토큰 예측에 도움 안 됨

// 복원 목적함수 K = X · Wk, V = X · Wv // 같은 X에서 파생 → 예측 이득 ≈ 0

제안 해결책

미래 토큰에서 타겟을 도출하는 인과적 Conv1D 사용

// LM 정렬 목적함수 V̂ = Conv1D(X0) · Wtarget // 인과적 패딩 → 미래 정보 누출 없음

닫힌 형식 업데이트 규칙

Wdown(i) = Wdown(i-1) + η · V̂[i]T · Z[i]

이론적 보장 (정리 1)

LM 정렬 타겟

정답 로짓: ≥ λ·cnorm²·calign 증가
오답 로짓: ≤ λ·ε·calign (노이즈 수준)

복원 타겟

정답 로짓 변화: ≤ λ·ε·calign
예측적 이득 없음
