MemDreamer

계층적 그래프 메모리와 에이전트 기반 검색 메커니즘을 통한
지각-추론 분리를 위한 장시간 비디오 이해

Decoupling Perception and Reasoning for Long Video Understanding
via Hierarchical Graph Memory and Agentic Retrieval Mechanism

문제 정의

현재의 시각-언어 모델(VLM)은 수 시간 길이의 비디오를 처리하는 데 심각한 한계를 겪고 있습니다.

토큰 폭발

전체 비주얼 시퀀스를 처리하면 토큰 수가 기하급수적으로 증가하여, 연산 비용이 감당할 수 없는 수준까지 치솟습니다.

주의력 희석

긴 시퀀스에서 어텐션이 분산되어, 핵심 정보에 집중하기 어렵고 모델의 추론 성능이 급격히 저하됩니다.

"수 시간 분량의 비디오를 한 번에 처리하는 것은 단순한 스케일 확장으로 해결할 수 없는 근본적 병목입니다."

핵심 아이디어

지각(Perception)과 추론(Reasoning)의 분리(Decoupling)

P
지각 단계
비디오 프레임을 효율적으로 인코딩하여 메모리에 저장
M
메모리 구축
계층적 그래프 구조로 정보를 조직화
R
추론 단계
질문에 따라 필요한 정보만 에이전트가 동적 검색

전체 비디오를 한 번에 처리하는 대신, 에이전트가 질문의 의도에 맞춰 메모리에서 필요한 정보만 선택적으로 검색합니다.

MemDreamer 전체 구조

비디오 입력

수 시간 길이의 비디오를 세그먼트 단위로 분할하여 입력

계층적 그래프 메모리

다층 구조로 시공간 정보를 압축·조직화

에이전트 검색

질문 기반으로 메모리에서 관련 정보 동적 조회

지각은 한 번만 수행하고, 추론은 질문마다 on-demand로 실행 → 효율성 극대화

계층적 그래프 메모리

비디오의 시공간 정보를 다층 그래프 구조로 압축하여 저장합니다.

프레임 레벨 (Frame-level)

개별 프레임의 시각적 특징을 노드로 저장

세그먼트 레벨 (Segment-level)

인접 프레임들을 그룹화하여 요약 노드 생성

장면 레벨 (Scene-level)

세그먼트 간 관계를 그래프 엣지로 연결하여 고수준 의미 구조 형성

그래프 엣지는 시간적 연속성, 의미적 유사성, 인과 관계를 포착합니다.

에이전트 기반 검색 메커니즘

고정된 검색이 아닌, 질문을 이해하는 에이전트가 동적으로 검색 전략을 결정합니다.

질문 분석

  • 질문의 시간적 범위 파악
  • 필요한 세부 수준(레벨) 판단
  • 관련 객체/이벤트 유형 식별

적응적 탐색

  • 그래프 상에서 관련 노드 탐색
  • 필요시 하위 레벨로 드릴다운
  • 충분한 정보 확보 시 탐색 종료

에이전트는 질문의 복잡도에 따라 검색 깊이와 범위를 스스로 조절합니다.

왜 분리가 효과적인가?

기존 방식

지각-추론 통합

  • 매 질문마다 전체 비디오 재처리
  • 토큰 수 = 비디오 길이에 비례
  • 길어질수록 어텐션 품질 저하
MemDreamer

지각-추론 분리

  • 지각은 한 번만, 메모리에 저장
  • 추론 시 토큰 수 독립적
  • 질문 관련 정보에만 집중

비디오가 길어질수록 분리의 이점이 극대화됩니다. 10분 비디오와 10시간 비디오가 추론 비용 측면에서 거의 동일해집니다.

작동 예시

질문: "영화 후반부에서 주인공이 어떤 결정을 내렸나요?"

1
질문 파싱

시간 범위: "후반부" → 장면 레벨에서 후반 노드 타겟팅

2
그래프 탐색

후반부 장면 노드에서 주인공 관련 엣지를 따라 세그먼트 레벨로 드릴다운

3
관련 프레임 검색

핵심 결정 장면의 프레임 레벨 노드만 선택적 검색

4
추론 응답

검색된 정보만으로 LLM이 정확한 답변 생성

주요 실험 결과

↑

장시간 비디오 벤치마크에서 기존 방법 대비 일관된 성능 향상

↓

추론 시 토큰 수 대폭 감소로 비용 효율적

→

비디오 길이가 늘어도 성능 안정적 유지

메모리 기준선 양호 보통 감소
MemDreamer 양호 우수 최소

핵심 기여 요약

새로운 패러다임 제안

장시간 비디오 이해를 위한 지각-추론 분리 프레임워크를 처음으로 체계화

계층적 그래프 메모리

다층 그래프로 비디오 정보를 시공간적으로 조직화하여 효율적 압축 달성

에이전트 기반 검색

질문 의도를 이해하고 적응적으로 메모리를 탐색하는 검색 에이전트 설계

스케일 불변성 달성

비디오 길이에 관계없이 일정한 추론 비용으로 고품질 답변 생성

시사점과 향후 방향

시사점

  • 긴 컨텍스트 처리는 "더 큰 모델"이 아닌 "더 나은 구조"로 해결
  • 에이전트 패러다임이 비디오 이해에 강력히 적용 가능
  • 메모리 구조 설계가 모델 능력의 상한을 결정

향후 방향

  • 멀티모달 메모리 (오디오, 텍스트 통합)
  • 메모리 업데이트·삭제 메커니즘 (장기 기억 발전)
  • 실시간 스트리밍 비디오로의 확장
