사전 대응형 인프라 LiDAR 프레임워크

Proactive Infrastructure LiDAR Framework for Intersection Safety Assessment

Tam Bang, Hussam Abubakr, Emiliano de la Garza Villarreal, Truc Phuong Nguyen, Austin Harris, Toru Hirano, Mina Sartipi, Yunfei Xu, Hoang H. Nguyen

University of Tennessee at Chattanooga · arXiv:2607.16156v1 · IEEE 2026

도시 교차로의 안전 문제

사고 다발 지역

도로 네트워크에서 교차로는 가장 위험한 위치로, 교통 사망사고와 중상의 상당 부분을 차지

취약 교통약자(VRU)

보행자와 자전거 탑승자는 충돌 사건 시 부상 위험이 매우 높은 집단

다중 에이전트 상호작용

차량, 보행자, 자전거 간 복잡한 상호작용이 연속적 실시간 모니터링을 요구

"사전 대응형 충돌 감지는 현재 상태를 관찰하는 것만으로는 불충분하며, 미래의 상호작용을 예측해야 한다."

기존 방법의 한계

  • 사후 분석용 오프라인 방법에 국한
  • 상위 인식 출력과 충돌 감지 간 연결 부족
  • 차량 중심 평가로 전체 교차로 관점 누락
  • 수동 주석 데이터에 의존

왜 인프라 LiDAR인가?

01

저조도 환경 강건성

조명 조건과 무관한 정밀 3D 공간 측정으로 야간 교차로 모니터링에 적합

02

프라이버시 보호

시각적 식별 정보를 캡처하지 않아 카메라 대비 프라이버시 우려 최소화

03

정밀 상태 추정

위치, 방향, 속도를 조명 독립적으로 정밀하게 추정 가능

카메라 기반 시스템의 한계

원근 왜곡, 제한된 깊이 정확도, 저조도 성능 저하, 프라이버시 문제

클라우드 의존의 문제

변동적인 네트워크 지연으로 인해 안전 핵심 실시간 작업에서 결정적 실행 불가

PRISA 프레임워크 개요

인프라 LiDAR를 활용한 실시간, 엣지 배포 가능, 모듈형 교차로 안전 모니터링 프레임워크

위치, 속도, 가속도, 헤딩 등 정형화된 데이터

모듈 분리의 장점

인식 계층과 위험 평가 모듈 간 완전한 결합 해제로:

  • 다양한 검출·추적 시스템과 호환
  • 표준 상태 벡터만 준수하면 연동 가능
  • 이종 교차로 환경으로의 일반화 용이

인식 출력의 현실적 문제

실제 인식 출력은 노이즈가 많고, 단편적이며, 불완전한 궤적을 포함 → 자동화된 데이터 정제 필요

Phase I: 자가지도 데이터 정제와 모델 학습

Self-Supervised Data Curation — 수동 주석 없이 사이트 특화 학습 데이터 자동 생성

축적된 인식 출력 활용

상위 인식 계층에서 축적된 원시 궤적 데이터를 학습 데이터로 직접 활용

자동 품질 필터링

노이즈, 단편적 궤적, 불완전한 트랙을 자동으로 식별하고 제거하여 신뢰할 수 있는 학습 샘플 생성

사이트 특화 모델 학습

정제된 데이터로 해당 교차로의 교통 패턴에 특화된 궤적 예측 모델 자동 학습

"기존 방법은 대규모 수동 주석 데이터셋에 의존하지만, PRISA는 수동 주석 전혀 없이 인식 출력만으로 학습 데이터를 자동 구성한다."

핵심 이점

  • 수동 주석 비용 완전 제거
  • 새 교차로 배포 시 자동 적응
  • 최소 기술 전문 지식으로 배포 가능
  • 노이즈로 인한 모델 성능 저하 방지

Phase II: 실시간 궤적 예측

Real-Time Trajectory Prediction — 학습된 모델을 배포하여 연속적 운동 예측 수행

입력

표준화된 상태 벡터 (위치, 속도, 가속도, 헤딩 등) — 상위 인식 계층에서 실시간 제공

출력

2.4초 예측 지평선에 걸친 다중 도로 사용자 미래 궤적

다중 에이전트 처리

차량, 보행자, 자전거 등 교차로 내 모든 도로 사용자의 상호작용을 동시에 예측

예측 모델의 역할

궤적 예측은 단순 추적을 넘어:

  • 반응형 SSM의 한계 극복
  • 사전 대응형 충돌 감지 가능
  • 더 긴 개입 리드 타임 확보

기존 방법과의 차이

MID, Trajectron++ 등 정확도가 입증된 예측 방법들이 존재하나, 인프라 측 위험 평가 파이프라인과의 통합이 제한적이었음. PRISA는 이를 엣지 환경에서 실제 배포

이중 대체 안전 측정(Dual SSM) 평가

Surrogate Safety Assessment — 두 가지 상호보완적 지표로 광범위한 충돌 유형 커버

TTCTime-to-Collision

종방향 추종 충돌에 적합

  • 후미 추돌 상황 평가
  • 관측된 현재 상태 기반 (반응형)
  • SSAM 표준 방법론 채택
  • 실시간 제약 내 운영

PPETPredicted Post-Encroachment Time

횡단 및 VRU 관련 충돌에 적합

  • 횡단·분기·합류 경로 평가
  • 예측 궤적 기반 (사전 대응형)
  • 보행자·자전거 상호작용 포함
  • 반응형 PET보다 조기 경고

단일 지표보다 이중 측정 전략을 채택하여 더 넓은 충돌 유형 커버와 더 빠른 경고 능력을 동시에 달성

실험 설정 및 실제 배포

R-LiViT 데이터셋

공개 인프라 LiDAR 교차로 데이터셋

  • 다중 도로 사용자 클래스 포함
  • 실제 교차로 환경의 궤적 데이터
  • 궤적 예측 성능 벤치마킹에 활용

현장 배포: 채터누가, 테네시

실제 신호 교차로에 라이브 배포

  • NVIDIA Jetson AGX Thor 엣지 장치
  • 실시간 운영 검증
  • 클라우드 의존 없는 독립 운영

실시간 운영의 의미

194ms 지연은 2.4초 예측 지평선 대비 약 12배의 여유 시간을 제공하며, 충돌 사전 경고 및 개입에 충분한 리드 타임 확보

주요 기여 및 결론

C1

모듈형 엣지 배포 프레임워크

원시 라이다 데이터를 실행 가능한 충돌 지표로 변환하는 엔드투엔드 파이프라인. 클라우드 없이 임베디드 엣지 하드웨어에서 실시간 운영

C2

Plug-and-Play 위험 평가 모듈

자가지도 궤적 예측과 이중 SSM 충돌 평가를 통합. 표준 상태 벡터 기반으로 상위 인식 시스템과 독립적 운영

C3

실제 환경 검증

공개 데이터셋 평가 및 채터누가 실제 교차로에 Jetson AGX Thor 엣지 배포로 실시간 운영 타당성 입증

PRISA는 최소 기술 전문 지식으로 다양한 도로변 환경에 배포 가능하며, 수동 주석 없이 자동 적응하고, 엣지에서 실시간으로 교차로 안전을 사전 대응형으로 모니터링하는 실용적 솔루션을 제시한다.
