문제 정의

지각적 앨리어싱:
같은 관측, 다른 행동

로봇 조작에서 가림(occlusion)과 상태 변화는 동일한 시각적 관측이 서로 다른 상호작용 이력에서 발생할 수 있게 합니다. 현재 시점의 관측만으로는 올바른 행동을 선택할 수 없습니다.

👁

비-마르코프 관측

동일한 관측 → 다른 행동이 필요. 관측 수준에서 의사결정이 비-마르코프적입니다.

🗜

의미 압축의 한계

기존 방법은 의미적으로 압축된 흔적을 사용하여 구별에 필요한 세밀한 지각 단서를 버립니다.

🔍

유사성 검색 실패

유사도 기반 검색은 지각적으로 유사하지만 의사결정과 무관한 에피소드를 반환합니다.

💡 핵심 통찰: 인간의 에피소드 기억(해마)에서 영감을 받아,
기하학 기반 멀티모달 토큰 + 미분 가능한 기억 스택으로 해결

🌺

복측 경로 (외형)

동결된 DINO 인코더가 전면/핸드 카메라의 패치 토큰을 추출하여 세밀한 지역 외형 정보를 보존합니다.

Vtv = DINO(Itv) ∈ RNv×d
📐

배측 경로 (기하학)

말단장치(EE) 포즈를 기하학적 앵커로 사용하여 각 패치에 대해 3가지 기하 특성을 계산합니다.

gt,iv = [u, r, ρ, cosθ] ∈ R7

🔄 교차 뷰 향상

양방향 잔차 크로스 어텐션으로 두 뷰 간 정보 융합. 어텐션 로짓은 3가지 성분을 결합합니다:

표준 어텐션 QKT/√d 에피폴라 실현 가능성 편향 EE 기반 기하 편향
Attn(Q,K,U) = softmax(QKT/√d + Bepi + ba1T + 1(bb)T)U

FiLM 스타일 변조

기하학이 특성 변조를 조건화: γ(C) ⊙ V̄ + β(C) → 최종 출력 xt

기억 모듈

미분 가능한 기억 스택

🎯

공간 앵커 (A개)

라우터 네트워크가 시각 토큰에 대한 라우팅 가중치 π를 생성하여 각 공간 앵커에 대한 시각적 요약을 형성합니다.

ua = Σ πa,i x̄i
⏳

시간 앵커 (B개)

각 공간 앵커가 다른 반감기를 가진 B개 시간 슬롯으로 확장됩니다. 어텐션으로 슬롯 특성 계산.

fa,b = Attn(qa,b, z̃)

🧠 선택적 상태 공간 기억

에피소드 기억

각 슬롯 (a,b)가 잠재 엔그램 mt을 유지. 콘텐츠 의존적 SSM 매개변수화로 업데이트합니다.

mt = SSMepi(mt-1; θt)

작업 기억

에피소드 읽기를 집계하여 작업 상태에 통합. 문맥 의존적 게이팅 수행.

ht = SSMwork(ht-1; rt)

🔍 계층적 기억 융합

각 레이어의 작업 읽기를 과제 조건부 소프트맥스 가중치로 융합하여 최종 기억 출력 ht 생성

αt,l = softmax(⟨qtask, Ul yt(l)⟩)   |   ht = Σ αt,l Encl(yt(l))
학습 & 정책

HoloHead & 정책 모듈

🔮

HoloHead: 잠재 상상

ht에 조건화하여 미래 경유점을 예측하는 보조 목표. 기억 상태의 패턴 완성을 강제합니다.

  • 앵커: Na개 연속 프레임 (단기)
  • 나침반: Nc개 로그 간격 경유점 (장기)
  • 2D (카메라 투영) + 3D (월드 프레임) 동시 예측
Lholo = Σ (||ŴA - WA||1 + ||ŴC - WC||1)
🎮

정류 흐름 매칭 정책

기억 읽기 ct = Wctx ht만으로 조건화. 정책에 전달되는 유일한 신호입니다.

  • 보간: xτ = (1-τ)x0 + τx1
  • 속도 네트워크: Transformer + AdaLN
  • 50 스텝 고정 ODE 솔버
  • 궤적 토큰화 + 셀프/크로스 어텐션
Lflow = E[||vθ(xτ, τ, ct) - (x1 - x0)||2]

패턴 분리 & 완성

🌌

패턴 분리

ht의 UMAP 시각화에서 관찰:

  • 겹치는 클러스터가 기억 정보 후 별개의 매니폴드로 분리
  • 시각적 구별 단서가 사라진 후에도 매니폴드 구분 지속
  • 순차 과제에서 ht가 공유 매니폴드의 서로 다른 단계를 따라 진화
🔭

패턴 완성

HoloHead 예측 롤아웃에서 확인:

  • 경유점 예측이 모호성 하에서도 잠재 목표와 일관
  • 무작위 시점에서의 롤아웃이 모든 과제에서 목표 일관성 유지
  • ht가 패턴 완성을 위한 과제 관련 잠재 상태를 인코딩

✅ 핵심 기여

  • 기하학 기반 지각: EE 앵커 배측 경로로 구별 단서 보존

  • HoloHead 상상 목표: 예측적 정규화로 패턴 완성 강제
