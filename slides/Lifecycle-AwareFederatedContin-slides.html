연합연속학습(FCL) 이중 시간척도 프레임워크 자율 로버 함대

Lifecycle-Aware Federated Continual Learning in Mobile Autonomous Systems

장기 미션 생애주기 동안 진화하는 지형에 적응하며 파괴적 망각을 방지하는 이중 시간척도 연합연속학습 프레임워크를 제안합니다.

arXiv: 2604.20745v1

행성 탐사 미션의 변화: 단일 탐사선에서 다수의 자율 로버 협력 체계로 진화

연합학습의 필요성: 행성간 인터넷의 대역폭·지연 제한으로 인해 원시 데이터 대신 모델 파라미터만 교환

연속적 적응의 필수성: 로버가 이동하면서 새로운 지형(예: 느슨한 토양 → 노출된 암반)을 접하게 되므로 지속적 학습 요구

파괴적 망각 문제: 새로운 지식을 학습하면서 이전 지식이 퇴화되는 치명적 문제 발생

이중 시간척도에서의 성능 저하: 학습 시간 규모(기울기 충돌)와 장기 규모(누적 드리프트) 두 가지 차원에서 망각 발생

한계 1 — 균일 보호 전략: 딥네트워크 계층별 망각 민감도의 이질성을 무시하고 모든 계층에 동일한 보호 적용

• 적응이 필요한 계층은 과보호되고, 안정화이 필요한 계층은 과소보호됨

한계 2 — 복구 메커니즘의 비효율성: LoRA, 지식 증류, 변분 추론 등은 반복적 기울기 기반 최적화에 의존

• 수렴 속도가 느리고 연합학습의 곱셈적 드리프트에 취약함

한계 3 — 시뮬레이션만의 검증: 실제 로버 함대의 시스템 이질성(온보드 저장 용량, 통신 자원 제한)을 반영하지 못함

• 시뮬레이션만으로는 실제 미션 배포의 신뢰성을 보장할 수 없음

생애주기 인식(Lifecycle-Aware): 학습 시간(사전 망각) 방어와 장기(사후 망각) 복구를 명시적으로 조율하는 구조

사전 망각(Pre-forgetting) 방어: 로컬 학습 중 즉각적 망각을 완화하는 LSR 전략

사후 망각(Post-forgetting) 복구: 장기 누적 드리프트 후 성능 저하된 모델을 적극 복원하는 RKR 전략

계층 선택적 설계: 백본 계층(지속적 적응)과 분류기 계층(강력한 안정화)의 요구를 분리하여 처리

실세계 검증 포함: 시뮬레이션뿐 아니라 실제 로버 테스트베드에서 시스템 수준 강건성 평가

목표: 로컬 학습 단계에서 발생하는 즉각적 파괴적 망각 완화

계층별 민감도 분석: 딥네트워크의 각 계층이 망각에 대해 가지는 이질적 민감도를 특성화

차별화된 보호 전략: 분류기 계층에는 의사결정 경계 보존을 위한 강한 안정화 적용

백본 계층의 가소성 유지: 표현력이 진화해야 하는 백본 계층은 균일 보호로 인한 과보호 방지

균일 보호의 문제 해결: 기존 "원사이즈핏올" 접근법이 야기하던 과보호/과소보호 trade-off를 현저히 개선

목표: 장기 누적 드리프트로 인해 성능이 저하된 모델의 빠른 복원

메타학습 기반 접근: 반복적 기울기 기반 최적화에 의존하지 않는 메타학습 복구 메커니즘 설계

낮은 연산·통신 오버헤드: 기존 복구 방법(LoRA, 지식 증류 등) 대비 훨씬 적은 자원으로 복구 수행

곱셈적 드리프트에 대한 강건성: 연합학습에서 반복적 집계로 인해 발생하는 누적 오차에 취약하지 않음

샘플 복잡도 경계: 재학습 대비 RKR의 성능에 대한 샘플 복잡도 상한을 이론적으로 도출

이질적 망각 동역학 특성화: 딥네트워크 계층 간 망각 민감도 차이를 수학적으로 정식화

장기 성능 저하의 불가피성 증명: 연합학습 환경에서 Non-IID 모델의 반복적 집계가 장기적 성능 저하를 필연적으로 초래함을 이론적으로 확립

LSR 보호 설계의 정당화: 계층 선택적 보호 전략이 균일 보호보다 우수한 이유를 형식적으로 입증

RKR 성능 경계 도출: 재학습(Retraining) 기준선 대비 RKR의 복구 성능에 대한 이론적 샘플 복잡도 경계 제공

이질적 환경 고려: 분산 로버 간 데이터 이질성이 망각 동역학에 미치는 영향을 이론적 프레임워크에 통합

데이터셋: MarsScapes, S5Mars, AI4MARS 등 3개 대표적 화성 지형 데이터셋 활용

mIoU 향상 성과: 최강 연합학습 베이스라인 대비 최대 8.3% mIoU 개선 달성

파인튜닝 대비 향상: 기존 파인튜닝 방식 대비 최대 31.7% mIoU 개선 달성

평가 지표: 평균 교차 영역(mIoU)을 주요 지표로 활용하여 의미론적 지형 분할 성능 측정

극단적 Non-IID 조건: 분산 로버 간 데이터 이질성이 극심한 상황에서도 일관된 성능 우위 확인

실제 로버 배포: 시뮬레이션을 넘어 실물 로버 테스트베드에 FCL 프레임워크 직접 배포

현실적 제약 조건 반영: 극단적 Non-IID 데이터 분포 및 자원 이질성(온보드 저장 용량, 통신 제한) 포함

시스템 수준 강건성 평가: 알고리즘 성능뿐 아니라 실제 시스템 통합 관점에서의 신뢰성 검증

시뮬레이션-실제 갭 검증: 순수 시뮬레이션 기반 연구가 놓칠 수 있는 실제 배포 이슈를 발견하고 해결

미션 배포 가능성 확인: 제안 프레임워크가 실제 행성 탐사 미션 환경에서 신뢰할 수 있는 성능을 제공함을 실험적으로 입증

이중 시간척도 프레임워크: 학습 시(LSR)와 장기(RKR) 두 시간척도를 모두 아우르는 생애주기 인식 FCL 설계

계층 선택적 접근의 효과: 균일 보호를 지양하고 계층별 가소성·안정성 요구를 분리 처리하여 망각 완화 극대화

빠른 저비용 복구: 기존 복구 방법의 느린 수렴 문제를 해결하고 낮은 연산·통신 오버헤드로 모델 복원

이론적 기반 확보: 망각 동역학 특성화, 장기 저하 불가피성 증명, RKR 샘플 복잡도 경계 등 탄탄한 이론적 뒷받침

실세계 검증 완료: 최대 8.3% mIoU 향상 및 31.7% 파인튜닝 대비 개선을 실제 로버 테스트베드에서까지 확인
