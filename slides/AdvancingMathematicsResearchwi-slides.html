AI 기반 형식적 증명 탐색으로
수학 연구의 지평을 넓히다

George Tsoukalas, Anton Kovsharov, Sergey Shirobokov, Anja Surina 외 · Google DeepMind

대규모 언어 모델의 수학적 추론 능력을 형식적 증명 검증과 결합하여,
연구 수준의 미해결 문제들을 자율적으로 해결하는 최초의 대규모 평가

왜 형식적 증명이 필요한가

자연어 증명의 한계

  • 미묘한 논리적 오류("환각") 발생 가능
  • 전문가 검토에 막대한 비용 소요
  • 중간 단계 오류가 증명 전체로 전파
  • 위임 가능한 작업의 복잡도 제한

형식적 증명의 해결책

  • 컴파일러가 모든 논리적 단계를 자동 검증
  • 오류 즉시 발견 → 피드백 루프 가능
  • 전문가 개입 없이 완전 자율적 탐색
  • 복잡한 연구 수준 문제에 도전 가능

"LLM이 생성한 자연어 증명은 미묘한 논리적 오류를 포함할 수 있어, 수학 연구 통합의 주요 장벽이다."

완전한 형식적 증명

AlphaProof Nexus 프레임워크

LLM 기반 증명 생성과 Lean 컴파일러 검증을 결합한 에이전트 프레임워크. 사용자는 정리와 EVOLVE-BLOCK, EVOLVE-VALUE 마커로 수정 가능 영역을 지정한다.

4단계 에이전트 설계 스펙트럼

A 기본 에이전트

독립적 하위 에이전트가 Ralph 루프(Gemini 3.1 Pro 기반)로 증명 스케치 반복 개선. 공유 상태 없음.

B AlphaProof 장착

기본 에이전트 + AlphaProof 도구 호출. RL 기반 올림피아드급 증명 시스템을 하위 목표 해결에 활용.

C 진화적 에이전트

AlphaEvolve에서 영감. 공유 인구 데이터베이스, Elo 평가, P-UCB 샘플링 기반 진화 탐색.

D 풀피처 에이전트

AlphaProof + 진화 탐색 통합. 본 논문의 핵심 실험에 사용된 최고 성능 에이전트.

Paul Erdős가 제기한 1,200개 이상의 미해결 문제 중 353개에 도전. Bloom이 유지하는 온라인 카탈로그에서 선정된 연구 수준의 난제들이다.

주요 해결 사례

  • 56년간 미해결이던 2개 문제 해결 (Schoen 2001, Baier 2004, Elsholtz 2016에서 언급)
  • 모든 해결이 완전 자율적 — 인간 개입 없이 Lean 증명 생성에서 검증까지 완수
  • 경쟁 수학을 넘어 실제 연구 수준 문제에서의 첫 대규모 성공 사례

OEIS 평가의 의미

OEIS (정수 수열 온라인 백과사전)는 수천 개의 정수 수열과 관련된 미증명 추측을 포함하는 대규모 데이터베이스

에르되시 문제보다 더 넓은 범위의 수학적 주제를 포괄하는 다양한 난이도의 문제들

단순히 정답을 맞히는 것이 아니라, 형식적으로 검증 가능한 완전한 증명을 자율적으로 생성

실제 수학 연구에의 배포

풀피처 에이전트는 단순한 벤치마크를 넘어, 실제 수학 연구 현장에서 다음과 같은 성과를 달성했다.

대수기하학

Hilbert 함수에 대한 15년 된 미해결 문제 해결

최적화 이론

볼록 최적화에서 새로운 알고리즘 파라미터 스케줄 발견으로 기존 한계 개선

조합론

Ben Green의 유명한 100개 문제 목록 중 미해결 문제 해결에 기여

양자 광학

진행 중인 양자 광학 연구에 현재도 활발히 활용

그래프 이론

그래프 이론 연구에서 지속적인 보조 도구로 운용 중

오류 발견

문헌 내 여러 형식화 오류(misformalization) 식별

에이전트 아키텍처 비교 분석

풀피처 에이전트가 해결한 9개 에르되시 문제를 대상으로 4가지 에이전트 변형의 사후 분석 수행

에이전트 구성 해결 문제 수 비용 효율성
A 기본 Ralph 루프만 9/9 어려운 문제에서 비용 증가
B AlphaProof 기본 + AP 도구 — —
C 진화적 기본 + 진화 탐색 — —
D 풀피처 AP + 진화 통합 9/9 가장 효율적

놀랍게도, 기본 에이전트(A)도 모든 9개 문제를 해결했다. 다만 더 어려운 문제에서 비용이 더 높았다. 이는 LLM이 충분히 강력해지면, 복잡한 에이전트 설계보다 단순한 루프만으로도 강력한 성능이 가능함을 시사한다.

핵심 발견과 시사점

주요 발견

1
실증적 입증
AI 형식적 증명 탐색이 연구 수준 문제 해결에 실제로 유효함을 대규모로 최초 입증
2
단순성의 힘
복잡한 진화/RL 시스템 없이도 기본 agentic 루프가 동일한 문제를 해결 가능
3
비용 효율성
문제당 수백 달러 수준으로 연구 실무에 충분히 활용 가능한 비용

패러다임 전환

전문적 훈련 시스템 → 일반적 에이전트 루프

LLM이 강력해짐에 따라, 문제 특화 RL 시스템에서 범용 LLM + 간단한 루프로의 이동 추세

경쟁 수학 → 연구 수준

올림피아드 문제를 넘어 실제 미해결 연구 문제로 영역 확장 달성

검증 없는 추론 → 형식적 검증

환각 문제를 컴파일러 기반 자동 검증으로 근본적으로 해결하는 방향 확립
