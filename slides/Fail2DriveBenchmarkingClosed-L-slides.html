💥 실패 분석 ① 행동 시나리오

가장 큰 일반화 격차를 보인 카테고리. 학습 데이터에 거의 없는 행동을 요구합니다.

🚫 FullyBlocked — 완전 도로 차단

PlanT -69.5% HM

트럭, 밴, 건초더미로 도로 완전 차단. 60초간 정지 후 제거. 모든 모델이 안정적으로 정지하지 못함.

TF++ 사례: LiDAR로 차량을 정확히 인식했지만, 최고 속도로 주행을 예측 → 충돌 발생

🧱 Wall — 이미지 벽 차단

모든 모델 → HM 0.0

도로 사진이 인쇄된 거대한 벽이 도로를 차단. 모든 모델이 ~100 HM에서 0.0 HM으로 붕괴.

SimLingo 사례: 벽을 인식하지 못하고 벽에 인쇄된 도로 이미지에 맞춰 예측을 조정 → 차선 변경 시도 → 충돌

🚶‍♂️ PedestriansOnRoad — 도로 위 보행자

SimLingo: 87% 충돌률

소규모 보행자 그룹이 자아 차선 위를 걸음. 감속하여 안전 추종 필요.

SimLingo: HM 98.50 → 19.68. 존재하지 않는 자동차나 자전거를 환각(hallucination)하며 보행자를 제대로 인식 못함.

🔍 근본 원인: 지름길 학습과 과적합

모델들은 일반화 가능한 개념을 학습하는 대신 특정 패턴에 대한 하드코딩된 연관에 의존합니다.

LiDAR 정보 무시

TF++는 LiDAR에서 장애물을 명확히 감지하지만 바운딩 박스를 예측하지 않음. 센서 데이터가 있어도 학습된 패턴과 맞지 않으면 무시.

패턴 연관 vs 개념 이해

"공사 경고 표지판 = 차선 변경"이라는 연관만 학습. 주행 가능 공간에 대한 직접적 추론 능력 없음. 표지판 제거 시 즉시 붕괴.

CARLA 특유 패턴 과적합

PlanT 2.0: 정확한 객체 크기와 위치에 심하게 과적합. 우편함 하나에도 불필요한 회피 기동 수행 → 충돌 위험 초래.

불확실 상황 대비 행동 부재

완전히 차단된 도로에서 정지하는 폴백(fallback) 행동 없음. 학습 데이터에 "정지하고 기다리기" 상황이 부족.

"모델은 주행 가능 공간을 직접 추론하지 않고,
'공사 경고 표지판이면 차선 변경'과 같은
하드코딩된 패턴 연관에 의존한다"
