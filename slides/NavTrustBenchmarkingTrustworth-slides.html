The Problem

Why Trustworthiness Matters

Existing benchmarks evaluate navigation agents under idealized conditions, ignoring real-world corruptions that cause critical failures.

👁 VLN Agents

Vision-Language Navigation agents follow natural language instructions to navigate environments.

Minor linguistic perturbations can reduce success rate by 25%

🎯 OGN Agents

Object-Goal Navigation agents navigate toward a specified target object in the scene.

Degrade sharply under small domain shifts like low lighting & motion blur

💡 NavTrust is the first unified benchmark systematically evaluating trustworthiness
across both VLN and OGN under realistic corruptions

Mitigation

Four Robustness Strategies

🎨

Data Augmentation

Per-frame transient, per-episode persistent, and distributed variants with corruption-weighted sampling.
PRS-SR 0.94 (RGB)
PRS-SR 0.75 (Depth)
🎓

Teacher-Student Distillation

Composite loss: imitation learning + policy-KL divergence + feature-MSE. Teacher uses 0.6-intensity augmentation.
PRS-SR 0.93 (RGB)
PRS-SR 0.85 (Depth)
🔌

Parameter-Efficient Adapters

1-3% added parameters with residual bottleneck + panoramic fusion with reliability weights.
Depth PRS 0.62→0.89
Only 4% params trained
🤖

Safeguard LLM

Fine-tuned LLaMA 3.2 (8-bit) or OpenAI o3 canonicalizes free-form inputs. Runs once per episode.
LLaMA: +0.14~0.32 PRS
o3: +0.03~0.20 PRS

💡 Distillation most effective for depth; Adapters best parameter-efficient; Safeguard LLM uniquely addresses instruction corruptions

Insights

Key Takeaways

🏗

Architecture Matters

Modular designs are more robust than end-to-end pipelines

👀

Pre-trained Encoders Win

BLIP-2 based models (VLFM) show inherent visual robustness

🔗

Fusion Strategy is Critical

Late-fusion with confidence gating outperforms early-fusion

📏

Depth Remains Limiting

Even with augmentation, depth robustness lags behind RGB

💬

Language Sensitivity

Tokenizer rigidity inversely correlates with instruction robustness

🎯

Training > Scaling

Explicit robustness objectives are more effective than model scaling

NavTrust provides a standardized, reproducible protocol for evaluating and improving
embodied navigation trustworthiness across both VLN and OGN paradigms
