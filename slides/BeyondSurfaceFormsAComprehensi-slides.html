표면 형태를 넘어서: LLM 기반 암호어 탐지를 위한
간접적 언어 인코딩의 메커니즘 중심 분류학

Beyond Surface Forms: A Comprehensive, Mechanism-Oriented Taxonomy of
Indirect Linguistic Encoding for LLM-Based Coded Language Detection

간접적 언어 표현(ILE)의 인코딩 메커니즘을 체계적으로 분류하고, LLM 기반 탐지 프레임워크를 제안

배경과 동기

모더레이션 회피

소셜 미디어 사용자들은 콘텐츠 필터링과 감시를 피하기 위해 민감한 의미를 위장하는 간접적 언어 표현(ILE)을 지속적으로 창출합니다.

다양한 형태로 등장

알고스피크, 완곡어법, 적대적 난독화 등 의도와 맥락에 따라 다르게 나타나며, 재귀적 인코딩 메커니즘을 포함합니다.

기존 접근법은 표면 형태에만 집중하여, 메커니즘적 본질을 간과하는 한계가 있습니다.

핵심 문제 정의

간접적 언어 인코딩(ILE)이란?

민감한 의미를 숨기기 위해 의도적으로 사용되는 모든 간접적 언어 표현. 단순한 대체어를 넘어 음운적, 의미적, 구문적, 시각적 차원에서 다양하게 작동합니다.

기존 접근법의 한계

접근법 방식 주요 한계
정규식 / 키워드 매칭 사전 등록된 단어 탐지 새로운 변종에 무력
표면 형태 분류 알고스피크/완곡어법 구분 메커니즘 중복 · 누락

핵심 결여: "어떻게" 인코딩되는지에 대한 메커니즘적 이해가 부재

제안: 메커니즘 중심 분류학

표면 형태(알고스피크 등)가 아닌, 언어가 어떤 메커니즘으로 인코딩되는지를 기준으로 체계적 분류를 제안합니다.

핵심 원칙

  • 형태보다 기능 우선
  • 메커니즘 간 독립성 보장
  • 중첩 및 조합 가능성 명시
  • LLM 탐지에 직접 활용 가능

기대 효과

  • 미지의 변종도 메커니즘으로 식별
  • 탐지 프롬프트 구조화 가능
  • 분류 범주 간 중복 감소
  • 연구 간 비교 가능성 향상

인코딩 메커니즘 분류 체계

음운·철자적

발음 유사성, 철자 변형, 생략, 대체 등 음성/문자 수준의 조작

예: "un-alive", "seggs", leetspeak

의미적

은유, 환유, 대용, 유사성 기반 치환 등 의미 차원의 우회

예: "우유" → 백색 약물, "조개" → 특정 집단

구문·담화적

문장 구조 변환, 생략, 문맥 의존적 표현, 화용적 전략

예: "그것 할 때 주의해" (맥락 의존)

시각·서식적

이모지 치환, 공백/기호 삽입, 유니코드 조작, 이미지 텍스트 등 시각적 인코딩

혼합·중첩적

복수 메커니즘의 동시 사용으로 탐지 난이도 극대화 (예: 은유 + 이모지 + 철자 변형)

분류학의 다층적 구조

1

의도 층 (Intent Layer)

왜 인코딩하는가? — 모더레이션 회피, 집단 정체성, 농담, 적대적 목적

2

메커니즘 층 (Mechanism Layer)

어떻게 인코딩하는가? — 음운·철자 / 의미 / 구문·담화 / 시각·서식 / 혼합

3

표현 층 (Surface Layer)

결과적으로 무엇이 나타나는가? — 알고스피크, 완곡어법, 적대적 난독화

4

맥락 층 (Context Layer)

어떤 환경에서 사용되는가? — 플랫폼, 커뮤니티, 시간적 변화, 문화적 배경

핵심 설계 철학

메커니즘 분류학을 프롬프트 구조화의 뼈대로 활용하여, LLM이 단순히 "이상한 단어"를 찾는 것을 넘어 "어떤 방식으로 위장되었는지"를 먼저 파악한 후 의미를 역추론합니다.

실험 설정

데이터셋 구성

  • 다양한 플랫폼 수집 (Reddit, X, TikTok 등)
  • 메커니즘별 레이블링 수행
  • 단일 메커니즘 + 혼합 메커니즘 포함
  • 다국어 표본 포함

비교 대상

  • 키워드/정규식 기반 탐지기
  • 전통적 ML 분류기 (SVM, BERT)
  • 제로샷 LLM (GPT-4, Claude 등)
  • 본 연구의 메커니즘 기반 프레임워크

평가 지표

Precision, Recall, F1 외에 메커니즘 식별 정확도, 혼합 인코딩 탐지율, 미지 변종에 대한 일반화 성능을 추가로 측정합니다.

특히 혼합 인코딩과 미지 변종에서 기존 방법 대비 유의미한 성능 격차를 확인

메커니즘별 분석 인사이트

상대적으로 탐지 용이

  • 음운·철자적: 패턴 규칙성 높음
  • 시각·서식적: 형태적 이상치 탐지 가능

정규화 후 의미 파악 비교적 수월

탐지 난이도 높음

  • 의미적: 문화적 배경 지식 필수
  • 구문·담화적: 문맥 없이는 정상 문장
  • 혼합·중첩적: 다계층 분해 필요

주요 발견

메커니즘을 명시적으로 식별하도록 LLM을 유도할 때, 의미 복원 정확도가 평균 15~25% 향상되었습니다. 특히 혼합 인코딩에서는 메커니즘 분해 단계가 필수적임이 확인되었습니다.

논의: 한계와 윤리적 고려

기술적 한계

  • 문화 특정적 ILE에 대한 커버리지
  • 멀티모달 요소(이미지, 오디오)의 완전한 처리
  • LLM의 환각 가능성에 의한 과탐지 위험
  • 실시간 처리를 위한 지연 시간 문제

윤리적 고려

  • 정당한 표현의 과도한 검열 위험
  • 특정 집단에 대한 편향적 탐지 가능성
  • 탐지 기술의 악용 가능성 (우회 기법 학습)
  • 투명성과 책임성 확보 필요

탐지 기술은 안전한 온라인 환경을 위한 도구일 뿐, 표현의 자체를 통제하는 목적이 되어서는 안 됩니다.

결론 및 향후 연구 방향

핵심 기여

  • 표면 형태를 넘어 메커니즘 중심의 포괄적 ILE 분류학 최초 제안
  • 의도-메커니즘-표현-맥락의 다층적 분석 프레임워크 구축
  • 분류학을 LLM 프롬프트에 통합하여 혼합 인코딩 탐지 성능 현저히 향상
  • 미지 변종에 대한 일반화 능력 입증

향후 연구 방향

  • 멀티모달 ILE (이미지+텍스트) 탐지
  • 다국어 분류학 확장
  • ILE 진화의 시간적 추적 연구
  • 과탐지 최소화를 위한 정밀 제어
