핵심 알고리즘

SGT-MCTS: 계통 재구성

Self-Guided Temporal Monte Carlo Tree Search — 방법론 진화 체인을 충실하게 복원

왜 MCTS가 필요한가?

방법론적 진보는 유향 비순환 그래프(DAG)를 형성합니다.

Transformer → BERT, GPT, ViT, DETR … 각 분기점에서 탐욕적 선택은 대안 경로를 폐기합니다.

하위 응용 1

그래프 기반 아이디어 평가

기존 평가 프레임워크를 기술적 인프라에서 결정론적으로 연산 가능하게 합니다.

차원별 신호

  • 중복 위험 탐지
  • 새로움(Novelty) 점수
  • 실행 가능성(Feasibility)
  • 구조적 갭 근거

차원 간 정규화

Ω_cross 정규화기를 통해 차원 간 상호작용을 모델링합니다.

순수 LLM 평가의 체계적 편향(안전한 개념 선호)을 그래프 기반 근거로 보정

우아한 폴백(fallback) 및 선택적 심판자(adjudicator) 메커니즘 제공

하위 응용 2

전략 기반 아이디어 생성

그래프에서 파생된 진화 맥락을 활용하여 구조적 갭에서 연구 아이디어를 생성합니다.

구조적 갭 패턴 식별

방법론 그래프에서 미탐색 방향을 체계적으로 탐지:

  • 해결되지 않은 병목
  • 분기되지 않은 진화 경로
  • 도메인 간 전이 기회

전략 조건부 제안

식별된 갭에 대해 전략을 조건으로 아이디어를 생성:

  • 특정 병목 해결 전략
  • 기존 메커니즘 적응 전략
  • 교차 도메인 융합 전략
