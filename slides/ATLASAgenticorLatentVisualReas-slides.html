Ziyu Guo, Rain Liu, Xinyan Chen, Pheng-Ann Heng

Meta AI · The Chinese University of Hong Kong

arxiv.org/abs/2605.15198v1

비전 언어 모델과 시각적 추론

VLM의 발전으로 multimodal intelligence가 인식(perception)에서 추론(reasoning)으로 진화하고 있음. 순수 텍스트 기반 추론만으로는 부족하며, 문제 해결에 중간 시각 상태(intermediate visual states)가 필요한 경우가 많음.

게임 풀이

각 조작 후 보드 상태를 업데이트해야 함

기하학 풀이

숨겨진 관계를 드러내기 위해 보조선 구성 필요

현재 VLM은 이러한 동적 시각적 추론 과정에서 여전히 한계를 보임

기존 패러다임의 한계

I. 통합 모델

픽셀 수준 이미지 생성

  • 추론 비용 극대화
  • 이미지 디코딩/인코딩 부담
  • 프레임워크 수준 아키텍처 변경 필요
II. 에이전트 방식

코드/도구 호출 기반

  • 컨텍스트 전환 지연 발생
  • 간단한 연산도 긴 코드 필요
  • 외부 실행 환경 의존
III. 잠재 방식

숨겨진 임베딩 기반

  • 특정 태스크로 일반화 제한
  • 순환 잠재 의존성 문제
  • 병렬 훈련 호환성 저하

ATLAS: 핵심 아이디어

단일离散 "단어"가 에이전트 연산이자 잠재 시각적 추론 단위로 동시에 기능

기능적 토큰 (Functional Token)

각 시각적 연산을 토크나이저 어휘의 표준离散 토큰으로 표현. 자연어 토큰과 동일한 시퀀스 내에서 일반적인 next-token prediction으로 생성됨.

에이전트 방식 vs ATLAS

복잡한 코드 생성·도구 호출·외부 실행을 단일 토큰으로 내부화

잠재 방식 vs ATLAS

표준 자기회귀 루프 유지, 시각적 감독 불필요, 병렬 훈련 호환

기능적 토큰의 분류체계

전체 어휘 𝒱 = 𝒱text ∪ 𝒱spec ∪ 𝒱func — 기존 어휘에 5개 기능적 토큰만 추가

⟨|Manip|⟩

이미지 조작

영역 확대, 자르기 등

⟨|Shape|⟩

도형 삽입

원, 사각형 등 시각적 요소

⟨|Line|⟩

보조선 구성

기하 관계 시각화

⟨|Arrow|⟩

방향성, 인과관계 표시

⟨|Text|⟩

텍스트 라벨

시각적 요소에 주석 추가

이미지 수준 감독 불필요 — 표준 CE 목적만으로 최적화

통합 시퀀스 모델링

기능적 토큰은 자연어 토큰과 동일한 자기회귀 시퀀스 내에서 생성됨

1
이미지 인코딩
입력 이미지 → 시각적 토큰
2
텍스트+기능 토큰
자연어와 기능적 토큰이 혼합된 시퀀스 생성
3
내부화된 연산
기능적 토큰이 시각적 연산을 내부적으로 수행
4
최종 답변
추론 완료 후 텍스트로 응답

핵심 장점

외부 도구, 픽셀 생성, 순환 잠재 의존성 모두 불필요

기반 모델

Qwen2.5-VL 기반 — 기존 VLM 파이프라인과 완전 호환

2단계 훈련 레시피

1

SFT 단계

ATLAS-178K 데이터셋으로 지도 미세조정

  • 40개 이상 시각 추론 태스크 포괄
  • 기능적 토큰 궤적(trajectory) 주석
  • 언제·어떻게 기능적 토큰을 호출할지 학습
  • 표준 cross-entropy 목적 함수 사용
2

RL 단계

표준 GRPO로 결과 기반 최적화

  • 아키텍처 수정 없이 직접 적용 가능
  • 답변 정확도 보상
  • 유효 기능적 토큰 사용 보상
  • 일관된 추론 행동 보상

LA-GRPO: 기울기 희석 문제 해결

RL 훈련 중 기능적 토큰이 지나치게 희소하여 일반 텍스트 토큰에 압도되는 "기울기 희석(gradient dilution)" 문제 발견

Latent-Anchored GRPO

정적 가중 토큰 수준 보조 손실

기능적 토큰 어휘에 고정된(auxiliary) 목적을 추가

지속적 학습 신호 제공

희소한 기능적 토큰에 강한 기울기 업데이트 보장

일관된 성능 향상

다양한 추론 태스크에서 안정적 개선 달성

표준 GRPO + 정적 가중 보조 손실 = 맞춤형 훈련 수정 불필요, 기존 RL 프레임워크와 완전 호환

핵심 기여 요약

1

새로운 패러다임 제안

시각적 연산을 이산적 기능적 토큰으로 표현하여, 중간 시각 상태 생성 오버헤드를 제거하면서도 확장 가능한 자기회귀 훈련과 호환

2

기울기 희석 문제 식별 및 해결

희소한 기능적 토큰의 최적화 부족을 LA-GRPO로 해결 — 토큰 고정 보조 손실로 강한 기울기 신호 제공

3

압축 단일 토큰 시각 추론 실현

복잡한 시각 연산을 단일 토큰으로 수행하여, 도전적 벤치마크에서 강력한 성능을 substantially reduced overhead로 달성
