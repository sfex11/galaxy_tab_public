문제 제기

사전 학습 지식과 환각의 모순

사전 학습 단계

대규모 말뭉치에서 다음 토큰 예측을 통해
대부분의 사실적 지식을 파라미터에 저장

사후 학습 단계

SFT, RLHF 등에서 파라미터 지식 외부의
새로운 사실이 도입되며 환각 유발

"새로운 지식에 대한 SFT가 환각을 악화시킨다는 것은 입증되었으나, 근본적인 메커니즘은 여전히 불명확하다."

연구 배경

기존 연구의 한계

  • Gekhman et al. (2024): 미지 사실 비율 증가 → 환전 증가를 행동 관찰로만 확인
  • Kang et al. (2024): 모델이 익숙하지 않은 예시의 응답 분포를 모방함을 발견
  • Wang et al. (2025): SAE 잠재 변수 활성화 차이 기반 정렬 해제 특징 식별
  • Minder et al. (2025): 협소한 미세조정은 아티팩트를 남길 수 있음을 경고

공통적 한계: 두 모델 상태 간의 정적 비교에 의존하며, 미세조정 조건의 연속적 변화에 따른 내부 표현의 변화를 추적하지 않음

미지 사실 비율과 에포크 수를 독립적으로 제어하여 인식론적(epistemic) 차원에서의 변화를 추적 — 주제적 협소성 아티팩트 회피

핵심 관찰

미지 사실 노출이 환각을 증가시킨다

↑ Unknown 비율

학습 데이터에 미지 사실이 많을수록 테스트 정확도가 지속적으로 하락. 세 모델 모두 일관된 패턴 확인.

↑ Epoch 수

동일한 Unknown 비율에서도 에포크가 증가할수록 환각 효과가 더욱 강화됨.

이 행동적 변화는 내부 활성화 공간에서 어떻게 나타나는가? → SAE 분석으로 접근

분석 도구

희소 오토인코더(SAE)와
잔차 흐름 분석

SAE의 역할

잔차 흐름(Residual Stream)의 고차원 활성화를 희소한 해석 가능한 특징으로 분해. 사전 학습된 SAE를 그대로 사용하여 미세조정 간 아티팩트 방지.

분석 대상

각 모델의 여러 체크포인트에서 잔차 흐름 활성화 측정. N(Unknown 비율 단계) × F(에포크 단계) 매트릭스 구성.

활성화 매트릭스:
āi,je = SAE 특징 j의 평균 활성화
(i: Unknown 비율 단계, e: 에포크)

방법론

MoRFI 알고리즘

Monotonic Relationship Feature Identification for Sparse Autoencoders

1
활성화 추출
모든 체크포인트에서 SAE 특징 평균 활성화 측정
2
단조성 검정
Mann-Kendall 등으로 각 특징의 단조적 추세 판별
3
부트스트랩
재표본추출로 일시적/허위 변동 제거
4
특징 선별
증가/감소 방향별 강건한 단조적 특징 집합 확보

핵심 차별점: 두 고정 상태의 차이가 아닌, 제어된 기울기 전체에서의 부트스트랩 단조 추세로 특징 선별 — 일반적 알고리즘으로 어떤 제어 변수에도 적용 가능

방법론 상세

MoRFI의 두 분석 차원

↑ Unknown 차원

에포크 고정, Unknown 비율을 0%→100%로 변화시키며 각 특징의 단조성 관찰

  • 미지 사실에 노출될수록 지속적으로 증가/감소하는 특징 식별
  • 지식 접근 경로의 변화를 직접적으로 포착

↑ Epoch 차원

Unknown 비율 고정, 에포크 수를 증가시키며 단조성 관찰

  • 학습이 반복될수록 강화되는 변화 방향 식별
  • 환전 악화의 누적 효과 측정

두 차원에서 식별된 특징 간에 유의미한 교집합이 존재 → 공통의 지식 관련 메커니즘 시사

심층 분석

합성 방향 vs 단일 잠재 변수

합성 방향 (Difference-in-Means)

Known vs Unknown 체크포인트 간 SAE 평균 활성화 차이 벡터. 표현 변화의 일관성과 보편성을 확인하지만, 개입 시 과도한 부작용 우려.

단일 잠재 변수 개입

MoRFI로 선별된 개별 특징만 조작. 합성 방향 대비 성능 격차가 큼 → 지식 관련 신호가 희소하게 분포함을 시사.

중요 발견: 합성 방향과 단일 잠재 변수 사이의 큰 성능 격차는 지식 접근 경로가 소수의 특징에 집중되어 있음을 나타냄

핵심 발견

개입의 비대칭성과 잠재 구조

억제(Suppression) > 증폭

MoRFI로 식별된 감소(↓) 잠재 변수를 억제하는 것이 증가(↑) 잠재 변수를 증폭하는 것보다 압도적으로 효과적.

→ 미세조정이 지식 접근을 "방해하는" 방향으로 특징을 변화시킴

Known vs Unknown 잠재 구조

미지 사실에 대한 학습이 모델의 저장된 지식 검색 능력을 잔차 흐름의 특정 방향들에서 파괴함.

→ 환전은 새로운 정보의 주입이 아닌 기존 접근 경로의 차단

두 차원(↑Unknown, ↑Epochs)에서 식별된 잠재 변수 간 유의미한 교집합 존재 → 공통의 내부 메커니즘

아티팩트 통제

협소한 미세조정 아티팩트 배제

통제 잠재 변수 검증

관련 없는 속성(예: 응답 길이, 특정 주제)에서 단조적 추세를 보이는 잠재 변수로 개입 시 지식 복원 효과 없음을 확인.

다양성 보장 설계

미세조정 데이터가 다양한 관계와 개체를 포함하도록 설계. Minder et al.이 지적한 협소성 아티팩트에 덜 취약한 구조.

  • 무작위 잠재 변수 개입 → 지식 복원 없음 (기준선 확보)
  • 통제 그룹 잠재 변수 개입 → 무효과로 아티팩트 가능성 배제
  • 사전 학습 SAE 사용 → 미세조정 간 SAE 적합화 아티팩트 원천 차단
결론

요약 및 기여

방법론 기여

  • MoRFI 알고리즘 제안 — 부트스트랩 + 통계 검정 기반 단조적 특징 식별
  • 모델 간 비교가 아닌 제어된 기울기에서의 추세로 분석하는 새로운 패러다임
  • SAE 분석에 적용 가능한 범용 특징 선별 절차

과학적 발견

  • 미지 사실 노출이 잔차 흐름의 지식 접근 경로를 파괴함을 인과적으로 입증
  • 환각은 새 정보의 주입이 아닌 기존 경로의 차단
  • 억제 개입이 증폭보다 우월한 비대칭성 발견

MoRFI 파이프라인은 서로 다른 모델 아키텍처에서 신뢰할 수 있게 지식 관련 잠재 변수를 발견하며, 단일 잠재 변수 개입만으로도 실질적인 지식 복원을 달성한다.
