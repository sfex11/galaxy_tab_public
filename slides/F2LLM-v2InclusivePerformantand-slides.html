핵심 기법

3가지 최적화 기법

1

마트료시카 표현 학습

양 학습 단계 모두 적용 · 최소 차원 8부터 다양한 차원에서 유용한 임베딩 생성 가능. 자원 제약 환경에서 차원을 자유롭게 축소하면서도 성능을 유지할 수 있는 유연한 임베딩 구조.

2

모델 프루닝

1단계 학습 후 0.6B 모델에서 80M, 160M, 330M 모델을 생성. 히든 크기 · MLP 중간 크기는 활성화 노름 기반으로 행/열을 프루닝하고, 레이어 차원은 처음 n개 레이어만 유지하는 방식.

3

지식 증류

프루닝 후 성능 저하를 MSE 기반 임베딩 증류로 보완. 교사 모델의 시퀀스 임베딩과 학생 모델의 시퀀스 임베딩 간 MSE 손실을 추가로 계산.
교사: 0.6B → 소형 모델 / 1.7B → 4B 모델

결론

핵심 기여 요약

  • 80M부터 14B까지 8가지 크기로 다양한 배포 환경 지원
  • 282개 자연어와 40+ 프로그래밍 언어를 포괄하는 진정한 다국어 임베딩
  • 중·저자원 언어에 대한 데이터 분포 균형으로 포용성 극대화
  • 마트료시카 학습 + 프루닝 + 지식 증류의 3중 최적화 기법
  • MTEB 11개 벤치마크 1위로 최고 수준의 성능 입증
  • 모델, 데이터, 코드, 체크포인트 전면 공개로 연구 투명성 확보
