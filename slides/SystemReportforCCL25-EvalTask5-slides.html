연구 배경

고전 한시 감상 연구의 현황과 한계

기존 성과

  • 고전 중국어 번역에서 유의미한 진전
  • 한시 생성 모델의 발전 (Jiuge, Deep Poetry 등)
  • LLM 기반 접근법의 확장

주요 한계

  • 정밀 번역 및 감성-의미 이해 연구 부족
  • 시 감상을 일반 NLP 문제로 취급
  • 도메인 특화 고품질 데이터셋 부재

고전 한시 감상은 왕조, 시인 생애, 역사적 사건에 대한 지식이 필요하며, 이를 일반 도메인 문제로 접근하는 것에는 근본적 한계가 있다.

문제 정의

왜 도메인 특화 접근이 필요한가?

1

과제 분해 부재

시 감상을 단일 과제로 처리하여 세밀한 평가 불가

2

데이터 부족

용어 해석·감성 추론을 포괄하는 전용 데이터셋 없음

3

모델 미특화

범용 모델로 시의 언어적·역사적 복잡성 해결 한계

도메인 특화 데이터셋 구축 + 전용 모델 개발이 필수적

고전 한시 감상의 3가지 하위 과제

핵심 발견

일관된 성능 우위

PoetryQwen은 3개 하위 과제 모두에서 베이스라인을 상회하며, 도메인 특화 명령어 튜닝의 유효성을 입증

과제별 어댑터의 효과

단일 모델 대비 각 하위 과제에 맞춘 독립 LoRA 어댑터가 전문성을 극대화

데이터셋의 역할

49K 고품질 명령어 쌍이 정밀 번역과 감성 이해 양측면에서 동시에 향상 기여

효율적 파인튜닝

전체 파라미터 조정 없이 LoRA만으로 유의미한 성능 향상 달성 — 자원 효율적 접근
