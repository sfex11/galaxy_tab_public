01 · 문제 정의

LLM의 공간 추론 한계

핵심 문제
LLM/VLM은 뛰어난 추론 능력에도 불구하고, 세밀한 공간 이해와 레이아웃 일관성 유지에서 어려움을 겪는다.
✗
기존 접근법의 한계
  • Zero-shot LLM은 충돌 및 제약조건 위반 발생
  • 비구조적 CoT는 다단계 편집에서 오류 누적
  • 235B 파라미터 모델도 정밀한 3D 배치 실패
✓
제안 솔루션
  • 장면 그래프 기반 구조적 추론
  • 단계별 그래프 편집 체인
  • 기하학적 보상을 활용한 강화학습
목표: 자연어 지시를 받아 3D 장면 그래프를 단계별로 편집하여
공간적 일관성을 유지하는 프레임워크 구축
02 · 핵심 기여

4가지 주요 기여

🏗️ 기여 1
3D-Layout-R1 프레임워크
언어 지시 기반 다단계 3D 장면 편집을 위한 해석 가능한 그래프 편집 체인 추론 접근법
📊 기여 2
새로운 벤치마크 데이터셋
정렬·배열·방 편집을 포함하는 15,000개 3D 장면 + 자연어 지시 + 추론 과정 포함
🎯 기여 3
기하학적 보상 함수
3D IoU 기반 보상 함수로 RL 정책 최적화에 연속적이고 기하학적으로 충실한 감독 신호 제공
🏆 기여 4
최첨단 성능 달성
7-8B 모델로 235B 규모 모델 대비 최대 20% IoU 향상, 25% 중심거리 오차 감소

📤 출력
편집된 장면 그래프
Stage 1 CoT-SFT
DeepSeek-R1이 생성한 구조적 사고 경로로 지도 미세조정
  • 5 에폭, 16x A100 GPU
  • 학습률: 2×10⁻⁷, AdamW
  • 최대 시퀀스: 10,240 토큰
  • bfloat16 + ZeRO-3 샤딩
Stage 2 GRPO 강화학습
CoT-SFT 체크포인트에서 초기화 후 보상 기반 정책 최적화
  • 15 에폭, 8 GPU
  • 학습률: 1×10⁻⁶
  • 응답 최대: 16,384 토큰
  • KL 계수: λ = 0.01
기본 모델: Qwen2.5-VL-7B-Instruct (VLM) / Qwen3-8B-Instruct (LLM) — 비전 백본(ViT)은 동결, 언어 부분만 학습
05 · 보상 함수

3요소 보상 설계

r = IoU(Gpred, G*) + λ₁ · Coll(Gpred) + λ₂ · Fmt(Gpred)
📐
IoU 보상
매칭된 객체 쌍의 평균 3D IoU 계산
riou = (1/N) Σ IoU(bi, bj)
🛡️
충돌 방지 보상
비컨테이너 객체 간 관통 페널티
Coll(G) = 1 − |C|/N
📋
포맷 보상
<think> 태그 + 유효한 JSON 출력 확인
Fmt = 1 (유효) | 0 (무효)
GRPO 목적함수: 그룹 내 정규화된 보상 편차를 이점(advantage)으로 사용하여, PPO 스타일 클리핑과 KL 정규화로 안정적 정책 업데이트

모든 대규모 Zero-shot 모델 대비 제안 방법(7-8B)이 우수한 성능을 보임. 구조적 CoT + GRPO가 각 개별 구성 요소보다 일관되게 높은 성능 달성.
정성적 비교: 제안 방법은 정답(Ground Truth)과 거의 일치하는 반면, 베이스라인(Qwen3-235B, DeepSeek)은 충돌 발생 및 제약 조건 위반
10 · 결론

요약 및 시사점

3D-Layout-R1
  • 구조적 추론: JSON 장면 그래프를 통한 해석 가능한 단계별 편집 체인
  • 2단계 학습: CoT-SFT → GRPO RL로 점진적 성능 향상
  • 기하학적 보상: 3D IoU + 충돌 방지 + 포맷 검증의 복합 보상
  • 효율성: 7-8B 모델이 235B 모델을 최대 20% IoU로 능가
  • 일반화: 창고 시뮬레이션, 실세계 로봇 배치로 전이 가능
💡 핵심 통찰
구조화된 추론 + 도메인 특화 보상이 모델 크기보다 중요하다
🔮 향후 방향
더 복잡한 장면, 동적 환경, 다양한 로봇 플랫폼으로 확장 가능
