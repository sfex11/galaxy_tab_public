GaP: Graph-as-Policy
다중 에이전트 자기학습을 통한
변이형 자동화 과제 해결

해석 가능한 로봇 프로그래밍과 모델 자유 정책의 개방 세계 적응성을 결합한 에이전트 코딩 하니스

UC Berkeley · NVIDIA · Carnegie Mellon University · Bosch

변이형 자동화(VA)란?

기존 자동화 분류와 구별되는 새로운 과제 클래스

고정 자동화 (FA)

동일한 형상·자세의 객체를 반복 처리

  • 스팟 용접, 분무 도장
  • 환경 변화 최소

변이형 자동화 (VA)

객체 기하·자세에 비자명한 변이가 존재하는 반복 과제

  • 상품 분류, 카페 커피 제조
  • 신뢰성 + 적응성 모두 필요

일반 로봇학 (GR)

다양한 과제를 다양한 환경에서 수행

  • 가사 로봇, 다목적 조작
  • 범용성 중심, 신뢰성 부족

왜 기존 접근법으로는 부족한가?

모델 자유 정책과 단일 에이전트 코딩의 한계

모델 자유 VLA 정책

  • 상업적 신뢰성 수준 달성 어려움
  • 장시간 지속 실행 시 누적 오류
  • 실패 원인 해석 불가

단일 에이전트 Code-as-Policy

  • 자유 형태 코드 → 구조화 부족
  • 컨텍스트 윈도우 과대 확장
  • 환각(hallucination) 및 "치팅" 문제

핵심 질문: 에이전트 코딩 시스템의 최신 발전으로 해석 가능한 로봇 프로그래밍과 모델 자유 정책의 개방 세계 적응성을 결합할 수 있는가?

Graph-as-Policy: 계산 그래프 구조

TAMP와 ROS에서 영감을 받은 방향성 계산 그래프

그래프의 장점

  • 모듈성: 각 노드가 독립적 기능 수행
  • 재사용성: 노드가 여러 그래프에서 공유
  • 조립성: 노드 조합으로 새로운 과제 해결
  • 해석 가능성: 그래프 구조로 실행 흐름 추적
  • 컨텍스트 제한: 각 에이전트의 윈도우 크기 축소

노드 타입 예시

지각 노드
카메라 프레임 획득, 객체 탐지·분할

추론 노드
VLM 기반 의미 추론, 파라미터 결정

계획 노드
운동 계획, IK 해결, 경로 생성

제어 노드
그리퍼 제어, 트레이젝토리 실행

MORSL: 모듈형 개방 로봇 스킬 라이브러리

모델 기반 절차(ROS)와 모델 자유 정책(GraspGen 등)을 통합하는 51개 초기 스킬

지각 스킬

  • 객체 탐지 및 분할
  • 시각-언어 추론
  • 그리핑 포인트 생성

조작 스킬

  • 파지(Grasping)
  • 이송(Transport)
  • 양손 조작(Bimanual)

기획·제어 스킬

  • 운동 계획 및 IK
  • 기하 연산
  • 로봇·환경 제어

오케스트레이션 에이전트

  • VA 과제를 기능 세그먼트로 분할
  • 각 세그먼트를 적절한 스킬 에이전트에 할당
  • 서브그래프를 집성하여 전체 그래프 와이어링
  • 자기학습 루프 조율

스킬 에이전트들

  • 할당된 세그먼트에 대한 국소적 서브그래프 합성
  • MORSL의 원자 스킬 노드를 선택·구성
  • 작은 컨텍스트 윈도우로 집중 최적화
  • 병렬 실행으로 전체 생성 시간 단축

치팅 방지: 그래프 요소의 생성과 테스트를 분리하여 개별 에이전트가 인위적으로 성공 지표를 조작할 유인 감소

핵심 기여 요약

본 연구의 6가지 주요 기여

① VA 과제 클래스 정의 — 고정 자동화와 일반 로봇학 사이의 새로운 문제 영역 식별

② 계산 그래프 정책 표현 — 에이전트 로봇공학을 위한 구조화된 그래프 형식 제안

③ GaP 다중 에이전트 하니스 — 자연어 명세에서 로봇 계산 그래프를 자동 생성하는 구현체

⑤ 자기학습 프레임워크 — Isaac-Lab 기반 그래프 리허설 및 반복 최적화 방법
