SKILL0

인컨텍스트 에이전틱 강화학습을 통한
스킬 내재화(Skill Internalization)

Zhengxi Lu, Zhiyuan Yao, Jinyang Wu, Chengcheng Han, Qi Gu,
Xunliang Cai, Weiming Lu, Jun Xiao, Yueting Zhuang, Yongliang Shen
강화학습 LLM 에이전트 스킬 내재화 제로샷
문제 정의

기존 스킬 활용 방식의 한계

현재 LLM 에이전트는 추론 시점에 외부 스킬을 검색·주입하여 사용합니다.
이 방식에는 세 가지 근본적 한계가 존재합니다.

🔍 검색 노이즈

스킬 검색 시 관련 없는 가이드가 함께 주입되어 에이전트 성능을 저하시킵니다.

📦 토큰 오버헤드

스킬 컨텍스트 주입으로 인해 스텝당 최대 2.21k 토큰이 소비됩니다.

🧠 지식 미습득

모델이 실제 지식을 학습하지 못하고, 단순히 주어진 지시를 따를 뿐입니다.

핵심 질문: 스킬을 추론 시점에 주입하는 대신, 모델 파라미터에 내재화할 수 있다면?

핵심 아이디어

SKILL0의 접근법

훈련 중에 스킬을 제공하고, 점진적으로 제거하여
모델이 스스로 스킬을 내재화하도록 합니다.

기존 방식

추론 시 스킬 증강

  • 매 스텝마다 스킬 검색 필요
  • 높은 토큰 비용 (~2.21k/스텝)
  • 검색 실패 시 성능 저하
  • 모델이 지식을 습득하지 못함
SKILL0

스킬 내재화

  • 훈련 시에만 스킬 사용
  • 최소 토큰 비용 (~0.38k/스텝)
  • 추론 시 스킬 검색 불필요
  • 제로샷 자율 행동 달성
훈련: 스킬 컨텍스트 ✓ → 점진적 제거 → 스킬 제거 ✗
추론: 스킬 없이 제로샷 수행
프레임워크

3단계 학습 프레임워크

01
관련성 기반 스킬 그룹화
스킬을 일반 스킬(탐색, 목표 추적)과 작업별 스킬로 계층적 분류. 검증 세트를 N개 하위 작업으로 분할.
02
인컨텍스트 강화학습 (ICRL)
스킬을 시각적 컨텍스트로 렌더링하여 훈련 롤아웃에 제공. 정책이 컨텍스트에서 파라미터로 지식을 점진적으로 전이.
03
유용성 기반 동적 커리큘럼
선형 감소 예산으로 스킬 수를 단계적으로 줄임. 각 스킬의 유용성을 평가하여 상위 스킬만 유지.
|𝒮(s)| ≤ M(s) = ⌈N·(Ns-s) / (Ns-1)⌉
선형 감소 예산: 매 단계마다 스킬 수가 균일하게 감소

유용성은 초기에 낮고 → 정책이 컨텍스트 활용을 학습하며 상승 → 커리큘럼이 내재화를 강제하며 하락합니다. 이는 스킬이 "효과적이지만 일시적인 발판"임을 실증합니다.

학습 목표

보상 함수 및 손실 함수

복합 보상 함수

r̃t = rt + λ · rcompt

rt — 작업 완수 정확도 보상
rcompt — 성공 시 ln(ct), 실패 시 0
λ — 작업-압축 트레이드오프 제어

SKILL0 손실 함수

ℒ = 𝔼[clip(r(θ), A, ε)]
   - β · DKL[πθ ∥ πref]

PPO 클리핑 기반 정책 최적화
KL 정규화로 참조 정책 이탈 방지

이론적 안정성 보장

DKL[π(s) ∥ π(s+1)] ≤ Lπ · LE · ⌈N/(Ns-1)⌉

선형 스케줄이 구간별 정상성을 보장하여 PPO 중요도 비율을 안정화합니다.

핵심 통찰

왜 SKILL0가 작동하는가?

1. 스킬은 "일시적 발판" 역할

초기에 스킬은 학습을 가속화하고, 정책이 컨텍스트 활용을 배운 후에는 점진적으로 제거되어 파라미터 내재화를 강제합니다.

2. 선형 감소가 안정적 전이를 보장

고정 예산(항상 있거나 항상 적거나)과 달리, 선형 감소 스케줄은 분포 변화를 부드럽게 만들어 PPO 중요도 비율을 안정화합니다.

3. 유용성 기반 선별이 핵심

단순한 제거가 아닌, 현재 정책에 실제로 도움이 되는 스킬만 유지하여 학습 효율을 극대화합니다. 랭킹 제거 시 -13.7% 붕괴가 이를 입증합니다.

4. 제로샷 > 스킬 증강

놀랍게도, [6,3,0] 스케줄로 훈련한 모델은 스킬 제거 후 오히려 +1.6% 성능이 향상되었습니다. 내재화된 지식이 노이즈 없는 추론을 가능하게 합니다.

기여 및 한계

주요 기여와 한계점

주요 기여

  • LLM 에이전트 최초의 체계적 스킬 내재화 프레임워크 제안
  • 점진적 스킬 제거를 위한 동적 커리큘럼 설계
  • 훈련 후 제로샷 자율 행동 달성 실증
  • 최소 토큰 오버헤드로 최고 성능 달성 (ALFWorld +9.7%, Search-QA +6.6%)
  • 구간별 정상성에 대한 이론적 안정성 보장 제시
  • 오픈소스 구현 공개로 재현성 보장

알려진 한계

  • 초기 SkillBank의 품질에 의존합니다.
  • 오프라인 스킬 그룹화는 새로운 작업 도메인 적용 시 재분할이 필요합니다.
  • 비전 인코더 기반으로, 텍스트 전용 모델에는 직접 적용이 어렵습니다.
