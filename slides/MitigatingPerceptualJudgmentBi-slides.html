다중모달 LLM 판사의
지각 판단 편향 완화

지각적 섭동과 보상 모델링을 통한 접근

Seojeong Park, Jiho Choi, Junyong Kang, Seonho Lee, Jaeyo Shin, Hyunjung Shim

arXiv: 2606.02578v1

배경: MLLM-as-a-Judge

다중모달 대형 언어모델을 자동화된 평가자로 활용하여, 비전-언어 응답을 규모 있게 평가하는 패러다임

등장 배경

  • 인간 평가의 높은 비용과 시간 소요
  • 평가자 간 일관성 부족 문제
  • 모델 규모와 벤치마크 확장에 따른 평가 수요 증가

핵심 한계

  • 언어적으로 그럴듯하지만 시각적으로 근거 없는 평가
  • 주로 SFT에 의존, 지각 편향 미해결
  • 시각-추론 갈등 시 신뢰도 저하

문제 정의: 지각 판단 편향

시각적 증거가 텍스트 단서와 충돌할 때, MLLM 판사가 지각적으로 올바른 응답보다 그럴듯한 서사에 더 높은 점수를 부여하는 체계적 편향

기존 판사의 전반적 오류율이 20% 이상으로 측정되며, 이는 두 가지 실패 모드에서 기인

실패 모드 A

불충분한 지각 능력

판사 자체의 시각 이해가 부정확하여, 논리적으로 일관되지만 시각적으로 틀린 응답에 높은 점수 부여

실패 모드 B

응답 앵커링

판사의 지각이 정확하더라도, 응답 텍스트에 포함된 시각 설명에 앵커링되어 자신의 지각을 무시

편향의 정량적 분석

통제된 시각 섭동 실험

0.8+

복합 오류(지각+추론) 식별 정확도

-10%p

지각 오류만 존재할 때 정확도 급락

핵심 관찰

  • 복합 오류는 비교적 잘 감지
  • 순수 지각 오류는 눈에 띄게 놓침
  • 텍스트의 논리적 일관성에 현혹됨
  • 지각 검증을 판단의 전제 조건으로 강제할 필요성

→ 지각적 일관성만 검증하면 기존 판사의 성능이 크게 하락함이 입증됨

PPJD: 지각적으로 섭동된 판단 데이터셋

최소 편집의 반사실적 응답을 구성하여 지각 오류를 분리하고, 검증 가능한 감독 신호를 제공

쿼드플렛 구조

r_c

완전 정답

지각 + 추론 모두 정확

r_{rp}

지각 오류만

시각적 불일치만 주입

r_{rp+r}

복합 오류

지각 + 추론 오류

x_i

입력

이미지 + 질문

일반적 선호 데이터셋과 달리, 미세한 지각 실패를 노출하는 입자도 높은 검증 가능한 감독 제공

PPJD 데이터 생성 파이프라인

1
소스 예제 수집
MMPR에서 정답 예제 추출 및 절대 정답성 검증
2
지각 속성 추출
응답에서 색상, 위치, 수량 등 지각적 속성 식별
3
지각 섭동 생성
추론 일관성을 유지하면서 지知觉 속성만 최소 변경
4
복합 섭동 생성
지각 + 추론 오류를 결합한 반사실적 응답 구성

거부된 접근 1: 추론 없이 지知觉만 섭동 → 비자연스러운 응답

거부된 접근 2: 지각-추론 동시 섭동 → 오류 분리 불가

검증 가능한 보상 모델링 (GRPO)

명시적 쌍별 레이블 없이, 일관된 전역 순서를 달성하는 통합 훈련 프레임워크

GRPO 핵심 메커니즘

상대적 이점 계산

그룹 내 n개 후보 응답의 보상을 정규화하여, 보상 크기에 불변하고 작업 간 분산에 강건한 이점 추정

가치 네트워크 불필요

전통적 PPO와 달리 별도의 가치 함수 없이 그룹 내 상대 비교만으로 안정적 최적화 달성

정책 업데이트

클리핑된 surrogate 손실 + 그룹 정규화 이점 + KL 정규화(β)로 참조 정책과의 거리 유지

보상 설계: 등급화된 순서 강제

판사가 완전 정답과 지각 오류를 구분하고, 복합 오류까지 명시적 등급 순서로 차별화하도록 설계

1

지각 검증을 높은 보상의 전제 조건으로 설정

시각적 근거가 틀리면 추론이 아무리 타당해도 낮은 보상

2

그럴듯한 서사에 의한 과보상 방지

응답 텍스트에 기술된 시각 단서에 현혹되지 않도록 제약

3

명시적 등급화된 보상 순서 부여

r_c > r_{rp} > r_{rp+r} 의 검증 가능한 전역 순서 학습

실험 설정

평가 프로토콜

  • 단일 점수 평가
  • 쌍별 비교
  • 배치 수준 순위

기반 모델

  • Qwen3-VL-4B-Thinking
  • Flex-Judge-7B
  • 대규모 모델 일반화

비교 대상

  • 기존 SFT 기반 판사
  • 상용 프라이빗 평가자
  • 편향 완화 베이스라인

평가 지표: Spearman 상관계수, F1/Recall(쌍별), Edit Distance(배치)

주요 결과: 단일 점수 & 쌍별 평가

+15%

Flex-Judge-7B 예측 정확도 향상

상용 프라이빗 평가자와 동등한 수준의 점수 기반 성능 달성

쌍별 비교

F1 및 Recall 지표에서 일관된 개선

지각적으로 틀린 응답을 정답보다 선호하는 비율이 크게 감소

Perception-Judge는 점수 기본 설정에서 선도적인 프라이빗 평가자와 성능을 일치시키며, 배치 수준 일관성에서는 이를 능가

Qwen3-VL-4B-Thinking 배치 메트릭 향상

평가 차원 기존 판사 Perception-Judge

순위 일관성 불안정 안정

시각-추론 갈등 처리 취약 강건

제거 연구

구성 요소별 기여도

  • PPJD 데이터 없이 훈련 시 성능 크게 하락
  • 등급화된 보상 없이 이진 보상만 사용 시 순위 일관성 저하
  • KL 페널티(β) 조정에 따른 안정성 변화 분석

섭동 유형별 효과

  • 순수 지각 섭동(r_{rp})이 핵심적 역할
  • 복합 섭동(r_{rp+r})은 추가적인 미세 조정 효과
  • 지각-추론 오류 분리가 학습 효율의 핵심

PPJD의 지각 오류 분리 구조와 등급화된 보상의 조합이 최적 성능의 필수 조건

핵심 요약

지각 판단 편향 발견 및 정식화

불충분한 지각 능력과 응답 앵커링이라는 두 실패 모드를 체계적으로 분석

PPJD 데이터셋 구축

최소 편집 반사실적 응답으로 지각 오류를 분리한 검증 가능한 감독 신호 제공

GRPO 기반 통합 훈련 프레임워크

명시적 쌍별 레이블 없이 등급화된 전역 순서를 학습하는 보상 설계

상당한 성능 향상

배치 메트릭 최대 +11%, 단일 점수 정확도 +15%, 상용 평가자와 동등 또는 우수
