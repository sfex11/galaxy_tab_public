VLMs는 자연 읽기 과정에서 LLMs보다
인간 정렬을 전역적으로 향상시키지 않을 수 있다

Jinzhou Wu, Zhengwu Ma, Jixing Li, Baoping Tang, Zitong Lu

Chongqing University · City University of Hong Kong · MIT

arXiv:2605.28818v1 · 2026년 5월

연구 배경
LLM과 인간 뇌의 정렬

대규모 언어 모델은 자연스러운 읽기, 이야기 듣기, 언어 이해 중 fMRI, MEG, ECoG 신호를 예측하며 인간 언어 처리와 놀라운 유사성을 보여줌

시각-언어 모델의 부상

VLM은 시각-언어 쌍 데이터로 학습되어 시각적으로 구현된 의미를 포착할 가능성이 있으나, 텍스트 전용 처리에서의 효과는 불명확

"시각-언어 학습이 자연 읽기 중 모델-인간 정렬을 변화시키는가? 그리고 그러한 효과가 전역적인지, 시각적으로 구현된 의미에 선택적으로 묶여 있는가?"

선행 연구의 한계
  • 다중모달 뇌 정렬 연구는 주로 시각·음시각 설정에서 VLM 우위를 보고했으나, 이것이 언어 표현의 변화인지 온라인 다중모달 처리인지 구분 불가
  • 많은 비교가 개별 개념에 집중하거나, 아키텍처·규모·훈련 데이터가 크게 다른 모델을 비교
  • 자연 읽기 상황에서 VLM이 인간 정렬을 향상시키는지에 대해서는 거의 알려지지 않음
  • 문장의 시각적 의미 강도(visualness strength)에 따라 효과가 달라지는지 검토 부족
필요한 접근

동일한 텍스트 전용 입력, 최소화된 모델 차이, 자연 읽기 맥락에서 fMRI + 안구운동 동시 평가

연구 기여점
1
전역적 우위 없음

다중모달 사전학습이 자동적으로 안정적인 전역적 정렬 우위를 산출하지 않음을 입증

2
내용 의존적 효과

VLM 우위는 강한 시각적 의미를 포함한 문장에서 선택적으로 나타남

3
새로운 연구 패러다임

정합 모델 쌍을 통제된 실험 도구로 사용하여 다중모달 경험의 역할 규명

참가자 및 실험 환경

참가자는 fMRI 스캐너 내에서 한 번에 하나씩 문장을 읽으며, 동시에 전뇌 피질 fMRI BOLD 반응과 안구운동(안점 추적)이 기록됨

이 설정은 자연스러운 읽기 과정에서의 연속적인 의미 통합과 안구운동 패턴을 포착

핵심 아이디어

문장의 시각적 의미 강도(visualness strength)가 높을수록 VLM의 다중모달 학습 이력이 더 큰 이점을 제공할 것이라는 가설 검증

시각 강도 측정
  • 문장별 시각 강도 점수 산출
  • LLM 기반 판정 절차 + 외부 LLM 판사
  • 다중 판사 융합 및 폴드별 보정
조건부 정렬 분석
  • 시각 강도에 따라 문장 분할
  • 각 조건에서 VLM−LLM fMRI 정렬 차이 평가
  • fMRI와 안구운동에서의 수렴 증거 확인
결과 1: 전역적 fMRI 정렬 — 향상 없음

시각적 사전학습이 자연 읽기 중 전역적 fMRI 정렬을 일관되게 향상시키지 않음

Qwen2 쌍
유의한 VLM 우위 없음
DeepSeek 쌍
유의한 VLM 우위 없음
Mistral/Idefics2 쌍
부분적 우세 불안정

텍스트 전용 언어 표현이 자연 읽기 중 신경 반응을 모델링하는 데 여전히 강력한 기반이며, 다중모달 사전학습만으로 전역적 향상이 보장되지 않음

결과 2: 회귀 안구운동 정렬 — 모델 의존적 개선

회귀 안구운동(퇴행성 눈물기) 정렬에서 제한적이고 모델에 따라 다른 VLM 개선 관찰

개선된 쌍

일부 모델 쌍에서 VLM이 회귀 안구운동 예측에서 LLM보다 약간 더 나은 정렬을 보였으나, 효과 크기는 작고 일관되지 않음

모델 의존성

이러한 개선은 특정 모델 쌍에 국한되며, 세 쌍 모두에서 재현되는 안정적인 패턴은 아님

→ 회귀 안구운동은 문장 재독과 관련된 행동 지표로, VLM이 일부 읽기 행동 측면에서 미미한 이점을 가질 수 있으나, 이것이 전역적 신경 정렬 향상으로 이어지지는 않음

결과 3: 시각 강도가 VLM−LLM 정렬 차이를 조절

가장 핵심적인 발견 — VLM 우위는 시각적 의미가 강한 문장에서 선택적으로 나타남

fMRI 증거

문장의 시각 강도가 높을 때 VLM−LLM fMRI 정렬 차이가 유의하게 증가. 시각 강도가 낮은 문장에서는 차이가 없거나 LLM이 우세

안구운동 증거

회귀 안구운동 정렬에서도 동일한 패턴 수렴. 시각적으로 생생한 문장에서 VLM이 더 인간에 가까운 안구운동 예측

"다중모달 사전학습은 전역적으로 기여하는 것이 아니라, 시각적으로 구현된 의미 처리에 선택적으로 기여한다"

인지적·신경과학적 함의
언어 처리의 핵심은 여전히 언어 내부 표현

자연 읽기에서 전뇌 피질 반응을 예측하는 주요 요인은 텍스트 전용 언어 표현의 질이며, 시각 경험이 이를 전역적으로 대체하지 않음

시각적 구현 인지의 선택적 역할

인간의 시각-지각 경험이 언어 이해에 기여하지만, 그 기여는 주로 시각적으로 풍부한 표현 처리에 국한됨

제어된 인실리코 실험의 가치

정합 모델 쌍은 인지 신경과학에서 인과적 추론을 위한 강력한 도구로, 다중모달 경험의 역할을 분리하는 데 기여

VLM 설계에 대한 시사

인간 정렬을 목표로 할 때 텍스트 코퍼스와 언어 목적함수의 질이 여전히 가장 중요한 설계 요소

결론 및 한계
핵심 결론
  • 다중모달 사전학습은 자연 읽기에서 전역적 인간 정렬 향상을 보장하지 않음
  • VLM의 이점은 시각적 의미가 강한 문장에 선택적으로 등장 (fMRI + 안구운동 수렴 증거)
  • 정합 모델 쌍 기반 평가 프레임워크가 다중모달 경험의 역할을 규명하는 유효한 패러다임임을 시사
한계점
  • 3개 모델 쌍으로 제한 — 더 다양한 규모·아키텍처에서의 검증 필요
  • 어텐션 특징만 사용 — FFN, 잔차 스트림 등 다른 표현과의 비교 부재
  • STEM 텍스트에 집중 — 다른 장르에서의 일반화 불확실
  • 시각 강도 측정의 간접성 — LLM 기반 판정 절차의 한계
