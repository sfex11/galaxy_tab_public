기존 접근의 한계

기존 패러다임: 수동 증거 소비

  • 이미 정리된 증거를 모델에 전달한다고 가정
  • 벤치마크 구성 시 미리 선택된 컨텍스트에 의존
  • 단일 모달리티(주로 텍스트)에 국한
  • 실제 임상 워크플로우와 괴리

실제 임상 워크플로우의 요구

  • 외부 의료 지식에서 능동적 검색
  • 원본 EHR 테이블에서 환자별 정보 탐색
  • 의료 영상에서 시각적 단서 추출
  • 다양한 출처의 증거를 통합적 합성

핵심 과제는 주어진 증거를 추론하는 것뿐만 아니라, 어디서, 무엇을, 어떻게 증거를 획득하고 통합할지 결정하는 것이다.

ClinSeekAgent: 핵심 아이디어

수집된 증거를 통합하여 최종 예측 도출

과제 정의와 상호작용 프로토콜

임상 과제 인스턴스

x = (p, t, q, M, Y)

  • p: 환자 식별자
  • t: 참조 타임스탬프 (예측 시점)
  • q: 임상 과제 지시문
  • M: 모달리티 메타데이터 (예: 영상 경로)
  • Y: 정답 스키마 또는 후보 레이블 공간

상호작용 루프

k

모델이 과제 인스턴스 x와 이전 상호작용 이력 hk-1을 관찰

도구 호출: 환경이 관찰 결과 ok 반환 → 다음 단계로

종료: 최종 예측 ŷ를 정답 스키마에 따라 출력

※ EHR 관련 과제에서는 ehr.load_ehr로 환자 DB를 먼저 로드하며, 모든 조회는 참조 시점 t 이전의 기록으로 제한

규칙 기반 검색 파이프라인과 달리, ClinSeekAgent는 증거 출처 간에 순서를 강제하지 않는다. 과제에 따라 모델이 자율적으로 탐색 전략을 결정한다.

다중모달 과제에서 텍스트 전용보다 훨씬 큰 폭의 개선을 보인다. 관련 증거가 EHR 테이블과 의료 영상에 분산되어 있어 능동적 탐색의 가치가 크기 때문이다. 평가된 5/6개 모델이 3개 CXR 과제 그룹 모두에서 향상되었다.

ClinSeekAgent 우위 분석

텍스트 전용: 위험 예측

사전 선별 컨텍스트가 누락하는 정보를 에이전트가 직접 발견

  • 길이 방향성 환자 기록에서 시간적 패턴 탐지
  • 여러 EHR 테이블 간 교차 검증
  • 의사결정 과제보다 위험 예측에서 더 큰 이점

다중모달: 구성적 도구 사용

시각·EHR·외부 지식을 연결하는 복합적 탐색

  • 흉부 X선 분류 → EHR에서 관련 검사 결과 확인
  • 영상 소견과 외부 지식의 통합 추론
  • 단일 모달리티로는 불가능한 근거 도출

실패 분석: 의사결정 과제

의사결정 과제에서는 개선이 제한적. 에이전트가 올바른 증거를 찾아도, 최종 선택 단계에서 여러 합리적 옵션 간 우선순위를 결정하는 데 어려움을 보임. 증거 탐색과 결정 합리화는 별개의 능력으로 보인다.

학습 시 검증: ClinSeek-35B-A3B

강한 교사 모델의 탐색 궤적을 소형 오픈소스 모델에 증류

학습 설정

  • 베이스라인: Qwen3.5-35B-A3B (MoE)
  • 교사: Claude Opus 4.6의 ClinSeekAgent 궤적 수집
  • SFT로 에이전트 탐색 행동 증류
  • 활성 파라미터 3B로 효율적 학습

AgentEHR-Bench 결과

34.0 평균 F1
  • 베이스라인 대비 +11.9 포인트 향상
  • 모든 오픈소스 베이스라인 최고 성능
  • Claude Opus 4.6 (36.0)에 근접

ClinSeekAgent는 추론 시 파이프라인일 뿐만 아니라, 임상 증거 탐색 행동을 오픈소스 모델에 확장 가능하게 증류하는 학습 파이프라인으로도 검증되었다.

학생 모델은 무엇을 학습하는가?

탐색 전략

어떤 도구를 언제 호출할지, 어떤 순서로 증거를 수집할지 학습

증거 선별

관련 없는 정보를 걸러내고 과제에 필수적인 증거에 집중하는 능력

증거 통합

다양한 출처의 증거를 결합하여 일관된 임상 결정을 도출

핵심 발견

  • 학생 모델은 교사의 도구 호출 패턴과 순서를 효과적으로 모방
  • 특히 EHR 스키마 탐색 및 SQL 쿼리 생성 능력이 크게 향상
  • 다만, 복잡한 다중모달 구성적 추론에서는 교사와 격차 존재
  • 3B 활성 파라미터로도 임상 에이전트 행동을 충분히 학습 가능

결론 및 시사점

주요 기여

  • 수동 증거 소비 → 능동 증거 획득 패러다임 제안
  • EHR + 웹 + 영상의 20개 도구를 갖춘 통일 에이전트 프레임워크
  • ClinSeek-Bench: 사전 선별 vs 자동 탐색 쌍대 평가 벤치마크
  • 추론 시·학습 시 양방향 검증 완료
