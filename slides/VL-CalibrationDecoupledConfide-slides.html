01

문제 정의

왜 LVLM의 신뢰도 보정이 필요한가?

핵심 문제: 과잉 확신(Overconfidence)

LVLM은 틀린 답변에도 높은 확신을 표현하여, 의료·법률 등 고위험 분야 배포를 저해합니다.

🔗

단일 신뢰도의 한계

기존 방법은 시각적 오류와 추론 오류를 하나의 점수로 합쳐, 오류의 원인을 구분할 수 없습니다.

📝

텍스트 편향

텍스트 전용 LLM용 보정 방법은 언어 사전확률이 시각적 불확실성 신호를 압도하는 문제가 있습니다.

출력 구조 (Trajectory)

τ = (z_vis, c_vis, z_reas, c_reas, y) z_vis → 시각 분석 (밀집 캡션) c_vis → 시각 신뢰도 [0-10] z_reas → 추론 체인 c_reas → 추론 신뢰도 [0-10] y → 최종 답변

통합 신뢰도: 조화 평균

Φ(ĉ_vis, ĉ_reas) = 2 · ĉ_vis · ĉ_reas / (ĉ_vis + ĉ_reas)

조화 평균은 최솟값에 의해 지배되므로, 한 쪽이 불확실하면 전체 신뢰도가 낮아집니다.

🔍

분리형 신뢰도

시각 확실성 추정
(VCE)

KL 발산 + 토큰 엔트로피로 시각 기반 여부를 비지도 방식으로 측정

⚖️

토큰 레벨 이점 재가중
(TAR)

시각적으로 근거 없는 토큰의 불이익을 선택적으로 증폭

전체 보상 함수

R(τ, y*) = λ_acc · R_acc + λ_cal · R_cal + λ_vis · R_vis R_acc = 𝟙{y ≡ y*} (정확도) R_cal = −(Φ − 𝟙{y=y*})² (보정) R_vis = −(ĉ_vis − S̃_vis)² (시각 확실성)
λ_acc = 1.0 λ_cal = 2.0 λ_vis = 0.4
04

시각 확실성 추정 (VCE)

비지도 시각 기반 라벨 생성

시각 근거 (KL 발산)

D_KL = 1/T Σ KL(π(·|I) ∥ π(·|I'))
  • I' = 이미지 80% 패치 마스킹
  • 높은 D_KL → 이미지에 강하게 의존
  • 낮은 D_KL → 환각 (이미지 무시)

내적 확실성 (토큰 엔트로피)

H = −1/T Σ Σ π(v) log π(v)
  • 분포의 날카로움을 측정
  • 낮은 엔트로피 → 높은 확신
  • 높은 엔트로피 → 불확실한 예측

통합 점수

S_vis = log(D_KL + ε) − log(H + ε) ↓ 배치 z-score 정규화 z = (S_vis − μ_B) / (σ_B + ε) ↓ 시그모이드 S̃_vis = σ(z) ∈ [0, 1] ↓ stop-gradient sg(S̃_vis) — RL 학습에서 그래디언트 차단
