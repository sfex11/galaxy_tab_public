기술 1

확장된 Cascade RL

Nemotron-Cascade 1 대비 훨씬 넓은 영역의 추론 및 에이전트 도메인을 포괄하도록 Cascade 강화학습을 대폭 확장.

  • • 계층적 검증 전략으로 솔루션 필터링
  • • 다단계 보상 계산을 통한 효율적 학습
  • • 수학, 코딩, 에이전트 도메인 통합
🔁
기술 2

다중 도메인 온-폴리시 증류

Cascade RL 과정 전반에 걸쳐 각 도메인의 최강 중간 교사 모델로부터 온-폴리시 증류를 수행.

  • • 도메인별 최적 교사 모델 자동 선택
  • • 벤치마크 성능 저하 효율적 복구
  • • 학습 전 과정에서 지속적 성능 향상
핵심 차별점: 다중 도메인 증류를 통해 RL 학습 중 발생하는
벤치마크 회귀(regression) 문제를 해결하면서도 전반적 성능 향상을 유지

v1 vs v2

Cascade v1 → v2 발전

Nemotron-Cascade 1
  • ● 제한적 도메인의 Cascade RL
  • ● 단일 도메인 학습
  • ● 벤치마크 회귀 발생 가능
  • ● 에이전트 능력 미지원
Nemotron-Cascade 2
  • ✓ 넓은 스펙트럼의 Cascade RL
  • ✓ 다중 도메인 온-폴리시 증류
  • ✓ 성능 회귀 효과적 방지
  • ✓ 강력한 에이전트 역량

핵심 발전: 다중 도메인 증류의 도입으로
RL 학습 과정에서 한 도메인의 개선이 다른 도메인의 성능을 저하시키는
고질적 문제를 해결하고 전 영역에서 균형 잡힌 성능 향상을 실현

결론

결론 및 시사점

  • 🚀
    극한의 효율성: 30B MoE (3B 활성)로 671B 모델에 근접한 추론 능력 달성. "지능 밀도" 패러다임 제시.
  • 🏆
    세계 대회 수준: IMO, IOI, ICPC에서 금메달 수준 — 수학과 코딩 추론 모두에서 최상위 성능 입증.
  • 🔧
    방법론 혁신: Cascade RL + 다중 도메인 온-폴리시 증류로 RL 학습의 성능 회귀 문제를 근본적으로 해결.
  • 🌐
    오픈소스 기여: 모델 체크포인트와 학습 데이터를 CC BY 4.0으로 공개. 연구 커뮤니티에 실질적 기여.
  • 💡
    실용적 함의: 소형 모델로도 강력한 추론이 가능함을 보여주며, 효율적 AI 배포의 새로운 가능성 제시.
