  • RoBERTa-large + Qwen2.5: 0.991

"C-ReD에서의 미세조정은 모든 도메인에서 극적인 성능 향상을 가져오며, 이 효과는 학습 시 보지 못한(OOD) 생성기에도 일관되게 확장됩니다."

핵심 발견

주요 발견 사항

효과적이었던 것

  • 미세조정이 ID/OOD 모두에서 극적인 성능 향상
  • ReMoDetect & IMBD: 전 도메인 0.85+ AUROC
  • 구조화된 도메인(Q&A, 영화)이 탐지에 용이
  • 뉴스/학술 도메인으로 학습 시 교차 도메인 일반화 우수

한계를 보인 것

  • 사전학습 RoBERTa: 0.4~0.5 수준으로 실패
  • LLM 제로샷 탐지: 자신이 생성한 텍스트도 판별 불가
  • Deepseek-R1: CoT 추론으로 인해 가장 탐지 어려움
  • 영화/Q&A → 타 도메인 전이 성능 불균일

프롬프트 복잡도 무관

단순화된 프롬프트 vs 복잡한 프롬프트: AUROC 차이 3% 미만. 정교한 LLM은 지시 수준에 관계없이 일관된 패턴을 생성합니다.

교차 도메인

교차 도메인 일반화 패턴

학습 도메인에 따른 타 도메인 전이 성능의 비대칭성

강한 일반화 소스

  • 뉴스 → 타 도메인: 0.65~0.85
  • 학술 논문 → 타 도메인: 유사하게 높음

유창하고 정보 밀도가 높은 도메인

약한 일반화 소스

  • 영화 리뷰 → 뉴스: 낮은 성능
  • Q&A → 학술: 0.45~0.55

스타일화/의견 중심 도메인

"탐지기는 학습 도메인에서 최고 성능을 달성하지만, 교차 도메인 일반화는 소스 학습 도메인과 타겟 테스트 도메인 모두에 의존하여 매우 비균일합니다."

기여

핵심 기여 사항

1

최초의 포괄적 중국어 벤치마크

9개 LLM(5개 국내 모델 포함), 6개 도메인, 128K+ 샘플 규모

2

3대 데이터셋 한계 해결

모델 다양성, 도메인 이질성, 현실적 프롬프트 문제를 동시에 해결

3

7가지 체계적 평가 프로토콜

도메인/생성기 무관, 교차 도메인, 교차 모델, LLM 탐지기, 프롬프트 절제, 번체 중국어, 외부 전이

4

다국어 검증 & 실용적 인사이트

간체/번체 중국어 모두 검증 | Deepseek-R1 탐지 난이도, 프롬프트 복잡도 무관성 등 발견

결론

결론 및 향후 과제

"C-ReD에서의 미세조정은 학습 시 본 생성기뿐 아니라 미지의 상용 모델과 외부 데이터셋에서도 상당한 성능 향상을 가져와, 그 대표성과 실용성을 입증합니다."

인정된 한계점

  • 중국어에만 초점 (번체 검증은 부분적)
  • 9개 LLM으로 모든 미래 아키텍처 포괄 불가
  • 적대적/고도 맞춤형 프롬프트 전략 미반영
  • 인간 참조 텍스트의 다양성 및 범위 제한
