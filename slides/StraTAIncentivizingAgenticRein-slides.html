Incentivizing Agentic Reinforcement Learning with Strategic Trajectory Abstraction

arXiv: 2605.06642v1

LLM 에이전트의 부상

상호작용형 에이전트

LLM이 단순한 텍스트 생성기를 넘어, 환경과 상호작용하며 의사결정을 수행하는 자율 에이전트로 진화

장기 궤적 문제

다단계 태스크에서는 수십~수백 번의 행동으로 이어지는 긴 궤적을 효과적으로 최적화해야 함

LLM 에이전트의 핵심 도전 과제는 장기horizon 의사결정을 어떻게 학습할 것인가이다.

문제 정의

현재 LLM 에이전트 최적화 방법은 대부분 순수 반응형(reactive) 방식에 의존한다.

1

탐색 부족

반응형 보상만으로는 다양한 행동 경로를 충분히 탐색하기 어려움

2

신용 할당 곤란

긴 궤적에서 어떤 행동이 최종 성과에 기여했는지 파악 불가

3

장기 전략 부재

즉각적 보상에만 반응하여 전략적 계획 수립 실패

반응형 접근의 근본적 한계

단계별 관측에만 의존

각 시점의 즉각적 상태만 보고 행동 선택, 궤적 전체 맥락 무시

즉각적 보건에 과도하게 민감

단기 피드백에 편향되어 지연 보상을 무시하는 myopic 행동 유발

행동 간 의존성 파악 불가

이전 행동이 미래 상태에 미치는 영향을 학습 신호로 활용하지 못함

핵심 인사이트

궤적을 전략적 수준으로 추상화하면, 반응형 방법의 한계를 극복하고 장기 의사결정을 위한 의미 있는 학습 신호를 생성할 수 있다.

기존 관점 (반응형)

행동 단위 → 단계별 보상 → 국소적 학습

StraTA 관점 (전략형)

추상화 단위 → 전략적 보상 → 구조적 학습

StraTA 방법론 개요

Strategic Trajectory Abstraction — 궤적을 의미 있는 전략 단위로 분할하여 강화학습 신호의 품질을 향상

  • 원시 궤적을 전략적 의미를 갖는 하위 궤적(sub-trajectory)로 추상화
  • 추상화된 단위 수준에서 보건을 재구성하여 신용 할당 문제 완화
  • 전략적 탐색을 장려하는 인센티브 메커니즘 설계
  • 기존 반응형 RL 파이프라인과 호환되는 플러그인 형태

전략적 궤적 추상화

긴 궤적을 전략적으로 의미 있는 청크로 분할하는 것이 핵심

1
원시 궤적
a₁, a₂, ..., aₙ 연속 행동 시퀀스
2
전략 분할
의미 단위 기반 하위 궤적 그룹화
3
추상 표현
각 청크의 전략적 특징 벡터 추출
4
재구성된 보건
추상 단위 수준의 학습 신호 생성

행동 수준의 잡음을 제거하고 전략적 패턴을 부각함으로써 신호 대 잡음비 극대화

탐색 인센티브 설계

기존 반응형 탐색

  • 단계 수준의 무작위扰动만 사용
  • 국소적 최적에 빠질 확률 높음
  • 장기적으로 유의미한 경로 발견 어려움

StraTA 전략적 탐색

  • 추상화된 전략 단위 수준에서 탐색
  • 새로운 전략 패턴 시도에 보너스 부여
  • 다양한 궤적 구조 발견 장려

신용 할당 개선 메커니즘

추상화를 통해 신용 할당의 해상도와 정확도를 동시에 향상

궤적 압축

수백 단계의 행동을 수십 개의 전략 청크로 압축하여 신호 전파 경로 단축

정밀한 귀인

전략적 맥락 내에서 어떤 하위 궤적이 성과에 기여했는지 식별 가능

계층적 할당

추상 수준(전략)과 구체 수준(행동)의 이중 신용 할당 수행

전체 학습 파이프라인

1
궤적 수집
에이전트가 환경과 상호작용하여 궤적 생성
2
추상화
StraTA가 궤적을 전략적 하위 단위로 분할
3
보건 재구성
추상 단위 수준에서 인센티브 신호 생성
4
정책 업데이트
재구성된 신호로 LLM 에이전트 강화학습

기존 RLHF / RLAIF 파이프라인에 드롭인 모듈으로 통합 가능 — 별도의 아키텍처 변경 불필요

실험 설정

벤치마크

  • 다단계 추론 태스크
  • 웹 기반 에이전트 환경
  • 도구 사용 시나리오
  • 장기horizon 게임 환경

비교 대상

  • 순수 반응형 RL 베이스라인
  • 표준 RLHF / RLAIF
  • 과거 궤적 보건 방법들
  • SFT (감독 학습) 기준

주요 실험 결과

방법 단기 태스크 장기 태스크 탐색 다양성
SFT 중간 낮음 매우 낮음
반응형 RL 높음 낮음 낮음
표준 RLHF 높음 중간 낮음
StraTA 높음 높음 높음

StraTA는 특히 장기horizon 태스크와 탐색 다양성 측면에서 기존 방법들을 일관되게 상회

핵심 기여 요약

문제 재정의

에이전트 RL의 핵심 병목을 반응형 접근으로 규명

새로운 패러다임

전략적 궤적 추상화라는 새로운 RL 프리미티브 제안

실증적 효과

다양한 벤치마크에서 장기 의사결정 성능 크게 향상

결론 및 전망

  • 반응형 접근이 에이전트 RL의 탐색과 신용 할당을 제한한다는 것을 입증
  • 전략적 궤적 추상화가 이 두 가지 문제를 동시에 완화
  • StraTA는 기존 파이프라인에 플러그인 형태로 적용 가능한 실용적 방법
  • 장기horizon 태스크에서 기존 RL 방법 대비 일관된 성능 향상 달성

향후 방향: 더 복잡한 멀티모달 에이전트 환경으로의 확장, 추상화 수준의 자동 적응 연구
