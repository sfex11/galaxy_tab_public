CLSGen

확률적 분류와 언어적 설명을 동시에 수행하는
이중 헤드 미세조정 프레임워크

Yoon, Zhu, Bulovic, Sehy, Gao, Dligach, Afshar, Miller

01 — 문제 정의

LLM의 근본적 딜레마

🎯

분류 성능 최적화

판별적 목적함수로 미세조정하면 확률 추정은 향상되지만...

  • 정확한 확률 값 출력
  • 높은 AUROC 달성
  • 신뢰할 수 있는 예측
💬

설명 생성 능력 상실

치명적 망각(Catastrophic Forgetting)으로 언어 생성 능력이 붕괴됩니다.

  • 파싱 불가능한 출력
  • 무의미한 텍스트 생성
  • 해석 가능성 제로
⚠️ 언어 붕괴 예시 (에폭 13 이후)
"Pom Pomuppy Pom Pom... waveformuppybagriprip"

→ 기존 방식으로 분류 미세조정 시, 파싱 가능한 출력이 ~85%에서 거의 0%로 급락

02 — 핵심 기여

CLSGen의 4가지 혁신

🏗️

이중 헤드 아키텍처

하나의 백본에 분류 헤드(CLS)와 언어 모델 헤드(LM)를 동시 부착하여 두 가지 능력을 보존합니다.

📊

결합 최적화

Ltotal = Lgen + Lcls 로 분류와 생성 손실을 동시에 최적화합니다.

📝

데이터 구축 전략

교사 모델을 활용한 약지도 학습으로 설명-레이블 쌍을 자동 생성합니다.

📐

일관성 평가 지표

AUROC-Alignment 등 새로운 메트릭을 도입하여 분류와 설명 간 정합성을 측정합니다.

03 — 아키텍처

분류 + 생성 이중 헤드

📄 입력: 시스템 프롬프트 + 문서 + 질문 + "Reasoning:"
▼
🧠 Llama-3.1-8B-Instruct (공유 백본) LoRA: rank=8, α=16, dropout=0.05
▼   hn (마지막 토큰 은닉 표현)   ▼
🎯 분류 헤드 (CLS) 2층 MLP, 히든 2048, 드롭아웃 0.1
💬 언어 모델 헤드 (LM) 자기회귀 텍스트 생성
→ 이진 확률 P(y|x)
→ 설명 + "Classification: [레이블]"
ℒtotal = ℒgen (인과적 언어모델링) + ℒcls (CrossEntropy(MLP(hn), y))
04 — 데이터 구축

교사 모델 기반 설명 생성

1

입력 샘플 준비

각 입력 xi와 정답 레이블 yi 쌍을 준비합니다.

↓
2

교사 모델 프롬프팅 (K=5회)

DeepSeek-R1-Distill-Llama-8B에 최대 5회 프롬프트하여 설명 ei와 예측 ŷi를 생성합니다.

↓
3

품질 필터링

ŷi = yi (정답 일치) AND 형식이 유효한 첫 번째 쌍만 채택합니다.

↓
4

학습 데이터셋 T 완성

최종 데이터: T = {(xi, yi, ei)} — 5회 모두 실패한 샘플은 제외됩니다.

⚠️ 소수 클래스 손실: LCD 30일 양성 샘플 57.7%, CliniFact 양성 56.3% 제외

09 — 핵심 인사이트

왜 CLSGen이 효과적인가

01

이중 목적함수의 상호 보완

Lgen은 언어 능력을 유지하고, Lcls는 확률 추정을 정교화합니다. 두 그래디언트가 공유 표현을 공동으로 형성합니다.

02

CNN 대비 장점

CNN은 AUROC는 높지만 F1과 설명 생성이 불가능합니다. CLSGen은 두 가지 모두 달성합니다.

03

추론 전용 대비 장점

자기일관성 등 제로샷 방법은 설명은 가능하지만 확률 보정이 약합니다. CLSGen은 미세조정으로 이를 해결합니다.

04

품질 기반 모델 선택

Squality = z(AUROCcls) + z(AUROCalign) + z(κ) 로 분류·정합성·일관성을 종합 평가합니다.

10 — 한계 및 향후 연구

한계점과 미래 방향

⚠️

현재 한계

  • 이진 분류만 지원 (다중 레이블 미지원)
  • 소수 클래스 데이터 손실 (최대 57.7%)
  • 임계값 의존적 지표 (F1, κ)
  • 제거된 샘플이 노이즈인지 어려운 사례인지 불명확
🚀

향후 연구 방향

  • 탐욕적 K-best 이외의 설명 샘플링 전략
  • 제거된 인스턴스의 정성적 분석
  • 다중 레이블 확장 연구
  • 임계값 일반화 전략 탐구

핵심 메시지

CLSGen은 정확한 확률 분류와 해석 가능한 설명 생성을 동시에 달성하는 최초의 이중 헤드 프레임워크로, 의료 AI 등 실세계 응용에 중요한 진전을 이루었습니다.
