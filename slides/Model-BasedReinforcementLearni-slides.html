문제 정의

실세계 시스템은 변한다

⚠️
기존 접근의 한계
대부분의 강화학습 알고리즘은 정상(stationary) 동역학을 가정합니다. 하지만 실제 시스템에서는 마모, 드리프트, 운용 조건 변화 등으로 동역학이 지속적으로 변합니다.
🎯
본 논문의 목표
시변(time-varying) 동역학 하에서도 잘 작동하는 모델 기반 강화학습(MBRL) 프레임워크를 설계하고 이론적 보장을 제공합니다.
에피소드 n에서의 시스템 동역학 xn,t+1 = f*n(xn,t, un,t) + wn,t

핵심 통찰: f*n이 에피소드마다 달라지면, 과거 데이터가 현재 시스템을 정확히 반영하지 못해 모델 보정(calibration)이 깨집니다.

이론적 프레임워크

비정상성 모델링: 변동 예산

변동 예산 (Variation Budget)
재생핵 힐베르트 공간(RKHS)에서 에피소드 간 누적 동역학 변화량에 상한을 둡니다.
N 에피소드에 걸친 총 변동량 제약 Σn=1N-1 ‖f*n+1 - f*n‖Hk ≤ PN
동적 리그렛 (Dynamic Regret)
정적 리그렛 대신, 매 에피소드마다 변하는 최적 정책과 비교합니다.
RN = Σn=1N [J(π*n, f*n) - J(πn, f*n)]
가우시안 프로세스 모델
각 상태 차원을 독립적인 GP로 모델링합니다. GP의 사후 평균과 분산이 불확실성을 정량화하지만, 비정상 환경에서는 시간적 편향이 추가로 발생합니다.

주의: GP 분산은 정상 함수 가정 하의 통계적 불확실성만 반영합니다. 시간에 따른 동역학 변화(temporal mismatch)는 포착하지 못합니다.

핵심 이론

드리프트 하의 보정 (Lemma 1)

비정상 환경에서 GP 신뢰 구간을 확장한 핵심 보조정리:
|f*n,j(z) - μm:n-1,j(z)| ≤ βn(δ, n-m) · σm:n-1,j(z) + ξn-m · Σs=mn-1 ‖f*s+1,j - f*s,j‖Hk
인식론적 불확실성 항
βn(δ, n-m) · σm:n-1,j(z)
표준 통계적 오차입니다. 데이터가 많아지면 감소합니다. 정상 환경에서의 기존 GP 신뢰 구간과 동일합니다.
시간적 편향 항 (신규)
ξn-m · Σ ‖f*s+1 - f*s‖Hk
비정상성으로 인한 새로운 오차항입니다. 버퍼 길이(n-m)가 길수록, 누적 드리프트가 클수록 증가합니다.

핵심 함의: 시간적 편향 항은 버퍼 길이에 따라 증가하므로, 오래된 데이터의 영향을 명시적으로 제한해야 합니다. → 이것이 망각 메커니즘의 이론적 근거입니다.

R-OMBRL & SW-OMBRL 알고리즘

Algorithm 1: R-OMBRL
정책 πθ, 모델 M0, 버퍼 D 초기화
for n = 1 to N:
if n mod H == 0: D ← ∅
πθ로 궤적 τn 수집
D ← D ∪ τn
D로 모델 Mφ 업데이트
(Mφ, D)로 πθ 업데이트
Algorithm 2: SW-OMBRL
정책 πθ, 모델 M0, 버퍼 D 초기화
for n = 1 to N:
πθ로 궤적 τn 수집
D ← D ∪ τn
w보다 오래된 데이터 제거
D로 모델 Mφ 업데이트
(Mφ, D)로 πθ 업데이트
