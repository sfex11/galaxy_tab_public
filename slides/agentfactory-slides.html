🎯 문제 정의

기존 LLM 에이전트의 한계

성공한 태스크 경험을 텍스트로 저장 → 재사용 어려움

❌ 텍스트 기반 경험

  • 자연어로 기록된 "교훈"
  • LLM이 매번 재해석 필요
  • 실행 신뢰성 낮음
  • 복잡한 시나리오에서 실패

✅ 코드 기반 경험

  • 실행 가능한 Python 코드
  • 그대로 실행하면 됨
  • 100% 재현 가능
  • 지속적 개선 가능

💡 AgentFactory: 핵심 혁신

🔧 실행 가능한 서브에이전트

  • 성공한 태스크를 .py 파일로 저장
  • SKILL.md로 문서화
  • 순수 Python → 이식성 뛰어남

🔄 Self-Evolve 메커니즘

  • 실행 피드백 기반 개선
  • README 서브에이전트: 3회 만에 자율 진화
  • 하드코딩 → LLM 파싱 → 정규식

🎯 Meta-Agent

  • 문제 분해 및 서브에이전트 할당
  • 관련 도구만 선별 → 탐색 공간 축소
  • 공통 패턴 인식 → 재사용

🌐 크로스 플랫폼

  • Claude Code에서 바로 사용
  • LangChain, AutoGen 호환
  • 프레임워크 독립적
