명시적 증거와 암시적 증거로부터

Personal Visual Memory from Explicit and Implicit Evidence

텍스트 중심 기억의 한계를 넘어, 시각 정보를 직접 활용하는
새로운 개인화 장기 기억 프레임워크

배경

개인화 AI 에이전트와 장기 기억

개인화 AI 에이전트는 사용자의 선호, 습관, 환경 등을 기억하고
이를 바탕으로 맞춤형 응답을 생성해야 합니다.

텍스트 기억

대화 내용, 명시적 선호 표현 등 텍스트 형태의 정보

시각 기억

사진, 영상, 환경 이미지 등 시각 형태의 정보

"장기 기억은 개인화 AI 에이전트의 핵심 역량이지만, 현실에서는 대부분 텍스트에만 의존하고 있다."

문제 정의

기존 방법의 근본적 한계

텍스트 중심 벤치마크

이미지가 포함되어도, 질문에 필요한 정보가 텍스트만으로도 복원 가능한 구조

이미지의 텍스트 환원

대부분의 기억 시스템이 이미지를 캡션으로 변환하여 시각 정보의 손실 유발

암시적 정보 무시

사용자가 직접 말하지 않았지만 이미지에서 추론 가능한 정보를 활용하지 못함

시각-텍스트 연결 부재

이미지 기반 증거와 텍스트 기반 증거를 통합하는 메커니즘 부족

핵심 아이디어

텍스트 환원 없는 순수 시각 기억

이미지를 캡션으로 압축하지 않고, 이미지 자체를 기억 단위로 유지하면서
명시적·암시적 증거 모두에서 개인 정보를 추출합니다.

명시적 증거 (Explicit)

사용자가 직접 언급한 정보와 관련된 시각 데이터
예: "제 방은 파란 벽입니다" + 방 사진

암시적 증거 (Implicit)

이미지에서 직접 관찰 가능하지만 언급되지 않은 정보
예: 사진에서 관찰된 실내 스타일, 식물 등

방법론

전체 파이프라인

1
입력 수집
사용자 대화 + 이미지 스트림
2
증거 분류
명시적 vs 암시적 증거 구분
3
시각 기억 저장
이미지 원본 + 임베딩 유지
4
검색 & 추론
질문에 맞는 시각 증거 검색

핵심 원칙: 이미지를 텍스트 캡션으로 축소하지 않고, 시각 임베딩 공간에서 직접 검색 및 추론 수행

방법론

명시적 증거 추출

사용자의 텍스트 발화에서 개인 정보를 식별하고,
관련된 이미지와 명시적으로 연결(grounding)합니다.

1

발화 분석

사용자 텍스트에서 개인 정보 엔티티 추출

2

이미지-텍스트 정렬

추출된 정보와 가장 관련성 높은 이미지 매칭

3

기억 메타데이터 구성

이미지 임베딩 + 텍스트 증거 + 관계 태그 저장

방법론

암시적 증거 추출

사용자가 언급하지 않았어도, 이미지에서 직접 관찰 가능한 개인 특성을 추출합니다.

환경 특성

실내 스타일, 색상 구성, 가구 배치

생활 패턴

식습관, 취미 용품, 활동 흔적

미적 선호

패션, 디자인, 색채 감각

비전-언어 모델을 활용하여 이미지에서 개인화 관련 속성을 자동으로 태깅

방법론

시각 기억 아키텍처

이미지를 캡션으로 변환하지 않고 원본 시각 표현을 유지하는 기억 구조

기존 방식

  • 이미지 → 캡션 텍스트 변환
  • 시각 세부 정보 손실
  • 텍스트 벡터 공간에서만 검색

제안 방식

  • 이미지 임베딩 직접 저장
  • 시각 세부 정보 보존
  • 멀티모달 공간에서 통합 검색
벤치마크

새로운 평가 벤치마크 설계

기존 벤치마크의 한계("텍스트만으로도 풀 수 있는 문제")를 극복하기 위해,
반드시 시각 정보가 필요한 질문으로 구성

실험 결과

성능 비교

시각 정보가 필수적인 질문(V-Only)에서 기존 방법과의 격차가 가장 크게 나타남

분석

주요 발견

캡션의 정보 손실

이미지를 캡션으로 변환할 때 평균 40% 이상의 시각 세부 정보가 손실되며, 이는 특히 암시적 질문에서 치명적

증거 통합의 효과

명시적 증거와 암시적 증거를 결합할 때 단일 증거만 사용할 때보다 평균 15~20% 정확도 향상

검색 정확도

멀티모달 임베딩 공간에서의 검색이 텍스트 공간 검색보다 관련 이미지 recall에서 대폭 우수

스케일러빌리티

기억 항목이 증가해도 시각 임베딩 기반 인덱싱으로 효율적인 검색 성능 유지

시사점

핵심 시사점 & 한계

시사점

  • 개인화 AI의 기억은 텍스트뿐 아니라 시각 정보를 직접 다뤄야 함
  • 사용자가 말하지 않은 정보(암시적 증거)가 개인화 품질에 핵심적 역할
  • 벤치마크 설계 시 "텍스트만으로 풀 수 있는지"를 반드시 검증해야 함

한계 및 향후 과제

  • 동영상 등 시간적 정보가 포함된 기억은 아직 다루지 않음
  • 프라이버시 보호 메커니즘과의 통합 필요
  • 실제 에이전트 시스템에서의 장기 운영 평가 부족
