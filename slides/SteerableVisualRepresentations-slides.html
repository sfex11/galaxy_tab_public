01 · 문제 정의

기존 ViT의 한계

DINOv2, MAE 같은 사전학습 모델은 가장 두드러진 시각적 단서에만 집중합니다.

🔒

쿼리 무관 인코딩

입력 이미지와 무관하게 항상 같은 방식으로 특징을 추출 — 덜 두드러진 개념을 무시

🔗

후기 융합의 한계

CLIP 같은 모델은 시각과 언어를 독립적으로 인코딩한 후 결합 — 진정한 조종 불가능

💰

대규모 MLLM의 비효율

InternVL3, Qwen3-VL 등은 조종 가능하나 수십억 파라미터 필요

🎯

품질 vs 조종성 트레이드오프

기존에는 표현 품질과 조종 가능성을 동시에 달성한 방법이 없었음

02 · 핵심 아이디어

조기 융합 (Early Fusion)

텍스트를 시각 인코더 내부 레이어에 직접 주입하여 표현 자체를 조종합니다.

기존 방식

후기 융합 (Late Fusion)

이미지 → 시각 인코더 → 특징
텍스트 → 텍스트 인코더 → 특징
→ 이후에 결합

조종 불가능

VS
SteerViT

조기 융합 (Early Fusion)

이미지 + 텍스트 → 시각 인코더 내부에서
교차 어텐션으로 텍스트 신호 주입
→ 조종된 특징 출력

완전한 조종 가능

핵심 통찰

비전을 언어에 맞추는 것이 아닌, 언어로 비전을 조종하는 새로운 패러다임

09 · 주요 발견

5가지 핵심 발견

1

텍스트로 시각 특징 조종

SteerViT은 쿼리된 개념 방향으로 시각 특징을 조종하며, 기존 인코더는 두드러진 시각적 단서에 붕괴합니다.

2

파레토 최적 달성

표현 품질을 희생하지 않으면서 텍스트로 조종 가능한 최초의 시각 표현 가족을 구성합니다.

3

텍스트 상세도 = 특징 세밀도

텍스트 조건의 상세 수준이 조종 가능한 시각 표현의 세밀도를 직접 제어합니다.

4

임베딩 공간 재구성

텍스트를 사용하여 의미적 추상화 수준과 클러스터링 기준을 모두 제어할 수 있습니다.

5

OOD 도메인 전이

자연어만으로 태스크별 학습 없이 분포 외 도메인으로의 전이가 가능합니다.
