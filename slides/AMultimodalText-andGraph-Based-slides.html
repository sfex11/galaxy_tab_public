문제 제기

기존 접근법의 두 가지 한계

폐쇄 영역 (Closed-Domain)
미리 정의된 이벤트 스키마에 의존 → 보지 못한 이벤트 유형에 일반화 불가
VS
개방 영역 (Open-Domain)
제한 없는 이벤트 유형 처리 가능하지만 휴리스틱/규칙 기반에 머물러 있음
LLM의 문서 수준 이해 부족
Lost-in-the-middle 현상과 Attention 희석으로 인해 문서 수준의 맥락, 구조, 의미 관계를 제대로 포착하지 못함
최근 LLM 미세조정 연구는 단순 파인튜닝만 수행하며, 문서 수준 추론을 명시적으로 모델링하지 않음

모듈 1

텍스트 인코더

T5의 인코더 부분을 사용하여 문서의 토큰 시퀀스를 문맥화된 토큰 수준 임베딩으로 변환합니다.

Htext = TextEncoder(D) ∈ ℝn×d
입력
문서 D의 n개 토큰 {t₁, t₂, ..., tₙ}
출력
토큰 수준 은닉 표현 Htext (n × d 차원)
T5 선택 이유: 인코더-디코더 아키텍처로 다중 모달(텍스트+그래프) 표현 통합과 조건부 생성(이벤트 5Ws)에 적합. Flan-T5 등 다른 인코더-디코더 LLM도 대체 가능.
모듈 2

그래프 인코더

문서의 토큰으로부터 문서 수준 토큰 그래프를 구성하고, GNN으로 노드 수준 임베딩을 생성합니다.

그래프 구성
정점 V: 문서의 각 토큰
간선 E: 시퀀스 인접 + 의존 구문 + 동시 출현 기반 연결
GNN 연산
그래프 구조를 통해 토큰 간 문서 수준 구조적, 의미적 관계를 명시적으로 모델링
Hgraph = GraphEncoder(G) ∈ ℝn×d
핵심 차별점: 외부 엔티티 식별 없이 문서 텍스트로부터 직접 그래프를 구성하며, 엔티티 → 그래프 → 추출의 다단계 파이프라인을 피하는 엔드투엔드 설계
모듈 3

주의 기반 게이티드 다중모달 융합

토큰 수준 LLM 임베딩과 노드 수준 그래프 임베딩을 통합하여 풍부한 다중모달 표현을 생성합니다.

주의 메커니즘
그래프 임베딩이 텍스트 임베딩의 어느 부분에 집중할지 계산
게이트
각 토큰마다 텍스트 vs 그래프 정보의 비율을 적응적으로 조절
융합
게이트 가중치로 가중합하여 최종 통합 임베딩 생성
Hfused = g ⊙ Htext + (1 - g) ⊙ (Attention(Hgraph, Htext))
이 메커니즘은 RAG 및 QA 연구에서 그래프 표현이 LLM 추론을 향상시킨다는 발견에서 동기부여를 받았습니다.
모듈 4 & 훈련

디코더와 훈련 전략

텍스트 디코더
T5 디코더가 융합된 다중모달 임베딩을 조건으로 이벤트의 5Ws를 엔드투엔드 생성 방식으로 출력합니다.
5Wse = Decoder(Hfused)
훈련 전략
교사 강제(Teacher Forcing)를 사용한 표준 시퀀스-투-시퀀스 학습. 손실 함수는 생성된 5Ws와 정답 간의 차이를 최소화합니다.
End-to-End Teacher Forcing Seq2Seq
엔드투엔드 설계의 의미: 엔티티 식별 → 그래프 구축 → 이벤트 추출의 다단계 파이프라인과 달리, 모든 과정이 단일 모델 내에서 jointly 최적화됩니다.

확장성

폐쇄 영역 이벤트 추출로의 일반화

MODEE는 개방 영역 설계가 주목적이지만, 대규모 폐쇄 영역 문서 수준 이벤트 추출 데이터셋에서도 기존 알고리즘을 능가합니다.

개방 영역
5Ws 프레임워크 기반, 사전 정의 스키마 없이 제한 없는 이벤트 추출
폐쇄 영역 적용
동일 아키텍처를 미세조정만으로 폐쇄 영역 데이터셋에 적용 가능
시사점
텍스트-그래프 다중모달 학습은 개방 영역뿐 아니라 폐쇄 영역에서도 효과적이며, MODEE의 설계 철학이 이벤트 추출 전반에 걸쳐 보편적으로 적용 가능함을 시사합니다.
요약

주요 기여

  1. 1
    GNN + LLM 하이브리드 아키텍처 — 개방 영역 문서 수준 이벤트 추출을 위한 최초의 다중모달 접근법
  2. 2
    주의 기반 게이티드 다중모달 융합 — 토큰 수준 LLM 임베딩과 노드 수준 그래프 임베딩의 효과적 통합 메커니즘
  3. 3
    문서 수준 추론 명시적 모델링 — Lost-in-the-middle 및 Attention 희석 문제를 그래프 구조로 완화
  4. 4
    확장 가능한 다중모달 패러다임 — 단일 주석원에서 텍스트-그래프 모달을 파생하여 병렬 주석 불필요
  5. 5
    폐쇄 영역 일반화 — 개방 영역에서뿐 아니라 폐쇄 영역 데이터셋에서도 기존 방법 능가

결론

MODEE는 그래프 기반 학습과 LLM 기반 텍스트 표현을 결합하여 개방 영역 문서 수준 이벤트 추출에서 최고 수준의 성능을 달성했습니다.

단일 주석원에서 파생된 텍스트-그래프 다중모달 학습은 비용 효율적이면서도 강력한 이벤트 추출을 가능하게 합니다.
