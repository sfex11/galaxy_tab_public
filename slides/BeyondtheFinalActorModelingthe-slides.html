4가지 텍스트 분류 체계

창작자(Creator)와 편집자(Editor)의 조합

✍
인간 작성 텍스트
창작: 인간 | 편집: 인간
순수하게 인간이 작성한 텍스트. LLM의 개입 없음.
🤖
LLM 생성 텍스트
창작: LLM | 편집: LLM
순수하게 LLM이 생성한 텍스트. 인간의 개입 없음.
📝
LLM 다듬은 인간 텍스트
창작: 인간 | 편집: LLM
인간이 초안을 작성하고, LLM이 문체를 다듬음.
🕵
인간화된 LLM 텍스트
창작: LLM | 편집: 인간
LLM이 초안을 생성하고, 인간이 탐지 회피를 위해 수정.
→ 창작자가 논리적 흐름을 확립하고, 편집자가 언어적 표현을 제어한다

핵심 발견: 수사 구조 분석

Rhetorical Structure Theory (RST)를 통한 창작자 서명 탐지

✍ 인간 창작자 특성

Attribution (출처 인용) 관계 과다 표현
Background (배경 설명) 관계 과다 표현

→ 출처를 인용하고, 논거를 근거에 기반하여 전개

🤖 LLM 창작자 특성

Elaboration (부연) 관계 과다 표현
Evaluation (평가), Enablement (활성화) 과다

→ 표면적 설명 위주, 깊은 근거 부족

4-클래스 분류
1. 이중 흔적 추출
End-to-end RST 파서로 텍스트를 이진 구성 트리로 변환.
EDU 시퀀스 (편집자 스타일) + 관계 노드 (창작자 논리) 추출.
18가지 대분류 수사 관계 레이블 사용.
2. 논리 인식 그래프 초기화
하위 스팬 풀링: PLM 임베딩의 평균 풀링으로 EDU 노드 표현.
정보 병목 투영: d_PLM → d_feat=128 차원 축소.
학습 가능한 노드 타입 임베딩 추가.
3. 수사 기반 메시지 전달
관계별 그래프 합성곱 네트워크 (RGCN)
관계별 변환 행렬 W_r, 기저 분해(B=10)로 파라미터 절감.
L=2 레이어, 은닉 차원 512.
4. 그래프 판독 & 분류
루트 노드에서 전역 표현 z_G 추출.
2층 MLP + Softmax 분류기.
손실 함수: 지도 대조 손실 + 교차 엔트로피 (τ=0.07).

핵심 수식

RGCN 메시지 집계 (수사 기반 전파)

hi(l+1) = σ( Σr∈R Σj∈Nr(i) [ hj(l) Wr(l) / Zi,r ] + hi(l) W0(l) )

기저 분해 (파라미터 효율화)

Wr(l) = Σk=1B αrk(l) Vk(l)   (B = 10 공유 기저 행렬)

최종 손실 함수

L = Lcon (지도 대조 손실, τ=0.07) + Lce (교차 엔트로피)

주요 기여 및 한계

✓ 핵심 기여

  • 과제 정의: 규제 정책에 부합하는 명시적 4-클래스 분류 과제 설정
  • 창작자-편집자 프레임워크: 논리적 기반과 문체 정제를 분리하는 개념적 모델 제안
  • RST 최초 적용: LLM 탐지 분야에 수사 구조 이론을 최초로 적용
  • 하이브리드 아키텍처: 의미 내용(하위 풀링)과 관계 토폴로지(RGCN)를 결합
  • SOTA 달성: 12개 베이스라인 대비 평균 TPR@1%에서 +3.36% 절대 향상

결론

RACE는 수사 구조 이론(RST)을 활용하여 창작자의 논리 흐름과 편집자의 문체 특성을 분리함으로써, 세밀한 LLM 생성 텍스트 탐지의 새로운 패러다임을 제시합니다.
