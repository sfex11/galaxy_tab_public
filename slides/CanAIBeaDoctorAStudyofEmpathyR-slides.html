01

왜 이 연구가 필요한가?

15만 명 이상의 임상의가 이미 AI 기반 시스템으로 환자 메시지를 작성하고 있습니다.
그러나 의료 소통의 질적 차원은 여전히 충분히 검증되지 않았습니다.

기존 연구의 한계
  • •정확성에만 집중, 공감·가독성 간과
  • •소셜 미디어·합성 데이터 의존
  • •단일 차원 평가로 다각도 분석 부재
  • •소규모 평가, 재현성 부족
본 연구의 접근
  • •의미 충실도 + 가독성 + 감정 공명 통합 평가
  • •실제 임상의 작성 권위 의료 데이터 활용
  • •전문가 + 환자 이해관계자 이중 평가
  • •다중 LLM + 프롬프트 전략 비교
핵심 전제: 환자 대면 의료 AI에서 오정렬은 불명확한 소통, 부적절한 안심, 안전하지 않은 권고로 이어져 환자 신뢰와 임상 의사결정에 직접적 영향을 미칩니다.
02

5개 연구 질문

RQ1

공감 (Empathy)

LLM은 임상 소통에서 의사와 비교하여 공감과 감정 인지를 어떻게 표현하는가?

RQ2

가독성 (Readability)

LLM이 생성한 응답은 의사 작성 답변과 비교하여 언어적 가독성에서 어떤 차이가 있는가?

RQ3

프롬프트 기반 정렬

공감 지향 프롬프트가 감정적 어조와 가독성을 개선하면서 의미 충실도를 유지할 수 있는가?

RQ4

인간-AI 협업

LLM이 의사 작성 응답을 협업적 재작성을 통해 명확성과 감정적 적절성을 향상시킬 수 있는가?

RQ5

전문가-환자 가치 정렬

다양한 LLM 구성이 전문가와 환자가 표현하는 서로 다른 선호를 얼마나 충족하는가?

3가지 프롬프트 전략

각 전략은 LLM의 행동을 체계적으로 조정하여 임상 소통 품질에 미치는 영향을 독립적으로 분석합니다.

Base Prompt
형식적 임상 답변
의료 질문에 대해 전문적이고 형식적인 임상 응답 생성 지시. SPIKES 프로토콜 기반 구조화.
Empathy Prompt
명확성 중심 공감 프롬프트
환자의 감정 상태를 인지하고, 공감적 언어를 사용하며, 일반 독자가 이해할 수 있는 수준으로 설명할 것을 추가 지시.
Rephrase Prompt
협업적 인간-LLM 편집
의사가 작성한 원본 응답을 바탕으로, 의미를 보존하면서 더 명확하고 감정적으로 적절하게 재작성할 것을 지시.

핵심 발견
+5.9~7.9%p 의사 대비 극단적 부정 감정 증가
• LLM은 환자의 고통을 인지하나, 감정적 극단성으로 표현
• 공감 프롬프트 적용 시 극단적 부정 감정 감소 효과 확인
• 감정 분포의 양극화 현상: 중립 감정 감소, 극단 감정 증가
경고: 환자가 느끼는 "공감"과 임상적 기준의 "적절한 감정적 참여"는 다릅니다. LLM의 극단적 부정 표현은 실제로 불안을 가중시킬 수 있습니다.

RQ5 전문가 vs 환자 평가

두 이해관계자 그룹은 서로 다른 기준으로 LLM 응답을 평가합니다.

전문가 평가
● 인식론적 기준 우선 — 정확성, 완전성, 임상적 적절성
● 어떤 모델도 의사를 능가하지 못함
● 재작성 변형에 대해 보수적 평가
환자 평가
● 경험적 기준 우선 — 명확성, 감정적 어조, 이해 용이성
● 재작성 변형
