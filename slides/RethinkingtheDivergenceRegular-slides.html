LLM RL은 왜 Off-policy인가?

학습-추론 불일치

롤아웃은 추론 엔진에서 생성되지만, 정책 업데이트는 학습 엔진에서 수행됨. 두 엔진 간의 미세한 수치적 차이가 존재하여, 데이터 생성 정책(μ)과 업데이트 대상 정책(π)이 다름.

정책 노후화 (Staleness)

대규모 배치로 수집된 궤적을 여러 미니배치로 분할하여 다중 그래디언트 스텝에 사용하는 것이 일반적. 업데이트가 누적될수록 π와 μ의 괴리가 커짐.

Off-policy 설정에서는 신뢰 영역(Trust Region) 제어이 안정적 최적화를 위해 필수적.

이 방법들은 모두 중요도 비율(Importance Ratio) rₜ = π/μ로 신뢰 영역을 정의.

문제: LLM의 대규모 롱테일 어휘에서 비율은 분포 변화의 좋은 대리지표가 아님.

Ratio 기반 접근의 근본적 한계

과제약 저확률 토큰

확률이 0.001에서 0.002로 증가해도 비율은 2배로 급증. 실제 확률 질량 변화는 미미하지만 신뢰 영역을 위반하는 것으로 간주됨.

과소제약 고확률 토큰

확률이 0.5에서 0.6으로 변해도 비율 변화는 20%에 불과. 실제로는 상당한 질량 이동이 발생하지만 제약이 약함.

고정된 비율 윈도우는 롱테일 어휘에서 저확률 토큰을 과제약하고 고확률 토큰을 과소제약하는 구조적 문제를 가짐.

DPPO: 발산 기반 마스크의 등장

DPPO의 개선점

  • 비율 기반 → 절대 확률 변화 기반
  • Binary-TV 기하학: |π − μ| ≤ δ
  • 롱테일 어휘에서 더 정확한 분포 변화 측정

DPPO의 한계

  • 이진 마스크: 경계 밖 그래디언트를 0으로 설정
  • 교정 신호 부재: 정책을 경계로 되돌릴 수 없음
  • 임계값 근처에서 급격한 변화 발생

DRPO: 발산 정규화 정책 최적화

Divergence Regularized Policy Optimization — 하드 마스크를 부드러운 advantage-가중 이차 정규화기로 대체

1
Binary-TV 제약을 토큰 적응적 비율 바운드로 재구성

|π(yₜ|sₜ) − μ(yₜ|sₜ)| ≤ δ를 토큰별 비율 제약으로 변환

2
SPO와 동일한 구조 적용

비율 바운드에 대해 advantage-가중 ℓ₂² 정규화기를 도입

3
DPPO의 신뢰 영역 기하학 보존 + SPO의 부드러움 결합

고정 비율 제약 → 절대 확률 변화 제약으로 기하학 전환

그래디언트 가중 메커니즘

각 토큰의 정책 그래디언트 기여도에 연속적 가중치를 곱함:

행동 정책에서 멀어질 때

가중치가 신뢰 영역 경계에서 0으로 감소. 경계를 넘어서면 교정 신호를 제공하여 정책을 경계 쪽으로 되돌림.

행동 정책으로 돌아갈 때

가중치가 증폭됨. 정책이 원래 분포로 회복되는 방향의 업데이트를 적극적으로 장려.

핵심: 가중치가 중요도 비율이 아닌 절대 확률 변화에 의존하므로, SPO처럼 저확률 테일에서 무한대로 발산하지 않고 항상 유계(bounded).

SPO vs DRPO: 그래디언트 가중치 비교

SPO의 문제

비율 기반 정규화기 사용:

μ(yₜ|sₜ) → 0일 때 가중치가 무한대로 발산. 롱테일 어휘에서 불안정한 업데이트 유발.

DRPO의 해결

절대 확률 변화 기반 정규화기 사용:

모든 토큰에 대해 가중치가 항상 유계. 저확률 테일에서도 안정적.

동일한 부드러운 정규화 구조를 사용하되, 신뢰 영역의 기하학을 비율에서 발산으로 전환하는 것이 핵심 차이.

효과적 정규화기의 3가지 설계 기준

Ablation 연구에서 도출된 실용적 원칙

1

안정적 경계

분포 변화와 정렬된 신뢰 영역 경계를 유도할 것. 비율 기반 경계는 롱테일에서 불안정.

2

유계 가중치

롱테일 어휘에서 토큰별 그래디언트 가중치가 유계여야 할 것. 무한 발산은 훈련 불안정의 원인.

3

부드러운 교정

정책이 신뢰 영역 밖으로 이동할 때 부드러운 교정 신호를 제공할 것. 이진 마스크의 급격한 차단 회피.

DRPO는 이 세 가지 기준을 모두 만족하는 단일 Binary-TV 정렬 정규화기.

주요 Ablation 발견

표준 KL/TV 페널티의 실패

표준 KL이나 TV 페널티의 그래디언트는 비율 기반 기하학을 재도입함. 페널티 항의 형태보다 그래디언트의 형태가 중요.

Advantage 가중의 역할

토큰별 페널티를 절대 advantage로 가중해야 신뢰 영역 경계가 보상 스케일과 독립적으로 유지됨.

이 발견들은 정규화기 설계에서 그래디언트 중심 관점(gradient-centered view)의 중요성을 강조함.

실험 결과

모델 스케일, 아키텍처, 정밀도 설정을 아우르는 광범위한 실험

방법 신뢰 영역 기하학 그래디언트 가중치 경계 밖 동작
PPO 고정 비율 경계 밖 0 그래디언트 차단
SPO 고정 비율 무계 (발산 가능) 선형 감쇠
DPPO 절대 확률 변화 이진 (0 또는 1) 그래디언트 차단
DRPO 절대 확률 변화 유계 & 연속 교정 신호 제공

DRPO는 여러 설정에서 훈련 안정성과 효율성을 동시에 개선.

핵심 요약

문제 진단

  • PPO/GRPO의 비율 기반 클리핑은 롱테일 어휘에서 부적절
  • DPPO의 이진 마스크는 교정 신호가 없어 아쉬움
  • SPO의 가중치는 저확률 토큰에서 무한 발산

DRPO의 기여

  • DPPO의 발산 기반 기하학 + SPO의 부드러움 결합
  • 유계·연속 그래디언트 가중치로 안정적 훈련
  • 경계 밖 교정 신호로 빠른 복원
  • 그래디언트 중심 정규화기 설계 관점 제시
