BAMI: Training-Free Bias Mitigation
in GUI Grounding

추가 학습 없이 GUI 그라운딩의 근본적 편향을 진단하고 완화하는 추론 기반 방법

Borui Zhang, Bo Zhang, Bo Wang, Wenzhao Zheng 등 · Tsinghua University · Lenovo Research

GUI Grounding이란?

자연어 지시와 스크린샷을 입력받아, 목표 UI 요소의 좌표를 정확히 찾아내는 작업

핵심 역할

GUI 에이전트가 클릭, 드래그, 타이핑 등 원자적 액션을 수행하기 위한 필수 전처리 단계

현재 한계

ScreenSpot-Pro 벤치마크에서 대부분 모델의 정확도가 50% 미만으로, 복잡한 시나리오에서 성능이 크게 저하됨

"GUI 시나리오는 고해상도·밀집 요소라는 독특한 도전 과제를 제시하며, 아이콘·텍스트·맥락 단서가 결합된 의미 구조를 갖는다."

오류 원인 진단: MPD 방법

Masked Prediction Distribution — 스크린샷을 무작위로 가려서 반복 예측 후 핫스팟 빈도를 집계하는 기법

1
마스킹
스크린샷의 일부를 무작위로 가림
2
반복 예측
가려진 이미지에서 좌표를 반복 추론
3
분포 집계
예측 좌표의 빈도를 히트맵으로 시각화
4
오류 귀인
모델의 주의 분포를 기반으로 오류 유형 분류
14%

지식 부족
(Knowledge Deficiency)

74%

귀납적 편향
(Inductive Bias)

두 가지 귀납적 편향

74%의 오류가 모델이 충분한 지식을 가지고 있음에도 발생하는 편향에서 기인

Precision Bias

정밀도 편향

고해상도 이미지에서 모델이 목표 근처에 예측하나 정확한 좌표를 찾지 못하는 현상

  • 고해상도 환경에서 두드러짐
  • 작은 UI 요소에서 심화됨
  • 탐색 공간이 너무 큰 것이 원인
Ambiguity Bias

모호성 편향

유사한 UI 요소 간에 잘못된 선택을 하는 현상 — 언어 공간의 편집 거리와 공간적 거리 간 불일치

  • 복잡한 인터페이스에서 발생
  • 유사 요소 간 혼동이 핵심
  • 선택 규칙의 부재가 원인

정밀도 편향 완화: Coarse-to-Fine Focus

일회성 지역화를 계층적 다단계 탐색으로 변환하여 탐색 공간을 점진적으로 축소

1
전체 이미지 입력
원본 스크린샷에서 대략적 좌표 예측
2
후보 영역 크롭
예측 좌표 중심으로 일정 비율 영역 추출
3
세밀 재예측
크롭된 영역에서 정밀 좌표 재추론
N
반복 정제
설정된 반복 횟수까지 점진적 좌표 정제

각 단계에서 이전 라운드의 후보 영역을 입력으로 사용하여 탐색 공간을 줄이고 예측 좌표의 해상도를 향상시킨다.

모호성 편향 완화: Candidate Selection

MLLM의 잘못된 선택 기호를 외부 보정 규칙을 통해 수정

문제 인식

MLLM은 언어 공간의 편집 거리(Edit Distance)를 기준으로 후보를 선택하나, 이는 공간적 좌표 거리와 일치하지 않음

해결 방법

지역화 작업에 특화된 선택 규칙을 정의하고, 이를 프롬프트로 주입하여 모델의 선택 기호를 교정

다중 후보 생성 → 규칙 기반 필터링 → 최종 선택

프롬프트에 정의된 선택 기준에 따라 가장 적합한 후보를 자동 선별

BAMI: 전체 프레임워크

Bias-Aware Manipulation Inference — 편향 인지 조작 추론

조작 1

Coarse-to-Fine Focus

  • 계층적 크롭 기반 점진적 정제
  • 탐색 공간 축소로 정밀도 향상
  • 크롭 비율·반복 횟수 조절 가능
조작 2

Candidate Selection

  • 프롬프트 규칙 기반 선택 보정
  • 편집 거리 vs 공간 거리 불일치 해결
  • 외부 모델로 로컬 보정 수행

핵심 아이디어: 일회성 지역화를 편향 인지 사전 정의 조작을 통한 재귀적 다단계 구조화 추론으로 변환한다.

Training-Free: 추가 모델 학습 없이 기존 오픈소스 백본에 직접 적용 가능한 추론 시간 방법론

주요 결과: ScreenSpot-Pro

모든 백본 모델에서 일관된 정확도 향상 달성

모델 원본 정확도 BAMI 적용 향상폭
OS-Atlas-7B 42.7% 46.8% +4.1%
UI-TARS-7B 49.5% 54.2% +4.7%
TianXi-Action-7B 51.9% 57.8% +5.9%

TianXi-Action-7B에 BAMI를 적용하여 ScreenSpot-Pro에서 51.9% → 57.8% 달성 — 최대 +5.9%p 향상

절제 연구: 정밀도 편향 제거

크롭 비율·반복 횟수의 영향

크롭 비율과 반복 횟수를 다양하게 조정한 결과, 다양한 파라미터 설정에서 안정적인 향상을 확인

  • 너무 작은 크롭은 정보 손실 초래
  • 너무 큰 크롭은 정제 효과 감소
  • 적절한 범위 내에서 강건한 성능

목표 유형별 효과

작은 UI 요소(아이콘, 작은 버튼)에서 정밀도 편향 완화 효과가 특히 두드러짐

  • 아이콘·작은 버튼: 큰 향상
  • 텍스트 필드: 안정적 향상
  • 큰 영역 요소: 제한적 효과

절제 연구: 모호성 편향 제거

프롬프트 설계의 영향

선택 규칙의 구체성과 표현 방식에 따라 후보 보정 정도가 달라지며, 명확한 규칙 정의가 핵심

보정 모델 선택

로컬 보정을 수행하는 외부 모델의 선택이 모호성 해소 품질에 영향을 미침

언어 공간에서의 Chain-of-Thought 확장은 GUI 시나리오에서 오히려 성능을 저해할 수 있으며, 이미지 공간에서의 구조화된 추론이 효과적임이 확인되었다.

핵심 시사점

진단

MPD 귀인의 가치

단순한 성능 평가를 넘어, 오류의 근본 원인을 시각적으로 진단하는 프레임워크 제공

설계

추론 시간 구조화의 효과

추론 경로를 확장하고 구조화하는 것이 기존 모델의 잠재력을 해방하는 비용 효율적 수단

범용성

백본 무관 적용

특정 아키텍처에 종속되지 않고 다양한 오픈소스 모델에 직접 적용 가능

실용성

Training-Free

추가 학습 없이 배포 가능하여 실제 GUI 에이전트 시스템에 즉시 통합 가능
