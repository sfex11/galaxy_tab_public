CVPR 2026 arXiv: 2603.24558

LensWalk

에이전틱 비디오 이해

비디오에서 "어떻게 볼 것인지"를 스스로 계획하는
능동적 시각 관찰 프레임워크

Keliang Li, Yansong Li, Hongze Shen, Mengdi Liu, Hong Chang, Shiguang Shan
파인튜닝 불필요 5%+ 성능 향상 플러그 앤 플레이
01 / 문제 정의

기존 비디오 이해의 한계

기존 방식

  • ✗ 정적 전처리 — 미리 선택된 프레임만 분석
  • ✗ 추론-인식 단절 — 추론 과정이 관찰을 제어하지 못함
  • ✗ 일회성 관찰 — 필요한 부분을 다시 볼 수 없음
VS

LensWalk

  • ✓ 능동적 관찰 — 에이전트가 관찰 범위를 직접 결정
  • ✓ 추론-계획-관찰 루프 — 이해가 관찰을 이끔
  • ✓ 반복적 탐색 — 증거가 부족하면 재관찰

핵심 통찰: 비디오 이해를 "한 번 보기"가 아닌 "계획적 탐색 과정"으로 재정의

Reasoner (Mr)

쿼리, 비디오 메타데이터, 축적된 히스토리를 통합 분석하여 관찰 행동을 결정

at := (ot, qt, It, ρt)
👁

Observer (Mo)

Vision-Language Model이 지정된 시간 범위와 샘플링 밀도로 시각 정보를 추출

et = Observe(ot, qt, It)
📝

메모리 시스템

Timestamp Anchors — 시간 정보 그라운딩  |  Subject Memory Table — 개체 속성 추적

03 / 관찰 도구

다중 해상도 관찰 도구 모음

🔍

Scan Search

광범위 탐색

지정된 시간 구간을 슬라이스로 분할하여 희소 샘플링으로 빠르게 스캔. 각 슬라이스별 독립 요약 생성. 병렬 처리 가능.

🎯

Segment Focus

정밀 집중

단일 연속 구간에 대해 고밀도 샘플링으로 세밀한 디테일 추출. 가설 검증, 속성 읽기, 모호성 해소에 이상적.

✔

Stitched Verify

교차 검증

비연속 다중 세그먼트의 증거를 통합. 세그먼트별 비대칭 샘플링 지원— 액션 장면은 높은 프레임 레이트, 정적 장면은 낮은 레이트로 효율적 관찰.

04 / 작동 예시

실제 추론 트레이스

에이전트가 비디오 속 동물을 식별하는 과정

1
Scan Search — 전체 비디오를 10개 슬라이스로 분할 스캔

"01:10~01:45 구간에서 작은 동물이 물가에서 관찰됨"

2
Segment Focus — 01:10~01:45 구간 고밀도 샘플링

초기 판단: "물밭쥐(water vole)로 추정"

3
Reasoner 반성 — Subject Memory에서 모순 발견

⚠ 이전 관찰에서 "두더지(mole)" 기록과 충돌

4
Stitched Verify — 두 구간을 비교 통합 관찰

비연속 세그먼트를 단일 배치로 합쳐 VLM에 제출

5
정답 도출 — 모순 해결, 올바른 종 식별

✓ 교차 검증을 통해 정확한 답변 생성

08 / 전략 분석

에이전트의 창발적 행동

LensWalk 에이전트가 자율적으로 발전시킨 6가지 관찰 전략

Direct Inquiry

간단한 질문은 최소 턴으로 해결

Progressive Zoom-in

넓은 범위에서 점차 좁혀감

Scope Partitioning

시간 범위를 체계적으로 분할

Strategic Reflection

모순 발견 시 전략적 재고

Integrative Verify

다중 증거를 통합하여 확인

Static Repetition

거의 사용하지 않음 (비효율적)

적응적 자원 배분

어려운 질문 → 더 많은 프레임과 관찰 턴
쉬운 질문 → 최소 비용으로 효율적 해결

정적 반복 회피

기존 에이전트의 단순 반복 대신
반성적 패턴(Strategic Reflection)을 자율 채택

10 / 한계 및 의의

한계점과 시사점

⚠ 한계점

오픈소스 추론기 한계

Observer 품질 의존성
