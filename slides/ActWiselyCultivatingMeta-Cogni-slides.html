01 — 문제 정의

맹목적 도구 호출
메타인지 결핍 문제

현재 멀티모달 에이전트는 심각한 메타인지 결핍을 겪고 있습니다

🔄

맹목적 도구 호출

충분한 시각적 맥락이 있는데도 반사적으로 외부 도구를 실행 — 도구 호출률 98%

⏱

지연 시간 병목

불필요한 도구 호출로 인한 과도한 지연 발생 및 추론 방해 노이즈 유입

핵심 딜레마: 보상 스칼라화의 한계

정확도 + 효율성을 하나의 스칼라 보상으로 결합하면 최적화 충돌 발생

Var(R_mix) = σ²_acc + α²σ²_tool + 2α·Cov(R_acc, R_tool)
→ 효율성 신호가 정확도 분산에 의해 소멸: O(α)
02 — 기존 접근법의 실패

왜 기존 강화학습은
실패하는가?

강한 도구 페널티

α 값을 높이면 필수적인 도구 사용까지 억제 → 정확도 급락

정확도: 매우 낮음
VS

약한 도구 페널티

α 값을 낮추면 효율성 신호가 분산에 완전히 흡수되어 효과 없음

도구 호출: 여전히 과다

정확도와 효율성을 하나의 보상으로 결합하는 한
이 딜레마는 수학적으로 해결 불가능

03 — 핵심 솔루션

HDPO
계층적 분리 정책 최적화

Hierarchical Decoupled Policy Optimization

핵심 아이디어

도구 효율성을 경쟁하는 스칼라 목표가 아닌 엄격한 조건부 목표로 재정의 — 두 개의 독립 채널로 분리

🎯

정확도 채널

모든 롤아웃에 대해 표준 GRPO 정규화로 정확도 평가

모든 궤적 → 정확도 어드밴티지 계산
⚡

효율성 채널

정확한 궤적 내에서만 조건부 정규화로 도구 효율 평가

오답 → 보상 0 (조기 종료 방지)
도구 보상 = 정답일 때: 1/(T_i + 1)  |  오답일 때: 0
가중치: w_acc = 1.0, w_tool = 0.15
