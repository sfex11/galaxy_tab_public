동기: 장기 추론의 한계

급격한 성능 저하

추론 단계가 길어지면 LLM의 성능이 급격히 하락함. 개별 하위 문제는 풀 수 있어도, 다단계 연쇄에서 실패

연구 환경의 부재

난이도를 체계적으로 조절할 수 있는, 검증 가능하고 확장 가능한 훈련 환경이 없었음

"수학·코딩 도메인은 검증 가능하지만 난이도 제어가 어렵고, 기존 합성 데이터는 제어 축이 불충분하다."

ScaleLogic: 개요

논리적 추론 환경으로, 난이도의 두 축을 독립적으로 제어

추론 수평선 (Depth D)

증명 트리의 깊이. 필수 추론 단계의 수를 의미하며, 장기 추론의 길이를 직접 조절

논리적 표현력

사용되는 논리 연산자의 종류와 구조적 특성. 단순 함의부터 일차 논리까지 확장

형식: 주어진 사실들과 후보 결론들 중, 논리적으로 도출 가능한 결론을 선택하는 객관식 문제

논리적 표현력의 5단계

단순한 논리에서 점진적으로 연산자를 추가하여 표현력을 확장

1
함의만
if-then 구조만 사용
2
+ 논리곱
AND 결합 추가
3
+ 부정
NOT 연산 추가
4
+ 논리합
OR 연산 추가
5
+ 한정량화
FOR ALL 추가

각 단계는 이전 단계를 완전히 포함하는 점진적 확장 구조. 표현력이 높아질수록 탐색 공간이 기하급수적으로 증가

문제 생성 과정

1

역방향 증명 트리 구성

원하는 깊이 D와 표현력 수준에 맞춰 결론에서부터 공리까지 역방향으로 증명 트리 생성

2

다지선다 인스턴스 조립

정답 트리 1개 + 공리를 하나씩 손상시킨 오답 트리 B-1개로 구성. 모든 후보에 동등한 추론 노력 요구

3

자연어 변환

형식 논리를 자연어로 변환하여 LLM이 자연스럽게 추론하도록 제시

4

Z3 검증

Z3 SMT 솔버로 논리적 건전성과 단축어 우회 가능성을 자동 검증

핵심 발견: 멱법칙 스케일링

T ∝ Dγ

훈련 컴퓨트 T는 증명 깊이 D에 대해 멱법칙을 따름

R² > 0.99

모든 표현력 수준에서 매우 높은 적합도

스케일링 지수 γ의 변화

표현력 수준 스케일링 지수 γ 해석
함의만1.04거의 선형적 스케일링
+ 논리곱1.52약간 초선형
+ 부정1.91초선형 가속
+ 논리합2.30초선형 가속 증가
+ 한정량화2.60가장 가파른 스케일링

표현력이 스케일링에 미치는 영향

단순 논리

γ = 1.04

깊이가 2배 되면 훈련이 약 2배 필요. 거의 선형적이므로 비교적 쉽게 스케일업 가능

중간 논리

γ ≈ 1.52 ~ 1.91

부정이 추가되면 탐색 공간이 급격히 확장. 초선형적 컴퓨트 증가 시작

고도 표현

γ ≈ 2.30 ~ 2.60

논리합과 한정량화가 추가되면 거의 제곱 스케일링. 동일 깊이라도 훈련 비용 폭발

"더 높은 표현력 설정은 동일한 깊이의 문제를 해결하기 위해 불균형적으로 더 많은 훈련을 요구한다."

커리큘럼 훈련의 효과

훈련 분포의 설계가 스케일링 효율에 강한 영향을 미침

정적 분포

  • 고정된 깊이 분포로 훈련
  • 깊이가 깊어질수록 불안정
  • 이른 조기 수렴 또는 발산 위험

커리큘럼 분포

  • 점진적으로 깊이를 증가시키는 분포
  • 훈련 안정성과 효율 크게 향상
  • 더 깊은 문제까지 성공적 도달

멱법칙 관계 자체는 유지되지만, 커리큘럼은 상수 인자를 개선하여 실제 필요 컴퓨트를 감소시킴

교차 알고리즘 견고성

멱법칙 스케일링은 특정 RL 알고리즘에 국한되지 않음

GRPO

멱법칙 확인

ReMax

멱법칙 확인

DRPO

멱법칙 확인

세 가지 서로 다른 RL 알고리즘에서 모두 T ∝ D^γ 관계가 성립하며, 지수 γ의 표현력 의존성 패턴도 일관됨

하류 전이 결과

합성 논리 추론 훈련이 실제 벤치마크 성능에 미치는 영향

+10.66

최고 표현력 설정에서
8개 벤치마크 평균 향상 (pp)

단조 증가

고표현력 훈련은
전이 성능이 조기 포화 없이 지속 상승

표현력별 전이 패턴

  • 고표현력 (한정량화 포함): 하향 전이 성능이 단조 증가하며, 컴퓨트 효율도 가장 높음
  • 저표현력 (함의만): 초기에는 향상되나, 일정 수준에서 조기 포화 (plateau)
  • 핵심 메시지: 얼마나 훈련하는가보다 무엇으로 훈련하는가가 하류 전이를 결정

분포 외 일반화

훈련에 사용되지 않은 깊이와 구조에 대한 일반화 능력

깊이 일반화

훈련 깊이 범위를 벗어난 더 깊은 문제에 대해서도 어느 정도 성능 유지. 단, 멱법칙에 따라 성능은 점진적 감소

구조 일반화

훈련 시 보지 못한 논리 구조 조합에 대해서는 일반화가 제한적. 표현력의 구조적 다양성이 중요함을 시사

이 결과는 합성 환경에서의 스케일링 법칙이 실제 응용으로의 전이 가능성과 한계를 동시에 보여줌

핵심 시사점

표현력이 스케일링의 본질을 결정한다

동일한 깊이라도 논리적 표현력에 따라 필요 훈련량이 선형(γ=1.04)에서 거의 제곱(γ=2.60)까지 변화

무엇을 훈련하느냐가 얼마나 훈련하느냐보다 중요하다

고표현력 데이터는 적은 컴퓨트로도 저표현력보다 더 큰 하류 전이 효과를 달성

RL 스케일링은 예측 가능하다

멱법칙은 여러 RL 알고리즘에서 성립하며, 커리큘럼으로 효율을 개선할 수 있어 자원 계획에 활용 가능

통제된 합성 환경의 가치

ScaleLogic은 난이도 축을 명확히 분리하여 RL 추론 스케일링 연구의 새로운 패러다임을 제시

결론

  • ScaleLogic: 수평선과 표현력을 독립 제어하는 합성 논리 추론 벤치마크 제안
  • RL 훈련 컴퓨트는 증명 깊이에 대해 멱법칙을 따르며 (R²>0.99), 지수 γ는 표현력에 따라 1.04→2.60으로 단조 증가
  • 멱법칙은 GRPO, ReMax, DRPO 등 다수 RL 알고리즘에서 견고하게 성립
  • 커리큘럼 기반 훈련이 스케일링 효율을 실질적으로 개선
  • 고표현력 훈련이 하류 수학·추론 벤치마크에서 최대 +10.66pp 향상, 컴퓨트 효율적 전이 달성
