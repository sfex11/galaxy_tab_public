재현성 위기와 평가의 한계

재현성의 중요성

  • 과학적 진보는 재현 가능한 발견에 의존
  • 사회·행동과학에서 원데이터 재분석 시 결과가 항상 복원되지 않음
  • 어떤 발견을 신뢰할 수 있는지 판단이 어려움

기존 평가 방식의 문제

  • 독립 연구자가 수동으로 원데이터 재분석 필요
  • 연구마다 맞춤형 분석 워크플로우 재구성 필요
  • 시간·비용 집약적 → 대규모 감사 불가능

"재현성 평가는 리소스 집약적이고 연구별로 특화되어, 실증 문헌의 체계적 감사를 비용 측면에서 어렵게 만든다."

핵심 연구 질문

"LLM이 생성한 분석이 게재된 연구의 통계적 결과와 실질적 결론을 어느 정도 재현할 수 있는가?"

데이터셋

  • 사전 정의된 통계적 주장 포함

모델 설정

  • 에이전트 워크플로우 구성
  • 연구별 5회 독립 실행
  • 문맥 정보 실험적 변환

Multi100 인간 분석가

LLM이 산출한 분석은 원연구의 실질적 결론을 인간 재분석보다 높은 비율로 복원했다. 이는 연속적 효과 크기 추정의 정밀도와 결론 도달의 견고성이 서로 다른 차원임을 시사한다.

핵심 발견

  • LLM 추정치는 원연구 효과 크기의 변동을 좇지 못함
  • 대각선 기준선 주변에 점들이 넓게 흩어짐
  • 그러나 이 상관관계 수준은 인간 재분석 간의 상관관계와 유사
  • 정성적 결론은 높게 일치하나, 연속적 효과 크기는 제한적 정렬

LLM은 인간 재분석가를 대체하는 것이 아니라, 대규모 문헌에 대한 1차 선별 도구로서 가치가 있다.

논의: 차이의 근원

LLM과 원연구 간의 차이는 두 가지 출처에서 기인할 수 있으며,
이를 완전히 분리하기는 어렵다.

개념적 모호성

  • 연구 가설은 짧은 언어적 진술에 불과
  • 분석 사양을 단일하게 결정하지 못함
  • 주장의 과소 명세화(underspecification)가 합당한 분석적 변이의 원인
  • 이는 모델 오류가 아닌 본질적 불확실성

모델 오류

  • 생성된 코드에 논리적 오류 포함 가능
  • 연구 자료의 분석적 결정을 오해할 수 있음
  • 데이터에 대한 지지되지 않는 가정 반영
  • 유효한 효과 크기 산출 자체가 불가한 경우도 존재

핵심 과제: 합당한 분석적 변이와 모델 오류를 구별하는 것은 자동 재분석의 중심적 도전 과제이다.

실질적 함의와 경고

기대 효과

  • 학술지 데이터 품질 검사의 부담 감소
  • 수동 검토가 불가능한 학술지로 검사 확대
  • 출판 전·후 주장 선별을 위한 1차 화면
  • 문헌 전체에 대한 체계적 강건성 감사 지원

주의 사항

  • 검사 통과를 위한 논문 최적화 유인 가능성
  • 오검출 비용의 비대칭성: 잘못된 비재현 플래그는 저자에게 치명적
  • 전문가 판단을 완전히 대체할 수 없음
  • 코드 생성 실패 케이스에 대한 대응 필요

LLM 기반 재분석은 전문가 판단을 대체가 아닌 보완하는 확장 가능한 1차 선별 도구로 설계되어야 한다.
