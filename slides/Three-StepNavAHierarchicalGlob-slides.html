Three-Step Nav

계층적 글로벌-로컬 플래너를 활용한

Wanrong Zheng1 · Yunhao Ge2 · Laurent Itti1

1University of Southern California · 2NVIDIA Research

문제 정의

VLN-CE: 연속 환경에서의
시각-언어 내비게이션

관측

에이전트는 현재 위치에서 360° 파노라마 RGB-D 관측(12방향 이미지)과 자연어 지시문 W를 받음

행동

사전 정의된 연결 그래프 없이, 회전·전진 등 저수준 행동을 직접 선택하여 목적지 도달

이산 그래프 기반 VLN과 달리, VLN-CE는 부분 관측 가능성, 축적된 위치 오차, 장애물 회피를 동시에 해결해야 함

기존 방법의 한계

MLLM 기반 플래너의 세 가지 실패 모드

1

궤적 이탈

현재 관측만으로 판단하면 글로벌 경로를 잃고 엉뚱한 방향으로 이동

2

조기 정지

하위 지시문을 일부 완료한 것으로 오인하여 목적지 도달 전 멈춤

3

주의 산만

관련 없는 객체에 주의가 분산되어 핵심 랜드마크를 놓침

근본 원인: 현재 시점의 RGB-D 관측 + 텍스트 행동 기록만으로는 글로벌 진행 상황을 판단할 수 없음

핵심 아이디어

Three-Step Nav: 세 가지 시선 전환

글로벌-로컬-글로벌 추론 루프를 통해 계층적 내비게이션 수행

1
Look Forward
지시문을 하위 지시문으로 분해하고 글로벌 랜드마크 추출 → 조악한 경로 스케치
2
Look Now
현재 관측을 활성 하위 목표와 정렬하여 세밀한 방향 선택
3
Look Backward
전체 궤적을 감사하여 축적된 드리프트 교정 및 하위 지시문 완료 검증

기울기 업데이트나 태스크 특화 파인튜닝 없이 기존 VLN 파이프라인에 플러그인 가능

Step 1

Look Forward

지시문을 구조화된 하위 지시문 시퀀스로 분해하여 글로벌 경로 계획 수립

입력

  • 자연어 지시문 W
  • 초기 파노라마 관측 o₀

출력

  • 순서화된 하위 지시문 {s₁, s₂, …, s_K}
  • 각 하위 지시문의 핵심 랜드마크

예: "거실을 지나 왼쪽으로 가서 침대 옆에 멈춰라" → s₁: 거실 진입 → s₂: 왼쪽으로 회전 → s₃: 침대 옆 정지

Step 2

Look Now

현재 시점의 후보 방향을 활성 하위 목표와 매칭하여 다음 웨이포인트 선택

후보 시점 생성
360° 파노라마에서 탐색 가능한 방향별 RGB 이미지와 짧은 설명 생성
MLLM 추론
활성 하위 지시문 sₖ와 가장 일치하는 후보 시점 v*ₜ 선택 (Chain-of-Thought)
저수준 행동 실행
선택된 방향으로 회전 및 전진, 새로운 관측 획득

하위 지시문 완료 조건 충족 시 → Look Backward로 전환

Step 3

Look Backward

완료된 궤적을 되돌아보며 축적된 드리프트를 감지하고 교정

궤적 시각화

지금까지 선택한 모든 시점을 시각적으로 나열하여 실행된 궤적 τ 구성

완료 검증

궤적 τ를 완료된 하위 지시문들과 대조하여 실제로 만족되었는지 감사

검증 결과에 따른 분기:

  • 완료 확인 → 다음 하위 지시문 sₖ₊₁로 이동 (Look Forward 재호출)
  • 드리프트 감지 → Backtrack 메타 스킬 발동
적응형 판별 모듈

Adaptive Judge: 4가지 메타 스킬

불확실성 상황에서 에이전트가 동적으로 자가 교정할 수 있도록 부여된 메타 능력

Stay

현재 하위 목표를 아직 달성하지 못함. 제자리에서 추가 관찰 후 재판단

Continue

현재 방향이 올바름. 다음 하위 지시문으로 진행

Backtrack

드리프트가 감지됨. 이전 결정점까지 되돌아가서 재탐색

Look Around

주변 환경 정보 부족. 360° 스캔하여 랜드마크 재확인

전체 파이프라인

글로벌-로컬-글로벌 추론 루프

F
Look Forward — 지시문 분해 → 하위 지시문 {s₁…s_K} 생성 및 랜드마크 추출
N
Look Now — 활성 sₖ와 현재 관측 매칭 → 방향 v*ₜ 선택 → 행동 실행
B
Look Backward — 궤적 τ 감사 → Adaptive Judge가 메타 스킬 판단

루프 반복 조건

Continue → 다음 sₖ₊₁로 이동 후 Look Now 재시작
Backtrack → 이전 결정점 복귀 후 Look Now 재시작
모든 하위 지시문 완료 → 최종 Look Backward 후 정지

실험 결과

RxR-CE 다국어 성능

RxR-CE는 영어·힌디어·텔루구어 3개 언어의 지시문으로 구성된 더 어려운 벤치마크

주요 발견

  • Three-Step Nav는 다국어 환경에서도 제로샷 SOTA 달성
  • 지시문 분해(Look Forward)가 언어 간 의미 구조를 보존하여 다국어 일반화에 기여
  • 궤적 수준 감사(Look Backward)가 번역 노이즈로 인한 오해석을 교정

제로샷 방법 중 유일하게 학습 기반 방법(DUET, ETPNav)과 경쟁 가능한 수준의 성능을 보임

절제 연구

각 구성요소의 기여도 분석

F

Look Forward
제거

글로벌 경로 없이 로컬만 수행 → SR 크게 하락

B

Look Backward
제거

드리프트 교정 불가 → NE显著 증가

J

Adaptive Judge
제거

불확실성 대처 불가 → 모호한 환경에서 성능 급감

결론: 세 구성요소 모두 상호 보완적이며, 누락 시 성능이 각각 유의미하게 저하됨

한계 및 영향

결론, 한계, 사회적 영향

기여 요약

  • 글로벌-로컬-글로벌 계층적 프레임워크 제안
  • 파인튜닝 없는 제로샷 SOTA 달성
  • 모듈형 설계로 기존 파이프라인에 즉시 적용 가능

한계

  • MLLM 추론 지연으로 실시간 로봇 적용에 제약
  • 복잡한 다층 구조에서의 글로벌 경로 정확도 한계
  • 시뮬레이터 환경(Habitat)에 한정된 평가

사회적 영향: 시각-언어 내비게이션 기술은 홈 로봇, 보조 내비게이션, 구조 활동 등 실제 응용에 직결됨. 내비게이션 오류가 사용자 안전에 영향을 줄 수 있으므로 실배치 전 철저한 검증 필수
