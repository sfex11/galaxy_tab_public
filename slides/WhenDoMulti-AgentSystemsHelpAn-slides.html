정보 병목(Informatino Bottleneck) 관점에서의 접근

Wendi Yu, Lianhao Zhou, Xiangjue Dong, Sai Sudarshan Barath, Declan Staunton, Byung-Jun Yoon, Xiaoning Qian, James Caverlee, Shuiwang Ji

Texas A&M University · Brookhaven National Laboratory

동기: MAS의 이점은 불일관적이다

경험적 관찰

  • 기반 모델이 강해질수록 MAS 이점 감소
  • 이점이 아키텍처가 아닌 연산량 증가에 기인
  • 설정에 따라 성능이 불일관하게 변동

근본적 질문

MAS가 SAS보다 유리한 조건을 결정하는 구조적 메커니즘은 무엇인가?

기존 연구는 경험적 비교에 머물러 근본 원인을 설명하지 못함

핵심 관찰: 정보 흐름의 차이

단일 에이전트 (SAS)

하나의 공유 컨텍스트에 모든 하위 작업과 추론 흔적이 누적

  • 정보 손실 없음
  • 컨텍스트 간섭 가능성
  • 중복/잡음 축적

다중 에이전트 (MAS)

격리된 로컬 컨텍스트가 릴레이 메시지로 연결

  • 릴레이 = 압축 인터페이스
  • 잡음 컨텍스트 억제 가능
  • 하류 관련 정보 손실 위험
두 효과 — 잡음 억제의 이점 vs 정보 손실의 비용 — 이 자연스럽게 정보 병목(Information Bottleneck) 원리로 포착됨

세 가지 통제 프로토타입

작업 분해(decomposition) 효과와 릴레이 기반 컨텍스트 격리 효과를 분리하기 위한 설계

1
SAS
단일 공유 컨텍스트에서 전체 작업 수행
2
SAS-contextflow
계획기 기반 하위작업 순서 + 공유 컨텍스트
3
MAS
동일 분해 + 압축 릴레이 메시지 연결

  SAS-contextflow와 MAS의 차이는 릴레이 압축 유무뿐 — 분해 효과를 통제한 상태에서 순수 릴레이 효과를 관찰 가능

MAS-SAS 등가성 정리

정리

릴레이 대역폭이 무제한일 때, MAS는 상류 전체 컨텍스트를 전달하여 임의의 SAS를 정확히 시뮬레이션할 수 있다.

함의 1

분해(decomposition) 자체는 MAS의 비자명한 이점을 만들지 않음

함의 2

MAS의 비자명한 이득과 손실은 유한 릴레이 압축에서 비롯됨

"분해가 아니라 압축이 핵심이다"

릴레이 압축 = 정보 병목

각 릴레이를 정보 병목(IB)으로 모델링 — 상류 컨텍스트 Mᵢ를 압축 메시지 mᵢ로 변환

 비용: 릴레이 정보 손실

하류 작업에 필요한 정보가 릴레이에서 누락되면 후속 워커가 복구 불가

Δᵢ(mᵢ) = 하류 관련 정보 손실량

 이점: 컨텍스트 감소

상류 무관/잡음 컨텍스트를 제거하여 하류 간섭 감소

H(Mᵢ | mᵢ) = 제거된 상류 컨텍스트의 엔트로피

  MAS 설계의 본질: 상류 무관 컨텍스트를 제거하면서 하류 관련 정보를 보존하는 릴레이를 찾는 최적화 문제

단계별 MAS 이득 분해

MAS 이득 = 컨텍스트 감소 이점 − β × 릴레이 정보 손실 비용

H(Mᵢ|mᵢ)

컨텍스트 감소 이점

제거된 잡음/중복 정보의 양

β

유효 파라미터

하류 모델의 정보 추출 능력

Δᵢ(mᵢ)

릴레이 정보 손실

하류에 필요한 정보 누락량

MAS 이득 > 0 일 조건: 컨텍스트 감소 이점이 능력 가중 릴레이 손실 비용을 초과할 때

β 파라미터: 모델 능력의 역할

β의 의미

하류 LLM이 중복 컨텍스트에서 유용한 정보를 추출하는 능력을 포착

  • β가 작음 → 약한 모델: 중복 속에서 정보 추출 어려움
  • β가 큼 → 강한 모델: 중복 속에서도 정보 추출 가능

β에 따른 이득 변화

β 증가(강한 모델)의 이중 효과:

  • 압축 이점 감소 — 이미 스스로 정보 추출 가능
  • 정보 손실 민감도 증가 — 버린 정보의 대체 불가

  핵심 예측: 동일한 릴레이가 약한 모델에는 이점이 되지만, 강한 모델에는 손해가 될 수 있다

릴레이 체제별 예측된 MAS 이득

릴레이 체제 약한 모델 (작은 β) 강한 모델 (큰 β)
거의 충분한 릴레이 이점 큼 이점 있음
부분 손실 릴레이 이점 있음 이점 감소 또는 역전
큰 손실 릴레이 이점 감소 손실 우세 (MAS 해로움)
공통 패턴: 릴레이가 하류 충분(near-sufficient)할 때 MAS 일관적 이점 — 릴레이가 정보 손실을 일으킬 때 이점 축소 또는 역전

MAS 설계를 위한 시사점

1

릴레이 설계가 핵심 — 단순한 작업 분해가 아닌, 무관 컨텍스트를 제거하면서 하류 관련 정보를 보존하는 릴레이 메시지 설계에 집중

2

모델 능력에 맞춘 설계 — 약한 모델을 위한 MAS는 적극적 압축이 유효하나, 강한 모델용 MAS는 정보 손실을 최소화하는 보수적 릴레이 필요

3

국소-하류 충분성(Local-to-Downstream Sufficiency) — 각 릴레이가 하류 작업에 충분한 정보를 담고 있는지 단계별로 검증

4

IB 최적화 관점 — MAS 설계를 정보 병목 목적함수의 최적화로 재구성하여 체계적인 릴레이 튜닝 가능
