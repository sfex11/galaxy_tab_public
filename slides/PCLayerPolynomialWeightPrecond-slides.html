LLM 훈련에서의 정규화: 두 가지 접근

출력 공간 정규화
  • BatchNorm, LayerNorm, RMSNorm
  • QK-Norm (attention logit 제어)
  • LLM 아키텍처의 사실상 표준
가중치 공간 정규화
  • Weight Norm, Spectral Norm
  • 최근: 행/열 정규화, 구형 제약 등
  • LLM 사전훈련에서 재조명되는 축
가중치 기하학(weight geometry)이 안정적인 LLM 훈련을 위한 점점 더 중요한 설계 축이 되고 있다.

동기: 신호 전파에서 스펙트럼 제어로

선형 변환 y = Wx에서 가중치 행렬의 특이값이 신호 전파를 지배한다.
σmin(W) · ‖x‖₂  ≤  ‖Wx‖₂  ≤  σmax(W) · ‖x‖₂
σmax가 너무 크면
신호 과증폭 → 불안정
σmin가 너무 작으면
신호 소실 → 기울기 소실
조건수 κ = σmax/σmin
스펙트럼 퍼짐 정도
핵심 관찰: 가중치 스펙트럼을 직접 제어하면 신호 전파를 안정화하고 최적화를 용이하게 할 수 있다.

왜 완벽한 직교성이 아닐까?

Hard 직교화 (Muon 등)
  • 모든 특이값을 1로 강제
  • 표현력 과도 제한 우려
  • 특정 옵티마이저에 의존
Soft 스펙트럼 성형 (PC Layer)
  • 작은 특이값은 증폭, 큰 것은 포화
  • 스펙트럼 퍼짐을 부드럽게 축소
  • 옵티마이저 무관 적용 가능
PC Layer는 "거의 직교"가 아니라 "잘 조건화된 스펙트럼"을 목표로 한다 — 표현력과 안정성 사이의 균형.

다항식 사전조건화의 핵심 아이디어

행렬 다항식은 스펙트럼에 직접 작용하는 편리한 메커니즘을 제공한다.
W = U · diag(σ₁, …, σᵣ) · V⊤  →  g(W)의 특이값 = { |g(σ₁)|, …, |g(σᵣ)| }

PC Layer 방법론

다항식 피팅
  • Gram 행렬 기반 사전조건화
  • 스펙트럼 노름으로 피팅 구간 [γ_L, γ_U] 설정
  • 작은 σ를 증폭, 큰 σ를 포화시키는 함수를 저차 다항식으로 근사
  • 차수-표현력 트레이드오프
효율적 계산 트릭
  • 동등한 효율 형태로 변환
  • 중간 행렬 B 캐싱 및 재사용
  • Horner 법으로 다항식 평가

해석
  • Muon은 근-직교화로 스펙트럼을 제어
  • PC는 보다 부드러운 스펙트럼 성형

PC Layer는 옵티마이저에 구애받지 않는 범용 가중치 조건화 도구로 작동한다.

가중치 스펙트럼 개선 분석

PC Layer가 실제로 스펙트럼을 어떻게 변화시키는지 분석
수정 조건수 κ̃
  • 사전조건화 후 특이값 스펙트럼의 유효 퍼짐을 측정
  • PC 적용 시 κ̃가 베이스라인보다 일관되게 낮음
  • 전역 수정 조건수(GMCN)도 감소
특이값 분포 비교
  • 베이스라인: 긴 꼬리, 넓은 스펙트럼
  • PC 적용: 하단 꼬리가 상승, 상단이 포화
  • 훈련 내내 안정적인 조건화 유지
과도한 스펙트럼 평탄화(overly aggressive flattening)는 오히려 성능 저하를 유지 — 적절한 차수 선택이 중요

이론: 스펙트럼 제어가 보장하는 수렴성

깊은 선형 네트워크에 대한 이론적 분석
정리 1 (Theorem 1)
다층 완전연결 선형 네트워크에서, 모든 층의 가중치 행렬이 유계 조건수를 갖는다면, 경사하강법은 기하급수적 수렴률로 전역 최솟값에 수렴한다. 수렴률은 가중치 조건수에 의존한다.
