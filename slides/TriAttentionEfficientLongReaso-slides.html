03 / 12 핵심 발견

Pre-RoPE 공간의 Q/K 집중 현상

핵심 관찰 (Observation 3.1)

"Pre-RoPE 공간에서 Q, K 벡터는 대부분의 어텐션 헤드에서 고정된 비영 중심(center) 주위에 고도로 집중되며, 이는 토큰 위치와 입력 컨텍스트에 관계없이 안정적"

(Math/Code/Chat)

삼각급수 재구성
상관관계
Mean Resultant Length:   R = ‖𝔼[q]‖ / 𝔼[‖q‖]
R → 1: 완전 집중  |  R → 0: 균일 분산

이 집중 현상은 모델 고유 특성으로, 학습 도메인에 무관하게 일관됨

04 / 12 이론적 기반

삼각함수 급수에 의한 거리 선호도 예측

Q/K 벡터가 중심에 집중될 때, RoPE 어텐션 로짓은 삼각함수 급수로 근사됩니다:

logit(Δ) ≈ Σf [ af·cos(ωf·Δ) + bf·sin(ωf·Δ) ]

Δ = pq - pk

쿼리-키 간 위치 거리

af, bf

Q/K 중심에 의해 결정되는 계수

ωf = θ-2f/d

RoPE 주파수 (θ=10000)

핵심 의미

각 어텐션 헤드는 특정 거리에서 예측 가능한 피크를 가지는 "어텐션-거리 곡선"을 형성합니다. 이를 통해 쿼리 없이도 키의 중요도를 사전에 추정할 수 있습니다.

캐시 프루닝

1. 캘리브레이션

캘리브레이션 데이터로 Pre-RoPE Q/K 분포의 중심(center)을 계산. 도메인 간 안정적 (50K~960K 토큰, 정확도 45.4~45.8%)

2. 중요도 계산

삼각급수 점수 Strig과 노름 점수 Snorm을 결합하여 각 키의 중요도를 실시간 추정

3. 캐시 관리

128 토큰 생성마다 프루닝 실행. 상위 점수의 키만 유지, 나머지 제거하여 예산 B 내 관리

최종 점수: S(k,Δ) = Strig(k,Δ) + Snorm(k)
평균화: S̃(k) = (1/|𝒟|) Σδ∈𝒟 S(k, Δ+δ),   𝒟 = {1, 2, 4, ..., 216}
06 / 12 중요도 스코어링

이중 스코어링 메커니즘

📐

삼각급수 점수 (Strig)

Strig(k,Δ) = Σf ‖𝔼[qf]‖ · ‖kf‖ · cos(ωf·Δ + φf)

φf = Q 중심과 현재 키의 위상 차이
집중된 헤드(R↑)에서 강력한 거리 선호도 예측

📏

노름 기반 점수 (Snorm)

Snorm(k) = Σf (1-Rf) · 𝔼[‖qf‖] · ‖kf‖

(1-Rf)로 스케일링
집중도가 낮은 헤드(R↓)에서 더 큰 기여

GQA (Grouped Query Attention) 처리

  • Z-score 정규화: 각 헤드 내 점수를 표준화 → Ŝ(g)(k) = (S̃(g)(k) - μg) / σg
  • 최대값 집계: Sfinal(k) = maxg(Ŝ(g)(k)) — 어떤 쿼리 헤드에서든 높은 점수의 키 보존
