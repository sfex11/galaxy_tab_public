LLM을 잡음 채널로 바라보다
Shannon 관점에서의 모델 용량과 스케일링 법칙
Xu Ouyang, Deyi Liu, Yuhang Cai, Jing Liu, Yuan Yang, Chen Zheng, Thomas Hartvigsen, Yiyuan Ma
ByteDance Seed · University of Virginia · UC Berkeley
arXiv:2605.23901v1 cs.LG
기존 스케일링 법칙의 한계
"스케일링만으로 충분하다"는 가정이 실제로는 한계에 부딪히고 있다.
전통적 관점
  • 모델 크기와 데이터 증가 → 단조적 성능 향상
  • 파워 법칙(power law) 기반 예측
  • DeepSeek-V4(1.6T), Kimi K2.6(1T) 등 거대 모델로 검증
문제점
  • U자형 손실 곡선 설명 불가
  • 과학습(Catastrophic Overtraining) 무시
  • 양자화 유발 성능 저하(QiD) 모델링 실패
단조적이지 않은 현상들
치명적 과학습
과도한 사전학습이
파인튜닝 성능을 오히려 저하시킴
양자화 유발 저하
더 크게 학습된 모델이
양자화에 더 취약해지는 역설
U자형 손실 곡선
성능이 향상되다가
임계점 이후 급격히 악화
전통적 파워 법칙은 이러한 비단조적 현상을 포착하지 못한다.
새로운 이론적 프레임워크가 필요하다.
핵심 통찰: LLM = 잡음 채널
LLM 학습을 통신 시스템의 정보 전송으로 재해석한다.
1
변조 (Modulation)
사전학습: 정보를 모델 가중치에 인코딩
2
전송 (Transmission)
추론: 입력 𝒳에서 출력 𝒴로 정보 전달
3
잡음 (Noise)
데이터·모델·간섭으로 인한 정보 손실
물리적 채널이 잡음으로 제한되듯, LLM도 데이터와 아키텍처에서 비롯되는 잡음에 의해 용량이 제한된다.
Shannon-Hartley 정리
C = B · log₂(1 + S/N)
채널 용량 = 대역폭 × log₂(1 + 신호대잡음비)
B 대역폭
채널이 처리할 수 있는
주파수 범위
S 신호 전력
전송하려는
유효 정보의 양
N 잡음 전력
채널을 통과하며
섞이는 불필요한 신호
SNR이 충분하지 않으면 대역폭을 늘려도 잡음만 증폭된다.
Shannon 스케일링 법칙
CLLM = aNα · log₂( 1 + bDβ / c(DN)γ + dDδ + e )
N: 모델 파라미터 수 · D: 학습 토큰 수 · α,β,γ,δ,a,b,c,d,e: 양의 상수
aNα → 대역폭
모델 파라미터 수에 비례하는 표현 용량
bDβ → 신호
학습 토큰에서 얻는 유효 학습 신호
분모 → 잡음
데이터·모델·간섭의 세 가지 잡음원
핵심: 잡음 항에 D·N 교차항이 존재하여 모델-데이터 상호작용을 명시적으로 모델링
잡음 항의 세 가지 구성 요소
1
c(DN)γ — 모델-데이터 간섭 잡음
모델 크기와 토큰 수가 동시에 증가할 때 발생하는 교차 잡음. U자형 곡선의 핵심 원인.
2
dDδ — 데이터 내재 잡음
토큰 수가 증가함에 따라 누적되는 데이터 자체의 노이즈. 중복·충돌·저품질 데이터.
3
e — 기저 잡음 (Baseline Noise)
모델 아키텍처 자체에서 발생하는 불가피한 최소 잡음 수준.
이 세 잡음원의 상대적 크기에 따라 스케일링이 도움이 될 수도, 해가 될 수도 있다.
용량을 손실로 연결: 핵심 메커니즘
SNR이 충분히 높은 영역 → 단조적 개선  |  SNR이 낮아지는 영역 → U자형 저하
고 SNR 영역
  • 신호가 잡음을 지배
  • 모델·데이터 확장 → 용량 증가
  • 전통적 파워 법칙과 일치
  • 사전학습 초기 단계
저 SNR 영역
  • 잡음이 신호를 압도
  • 확장이 잡음을 증폭
  • 성능 역전·U자형 곡선 발생
  • 과학습, 양자화 저하 영역
perturbation(양자화, SFT 등)이 잡음 항을 동적으로 증가시켜 고 SNR → 저 SNR 전이를 유발한다.
실험: 세 가지 perturbation 설정
Pythia (70M~12B) 및 OLMo2 모델 계열 사용
Perturbation 방식 관찰 결과
가우시안 잡음 가중치에 다양한 σ의 가우시안 잡음 주입 잡음 수준 증가에 따라 명확한 U자형 곡선 출현
지도 파인튜닝(SFT) 수학(GSM8K), QA(SiQA), 코드(StarCoder) 데이터셋 학습률 조절에 따른 loss basin 형성 관찰
양자화(Quantization) FP16 → INT8, INT4, INT3 등 다양한 정밀도 정밀도 하락 시 U자형 곡선 및 loss basin 출현
사전학습만의 궤적은 고 SNR 특수 사례로, Shannon 법칙이 단조적 법칙을 포함함을 확인.
외삽 성능: 보지 못한 모델 예측
≤6.9B Pythia 모델, ≤180B 토큰으로 학습 → 보지 못한 12B 모델(307B 토큰) 예측

적합 지수가 말해주는 것
fitted exponent들의 상대 크기가 스케일링의 효과 방향을 결정한다.
γ > α일 때: 모델 스케일링이 역효과
잡음의 모델-데이터 교차항이 대역폭보다 빠르게 성장 → 모델을 키울수록 잡음이 더 크게 증폭 → "클수록 나쁨"
δ > β일 때: 토큰 잡음이 항상 승리
데이터 내재 잡음이 신호보다 빠르게 누적 → 토큰을 무한히 늘려도 극한에서는 잡음 지배 → "더 많이 학습해도 소용없음"
D·N 교차항(c(DN)γ)의 필수성: 이 항을 제거하면 U자형 곡선과 loss basin을 전혀 포착할 수 없음을 실험적으로 확인.
결론 및 시사점
이론적 기여
  • Shannon-Hartley 정리를 LLM 스케일링에 최초로 정식 연결
  • 단조적·비단조적 스케일링을 통일하는 프레임워크 제공
  • LLM 용량의 근본적 한계(Shannon capacity) 정의
  • D·N 교차항의 이론적·실증적 필수성 입증
실무적 시사점
  • SNR 모니터링으로 과학습 조기 감지 가능
  • 양자화 전 SNR 평가로 QiD 사전 예측
  • 적은 파라미터로 효율적 스케일링 설계 가능
  • "더 크게, 더 많이"가 항상 정답이 아님을 정량화
스케일링의 방향이 아니라 신호대잡음비의 보존이 LLM 성능의 본질적 열쇠다.
요약
1
기존 단조적 스케일링 법칙은 과학습·QiD 같은 비단조적 현상을 설명할 수 없다
2
LLM을 잡음 채널로 모델링하여 Shannon-Hartley 정리에서 유도된 Shannon 스케일링 법칙을 제안
3
대역폭(모델)·신호(토큰)·잡음(교차항+데이터+기저)의 구조로 U자형 곡선을 자연스럽게 포착
4
가우시안 잡음·SFT·양자화 실험에서 기존 법칙 대비 압도적 R²=0.847 외삽 성능 달성
5
γ>α, δ>β 조건이 언제 스케일링이 해가 되는지 정량적으로 판별 가능하게 함
