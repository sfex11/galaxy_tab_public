학습 방법 비교

SFT vs 강화 학습 (RL)

지도 미세조정 (SFT)

  • 초기에 빠르게 성능 향상
  • 장기 학습 시 과적합으로 급격한 성능 저하
  • 최고 성능(peak)이 RL보다 높거나 같음

강화 학습 (Dr.GRPO)

  • 에폭 전반에 걸쳐 안정적
  • SFT의 최고 성능을 절대 초과하지 못함
  • 과적합을 방지하지만 능력 한계는 동일
⚠️ 핵심: RL은 학습을 안정화하지만, 새로운 전이 또는 확장 능력을 해금하지 못함
→ 최고 성능은 항상 최적의 SFT 모델과 동일

종합 분석

학습 파이프라인 단계별 역할

  1. 데이터 커버리지가 능력의 상한선을 설정
    커버리지가 부족하면 다른 어떤 방법으로도 보상 불가
  2. 강화 학습은 학습을 안정화하지만 한계를 확장하지 못함
    과적합 방지에는 효과적이나 새로운 능력 해금 없음
  3. 추론 시간 확장은 성능을 향상시키지만 구조적 실패를 해결 못함
    목표 지향적 선택(최단 경로)이 가장 효과적
