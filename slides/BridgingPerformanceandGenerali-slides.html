기존 RL 정책은 훈련되지 않은 트랙에서 단 한 바퀴도 완주하지 못하고 충돌

인간 조종사는 초보자라도 처음 보는 트랙을 충돌 없이 완주할 수 있다. 반면 RL 에이전트는 훈련 트랙에서는 초인적 성능을 보이면서도, 미지의 트랙에서는 즉시 실패한다. 이 간극을 어떻게 메울 것인가?

핵심 개념

제로샷 일반화(ZSG)란 무엇인가

트랙별 훈련이나 파인튜닝 없이, 처음 보는 환경에서 안전하고 효과적으로 비행하는 능력. 드론 레이싱에서 단 한 번의 충돌도 기체를 완전히 파괴할 수 있으므로, ZSG는 실제 배포를 위한 필수 불가결한 능력이다.

기존 방법의 근본적 한계

Domain Randomization

고정된 파라미터를 무작위로 변화 — 고차원 task 변동성에 부족

Adaptive Difficulty (Wang et al.)

일반화는 개선되나 비행 속도가 크게 저하되는 치명적 trade-off

제안 방법

제안 프레임워크 개요

학습 진행에 기반한 적응적 task 전환과 물리적으로 정보가 있는 절차적 트랙 생성기를 결합하여, 테스트 시간 적응 없이 빠르고 강건한 범용 정책을 학습한다.

1
절차적 트랙 생성
스플라인 기반 물리적 제약을 갖춘 다양한 트랙 자동 생성
2
대규모 병렬 학습
다수의 환경에서 동시에 PPO 기반 정책 학습
3
적응적 Task 전환
학습 신호가 약해진 task를 감지하여 새로운 task로 교체
4
범용 정책 산출
속도 저하 없이 제로샷 일반화 달성
방법론

문제 공식화: UPOMDP

미지정 부분 관측 마르코프 결정 과정(Underspecified POMDP)으로 모델링하여, 연속적인 task 분포에 대한 최적의 단일 정책을 찾는다.

Task 공간 T

게이트 위치와 방향(트랙 레이아웃)으로 정의되는 환경 파라미터

관측 공간 O

상태 벡터 또는 84×84 이미지를 통한 부분 관측 처리

목표 π*

단일 task가 아닌 task 분포 전체에 대한 기댓값 최대화

핵심 아이디어: 이산적 task 집합이 아닌 연속적 task 분포 p(T)에서 샘플링하여 훈련하고, 테스트는 완전히 분리된 T_test에서 평가한다.

방법론

정보 기반 절차적 트랙 생성

단순한 무작위 게이트 배치가 아닌, 물리적 비행 역학을 반영한 스플라인 기반 생성으로 의미 있는 다양성을 확보한다.

1

제어점 샘플링

공간 내에서 제어점들을 무작위로 배치하여 곡선의 기본 형태 결정

2

주기적 스플라인 구성

폐곡선 스플라인을 통해 부드럽고 연속적인 경로 생성

3

게이트 배치

스플라인 경로를 따라 게이트를 배치하며 곡률과 간격 제약 적용

4

시작 자세 제약 검증

출발점에서 물리적으로 비행 가능한 자세인지 검증하여 불가능한 트랙 제거

방법론

적응적 Task 전환 메커니즘

에이전트가 특정 task에서 더 이상 유의미한 학습 신호를 받지 못할 때, 해당 task를 새로운 task로 교체하여 과적합을 방지한다.

Spearman 방법

  • 에피소드 보상의 순위 상관계수(Spearman) 추적
  • 상관계수가 임계값 아래로 떨어지면 학습 정체로 판단
  • 구현이 단순하고 계산 비용이 낮음

Kalman 방법

  • 칼만 필터로 보상의 추세선 예측
  • 추세의 기울기가 0에 수렴하면 전환 결정
  • 노이즈에 더 강건한 신호 감지

핵심 통찰: 고정된 훈련 셋으로 학습하면 과적합이 발생한다. 에이전트의 학습 진행 상태를 모니터링하여 훈련 task 자체를 동적으로 갱신하는 것이 일반화의 핵심이다.

방법론

정책 학습: 상태 → 비전

두 가지 관측 모달리티를 통해 프레임워크의 범용성을 검증한다.

상태 기반 정책

  • 회전 행렬, 선속도, 각속도
  • 다음 게이트 4개 코너의 상대 위치 (12차원)
  • 게이트 간 상대 위치 정보 (12차원)
  • 이전 액션 기록

비전 기반 정책

  • 84×84 이미지 입력
  • 3-layer CNN → 128차원 임베딩
  • 부분 상태 정보 + 이전 액션과 결합
  • 명시적 상태 추정 없는 end-to-end 제어

출력: 집단 추력 c ∈ ℝ + 바디레이트 설정값 ω^ref ∈ ℝ³ (롤, 피치, 요)

주요 결과

비전 기반 실험: 최초의 성공

비전 기반 드론 레이싱은 본질적으로 부분 관측 가능하며, 시각적 단서로 보이지 않는 트랙 기하를 추론해야 하므로 현존 모든 이전 방법이 일반화에 실패한 영역이다.

기존 방법의 결과

  • 단일 트랙 훈련: 미지 트랙에서 즉시 충돌
  • Domain Randomization: 일반화 불가
  • 적응 난이도: 비전 설정에서도 실패

제안 방법의 결과

  • 미지 트랙에서 성공적 제로샷 완주
  • 명시적 상태 추정 없는 end-to-end 제어
  • RL 비전 정책이 고속 레이싱에서 일반화 가능함을 최초로 증명
분석

일반화에 기여하는 요인 분석

논문에서는 일반화에 영향을 미치는 다양한 전략을 체계적으로 분석하고 검증한다.

효과가 확인된 요인

  • 적응적 task 전환 (가장 중요)
  • 물리적으로 정보 있는 트랙 생성
  • 대규모 병렬 학습
  • 다양한 task 노출의 지속적 갱신

효과가 제한적인 요인

  • L2 정규화: 단독으로는 불충분
  • 경험 재생 메모리: 치명적 망각 위험
  • 단순 무작위 트랙: 물리적 타당성 부족

치명적 망각(Catastrophic Forgetting) 분석: 경험 재생 버퍼를 사용할 경우, 오래된 task의 경험이 새로운 task 학습을 방해하여 오히려 일반화가 저하되는 현상을 확인했다.

실세계 검증

시뮬레이션 → 실세계 배포

시뮬레이션에서 학습한 정책을 테스트 시간 적응 없이 그대로 실세계 드론에 배포하여 검증한다.

상태 기반 배포

다양한 미지 트랙에서 강건한 비행 성능 확인. SOTA 모델 기반 제어기보다 우수한 성능

비전 기반 배포

명시적 상태 추정 없이 카메라 입력만으로 미지 트랙 완주. 이전 방법은 모두 실패

Sim-to-Real 간극

도메인 무작위화를 통한 견고성 확보와 절차적 생성의 다양성이 결합하여 간극 최소화

학습은 전적으로 시뮬레이션에서 수행되며, 실세계 배포 시 어떠한 미세조정(fine-tuning)도 수행하지 않는다.
