해석

SW 좌표 동결이 LoRA 성능에 영향을 주지 않는다는 것은, LoRA의 저랭크 구조가 SW 위치를 암묵적으로 보호하면서도 레이어 전체에 걸쳐 조율된 업데이트를 생성함을 의미한다. 개별 좌표 제한은 저랭크 분해의 전역적 특성에 의해 자연스럽게 상쇄된다.

이론적 분석

본질 차원성 관점

  • 미세조정 부분공간은 저랭크 구조를 가짐
  • 희소 업데이트는 이 부분공간의 무시할 수 있는 비율만 포착
  • 좌표 정렬 문제: SW 위치가 미세조정 방향과 정렬되지 않음

결과: 희소 학습이 "수학적으로" 실패하는 이유를 차원 정렬 관점에서 설명

최적화 동역학 관점

  • SW 위치에서 헤시안 곡률이 높음
  • 높은 곡률 → 작은 스텝 크기 → 업데이트 억제
  • 가중치 감쇠가 SW 위치에서 암묵적 정규화 역할

검증: 활성화 증폭, 곡률 스케일링, 섭동 강건성 3가지 실험으로 지지

핵심 관찰

관찰 1

실패는 SW 좌표에 특정

실패는 희소성이나 모듈 선택 때문이 아니다.

  • 동일 수의 랜덤 위치 학습 → 성공
  • 동일 모듈 저랭크 업데이트 → 성공
  • SW 좌표 타겟팅 → 실패
관찰 2

LoRA는 레이어 전체 조율

LoRA의 저랭크 구조가 생성하는 업데이트는 전체 레이어에 걸쳐 조율된다.

  • 0.16% 파라미터로 베이스라인 초과 달성
  • SW 좌표 동결해도 성능 변화 없음
  • 개별 최대 업데이트 위치는 중요하지 않음

결론 및 시사점

핵심 발견

파라미터 중요성 ≠ 파라미터 독립 학습 가능성

제거 시 치명적이라도, 격리하여 학습할 수 있는 것은 아님

구조적 분해가 핵심

개별 중요 가중치 타겟팅이 아닌 레이어 전체 구조 활용

실무적 시사점

  • 중요 파라미터를 식별한 뒤 선택적 미세조정하는 접근법은 효과적이지 않음
  • LoRA의 성공은 저랭크성 + 레이어 전체 조율의 결합 효과
  • 프루닝 민감도와 미세조정 가능성은 독립적인 속성
  • 향후 PEFT 설계는 개별 좌표 중요도가 아닌 구조적 커버리지를 고려해야 함

한계점

모델 범위

학습 실험은 OLMo-1B/7B에만 수행. 프루닝은 10개 모델에서 검증했으나, 학습 실험의 범용성은 추가 확인 필요
