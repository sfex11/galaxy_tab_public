배경 및 동기

수상 모듈형 자가재구성 로봇(MSRR)

다수의 로봇 유닛이 물리적으로 연결 및 재배열되어 더 큰 구조물을 형성하는 시스템

  • 고정 인프라 구축 비용 절감
  • 환경 모니터링 플랫폼 구성
  • 해상 작업용 임시 인프라
  • 작업 및 환경 변화에 적응적 재구성

핵심 과제

다수 에이전트의 운동을 조율하면서 충돌 회피와 물리적 연결을 동시에 달성해야 함

"중앙 집중식 방식은 에이전트 수 증가에 따라 확장성이 떨어지며, 단일 장애점(single point of failure)을 생성함"

기존 접근법의 한계

접근법 장점 한계
중앙 집중식 제어 전역 최적해, 순차 조립 가능 확장성 부족, 단일 장애점
포텐셜 필드 기반 분산 실행, 지역 정보만 사용 국소 최적점 문제, 정식 안전 보장 없음
분산 MPC (ADMM) 병렬 계산, 예측적 행동 비볼록 문제로 충돌 제약 보장 불가
CBF 단독 사용 정식 안전 보장, 순방향 불변성 근시안적, 밀집 환경에서 교착상태 발생

 MPC의 예측적 궤적 계산과 CBF의 정식 안전 보장을 결합하면 두 한계를 동시에 해결할 수 있음

운동학 모델 및 CBF 이론

단일 적분기 운동학 모델

ṗᵢ = uᵢ,   i ∈ 𝒩 := {1, …, N}

속도 제약: ‖uᵢ‖∞ ≤ u_max

하위 제어기가 고주파수로 속도 루프를 닫으므로 단일 적분기로 근사 가능. 피드백 선형화가 비선형 유체역학을 보상함

제어 장벽 함수 (CBF)

안전 집합: 𝒞 = {x ∈ ℝⁿ : h(x) ≥ 0}

CBF 조건: 존재하는 확장 클래스 K 함수에 대해

sup [Lf h(x) + Lg h(x) · u] ≥ −α(h(x))

CBF는 안전 집합의 순방향 불변성을 보장하여, 초기 상태가 안전하면 시스템이 항상 안전 집합 내에 유지됨

실행 전략

고정된 ADMM 반복 횟수 후 첫 번째 시간 스텝만 적용 (Receding Horizon). 실시간성을 위해 반복 횟수를 제한하며, 수렴하지 않아도 CBF 필터가 안전을 보장함

CBF 안전 필터 및 거리 적응

분산 CBF 안전 필터

MPC가 생성한 속도 참조값을 CBF 필터가 수정하여 충돌 회피 보장

CBF-QP:

min ‖u − u_mpc‖²

s.t. CBF 제약 만족

  • 쌍별 충돌 회피만 고려 → 분산 가능
  • 작은 규모 QP로 실시간 해결
  • MPC 참조값에서 최소 수정

도킹을 위한 거리 적응

일반 이동 시와 도킹 시 안전 거리를 동적으로 조절

이동 단계
충돌 회피 거리 d_safe 유지
접근 단계
d_safe → d_dock로 점진 축소
도킹 단계
자기 래칭 메커니즘으로 물리적 연결

제거 연구: 각 구성요소의 역할

ADMM만 사용

비볼록 충돌 제약으로 인해 ADMM이 수렴하지 못하고 안전 위반이 발생

예측적 궤적은 좋으나 보장 없음

CBF만 사용

안전은 보장되나 교착상태(deadlock) 발생. 에이전트가 서로 멈춰 형성 불가

근시안적 반응으로 장기 계획 부재

ADMM + CBF

MPC가 예측적으로 교착상태를 완화하고, CBF가 안전을 보장

상호 보완적 결합으로 두 한계 해결

핵심 인사이트: ADMM은 "어디로 가야 할지"를 제시하고, CBF는 "거기까지 안전하게 가는 방법"을 보장함

확장성 한계 분석

요인 영향
예측 수평선 길이 길어질수록 계산량 증가
이웃 크기 밀집도에 따라 국소적 증가
중앙 집중형 할당 에이전트 수 증가 시 병목

 분산 MPC 자체는 확장 가능하나, 중앙 집중형 목표 할당이 전체 시스템의 확장성 병목으로 작용할 수 있음
