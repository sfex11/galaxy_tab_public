01 / 문제 정의
WER의 근본적 한계
자동 음성 인식(ASR) 평가의 표준 지표인 WER은 의미를 무시하고, 인간 지각과의 상관관계가 낮습니다.
  WER의 문제
  • 어휘 수준에서만 비교 — 의미 맥락 무시
  • 치명적 오류와 사소한 오류를 동일하게 처리
  • 다운스트림 작업과 상관관계 부족
  • 인간 지각과 낮은 일치도 (63%)
  • 대소문자 구분 등 경직성
VS
  필요한 지표
  • 의미적 유사성 기반 평가
  • 오류 심각도 가중치 부여
  • 인간 판단과 높은 상관관계
  • 해석 가능한 결과 제공
  • 실제 사용자 경험 반영
WER 개선이 반드시 사용자가 체감하는 품질 향상으로 이어지지는 않습니다. 평가 지표의 전환이 시급합니다.
02 / 관련 연구
의미 기반 평가 지표의 발전
정적 임베딩에서 문맥적 인코더 모델까지, 의미 평가 지표는 지속적으로 발전해 왔습니다.
1세대
정적 워드 임베딩 (Le et al., 2016)
Word2Vec, GloVe 등을 활용하여 ASR 평가에 임베딩 기반 접근을 최초로 도입. 문맥 무시로 인해 한계 노출.
2세대
문맥적 인코더 모델 (BERT, SemDist)
BERT의 양방향 인코더를 활용한 SemDist(Kim et al., 2021)가 인간 지각과 WER보다 우수한 상관관계를 입증. 가중 WER 등 해석 가능성 연구도 진행.
미탐구 영역
디코더 기반 생성적 LLM
GPT, Llama, Gemma 등 디코더 아키텍처 LLM은 다양한 작업에서 뛰어난 성능을 보여주었으나, ASR 평가에는 거의 활용되지 않음.
본 논문의 핵심 질문: 디코더 기반 LLM이 ASR 평가에 얼마나 적합한가?
03 / 연구 방법
LLM을 활용한 세 가지 평가 접근법
두 가지 보완적 축 — "LLM as Judge"와 임베딩 품질 평가 — 에서 세 가지 실험을 수행합니다.
접근법 1
  최적 가설 선택
두 개의 ASR 가설 중 참조 텍스트에 더 가까운 것을 LLM이 선택. Chain-of-thought 추론을 활용하여 의사결정 과정을 노출.
접근법 2
  생성적 임베딩 기반 의미 거리
디코더 LLM에서 추출한 임베딩에 다양한 풀링(pooling) 전략을 적용하여 의미 유사도 메트릭 구축. 인코더 임베딩과 비교.
접근법 3
  오류의 정성적 분류
LLM이 전사 오류에 정성적 라벨을 부여하여 해석 가능한 평가 수행. 오류 유형과 심각도를 자동으로 분석.

05 / 핵심 실험
LLM을 활용한 최적 가설 선택
One-shot 프롬프트로 두 가설 중 더 나은 전사를 선택하도록 LLM에 요청합니다.
User
참조 텍스트는 오디오의 정확한 전사입니다. 두 개의 잘못된 가설이 제시됩니다. 추론을 설명하고 마지막에 'A', 'B', 'a', 또는 'b'를 작성하세요.
참조: it's to him self
가설 A: are to him self
가설 B: it's uh to him-self
Assistant
가설 B는 불필요한 발화('uh')를 포함하지만, 단어와 의미 면에서 참조에 훨씬 더 잘 대응합니다. 불필요한 발화가 원본에 포함된 것이라면 허용될 수 있지만, 가설 A의 문법 오류는 더 문제적입니다. 따라서 가장 수용 가능한 전사는 가설 B입니다.
Chain-of-thought를 활용하여 모델의 추론 과정을 노출하고, 최종 결정만 자동 추출합니다.

07 / 임베딩 품질
디코더 LLM 임베딩의 의미 표현 능력
디코더 LLM에서 추출한 임베딩이 인코더 모델과 비교해 어떤 성능을 보이는지 분석합니다.
핵심 발견
  인코더 대비 성능
디코더 기반 LLM 임베딩은 인코더 모델(CamemBERT)과 유사한 수준의 의미 유사도 측정 성능을 보입니다.
  풀링 전략의 중요성
평균 풀링, 최대 풀링, CLS 토큰 등 풀링 방식에 따라 성능이 크게 달라지며, 최적 전략은 모델별로 상이합니다.
풀링 전략 비교
임베딩 집계 방법
Mean Pooling Max Pooling Last Token CLS Token Weighted Mean

각 풀링 전략을 Cosine Similarity 기반 의미 거리 계산에 적용하여, 문장 수준의 의미 보존 정도를 정량화했습니다.
디코더 LLM은 단순한 텍스트 생성뿐만 아니라, 의미적 표현 공간에서도 인코더에 필적하는 품질을 제공합니다.
08 / 결론
ASR 평가의 새로운 패러다임
생성적 LLM은 ASR 평가에서 인간 지각에 가장 가까운 판단을 제공하며, 해석 가능성까지 확보합니다.
압도적 성능: 최고 LLM은 인간 주석자와 94% 일치율을 달성하며, WER(63%)은 물론 인코더 기반 의미 지표(90%)도 능가했습니다.
임베딩 가치 입증: 디코더 LLM 임베딩이 인코더 모델에 필적하는 의미 표현 능력을 보여주어, 새로운 의미 메트릭 구축의 가능성을 열었습니다.
해석 가능성: LLM이 오류 유형과 심각도에 정성적 라벨을 부여할 수 있어, "0.62" 같은 불투명한 점수 대신 의미 있는 피드백 제공이 가능합니다.
개방성: qwen3.5-35B 같은 오픈 액세스 모델이 GPT-4.1에 필적하는 성능을 보여, 상용 API 의존 없이도 고품질 평가가 가능합니다.
향후 방향: 다국어 확장, 실시간 평가 파이프라인 통합, 오류 심각도의 정량화 등이 연구 과제로 남아 있습니다.
