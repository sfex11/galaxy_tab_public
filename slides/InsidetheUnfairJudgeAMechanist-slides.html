LLM-as-Judge와 편향 문제

LLM 판사는 답변 품질 평가, 후보 비교, RLHF 보상 신호 생성에 널리 사용되지만, 품질과 무관한 표면 단서에 의해 점수가 왜곡된다.

편향 유형
Prestige, Verbosity, Bandwagon, Authority, Sentiment, Refinement, Diversity

9

벤치마크
다양한 도메인의 판단 작업

7

판사 모델
Llama, Qwen, Gemma 등

편향은 벤치마크 유효성을 훼손하고, RLHF 파이프라인을 통해 평가 대상 모델에까지 묵묵히 전파된다.

기존 접근의 한계

측면 기존 연구 (입출력 수준) 본 연구 (표현 수준)
관점 입력 변형 → 점수 변화 측정 은닉 상태의 기하학적 구조 분석
완화 방법 프롬프트 엔지니어링, 앙상블 활성화 조향(Steering)을 통한 인과적 제어
일반화 관측된 편향에 국한 미검증 벤치마크에서의 판사 실패 예측
메커니즘 블랙박스 기술 편향 방향의 식별과 조작

"판사가 불공정한 점수를 내릴 때, 모델 내부에서 무슨 일이 일어나는가?"

핵심 아이디어

LLM-as-Judge 편향은 표면 단서에서 점수 예측으로의 내부 매핑이며, 판사의 은닉 상태에 저차원 기하학적 구조로 존재한다.

1
기하학
편향이 저차원 부분공간에 집중됨
2
인과 제어
해당 방향으로 조향하면 점수 조작 가능
3
운용적 예측
미검증 도메인에서 판사 실패 예측

입출력 노이즈가 아닌 활성화 기하학으로 편향을 읽으면, 기하 구조·인과 제어·운용 예측을 단일 프레임워크로 통합할 수 있다.

방법론: 통제 데이터 생성

동일한 질문과 사실적 내용을 공유하면서 의미와 무관한 표면 프레이밍만 다른 삼중집합(triple)을 구성한다.

𝒟base

중립 프레이밍
편향 단서 없는 기준 입력

𝒟pos

긍정 편향 프레이밍
점수 상승 유도 단서 포함

𝒟neg

부정 편향 프레이밍
점수 하락 유도 단서 포함

예: 동일한 답변에 "전문가가 작성함"(Authority+), "초보자가 작성함"(Authority−) 라벨만 부착

방법론: 활성화 공간 분석

판사 모델의 잔차 스트림(residual stream)에서 각 레이어의 은닉 상태를 추출하여 편향의 기하학적 특성을 분석한다.

  • 방향 추정기(Direction Estimator): 평균 차이 벡터
  • 판별 추정기(Discriminative Estimator): 선형 SVM 기반
  • 세 가지 독립 목적의 추정기 패밀리가 동일 부분공간 복원

유효 편향 기질(Effective Bias Substrate)

base 대비 편향 입력의 활성화 변위가 집중되는 저차원 부분공간

편향 벡터(Bias Vector)

해당 부분공간을 대표하는 단위 방향 벡터. 유형별로 고유함

발견 1: 편향의 기하학적 구조

  • 기준 활성화는 조밀한 매니폴드 ℳbase에 위치
  • 편향 입력은 유형별 저차원 부분공간을 따라 변위
  • 부분공간은 층이 깊어질수록 선명하게 집중
  • 행동적 비대칭성(부정 단서에 대한 큰 페널티)은 이 기하학의 하류 서명으로 해석됨

핵심 관찰

입출력 연구에서 문서화된 "부정 단서에 대한 큰 점수 하락, 긍정 단서에서의 좁은 분포"라는 비대칭성은 1차 발견이 아니라 활성화 기하학의 2차적 결과물이다.

세 가지 독립 추정기 패밀리가 일관되게 동일 방향을 복원함

발견 2: 인과적 제어

복원된 편향 방향을 따라 은닉 상태를 조향(steering)하면 점수를 양방향으로 제어할 수 있다.

순방향 조향 (Attack)

깨끗한 입력에 편향 방향을 더하면 편향 점수가 재현됨

역방향 조향 (Defense)

편향 입력에서 편향 방향을 빼면 기준 점수가 복원됨

통제 1 동일 노름의 무작위 방향은 편향 방향보다 약 10배 작은 효과만 발생

통제 2 타 편향 유형 방향으로 조향하면 해당 유형의 점수 변화만 발생 (교차 불가)

발견 3: 운용적 예측

편향 방향 특징에 대한 단순 선형 투영으로 판사가 본 적 없는 벤치마크에서의 실패를 예측한다.

0.82

활성화 특징 기반 AUC

3개의 미검증(unseen) 벤치마크에서의 교차 도메인 예측 성능

0.63

텍스트 기반 베이스라인 AUC

입력 텍스트 특징만 사용한 기존 방식

활성화 기하학이 학습 도메인을 넘어 전이됨을 시사한다. 텍스트를 읽지 않고 은닉 상태만으로 판사 붕괴를 감지할 수 있다.

분석 대상 층

잔차 스트림 전체 레이어에서 활성화 추출 및 깊이별 프로파일링

엄격한 통제 조건

무작위 방향 통제
노름 매칭된 랜덤 벡터로 대체

편향 유형 교환 통제
타 유형의 편향 벡터로 조향

텍스트 공격 비교
동일 예산의 입력 텍스트 변형

결과 요약

발견 핵심 결과 의의
기하학 편향이 저차원·유형별 부분공간에 집중, 깊이에 따라 선명화 편향의 내부 구조 최초 특성화
인과 제어 양방향 조향 성공, 무작위 방향보다 10× 큰 효과 수동적 상관이 아닌 인과적 조작 핸들 확보
운용 예측 미검증 벤치마크에서 AUC 0.82 (텍스트 기반 0.63) 은닉 상태만으로 판사 실패 사전 감지 가능

세 가지 모델(Llama, Qwen, Gemma)에서 아키텍처 간 일관된 편향 방향 코사인 유사도가 관측되어, 발견이 특정 모델에 국한되지 않음을 시사한다.

시사점과 한계

기여

  • 편향을 입출력 노이즈가 아닌 내부 기하학으로 재개념화
  • 공격(점수 조작)과 방어(점수 복원)를 단일 벡터로 통합
  • 텍스트 없이 활성화만으로 판사 품질 모니터링 가능
  • 기계론적 해석 도구를 평가자 역할에 최초로 본격 적용

한계

  • 인과적 충분성(sufficiency) 범위: 판사의 유일한 자연 경로임을 주장하지 않음
  • 경로 패칭·인과 추적(mediation)은 별도 검증 필요
  • 화이트박스 접근으로 블랙박스 API 판사에는 직접 적용 불가
  • 7가지 편향 유형 외의 편향으로의 확장은 추가 연구 필요

결론

편향을 활성화 기하학으로 읽으면, 기하 구조·인과 제어·운용 예측을 단일 프레임워크 안에 통합할 수 있다.

기하학 — 편향은 저차원, 유형별, 깊이-선명화 부분공간에 존재
인과 제어 — 단일 벡터로 공격과 방어 모두 달성
운용 예측 — 미검증 도메인에서 텍스트 기반 방법을 크게 상회

LLM 판사의 공정성은 프롬프트 수정만으로는 보장될 수 없다. 내부 표현 수준에서 편향을 이해하고 제어하는 것이 다음 단계다.
