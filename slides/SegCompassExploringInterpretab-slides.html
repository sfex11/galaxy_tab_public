추론 세그멘테이션의 해석 가능성 문제

LLM은 강력한 구성적 추론 능력을 갖추고 있으나, 기존 파이프라인은 추론 과정을 시각적 지각과 투명하게 연결하지 못합니다.

잠재 쿼리 정렬

End-to-end 학습은 가능하지만, 중간 결정이 불투명한 "블랙박스"로 동작합니다.

LISA, PerceptionGPT, PixelLM 등

텍스트 지역화 읽기

사후 처리 단계로 텍스트 단서를 생성하나, 진정한 해석 가능성이 아닌 단순 가독성만 제공합니다.

SAM4MLLM, Seg-Zero, VisionReasoner 등

"추론 과정에서 최종 마스크 예측까지 해석 가능하고 검증 가능한 연결 메커니즘이 여전히 부재하다."

핵심 아이디어: SAE를 통한 "화이트박스" 연결

희소 오토인코더(SAE)가 학습하는 이산적, 고차원, 의미론적 특징은 해석 가능한 개념 포착에 본질적으로 적합합니다.

SegCompass는 SAE를 활용해 CoT 추론과 시각적 토큰을 공유된 고차원 희소 개념 공간으로 매핑하여, 추론에서 마스크 생성까지 투명한 경로를 구축합니다.

정렬 전략 비교

잠재 쿼리 정렬

LLM 잠재 벡터 → 쿼리

  • End-to-end 학습
  • 불투명한 중간 표현
  • 해석 불가

텍스트 지역화 읽기

CoT → 좌표/패치 → SAM

  • 텍스트 가독성
  • 사후 처리 단계 분리
  • 공간 단서 불명확

SAE 해석 가능 정렬

CoT+시각 → SAE → 히트맵

  • End-to-end 학습
  • 투명한 중간 표현
  • 해석 가능한 개념 공간

SAE 기반 희소 개념 인터페이스

희소 인코딩 과정

SAE는 MLLM의 특정 레이어 은닉 상태를 희소 벡터로 변환합니다. ReLU 활성화를 통해 대부분의 차원이 0이 되며, 소수의 활성 차원만이 의미 있는 개념을 부호화합니다.

CoT 은닉 상태 → SAE → 희소 개념 벡터
시각 토큰 → SAE → 희소 개념 벡터

쿼리 코드북 & 집계

코드북 C ∈ R^{d_sae × d_c}가 희소 특징에서 핵심 개념을 선택하고, 트랜스포머 인코더 블록이 이를 K_s개의 개념 표현으로 집계합니다.

희소 벡터 × 코드북 → 개념 표현
트랜스포머 집계 → K_s개 표현

결과: 언어 추론과 시각 정보가 동일한 개념 공간에서 만나며, 각 활성화 차원이 특정 개념(예: "하얀 도자기 그릇")에 대응합니다.

다중 슬롯 히트맵과 마스크 생성

슬롯 매퍼 (Slot Mapper)

집중 토큰 임베딩과 SAE 개념 표현을 융합하여 K_s개의 쿼리를 구성합니다. 이 쿼리들이 비전 인코더에서 추출된 이미지 키에 어텐션을 수행합니다.

결과적으로 K_s-슬롯 히트맵이 생성되며, 각 슬롯은 명령어의 각 부분이 이미지의 어느 영역과 대응하는지를 시각적으로 보여줍니다.

마스크 디코더

다중 슬롯 히트맵을 디코더에 입력하여 최종 세그멘테이션 마스크를 예측합니다.

핵심 장점: 히트맵을 통해 "모델이 왜 해당 위치를 세그멘테이션했는지"를 직접 관측할 수 있습니다.

"CoT에서 식별된 개념(예: '하얀 도자기 그릇')이 다중 슬롯 히트맵을 통해 이미지 상에 명시적으로 정착됩니다."

통합 최적화 전략

SegCompass는 강화학습과 지도학습을 단일 학습 목표 하에 통합합니다.

추론 경로 — GRPO

Group Relative Policy Optimization (GRPO)를 적용하여 CoT 생성 품질을 향상시킵니다.

  • 그룹 내 상대적 보상 기반
  • 마스크 품질을 보상 신호로 활용
  • KL 정규화로 안정적 학습

세그멘테이션 경로 — 지도학습

표준 세그멘테이션 손실 함수로 마스크 예측을 직접 감독합니다.

  • BCE + Dice 손실
  • SAE 재구성 손실
  • 희소성 정규화

두 경로가 end-to-end로 공동 최적화되어 추론 품질과 세그멘테이션 정확도가 상호 향상됩니다.

실험 설정 (RQ1)

벤치마크 데이터셋

  • ReasonSeg — 복합 추론 세그멘테이션
  • gRefCOCO — 생성형 참조 세그멘테이션
  • RefCOCO / RefCOCO+ / RefCOCOg

평가 지표

  • Generalized IoU (gIoU)
  • cIoU (ReasonSeg 전용)
  • 정밀도 @ 0.5 / @ 0.7 IoU

비교 대상: LISA, PerceptionGPT, PixelLM, SAM4MLLM, Seg-Zero, VisionReasoner, LIRA, RAS 등 최신 방법론들과 동일 평가 프로토콜 하에 비교 수행.

해석 가능성 분석 (RQ2)

학습된 희소 개념의 질과 최종 마스크 정확도 사이에 강한 상관관계가 확인되었습니다.

정량적 분석

  • SAE 활성화 패턴과 IoU 간 높은 상관계수
  • 개념 특징 품질이 높을수록 마스크 정확도 향상
  • 희소성-재구성 트레이드오프 최적점에서 최고 성능

정성적 분석

  • 다중 슬롯 히트맵에서 명령어 각 부분의 시각적 정착 확인
  • 활성화 토큰 시각화로 개념 공간의 의미론적 구조 관측
  • 잘못된 세그멘테이션 시 희소 개념의 비정상적 활성화와 연관됨

"SegCompass는 단지 높은 성능을 달성하는 것이 아니라, 검사 가능한 정렬을 통해 그 성능을 달성한다."

결론: 모든 구성 요소가 성능에 기여하며, 특히 SAE 제거 시 감폭이 가장 크게 나타나 해석 가능 정렬 경로의 핵심적 역할이 입증됩니다.
