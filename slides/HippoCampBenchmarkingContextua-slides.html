실패 분석

주요 실패 모드 4가지

1. 검색 불일치 (Retrieval Mismatch)

RAG 시스템이 의미적으로 유사하지만 맥락적으로 무관한 문서를 검색함

2. 근거 회피 (Grounding Avoidance)

에이전트가 증거를 찾았지만 구체적 답변을 회피하고 모호하게 응답함

3. 강한 증거 환각 (Hard Evidence Hallucination)

터미널 에이전트가 파일 경로와 메타데이터를 조작하여 존재하지 않는 증거를 생성

4. 엔티티 오귀속 (Entity Misattribution)

올바른 증거를 찾았지만 잘못된 엔티티에 연결하는 귀속 오류 발생

증거 근거의 어려움

에이전트가 관련 파일을 찾더라도 정확한 증거에 기반한 응답을 생성하는 데 실패

📏

장기 검색의 한계

증거의 범위와 모달리티가 넓어질수록 성능이 급격히 하락

🔀

교차 모달 추론 부족

서로 다른 파일 형식에 분산된 정보를 종합하여 추론하는 능력이 현저히 부족
