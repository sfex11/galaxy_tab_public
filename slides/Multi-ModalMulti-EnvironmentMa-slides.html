Multi-Modal, Multi-Environment Machine Teaching

Ali Larian, Qian Lin, Chang Zong Wu, Daniel S. Brown

왜 다중 환경 티칭이 필요한가?

기존의 한계

  • 단일 환경에서만 피드백 수집
  • 데모stration만을 피드백으로 사용
  • 환경 특정 동역학에 과적합
  • 새로운 환경 배포 시 성능 저하

현실적 요구

  • 다양한 배치 환경에서 일관된 행동
  • 이질적인 피드백 양식 활용
  • 동역학 변화에 강인한 보상 함수
  • 제한된 예산 내 효율적 학습
서비스 로봇이 다른 물리적 레이아웃, 제약 조건에서도 일관된 인간 의도에 따라 행동해야 함

핵심 연구 질문

인간이 이질적인 피드백을 어떻게 활용하여,
에이전트의 행동이 여러 운영 조건에서도
인간 의도와 정렬되도록 가르칠 수 있는가?
Q1

각 피드백 유형이 보상 식별에
어떻게 제약을 가하는가?

Q2

단일 MDP에서의 보상 식별 가능성은
환경에 어떻게 의존하는가?

Q3

다중 MDP에서 환경과 피드백을
어떻게 효율적으로 선택하는가?

배경 지식

마르코프 결정 과정 (MDP)

M = (S, A, T, γ)

  • S: 상태 공간
  • A: 행동 공간
  • T: 전이 함수 (동역학)
  • γ: 할인 인자

보상 함수 R은 학습 대상으로,
환경 동역학 T와는 독립적

역강화 학습 (IRL)

인간 피드백으로부터 보상 함수 R 추론

  • 단순 모방이 아닌 의도 파악
  • 다양한 환경에 일반화 가능
  • 머신 티칭: 최소 피드백으로 목표 보상 유도

기존: 단일 MDP + 데모stration만

피드백 모달리티 분류

데모stration

최적 궤적 제공

환경 동역학과 보상 정보가 혼합

비교 (Comparison)

두 궤적 중 선호 표현

동역학 독립적 제약 부과

교정 (Corrective)

특정 상태-행동 쌍에 대한 피드백

국소적 제약, 낮은 비용

비상정지 (E-stop)

위험 상태에서 행동 중단

매우 국소적 제약, 최소 비용

각 모달리티는 보상 공간에 서로 다른 기하학적 제약을 형성

피드백 정보성 이론 분석

무한 데이터 체제

모달리티전역 제약 강도
비교가장 강함
데모stration중간
교정약함
비상정지가장 약함

비교는 동역학에 독립적인 strictly stronger global constraints 부과

제한된 예산 체제

모달리티쿼리당 효율
데모stration가장 효율적
비교중간
교정중간
비상정지낮음

예산이 타이트할 때 데모stration이 가장 제약 효율적

무한 데이터 vs 제한 예산에서 최적 모달리티가 다름 — 다중 모달 접근의 당위성

단일 MDP의 근본적 한계

환경 의존적 보상 식별 가능성

단일 MDP에서 무한한 피드백을 주어도
잔여 보상 모호성(residual reward ambiguity)이 존재

1
단일 환경
데모stration = 보상 × 동역학 혼합
2
보상 모호성
동역학에 의해 보상 방향 가려짐
3
일반화 실패
새 환경 배포 시 성능 저하

해결책: 다중 환경이 서로 다른 상보적 제약 방향을 노출

HSCOT: 계층적 집합 덮기 최적 티칭

Hierarchical Set Cover Optimal Teaching

보상 공간의 제약을 집합 덮기 문제로 공식화
환경 선택 → 피드백 선택의 2단계 계층 구조

외부 수준 (환경 선택)

상보적 제약 방향을 노출하는 가장 정보성이 높은 환경을 탐욕적으로 선택

내부 수준 (원자 선택)

선택된 환경 내에서 저비용 피드백을 전략적으로 쿼리하여 제약 원자 선정

HSCOT 알고리즘 상세

1
제약 우주 정의

모든 (환경, 모달리티, 인스턴스) 조합에 대해 초평면 제약 원자 생성

2
외부 루프: 환경 탐욕 선택

아직 덮이지 않은 제약 방향을 가장 많이 노출하는 환경 선택

3
내부 루프: 원자 탐욕 선택

해당 환경에서 새로운 제약 방향을 가장 많이 덮는 피드백 원자 선택

4
예산 소진까지 반복

피드백 예산이 소진될 때까지 2-3 반복

탐욕 집합 덮기의 ln(n)-근사 보장이 계층적 구조에서도 유지됨

실험 설정

도메인

  • GridWorld (다중 레이아웃)
  • 다양한 장애물 배치
  • 서로 다른 전이 동역학

베이스라인

  • 균일 티칭 (Uniform)
  • 단일 환경 티칭
  • 단일 모달리티 티칭

평가 지표

  • Held-out Regret
  • 제약 커버리지
  • 실현 가능 영역 부피

모든 방법에 동일한 피드백 예산 부여하여 공정 비교

실험 결과

Held-out Regret

HSCOT이 균일 티칭 대비 유의미하게 낮은 regret 달성

  • 보지 않은 환경에서도 강한 일반화
  • 예산이 작을수록 격차 더 벌어짐

제약 커버리지

HSCOT이 더 높은 제약 커버리지 확보

  • 보상 공간의 더 많은 방향 제약
  • 실현 가능 영역 부피 축소

선택적 환경 활성화

HSCOT은 모든 환경을 균등하게 사용하지 않고, 상보적 제약을 제공하는 핵심 환경에 집중적으로 피드백 할당

핵심 발견 요약

1

무한 데이터에서 비교 피드백이 가장 강한 전역 제약, 제한 예산에서는 데모stration이 가장 효율

2

단일 MDP에서는 환경 의존적 모호성이 불가피하며, 다중 환경이 필수적

3

HSCOT의 계층적 선택이 균일 티칭보다 동일 예산에서 월등한 성능

4

보지 않은 환경에서도 낮은 regret — 동역학 강인한 보상 학습 달성
