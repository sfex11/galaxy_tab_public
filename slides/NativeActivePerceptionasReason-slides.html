Zhenghao Xing, Ruiyang Xu, Yuxuan Wang, Jinzheng He, Ziyang Ma, Qize Yang, Yunfei Chu, Jin Xu, Junyang Lin, Chi-Wing Fu, Pheng-Ann Heng

문제 정의: 수동적 비디오 이해의 한계

Watch-It-All 패러다임

기존 모델은 비디오 프레임을 질문 난이도와 무관하게 균일하게 처리합니다. 비디오 길이가 길어질수록 계산 비용이 선형적으로 증가합니다.

컨텍스트 비용 폭발

시공간 데이터의 고차원성으로 인해 계산 비용이 시퀀스 길이에 대해 초선형적으로 증가하며, 장시간 비디오 처리는 사실상 불가능해집니다.

"인간의 지각은 정적인 스냅샷 처리가 아니라, 능동적이고 지속적인 신호 탐색 과정이다."

기존 접근법과 한계점

한계 외부 모듈 의존형
  • LLM이 전문 모듈(캡션, ASR 등)을 호출하는 방식
  • 중간 모듈이 정보 병목 생성
  • 추론과 지각 간 기울기 흐름 단절
한계 반수동형 (Think with Image)
  • 시간적 클리핑, 공간적 줌 등 변환 도구 사용
  • 전역 사전 스캔이 필수적
  • 밀집 시각 버퍼 유지 → 시간 단위 비디오 불가

핵심 문제: 추론 복잡도를 비디오 길이로부터 진정으로 분리하지 못함

OmniAgent: 핵심 아이디어

비디오 이해를 POMDP 기반의 반복적 Observation-Thought-Action (OTA) 사이클로 공식화합니다.

O
Observation
메타데이터, 기존 메모리 기반 현재 상태 인식
T
Thought
쿼리 조건부 탐색 계획 및 추론
A
Action
프레임/오디오/클립 요청 후 텍스트로 증류
단일 네이티브 모델

환경은 원시 미디어 추출만 수행하며, 모든 지각과 추론을 동일한 모델이 수행합니다.

복잡도 분리

내부 상태는 추론 궤적의 복잡도에만 의존하며, 비디오 원시 길이와 무관합니다.

메모리 통합: 정보 증류 아키텍처

두 가지 상태 공간
지속적 텍스트 메모리 ℳ

쿼리, 메타데이터, 모든 OTA 턴의 기록이 텍스트로 축적됩니다. 일정 차수의 미디어 오버헤드를 유지합니다.

일시적 멀티모달 지각 ℰ

각 턴에서 환경으로부터 반환된 고차원 원시 미디어(프레임, 오디오, 클립)입니다. 턴 종료 후 폐기됩니다.

액션 종류
Browse (프레임)
특정 타임스탬프의 프레임 요청
Listen (오디오)
특정 구간의 오디오 세그먼트 요청
Watch (오디오-비주얼 클립)
짧은 비디오 클립 요청
Answer (최종 응답)
충분한 증거 수집 후 종료

2단계 최적화 전략

Stage 1: Agentic SFT

네이티브 능동 지각을 부트스트래핑하기 위한 지도 미세조정

  • Best-of-N 궤적 합성 파이프라인
  • 이중 단계 품질 제어 (Dual-Stage QC)
  • 랜덤화를 통한 탐색 인센티브 부여
  • 기본 액션 실행 능력 확보
Stage 2: Agentic RL

추론 기반 지각을 정제하기 위한 강화학습 + TAURA

  • GRPO 기반 정책 최적화
  • TAURA: 엔트로피 기반 크레딧 할당
  • 핵심 발견 턴에 보상 집중
  • 추론-지각 통합 정제

Agentic SFT: Best-of-N 궤적 합성

1
탐색
랜덤화된 환경에서 N개 궤적 샘플링
2
1차 QC
형식 준수, 액션 유효성 필터링
3
2차 QC
답변 정확도 기반 순위 평가
4
선택
최고 궤적 1개를 SFT 데이터로 사용
1차 품질 제어

형식적 오류, 무효 액션, 중복 탐색 등을 자동으로 필터링하여 고품질 후보만 2차로 전달합니다.

2차 품질 제어

정답 일치도를 기준으로 궤적을 순위화하고, 가장 효율적인 탐색 전략을 보여주는 궤적을 선택합니다.

Advantage Homogenization 문제

표준 GRPO를 다중 턴 에이전트 추론에 적용하면 발생하는 근본적 한계

문제: 균일 궤적 수준 어드밴티지

전체 궤적에 단일 어드밴티지 값을 부여하면, 핵심 발견 턴(중요 정보를 처음 발견한 순간)과 사소한 액션 턴(단순 확인 등)이 동일한 보상을 받습니다.

결과적으로 모델이 어떤 턴이 진정 중요한지 학습하지 못합니다.

해결: TAURA

Turn-aware Adaptive Uncertainty Rescaled Advantage

  • 턴 수준 엔트로피를 불확실성 프록시로 활용
  • 높은 엔트로피(불확실성) 턴에 더 높은 크레딧 할당
  • 핵심 발견 턴에서의 정책 업데이트 가중치 증폭

TAURA: 엔트로피 기반 크레딧 할당

핵심 발견 턴에서 모델의 불확실성(엔트로피)이 높다는 관찰에 기반합니다.

결론 및 기여 요약

1

지각·추론·액션을 단일 모델 내에서 통합하고, POMDP 공식화로 추론 복잡도를 비디오 길이로부터 분리

2

3
