01 / 연구 배경

비균일 지역 커버리지 문제

다중 에이전트가 특정 공간 밀도 분포에 맞게 배치되어야 하는 임무 — 감시, 수색/구조, 환경 모니터링 등에서 핵심적인 과제입니다.

기존 방법의 한계

  • 오일러 방정식(PDE) 기반 — 계산 비용 과다
  • 가우시안 혼합 모델(GMM) — 매개변수 가정 필요
  • 휴리스틱 방법 — 최적성 보장 불가

본 논문의 해결책

  • 라그랑주 입자 기반 접근
  • 비모수 목표 밀도 지원
  • 엄격한 수렴 보장 제공

02 / 시스템 모델

확률적 LTI 동역학

이산 시간 확률적 시스템

x_i^(k+1) = A_i · x_i^k + B_i · u_i^k + w_i^k, w_i^k ~ N(0, Σ_w)
y_i^k = C_i · x_i^k + v_i^k, v_i^k ~ N(0, Σ_v)
x_i^k ∈ ℝⁿ : 상태 u_i^k ∈ ℝᵐ : 제어 입력 y_i^k ∈ ℝᵈ : 출력
가정 1

각 에이전트의 (A_i, B_i)는 완전 가제어(completely controllable)입니다.

가정 2

A_i는 최소한 한계 안정(marginally stable) — 모든 고유값이 닫힌 단위원 내에 존재합니다.

출력 상대 차수 (Output Relative Degree)

정의: C_i · A_i^(r-1) · B_i ≠ 0 을 만족하는 최소 양의 정수 r.
제어 입력이 출력에 k+r 스텝 후 영향을 미칩니다.

03 / 이론적 기반

최적 수송과 와서슈타인 거리

2-와서슈타인 거리 (Kantorovich 문제)

W₂(ρ, ν) = ( min_{π_ij ≥ 0} ΣΣ π_ij · d(y_i, q_j)² )^(1/2)

제약조건: Σ_j π_ij = α_i,   Σ_i π_ij = β_j,   ΣΣ π_ij = 1

시간 평균 경험적 분포

ρ^k := 1/((k+1)·n_a) · Σ_i Σ_t δ_{y_i^t}

에이전트의 과거 출력 위치를 시간에 대해 평균화한 경험적 분포입니다.

목표 분포

ν := 1/N · Σ_j δ_{q_j}

비모수적(non-parametric) 목표 밀도를 이산 샘플 포인트로 표현합니다.

핵심 목표: 시간이 지남에 따라 경험적 분포 ρ^k가 목표 분포 ν에 와서슈타인 거리 의미에서 수렴하도록 제어합니다.

04 / 핵심 공식화

D²OC 최적 제어 공식화

비용 함수

min_{U_i} J(U_i) := E[ Σ_{h=r}^{H+r-1} (W_i^{k+h})² ] + ||U_i||²_R

와서슈타인 비용 + 제어 정규화 최소화

명제 1 — 이차 재구성

와서슈타인 비용이 이차 형태로 변환됩니다:

J = U^T · H_i · U + 2f_i^T · U + const.

H_i = (Ω_i·Θ_i)^T·(Ω_i·Θ_i) + R
f_i = (Ω_i·Θ_i)^T·Ω_i·(Φ_i·E[x_i^k] − Q̄_i)

정리 1 — 최적 제어 법칙

비제약 해:

U_i^{k|H★} = −H_i⁻¹ · f_i

구 제약 투영 (따름정리 1):

u_i^{k+τ,★} = min(1, u_max / ||u_i^{k+τ,uncon}||) · u_i^{k+τ,uncon}

반복 솔버 없이 닫힌 형태 투영으로 해를 구합니다.

05 / 도달가능성 분석

도달가능성 기반 목표 선택

확률적 h-스텝 도달가능 집합

R_i^{k+h}(α) := { x ∈ ℝⁿ | ∃ 제어 입력 U ⊆ 𝒰, (x − μ)^T · Σ_{w,h}⁻¹ · (x − μ) ≤ χ²_{n,α} }

신뢰 수준 α에서 에이전트가 h-스텝 내 도달할 수 있는 상태 공간 영역을 정의합니다.

지역 샘플 선택 전략

min_{q_j, π_j} Σ_{j ∈ S_i} π_j · ||A_i^h · μ_i^k − q_j||²

공칭 예측에 가장 가까운 이웃을 순차적으로 선택하고, 샘플 용량 β를 고려하여 질량을 할당합니다.

투영 접근법

q̃ = argmin_{z ∈ C·M} ||z − q̄||²

무게중심이 도달가능 집합 밖이면 가장 가까운 점으로 투영합니다.

연성 제약 접근법

min Σπ_j||q_j − q̄||² + λ||q̄ − q̂||²

λ > 0으로 원하는 무게중심과 실현 가능성 사이의 트레이드오프를 조절합니다.

06 / 알고리즘 프레임워크

3단계 분산형 D²OC

1

도달가능성 기반 MPC

각 에이전트가 근접성과 잔여 가중치 기반으로 샘플 포인트를 선택하고, 지역 와서슈타인 거리를 최소화하는 제어 입력을 계산합니다.

2

가중치 업데이트

지역 와서슈타인 문제를 풀어 샘플 가중치를 갱신합니다. 최근 커버된 위치의 가중치를 감소시켜 에이전트가 새로운 영역을 탐색하도록 유도합니다.

3

가중치 공유 및 합의

통신 범위 내 이웃과 샘플 가중치를 교환합니다. 최소 가중치 합의(min-weight consensus)를 통해 과대평가를 방지합니다.

β_i,j ← min(β_i,j, β_{이웃,j})
MPC 계산
→
가중치 갱신
→
분산 합의
→
반복
07 / 알고리즘

분산형 D²OC 알고리즘

Algorithm 1: Decentralized D²OC with Reachability-Aware MPC
초기화: μ_i⁰, β_i,j⁰ 설정
for k = 0, 1, 2, ... do
// 단계 1: 도달가능성 기반 MPC
  for h = r to H+r-1:
    - 공칭 예측 A_i^h · μ_i^k 계산
    - 지역 샘플 Q_i^{k+h} 식별
    - OT로 무게중심 q̄_i^{k+h} 계산
    - 투영 또는 연성 제약 적용
  최적 제어 u_i^k 계산 → 적용
// 단계 2: 가중치 업데이트
  지역 OT → β_i,j^{k+1} 갱신
// 단계 3: 합의
  이웃과 β 교환
  β_i,j ← min(β_i,j, β_{이웃,j})
end

핵심 특성: 각 에이전트는 지역 정보만으로 독립적으로 제어를 계산하며, 이웃과의 간헐적 통신만으로 전역 커버리지를 달성합니다.

08 / 수렴 분석

수렴 보장 (정리 2)

정리 2 — 주요 수렴 결과

확률적 LTI 동역학 하에서, 분산형 가중치 공유를 통한 MPC 해를 적용하면:

E[W₂²(ρ^{k+1}, ν)] ≤ (1 − c/(k+1)) · E[W₂²(ρ^k, ν)] + C/(k+1)²

상수 c, C > 0은 가제어성, 노이즈, 통신 조건에 의존합니다.

극한 거동

lim_{k→∞} E[W₂²(ρ^k, ν)] ≤ ε_h

잔여 오차 ε_h는 통신 빈도, 프로세스/측정 노이즈 강도, 투영 오차에 의해 결정됩니다.

수렴 증명 핵심 단계

  • 엄격한 볼록성 → 유일한 최적 제어 보장 (정리 1)
  • 오차 분해: 투영 오차 + 확률적 추적 오차
  • L-부드러움을 통한 와서슈타인 하강 부등식
  • 소산적 드리프트 — 분산 합의가 과대평가 완화
  • Chung 보조정리 → O(1/k) 수렴 속도
