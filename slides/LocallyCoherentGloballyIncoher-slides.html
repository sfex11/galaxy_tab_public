Locally Coherent, Globally Incoherent: Bounding Compositional Incoherence in Multi-Component LLM Agents

Anany Kotawala

문제 정의: 국소 일관, 전역 불일관

다중 컴포넌트 에이전트

플래너가 검색, 산술, 확률 평가를 전문 서브에이전트에 분배하고, 각 컴포넌트는 공동 문제의 일부만 관찰함

핵심 모순

모든 컴포넌트가 국소적으로 일관(coherent)하더라도, 집계된 신념은 기본 확률 공리를 위반할 수 있음

"연구 컴포넌트가 P(공화당) = 0.6, 예측 컴포넌트가 P(민주당) = 0.6을 출력하면 1.2 질량을 할당하게 되어, 어떤 확률 측도도 부여할 수 없는 Dutch-book 노출이 발생한다."

기존 방법(보정, 자기일관성, conformal prediction)은 모두 개별 출력에만 작동하며 교차 컴포넌트 논리적 제약을 볼 수 없음

요구 제약: Σpᵢ = 1 (분할 공리)

실제 합산: Σpᵢ = 2.50

각 전문가는 자신의 섹터를 단일 베르누이 질문으로만 인식 — 섹터들이 분할을 이룬다는 사실을 아무도 보지 못함 → ε⋆ = 0.749로 실패 인증

형식화: 구성적 잔차 ε⋆

클리크와 일관 다면체

클리크 C = (Q₁, ..., Qₘ, R): m개 베르누이 질문과 논리적 관계 R의 튜플
일관 다면체 M_C: R과 호환되는 확률 측도가 존재하는 r ∈ [0,1]ᵐ의 집합 → 닫힌 볼록 다면체

구성적 잔차 정의

ε⋆(p̂) = ‖p̂ − Π_C(p̂)‖₂
구성된 인용과 일관 다면체 간의 L₂ 거리
런타임에 시스템 출력과 교차 컴포넌트 결합 제약만으로 계산 가능

ε⋆의 핵심 속성

  • 런타임, 분포 자유(distribution-free) 시스템 수준 일관성 인증서
  • 구성된 인용과 교차 컴포넌트 제약만으로 계산 가능
  • 소유자 선택 집계(owner-selected aggregation) 하에서 작동

곱 구조 이분법: 정리 3.3

소유자 선택 집계 하에서, 컴포넌트 일관성이 모든 입력에 대해 시스템 일관성을 보장하는 것은 오직 공동 다면체가 국소 다면체들의 데카르트 곱으로 인수분해될 때뿐이다.

곱 구조 (M⋆ = M₁ × M₂ × ...)

컴포넌트 간 결합 제약 없음
→ 국소 일관성 ≡ 전역 일관성
→ ε⋆ = 0 항상 성립

더 tight한 결합

교차 컴포넌트 논리 관계 존재
→ 국소 일관인 예측의 합성이 전역 불일관 가능
→ ε⋆ > 0 으로 실패 모드 인증

따라서 (Cor. 3.4) 더 tight한 결합 하에서는 국소 일관성만으로는 부족하며, 반드시 구성 수준의 검증이 필요함

크기 예측: Rayleigh-몫 형태 (Cor. 3.9)

존재적 이분법을 전문가 패널 공분산만으로 계산 가능한 폐쇄형 크기 예측으로 변환

Dutch-book 노출 한계

Exposure⋆ ≤ √(m⋆) · ε⋆
자산 가격 기본 정리(FTAP) 기반 시스템 수준 Dutch-book 노출의 상한

불일치가 잔차를 지배

컴포넌트 간 불일치(disagreement)가 구성적 잔차의 주요 구동 인자
컴포넌트 공분산 구조가 예측 가능성의 핵심

관계 클래스별 경도 순서 (이론 예측)

1
분할(Partition)
가장 tight한 제약 → 최대 잔차
2
부정(Negation)
강한 제약
3

해결책: 계층적 Boyle-Dykstra 투영

1
국소 JCD
각 컴포넌트를 국소 일관 다면체로 투영
2
결합 제약 투영
교차 컴포넌트 결합 제약으로 순차 투영
3
순환 반복
Boyle-Dykstra 사이클로 수렴까지 반복
4
ε⋆ → 0
결정론적 수리, 노출 한계를 수치 하한으로

핵심 장점

  • 전문가 라우팅 구조 보존
  • OSQP 기반 QP求解, KKT 잔차 ≤ 1.4×10⁻⁵
  • 폐쇄형 국소 투영 공식 제공 (부정, 논리곱, 선택, 분할)

핵심 메시지

다중 컴포넌트 LLM 에이전트에서 국소 일관성은 전역 일관성을 보장하지 않으며, 이 간극은 프롬프트나 검색으로 메울 수 없다. 구조적 투영이 필수적이며, 예측 가능하고 수리 가능하다.
