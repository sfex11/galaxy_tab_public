DiscoverPhysics

벤치마크를 통한 LLM의 과학적 발견 능력 평가

Matt L. Wiemann, Lindsay M. Smith, Peter Melchior, Siddharth Mishra-Sharma, Andrew Gordon Wilson, Pavel Izmailov, Carolina Cuesta-Lázaro

Princeton University · Boston University · NYU · Flatiron Institute · Institute for Advanced Studies

기존 벤치마크의 한계

지식 벤치마크 (GPQA 등)

모델이 무엇을 학습했는지 측정

최신 LLM은 GPQA에서 90% 이상 달성 — 하지만 이는 "앎"을 보여줄 뿐 "발견" 능력과는 다름

추론 벤치마크 (ARC 등)

주어진 입력에 대한 추론 측정

과학적 발견은 노이즈 환경에서 실험을 통해 관련 특징을 스스로 추론해야 함

기호 회귀 (LLM-SR 등)

데이터에서 방정식 복원

복잡한 함수 형태로 보간하는 경우가 많아, 생성 메커니즘과 무관할 수 있음

기존 인터랙티브 벤치마크

표준 법칙에 파라미터 변형

관련 변수 집합이 이미 알려져 있어, 과학적 발견의 개념적 부분을 탐색하지 못함

DiscoverPhysics란 무엇인가

우리 물리학과 의도적으로 다른 시뮬레이션 세계의 운동 법칙을 LLM 에이전트가 직접 발견하도록 요구하는 인터랙티브 벤치마크

  • 에이전트가 직접 실험을 설계하고, 노이즈가 있는 궤적 데이터를 관찰
  • 가설을 반복적으로 수정하며 장기적 추론(long-horizon reasoning) 수행
  • 최종적으로 자연어 설명 + Python 구현 제출
  • 22개의 비표준(non-canonical) 물리 세계로 구성

 예측 정확도

에이전트가 발견한 법칙이 hold-out 테스트 입자의 동역학을 얼마나 잘 예측하는지 측정

데이터에 맞추는 것만으로도 낮은 MSE 가능

설명 점수

 개념적 이해

전문가 작성 루브릭 기반 LLM 판사가 각 세계의 개념적 특징을 올바르게 식별했는지 평가

단순 예측을 넘어 원리 이해 필요

"로렌츠 변환은 경험적으로 도출되었으나, 아인슈타인이 단일 원리(광속 불변)에서 비롯됨을 인식하기 전까지는 근본적 이해가 불가능했다" — 예측과 설명의 분리의 역사적 예시

~50%

비표준 물리 세계

다양한 유형의 비정상 법칙

  • 잠재 구조 발견이 필요한 세계(숨겨진 입자, 추가 차원 등)에서 일관되게 실패
  • 좋은 예측 정확도가 높은 설명 품질을 보장하지 않음

예측과 이해의 분리

MSE 최저 모델 ≠ 설명 점수 최고

궤적 MSE가 가장 낮은 모델이 설명 점수에서도 가장 높은 것은 아님

추가 실험 라운드의 효과 차이

최저 MSE 모델은 추가 라운드의 이득이 적음 — 데이터에 맞추기만 하고 개념적 그림을 수정하지 않는 경향

핵심 기여

  • DiscoverPhysics 벤치마크 도입 — 비표준 물리 세계에서 능동적 실험을 통한 과학적 발견 능력 탐색
  • 듀얼 평가 프레임워크 정의 — 예측 정확도(궤적 MSE)와 설명 적절성(LLM 판사 점수)의 이중 축 평가
  • 11개 프론티어 및 오픈소스 LLM 벤치마크 — 최강 에이전트도 약 50% 통과율에 그침
  • 예측 정확도와 개념적 이해의 분리 현상 발견 — MSE 최저 모델이 설명 점수 최고가 아님
  • 오픈소스 모델의 현저한 격차 확인 — 실험 설계 및 결론 도출 양면에서 상용 모델 크게 뒤처짐
  • 시뮬레이터, 공개 세계 정의, 평가 프레임워크 공개 — 커뮤니티 확장 가능
