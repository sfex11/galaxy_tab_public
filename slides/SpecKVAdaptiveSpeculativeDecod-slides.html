Adaptive Speculative Decoding with Compression-Aware Gamma Selection

Shikhar Shukla · University of Kentucky

2026년 5월

LLM 추론 비용 문제

LLM 서비스의 총 소유 비용에서 추론 비용이 지배적인 상황에서, 추측 디코딩(Speculative Decoding)은 널리 채택된 가속 기법입니다.

추측 디코딩의 원리

작은 드래프트 모델이 γ개의 후보 토큰을 제안하면, 큰 타겟 모델이 단일 순전파로 검증합니다.

고정 γ의 한계

거의 모든 시스템이 γ=4로 고정하지만, 최적값은 작업 유형에 따라 최대 41.2% 차이가 납니다.

"압축과 추측 길이는 독립적이 아니라 서로 결합되어 있습니다."

핵심 인사이트

압축이 출력 분포를 변화시킨다

FP16 → INT8 → NF4로 압축될수록 타겟 모델의 출력 분포가 변하고, 이는 드래프트 토큰의 수락 확률에 직접 영향을 미칩니다.

최적 γ가 압축에 따라 이동한다

압축되지 않은 모델에 최적인 γ가 4비트 양자화 모델에서는 비최적이 될 수 있습니다.

압축 수준별 최적 γ 변화
FP16
최적 γ = 작업 의존
INT8
최적 γ가 이동
NF4
추가 γ 이동 발생

추측 디코딩의 작동 원리

1
드래프트 생성
작은 모델 Md이 γ개 후보 토큰 자가회귀 생성
2
타겟 검증
큰 모델 Mt이 단일 순전파로 전체 후보 평가
3
탐욕적 수락
일치하는 토큰 수락, 첫 불일치에서 타겟 예측으로 교체
4
k+1 토큰 출력
k개 수락 + 타겟 보정 토큰 1개 = 총 k+1개

수락률

k / γ — 드래프트가 얼마나 정확한지를 측정

γ의 트레이드오프

큰 γ → 잠재적 이득 ↑, 단계별 지연 ↑, 낭비 ↑

실험 설정

모델 구성

  • 드래프트: Llama 3.2 1B
  • 타겟: Llama 3.2 3B
  • NVIDIA RTX 3090

압축 수준

  • FP16 (무압축)
  • INT8 (BitsAndBytes)
  • NF4 (4비트 양자화)

γ 후보

  • γ ∈ {2, 4, 6, 8}
  • 4개 작업 범주
  • 총 5,112 단계 기록
코드 생성 수학 추론 오픈 채팅 요약 — 각 범주당 5개 프롬프트, 총 20개

신호 추출: 드래프트 모델로부터

추측 디코딩 과정에서 이미 계산되지만 버려지는 드래프트 확률 분포를 활용합니다. 추가 연산 비용은 제로(0)입니다.

평균 드래프트 엔트로피 H̄

γ개 후보 토큰의 엔트로피 평균. 낮을수록 드래프트가 확신 있음.

평균 드래프트 신뢰도 c̄

각 위치의 top-1 확률 평균. 높을수록 타겟과 일치 가능성 증가.

최대 엔트로피 Hmax

단계 내 가장 불확실한 토큰의 엔트로피. "병목" 신호 역할.

최소 신뢰도 cmin

가장 낮은 top-1 확률. 체인의 가약한 링크를 식별.

컨텍스트 벡터: x = [H̄, c̄, Hmax, cmin, comp]

수락률 예측과 γ 선택

γ 선택을 문맥적 결정 문제로 공식화합니다.

1
신호 관측
컨텍스트 벡터 x 추출
2
수락률 예측
f(x, γ) → â (회귀 모델)
3
기댓값 계산
E[tokens] = â · γ + 1
4
최적 γ 선택
argmax E[tokens]

결과 1: 압축이 최적 γ를 이동시킨다

FP16 베이스라인 프로파일링에서 작업별로 최적 γ가 다릅니다:

FP16 작업별 최적 γ
  • 코드 생성: γ=4 (119.7 tok/s)
  • 수학 추론: γ=6 (128.7 tok/s)
  • 오픈 채팅: γ=2 (108.0 tok/s)
  • 요약: γ=2 (111.6 tok/s)
핵심 관찰
  • 높은 γ에서는 모든 작업에서 처리량 저하
  • 저하 속도는 작업 의존적
  • INT8, NF4 압축 시 최적 γ가 추가 이동

고정 γ=4는 단일 압축 수준에서도 최적이 아닌 경우가 대부분이며, 압충 수준이 바뀌면 더 심화됩니다.

결과 2: 드래프트 신호가 수락률을 예측한다

≈ 0.56

드래프트 신호와 수락률 간 상관계수

강건한 예측성

  • 엔트로피와 신뢰도가 강한 예측 변수
  • 상관관계가 압축 수준 간에 일관되게 유지
  • 추가 연산 비용 없이 계산 가능

직관적 해석

  • 낮은 엔트로피 → 드래프트 확신 → 높은 수락률
  • 높은 신뢰도 → 타겟과 분포 유사 → 큰 γ 효과적
  • Hmax가 높으면 체인 내 병목 존재

"버려지던 정보가 강력한 의사결정 신호가 됩니다."

논의: 왜 압축이 최적 γ를 이동시키는가?

분포 왜곡 효과

양자화는 타겟 모델의 출력 확률 분포를 근사합니다. FP16에서 드래프트와 잘 일치하던 영역이 NF4에서는 왜곡되어 수락률이 변합니다.

γ-수락률 곡선 변화

압축으로 인해 E[tokens] = â·γ+1의 최댓값이 나타나는 γ 위치가 이동합니다. 드래프트의 "유효 범위"가 달라지는 것입니다.

한계점

  • 프롬프트 20개로 제한된 작업 다양성
  • 단일 모델 패밀리(Llama 3.2)만 검증
  • 오프라인 시뮬레이션 기반 평가
  • KV 캐시 압축(DMS 등)과의 결합 미탐구
