핵심 질문: 열린 문제

AdamW는 중꼬리 잡음 하에서 부호 기반 옵티마이저와 동일한 수렴률을 달성할 수 있는가,
아니면 2차 모멘트 누적기가 본질적 장애물을 만드는가?

긍정적 답의 의미

Lion/Muon의 실증적 우위가 수렴 이론 밖의 다른 요인(예: 일반화, 메모리)에서 비롯됨을 시사

부정적 답의 의미

이론적으로 AdamW가 중꼬리 LLM 학습에서 Muon에 지배될 수 있음을 설명

설정: AdamW 업데이트 규칙

모멘텀: mt = β1 mt−1 + (1 − β1) gt
2차 모멘트: vt = β2 vt−1 + (1 − β2) gt ⊙ gt
업데이트: xt+1 = (1 − ηtλ) xt − ηt mt / (√vt + εt)

분모의 역할

√vt은 과거 기울기의 2차 모멘트를 누적하여 적응적 학습률을 제공

핵심 모순

중꼬리 환경에서 2차 모멘트가 무한일 수 있는데, 분모가 이를 누적함

본 논문의 주요 분석 대상은 λ = 0 (가중치 감쇠 없음)인 핵심 경우

수학적 설정: 두 가지 핵심 가정

GC 기울기-곡률 조건

D(x) = diag(L0 + L1 ⊙ |∇f(x)|)로 정의되는 좌표별 곡률 행렬

∇f(x') − ∇f(x)의 D(x)−1 노름 ≤ x' − x의 D(x) 노름

비균일 곡률을 허용하는 일반화된 평활성 조건

HT 중꼬리 잡음 모델

무편향: E[gtb | Ft−1] = ∇f(xt)

E[|gt,ib − ∇if(xt)|p] ≤ σ0,ip + σ1,ip |∇if(xt)|p

p ∈ (1, 2]에서만 유한 모멘트 요구 (2차 모멘트 불필요)

부분 결과에서는 단순화된 경우 L1 = σ1 = 0 (상수 곡률, 기울기-독립적 잡음)를 고려

목표 수렴률: 비교 기준

부호 기반 옵티마이저의 달성률 (Yu et al. 2026)

O(T−(p−1)/(3p−2))

T에 대해 tight한 비증분(non-asymptotic) ℓ1-정상성 상한

p = 2

기존 유한 분산
O(T−1/4)와 일치

p → 1⁺

매우 무거운 꼬리
수렴이 극도로 느려짐

연속적

p의 연속적인 함수로
잡음 무거움 반영

핵심 질문: AdamW가 HT 가정만으로 이 동일한 rate를 달성할 수 있는가?

부분 결과: 가중 메트릭 벤치마크

Proposition 3.1 — 가중 측도 St에 대한 긍정적 결과

at,i = |mt,i| / (√vt,i + εt)   →   St = ⟨at, |∇f(xt)|⟩

(1/T) Σ E[St] ≤ O(T−(p−1)/(3p−2))  목표 rate 달성!

성취

가중 메트릭 St가 부호 기반 방법과 동일한 날카로운 rate를 달성함을 증명

한계

St ≠ ‖∇f(xt)‖1. 가중 진전을 순수 ℓ1 정상성으로 변환하는 것이 미해결

Corridor Lower-Bound 메커니즘

과거의 큰 기울기가 vt에 "복도(corridor)"를 만들어, 이후 단계에서 실제 기울기가 크더라도 at,i가 작게 묶인다. 부호 기반 방법은 이 문제가 없음 — 부호는 크기에 무관하게 ±1이므로. 이것이 St → ‖∇f‖1 변환의 본질적 장벽이다.

기존 이론과의 비교

옵티마이저 잡음 가정 수렴률 중꼬리 대응
AdamW (Li et al. 2025) 유한 분산 (p=2) O(√d / T1/4) 불가
AdaGrad 유한 p차 모멘트 수렴 보장 가능
Lion / Muon 유한 p차 모멘트 O(T−(p−1)/(3p−2)) Sharp & Tight
AdamW (본 논문) 유한 p차 모멘트 미해결 Open Problem

AdaGrad는 비재규적 분모로 인해 중꼬리 환경에서도 수렴 가능. AdamW의 재규적 2차 모멘트 누적이 핵심 차이점.

문제의 의도적 한정성

이 열린 문제는 의도적으로 좁게 설정됨:

  • HT 가정만 사용 (유한 분산 금지)
  • 단순한 경계 가정 배제
  • AdamW 특유의 장애물을 드러내야 함
  • 기존 유한 분산 하강 대수학의 단순 반복 금지

요약 및 전망

본 논문의 기여

  • AdamW의 중꼬리 수렴을 정식 열린 문제로 제형화
  • GC + HT 조건 체계 제안
  • 가중 메트릭 St에 대한 긍정적 벤치마크 증명
  • 분모 메모리의 corridor lower-bound 장벽 식별
  • St → ‖∇f‖1 변환의 AdamW 특유 난점 규명

향후 연구 방향

  • ℓ1 정상성 상한 증명 (긍정적 해결)
  • 하한 인스턴스 구성 (부정적 해결)
  • L1, σ1 > 0인 일반 경우로 확장
  • λ > 0 (가중치 감쇠 포함) 분석
  • 분모 메모리 장벽을 우회하는 AdamW 변형 탐색
