02 — 연구 방법론

③ 사용자 턴 (u)

✅ 진정한 후속 응답

어시스턴트의 응답에 기반한 맥락적으로 적절한 반응

  • 추가 질문
  • 명확화 요청
  • 응답에 대한 피드백

❌ 퇴화 출력

맥락을 무시하는 비정상적 생성

  • 이전 프롬프트 재진술
  • 어시스턴트 응답 복사
  • 메타 계획 텍스트
  • 역할 혼동

09 — 사후 학습 실험

상호작용 인식은 훈련 가능한가?

🎯 Qwen3.5-2B 대상 CollabLLM 레시피 적용 (~5,000 멀티턴 예시)

지도 미세조정 (SFT)

  • 후속 응답률: 1~2% → 46~48%
  • HealthBench: 36.7% → 54.4%
  • 정확도 트레이드오프: GSM8K 62.9% → 40.3%

강화 학습 (RL)

  • 과제 정확도 유지: 67.4%
  • 후속 응답률 적당한 향상
  • 명시적 사용자 턴 학습 없이도 개선

✅ 상호작용 인식은 학습 가능한 별도의 행동 차원이며,
강화 학습으로 정확도를 유지하면서 개선 가능

10 — 시사점

연구의 의의와 한계

🚀 시사점

  • 멀티 에이전트 시스템 — 에이전트 간 현실적 대화 연속성 필요
  • 셀프 플레이 학습 — 사용자 턴 생성 품질이 학습 루프 품질 결정
  • 배포 시나리오 — 대화형 AI의 맥락 인식 능력 재평가 필요
  • 평가 패러다임 — 어시스턴트 턴만으로는 불완전한 평가

⚠️ 한계

  • 판별기 검증이 GPT-5.4 계열에 한정
  • 영어 전용 — 다국어 일반화 미검증
  • 최선의 경우에도 후속 응답이 소수
  • 하류 과제 활용 미탐색
