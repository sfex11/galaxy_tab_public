세 가지 연쇄적 실패

1

경제적 실패

무상태 재주입으로 세션 비용 10배 증가. 동일 10,000작업 기준 CLI $3.20 vs MCP $55.20

2

인지적 실패

컨텍스트 활용률 ~70% 돌파 시 추론 품질 급락 — 파라미터 환각, 도구 혼동, 작업 기억 상실

3

적대적 실패 — Tool Poisoning

스키마 텍스트가 어텐션 마스크를 형성하므로, 악의적 도구 설명에 적대적 지시문 삽입으로 제어 흐름 탈취 가능

Tool Attention — 핵심 직관

"Scaled Dot-Product Attention이 시퀀스 모델링에서 순환을 동적 토큰 간 어텐션으로 대체했듯,
Tool Attention은 eager 스키마 주입을 동적 쿼리 조건부 도구 선택으로 대체한다."

구성 요소 상세 해부

구성 1 ISO 점수

사용자 쿼리 임베딩 q와 도구 스키마 요약 임베딩 s 간 코사인 유사도 계산. 상용 sentence-transformers 사용으로 모델 독립적. 낮은 ISO 점수 = 해당 도구에 대한 어텐션 에너지 기여 무의미 → 안전 제외.

구성 2 상태 인식 게이팅

ISO 점수에 논리적 게이트 결합: 인증 상태, 선행조건 만족, 도메인 접근 범위. 환각 게이트(Hallucination Gate) 의미론 — 모델이 호출할 수 없는 도구는 프롬프트에 진입 불가.

구성 3 2단계 지연 스키마 로딩

Phase 1 — 요약 풀

모든 도구의 자기 기술적 요약(1~2문장)을 컨텍스트에 상주. 전체 스키마 대비 ~95% 토큰 절약.

Phase 2 — 전체 스키마 승격

게이팅 통과 top-k 도구만 완전 JSON 스키마를 프롬프트에 주입. k는 임계값 기반 자동 결정.

이론적 근거: Total Attention Energy

MindGuard(2025)의 의존성 결정 그래프(DDG)와 총 어텐션 에너지(TAE) 공식에서 영감.

스키마 토큰에 대한 어텐션 가중치가 다운스트림 도구 호출에 대한 인과적 영향력과 강하게 상관됨.

방어적 활용: 특정 쿼리에 대해 TAE 기여가 무시할 수준인 도구 = 프롬프트에서 안전하게 제외 가능한 도구

Tool Poisoning 방어: 낮은 ISO 점수의 악의적 스키마가 어텐션 마스크에 진입하는 것 자체를 차단

직교적 조합: FlashAttention, KV-캐시 양자화와 함께 사용 시 프롬프트 토큰 감소가 선형적으로 효과 누적

투명성 원칙

중요 토큰 감소율은 직접 측정값. 작업 성공률, 지연, 비용은 측정 토큰 수 + 공개 원격 측정으로부터의 프로젝션이며, 라이브 에이전트 실행에서의 측정이 아님.

추가 분석 및 견고성

추론 품질

컨텍스트 활용률 70% 파단점 아래로 유지 → 모델이 에피소딕 작업 기억과 정밀한 파라미터 추론을 유지. 도구 혼동 현상 원천 차단.

스케일링 거동

도구 수 증가에 따른 절감 효과 비선형 증가. 120→500 도구 확장 시 절감률 추가 향상 예상.

절제 연구(Ablation)

ISO만 사용 vs 게이팅 추가 vs 2단계 로딩 추가 — 각 구성 요소의 독립 기여도 정량화. 세 구성 모두 필수적.

적대적 견고성

Tool Poisoning 공격에 대한 프로젝션: 낮은 ISO 점수의 악의적 스키마가 게이트를 통과할 확률 극히 낮음. TAE 기반 방어 메커니즘 내재.

한계 및 향후 과제

인정된 한계

  • 라이브 에이전트 측정 아님 — 시뮬레이션 기반
  • 적대적 의역(Paraphrase) 공격에 대한 추가 검증 필요
  • 턴 간 상태 인식 게이팅은 초기 구현

향후 방향

  • 학습 기반 게이팅 (현재 휴리스틱)
  • 코드 실행 패턴과의 조합
  • MOQT 프로토콜 수준 수렴 시 자연 폐기
  • 벤치마크 표준화 제안

"프로토콜 수준 효율성이, 원시 컨텍스트 길이가 아니라, 확장 가능한 에이전트 시스템의 결속적 제약( binding constraint)이다."

핵심 요약

문제 정의: MCP의 무상태 설계가 매 턴 10k~60k 토큰의 Tools Tax를 부과하고, 비용·추론·보안 세 축에서 연쇄적 실패 유발

해결책: Tool Attention — ISO 점수 + 상태 게이팅 + 2단계 지연 로딩의 모델 비의존적 미들웨어 메타 레이어

결과: 120도구 6서버 환경에서 토큰 95% 감소, 유효 컨텍스트 활용률 24%→91%

배포: 드롭인 미들웨어 — 오늘 배포 가능, MOQT 도입 시 자연 폐기. 오픈소스 참조 구현 공개.
