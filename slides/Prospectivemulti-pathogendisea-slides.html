배경

전염병 예측의 중요성

공중보건 핵심 도구

감염병의 확률적 예측은 자원 배분, 백신 접종 계획, 대응 전략 수립의 기반이 됩니다.

다병원체 동시 모니터링

인플루엔자, COVID-19, RSV 등 여러 병원체를 동시에 추적하고 예측할 필요가 증가하고 있습니다.

정확한 예측은 조기 경보 시스템의 핵심이며, 지연된 대응은 막대한 사회적 비용을 초래합니다.

문제 정의

수동 모델 구축의 병목

  • 전문가 팀이 각 병원체·지역별로 수동으로 모델을 설계·조정
  • 막대한 인적 자원과 시간 소요 — 주 단위의 수작업
  • 세분화된 지리적 해상도(군·시 단위)로 확장 불가
  • 신규 병원체 출현 시 신속한 대응 어려움

이른바 "Bespoke Development" 방식이 확장성의 근본 장벽이다.

핵심 메커니즘

LLM 유도 트리 탐색

트리 구조

전처리, 특성 엔지니어링, 모델 선택, 하이퍼파라미터 등 각 결정 노드가 트리를 형성합니다.

LLM의 역할

LLM은 도메인 지식을 활용하여 탐색 방향을 지능적으로 유도하고 불필요한 가지를 가지치기합니다.

전통적 그리드 서치나 무작위 탐색과 달리, LLM은 이전 탐색 결과와 도메인 지식을 결합하여 더 효율적으로 최적 조합을 발견합니다.

단일 모델이 아닌, 각 병원체의 특성에 맞춘 모델을 자동으로 구성합니다.

RSV 소아 집중 발생 연령 구조 반영 모델

실험

실험 설정

데이터

  • 미국 다수 주/군 수준 데이터
  • 다병원체 동시 예측 기간
  • Prospective (전망적) 평가

평가 지표

  • WIS (Weighted Interval Score)
  • 피해 폭 (Sharpness)
  • 신뢰 구간 적합도 (Calibration)

핵심: Retrospective가 아닌 Prospective 평가 — 실제 운영 환경과 동일한 조건에서 검증합니다.

핵심 발견

1

LLM의 도메인 지식이 탐색 효율을 극대화합니다 — 무작위 탐색 대비 훨씬 적은 평가 횟수로 최적 모델 발견

2

병원체별로 서로 다른 최적 모델 구성이 자동으로 발견되며, 이는 각 질병의 전파 역학을 반영

3

Prospective 평가에서 과적합 리스크가 통제됨 — 실 운영 배치의 신뢰성 확보

4

세분화된 지리적 해상도에서도 성능 저하가 제한적 — 지역 보건 당국에 실질적 가치 제공

  • 완전히 새로운 전파 패턴에 대한 일반화 미검증

향후 방향

  • 실시간 데이터 스트림과의 연동
  • 전 세계 다국가로의 확장
결론

요약 및 시사점

LLM 유도 트리 탐색 기반 자율 시스템은 전염병 예측 모델 구축의 수동 병목을 해소하면서도 전문가 수준의 예측 품질을 달성합니다.

  • 다병원체, 다지역 동시 예측의 확장성 확보
  • 신규 병원체 출현 시 신속한 모델 적응 가능
  • Prospective 검증으로 실 운영 신뢰성 입증
  • 공중보건 예측 인프라의 패러다임 전환 시사
