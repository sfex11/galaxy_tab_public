LACUNA

LLM 언러닝의 국소화 정밀도 평가를 위한
테스트베드

Matteo Boglioni, Thibault Rousset, Siva Reddy, Marius Mosbach, Verna Dankers

Mila – Quebec AI Institute · McGill University

문제 제기

LLM은 민감한 정보를 기억한다

프라이버시 위험

  • LLM은 학습 데이터 내 PII를 암기
  • 멤버십 추론 및 재구성 공격으로 유출 가능
  • 2차 프라이버시 위험: 추가 PII 암기 촉진

재학습의 한계

  • PII를 제거하고 재학습하는 것이 이상적
  • 하지만 현대 LLM 재학습 비용은 기하급수적
  • 사후 제거(post hoc) 방법의 필요성 대두
배경

LLM 언러닝이란?

학습 완료된 모델에서 특정 지식을 제거하여, 더 이상 원치 않는 출력을 생성하지 않도록 하는 알고리즘

언러닝이 만족해야 할 4가지 기준

1
유효성
forget 프롬프트에 대해 원치 않는 응답 편차
2
일반화
의미적으로 유사하지만 보지 못한 프롬프트에도 적용
3
유용성
retain 집합에 대한 출력 유지
4
비용 효율
처음부터 재학습보다 저렴해야 함

PII를 사전 정의된 파라미터에만 주입하여, 언러닝이 실제로 지식을 저장하는 가중치를 타겟하는지 직접 평가

구성 방법

Phase 1: PII 주입 파이프라인

학습 데이터 구성

  • PII 데이터: 합성 개인의 식별 가능 정보 생성
  • QA 데이터: PII 관련 질의응답 쌍 구성
  • 일반 데이터: Dolma 코퍼스의 서브셋 혼합

Masked 지속 사전학습

  • PII 토큰에 대응하는 파라미터만 업데이트 허용
  • 나머지 파라미터는 동결 (frozen)
  • 분산 학습과 완전 호환되는 설계

이후 Instruction Tuning을 수행하여 모델의 일반적 능력을 회복하면서도 PII 암기를 유지

구성 방법

Forget/Retain 집합 구성 및 주입 검증

Forget 집합

합성 개인의 PII를 직접 또는 간접적으로 조회하는 프롬프트-응답 쌍. 언러닝이 이 응답을 생성하지 않아야 함.

Retain 집합

PII와 무관한 일반 지식 질의응답. 언러닝 후에도 성능이 유지되어야 함.

주입 성공 검증

  • 능력 보존: 일반 벤치마크 성능 유지 확인
  • 암기 성공: forget 프롬프트에 대해 PII 정확히 생성하는지 검증
  • 마스크 외 파라미터에는 PII가 누출되지 않음을 확인
실험 설정

평가 대상 언러닝 방법

카테고리 방법 특징
기반 기반 GradAscent Forget loss에 대해 경사 상승
NPO Negative preference optimization
국소화 기반 MemFlex 암기 국소화 + 선택적 파라미터 수정
AlphaEdit Alpha 값 기반 파라미터 편집
오라클 OracleGrad Ground-truth 마스크를 알고 있는 이상적 기반 기반 언러닝
평가 지표

두 가지 차원의 평가 체계

출력 수준 지표

  • Unlearning Score: forget 응답과의 거리
  • Utility Score: retain 응답과의 유사도
  • Resurfacing: 공격 후 PII 재출현 여부

국소화 정밀도 지표

  • Overlap: 수정된 파라미터 ∩ ground-truth 마스크
  • Precision: 수정 중 마스크 내 비율
  • Recall: 마스크 중 수정된 비율
  • AUC: 파라미터 중요도 순위의 국소화 품질
핵심 결과

SOTA 방법의 국소화 정밀도는 극히 낮다

~0%

국소화 Precision

MemFlex, AlphaEdit 등 SOTA 방법들은
거의 무작위 수준의 정밀도

70%+

출력 수준 언러닝 점수

같은 방법들이 출력에서는
높은 언러닝 성능을 보여줌

출력 수준에서 "성공"한 언러닝도, 파라미터 관점에서 보면 지식을 은폐했을 뿐 진정한 삭제가 아님

이로 인해 재표면화(resurfacing) 공격에 취약해짐

오라클 분석

정확한 국소화가 언러닝의 핵심

OracleGrad

Ground-truth 마스크를 알고 있다고 가정한 단순 기반 기반 언러닝

  • 복잡한 알고리즘 없이 경사 상승만 사용
  • 마스크 내 파라미터만 수정

오라클의 성과

  • 모든 SOTA 방법보다 높은 언러닝 점수
  • 재표면화 공격에 강한 견고성 달성
  • 유용성(Utility)도 우수하게 유지

"국소화가 성공하면, 단순한 기반 기반 방법조차 현재 SOTA를 능가하는 삭제와 견고성을 달성한다"

결론

핵심 메시지

1
출력 수준 평가만으로는 진정한 지식 삭제를 보장할 수 없다
2
현재 SOTA 언러닝 방법들은 국소화 정밀도가 극히 낮다
3
정확한 국소화가 강건한 언러닝의 핵심 전제 조건이다
4
LACUNA는 행동 평가를 보완하는 새로운 평가 차원을 제공

LACUNA 공개 내용

  • 1B / 7B OLMo 기반 PII 주입 모델
  • 합성 개인에 대한 forget / retain 집합
  • Ground-truth 파라미터 마스크
  • 국소화 정밀도 평가 메트릭
