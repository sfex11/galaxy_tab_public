Reinforcement Learning without Ground-Truth Solutions can Improve LLMs

Yingyu Lin, Qiyue Gao, Nikki Lijing Kuang, Xunpeng Huang, Kun Zhou 등

기존 RLVR의 한계

기존 방식: 정답 매칭

  • 생성된 응답이 알려진 정답과 일치하는지 확인
  • 이진 보상 (0 또는 1) 제공
  • 수학·코딩에서 강력한 성능 달성

핵심 문제

  • 정답이 존재하지 않는 태스크에는 적용 불가
  • 최적해를 모르는 NP-hard 최적화 문제 등
  • 풍부한 실행 피드백이 있어도 활용 못함

"정답 매칭 RLVR은 강력하지만 좁은 패러다임이다. 많은 중요한 추론·코딩 태스크에는 단일 정답이 존재하지 않는다."

새로운 문제 설정: 점수 기반 최적화

AtCoder Heuristic Contest, Topcoder Marathon Match 같은 휴리스틱 알고리즘 대회를 학습 환경으로 활용

원시 점수를 그대로 쓸 수 없는 이유

도전 1 스케일 지배 (Scale Dominance)

인스턴스별로 점수 범위가 다름 (크기, 그래프 구조, 가중치 스케일 등)

  • 점수 범위가 큰 인스턴스가 보상 신호를 지배
  • 정책 업데이트가 의미 있는 품질 개선이 아닌 수치 스케일에 의존

도전 2 빈도 지배 (Frequency Dominance)

유사한 하위 최적 해가 반복 샘플링될 수 있음

  • 드물게 발견된 강한 솔버보다 반복된 약한 솔버가 더 큰 기여
  • 해 품질이 아닌 샘플링 빈도가 업데이트를 왜곡

RiVER 프레임워크

Ranking-induced VERifiable Framework

1

인스턴스별 순위화

각 히든 테스트 인스턴스 내에서만 후보 솔버를 상호 비교하여 순위 부여

스케일 지배 문제 해소

2

승자 가중 보상 형성

그룹 내 최고 솔버를 분리 강조하되, 유효한 비승자에게도 유한 바운드 피드백 제공

빈도 지배 문제 완화

"응답이 미리 계산된 해와 일치하는가?" → "생성된 해 A가 해 B보다 실행 평가에서 더 나은가?"

인스턴스별 순위화 상세

같은 인스턴스에서 실행된 솔버들끼리만 비교하여 순위 기반 점수로 변환

항목 원시 점수 사용 인스턴스별 순위화
인스턴스 간 스케일 보정 없이 혼합 → 왜곡 각 인스턴스 내에서만 정규화
이상치 영향 극단 점수가 지배적 순위로 대체하여 영향 최소화
정보 보존 점수 간격 유지 상대적 품질 순서 보존

핵심 아이디어: "몇 점인가?"가 아니라 "이 인스턴스에서 누가 더 나은가?"를 학습 신호로 사용

승자 가중 보상 형성

승자 (Winner)

그룹 내 최고 순위 솔버 → 높은 보상

유효한 비승자

제약 통과 but 하위 순위 → 유한 바운드 음수 보상

무효 솔버

제약 위반 → 고정 페널티

빈도 지배 완화 원리

비승자 보상이 하한 바운드로 제한되므로, 하위 해가 아무리 많이 샘플링되어도 기여도가 누적 폭발하지 않음

반면 승자는 명확히 구분된 높은 보상을 받아, 드문 발견이라도 정책 업데이트에 지배적 영향

  • SFT: 정답 매칭 데이터로 지도 학습

RiVER는 두 기반 모델 모두에서 Raw-GRPO 대비 월등한 ALE 평가 등급 향상을 달성

핵심 발견: 정답 매칭 벤치마크로의 전이

정답 없는 태스크로만 학습했는데, 정답이 있는 태스크도 개선됨

Raw-GRPO

  • ALE 점수 기반 평가는 개선
  • LiveCodeBench 전이 실패
  • USACO 전이 실패

실행 피드백만으로는 부족

RiVER

  • ALE 점수 기반 평가 개선
  • LiveCodeBench +2.4%p 절대 향상
  • USACO +3.5%p 절대 향상

적절한 보상 보정이 전이의 핵심

난이도별 분석 (LiveCodeBench)

RiVER는 Easy → Medium → Hard 모든 난이도에서 일관되게 성능 향상을 보임

Easy

↑

기존 강한 모델도 추가 개선

Medium

↑↑

가장 큰 향상 폭

Hard

↑

어려운 문제에서도 긍정적 전이

난이도에 관계없이 일관된 전이가 일어난다는 것은, RiVER가 단순한 트릭이 아닌 일반적인 코딩 능력을 향상시킴을 시사

정보이론적 관점: 그룹 상대 학습 신호

논문 부록에서 그룹 상대 학습 신호를 정보이론적으로 분석

그룹 피드백 충돌

여러 인스턴스에서 상반된 순위 신호가 동일 샘플에 부여될 수 있음

RiVER의 인스턴스별 순위화는 이 충돌을 자연스럽게 완화하는 구조

학습 신호의 정보량

승자 가중 형성은 강한 솔버의 신호에서 더 많은 정보를 추출하도록 설계

반복된 약한 신호의 엔트로피 증가를 억제하여 효율적 학습

결론 및 시사점

핵심 기여

  • 정답 없는 최적화 태스크를 LLM 학습 환경으로 제안
  • RiVER: 순위 유도 검증 가능 RL 프레임워크
  • 스케일 지배 & 빈도 지배 문제 식별 및 해결
  • 점수 기반 학습이 정답 매칭 벤치마크에도 전이됨을 실증

광범위한 시사점

  • 최적화 문제 = 평가 대상뿐 아니라 학습 환경
  • 보상 보정이 전이 가능성의 핵심 열쇠
  • RLVR를 정답 매칭 밖으로 확장하는 새로운 방향
  • 인간 라벨·참조 해 없이도 밀도 높은 학습 신호 확보 가능

"적절한 보상 보정과 결합한 점수 기반 최적화 태스크는 정답 없이도 일반적인 코딩 능력을 향상시키는 효과적인 학습 환경이 될 수 있다."
