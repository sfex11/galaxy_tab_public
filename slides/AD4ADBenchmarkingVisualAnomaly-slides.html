자율주행은 왜 위험한가?

학습 데이터에 포함되지 않은 상황에서 인식 성능이 급격히 저하됩니다

💥 안전 위협
📊
학습 분포 의존성
머신 비전 시스템의 신뢰성은 학습 데이터 분포에 크게 의존합니다. 비정형 장애물이나 이례적 상황에서는 인식 능력이 현저히 저하됩니다.
🚨
치명적 결과
다른 도메인과 달리, 자율주행에서의 오류는 승객, 보행자, 다른 도로 사용자에게 직접적인 물리적 위험을 초래합니다.
💡 해결책: 시각적 이상 탐지 (VAD)
학습 데이터에 없는 이상 객체를 탐지하여 운전자에게 경고하고, 픽셀 수준의 이상 맵으로 주의가 필요한 영역을 안내합니다.
핵심 기여

이 논문의 5가지 기여

  • ❶ 자율주행 도메인에서 VAD의 최초 체계적 평가 수행
  • ❷ AnoVox 데이터셋 기반 8개 최신 VAD 기법 벤치마크
  • ❸ 다양한 크기의 4가지 백본 아키텍처 비교 분석 (CNN + ViT)
  • ❹ 엣지 디바이스 배포를 위한 정확도-효율성 트레이드오프 분석
  • ❺ VAD가 도로 장면 탐지에 효과적으로 전이됨을 입증
🎯 핵심 발견
Tiny-Dinomaly가 엣지 배포에서 최고의 정확도-효율성 균형을 달성 — 전체 규모 모델과 동일한 위치 파악 성능을 메모리의 일부 비용으로 구현

평가 기법

8가지 VAD 기법

메모리 뱅크
PatchCore
정상 패치의 압축 메모리 뱅크를 구축하고, 테스트 패치와 최근접 이웃 간 거리로 이상 탐지
메모리 뱅크
PaDiM
각 공간 위치를 다변량 가우시안으로 모델링하고, 마할라노비스 거리로 편차 탐지
메모리 뱅크
CFA
정상 패치 임베딩의 메모리를 생성하고 결합된 초구에 적응시키는 방식
교사-학생
STFPM
지식 증류 기반 교사-학생 구조, 특징 맵 간의 불일치로 이상 탐지
교사-학생
RD4AD
STFPM을 개선한 오토인코더 접근법, 교사를 인코더로 활용
하이브리드
SSNet (SuperSimpleNet)
CFA와 유사한 특징 어댑터를 사용하며, 특징 수준에서 합성 이상을 생성
정규화 흐름
FastFlow
정규화 흐름 모델 기반, 입력 분포를 정규 분포로 변환하여 우도 기반 탐지
재구성
Dinomaly
동결된 사전학습 ViT 인코더와 디코더 구조, 정상 샘플로만 재구성 학습

🔍 비전 트랜스포머의 강점
ViT가 생성하는 표현은 더 풍부한 공간 정보를 인코딩하여 이상 위치 파악에 유리합니다. PaDiM의 P-AP가 WideResNet(0.28)에서 DeiT-Small(0.46)로 64% 향상되었습니다.

PRO 지표의 중요성
PRO는 크기에 관계없이 모든 이상 영역에 동일한 가중치를 부여합니다. 작은 이상도 큰 이상과 동일하게 중요하게 평가되어, 자율주행에서 핵심적인 소형 장애물 탐지 능력을 정확히 측정합니다.

한계점

반복되는 실패 유형

Dinomaly(DeiT-Small)의 이상 맵 분석에서 확인된 주요 실패 패턴

🔍 소형/원거리 객체 미탐지
매우 적은 픽셀을 차지하는 이상 객체는 탐지에 실패하는 경향이 있습니다. 패치 기반 특징 표현의 수용 영역이 너무 넓어 작은 이상을 포착하기 어렵습니다.
🛣 도로 곡선에서의 왜곡
커브 구간에서 원근 왜곡이 예측된 이상 맵의 공간 일관성을 저하시켜, 잘못된 이상 맵을 생성합니다.

✅ 성공 사례

  • •집중된 활성화 영역이 GT 이상의 공간 범위를 정확히 추적
  • •허위 활성화가 제한적
  • •P-F1 0.67, P-AP 0.66 달성

❌ 실패 사례

  • •소형 객체 누락
  • •곡선 도로 원근 왜곡
  • •수용 영역 한계로 미세 이상 미포착
결론 및 향후 과제

더 안전한 자율주행을 향하여

🏆
최고의 엣지 모델
Tiny-Dinomaly가 정확도-효율성 최적 균형 달성 (P-AP 0.66, 37.6MB, 19ms)
🧠
ViT의 우수성
비전 트랜스포머가 CNN 대비 이상 위치 파악에서 일관되게 우수한 공간 표현 제공
📋
최초 체계적 평가
자율주행 도메인에서 VAD의 최초 종합 벤치마크, 향후 연구를 위한 강력한 기준선 제공
🔭
향후 과제
소형 객체 탐지, 곡선 도로 강건성, LiDAR 다중모달 융합, 하류 계획 모듈 통합
