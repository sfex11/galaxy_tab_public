대형 언어 모델에서
가역적 SFT 행동 설계하기

Crafting Reversible SFT Behaviors in Large Language Models

Yuping Lin, Pengfei He, Yue Xing, Yingqian Cui, Jiayuan Ding, Subhabrata Mukherjee, Hui Liu, Zhen Xiang

Michigan State University · Hippocratic AI · University of Georgia

문제 제기

SFT는 행동을 유도하지만, 구조적 보장이 없다

지도 미세조정(SFT)은 안전 정렬, 지시 따르기, 도메인 적응 등 핵심 행동을 모델에 부여하지만, 이 행동들이 모델 내부에 어떻게 분포하는지에 대한 구조적 제약을 전혀 부과하지 않습니다.

문제 1: 인과적 진단 불가

사후 회로 발견은 상관관계만 식별합니다. 행동이 널리 퍼져 있으면, 어떤 하위구조가 진정으로 필요한지 결론을 내릴 수 없습니다.

문제 2: 인과적 제어 불가

기존 방법들은 파라미터 수준에서 개입하며, 고정된 가중치 하에서 입력만으로 주소 지정 가능한 하위구조를 만들지 않습니다.

핵심 통찰

상관관계 ≠ 인과적 필요성

기존 접근: 사후에 희소 하위네트워크를 발견하려 한다 → 상관관계만 얻음
본 연구 접근: 훈련 중에 희소 구조를 설계한다 → 인과적 필요성 확보

표준 SFT 모델에서는 행동이 광범위하게 분산되어 중복 경로가 보상을 상쇄할 수 있습니다. 반면, 설계에 의해 구조가 부과되면 캐리어 외부의 컴포넌트는 기본 모델 상태로 유지되어, 고정 가중치 하에서의 입력 개입만으로 인과적 병목을 검증할 수 있습니다.

연구 질문

SFT 행동을 희소하고 제어 가능한 캐리어에 압축할 수 있는가?

희소 행동 캐리어 (Sparse Behavioral Carrier)

SFT로 유도된 행동이 의도적으로 압축된 희소 파라미터 하위구조로, 두 가지 속성을 만족해야 합니다:

  • 행동 유지: 정상 추론 시 SFT 행동을 지속 가능한 최소 파라미터 구조
  • 가중치 고정 제어: 가중치 수정 없이 입력 개입만으로 SFT 행동을 신뢰성 있게 억제

핵심 전제: 희소성(Sparsity)은 두 속성 모두에 필수 — 압축된 구조만이 추론 시 정밀하게 표적화 가능

방법 개요

LCDD + SFT-Eraser 파이프라인

1
표준 SFT
행동이 모델 전체에 광범위하게 분포
2
LCDD
희소 캐리어로 행동 압축, 외부는 기본 모델 상태로 복원
3
SFT-Eraser
소프트 트리거 최적화로 캐리어 활성화를 기본 모델로 복원

정상 추론 (트리거 없음)

캐리어가 SFT 행동 유지

트리거 입력 시

캐리어 활성화가 기본 모델로 회귀

기초

가중치 변화 매개변수화와 마스크 구현

가중치 델타 정의

W = Wbase + M ⊙ ΔW

ΔW = Wft − Wbase : SFT로 유도된 파라미터 변화
M ∈ {0, 1} : 이진 라우팅 마스크 (어떤 파라미터가 캐리어에 속하는지)
M=1인 위치만 SFT 변화가 적용되고, M=0인 위치는 기본 모델 상태 유지

게이트-가중치 등가성

FFN 및 어텐션 레이어에서 활성화 게이팅 마스크가 이진 가중치 마스크와 수학적으로 등가임을 증명 (부록 A). 이를 통해 미분 가능한 게이트 최적화로 이진 마스크를 학습합니다.

Stage 1

LCDD: 효용 제약 희소 캐리어 설계

Loss-Constrained Dual Descent는 라우팅 마스크 M과 모델 가중치 ΔW를 명시적 효용 예산 하에서 공동 최적화합니다.

M

이진 라우팅 마스크
캐리어 구조 정의

ΔW

가중치 델타
캐리어 내 행동 학습

B

희소성 예산
캐리어 크기 제한

이중 하강 최적화

두 최적화 루프가 교대로 실행: (1) 가중치 고정 후 마스크 M을 경사 하강으로 갱신 → 캐리어 축소, (2) 마스크 고정 후 ΔW를 경사 하강으로 갱신 → 행동 복원. 회로 MSE 항이 표적 정밀도를 향상시키고 효용 비용을 감소시킵니다.

Stage 2

SFT-Eraser: 트리거 기반 행동 역전

소프트 프롬프트(트리거)를 최적화하여, 가중치 수정 없이 캐리어 채널의 활성화를 기본 모델 분포로 끌어당깁니다.

캐리어 채널 추출

LCDD 마스크 M에서 표적 잔차 채널을 식별. 트리거 최적화는 이 채널들에 대해서만 활성화 매칭을 수행합니다.

트리거 목표 함수

캐리어 채널에서의 활성화를 기본 모델 활성화에 매칭. 행동 억제와 분포 회귀를 동시에 달성하는 것이 성공의 기준입니다.

백도어 트리거와의 차이: 백도어 트리거가 특정 삽입 행동을 활성화하는 반면, SFT-Eraser 트리거는 SFT 이전의 기본 모델 기능을 복원합니다.

다양한 행동 유형과 모델 패밀리에서 검증

셰익스피어 스타일

LCDD 캐리어는 높은 LLM 판사 점수 유지
SFT-Eraser 트리거 시 스타일 점수가 기본 모델 수준으로 회귀
내용 유지율도 양호하게 유지

교차 작업 관찰: 한 작업(예: 고정 응답)용으로 설계된 캐리어의 트리거는 다른 작업(예: 안전)의 행동에 영향을 주지 않음 → 작업별 독립성 확인

핵심 증명

절제 연구: 희소 구조가 결정적 인자

발견 1: 희소 캐리어 구조가 결정적

동일한 트리거 최적화를 표준 SFT 모델에 적용 → 역전 실패

트리거 설계가 아닌 희소 구조 자체가 역전의 전제조건임을 확립합니다. 이는 학습된 캐리어가 단순한 상관관계가 아니라 인과적으로 필요함을 직접적으로 입증합니다.

발견 2: 회로 MSE 항의 역할

회로 MSE 정규화 항이:

표적 정밀도 향상 — 캐리어가 정확히 관련 컴포넌트에 집중
효용 비용 감소 — 더 적은 파라미터로 동등한 행동 유지

추가 절제 (부록 E): 마스크만 최적화 → 비교 불가능한 희소성 달성 불가 | 얕은 희소성 → 더 약한 병목

의의

기존 방법과의 본질적 차이

방법 구조 생성 가중치 고정 제어 인과적 필요성
사후 회로 발견 ✗ (발견만) ✗ 상관관계만

LCDD + SFT-Eraser ○ 설계 ○ 입력만 ○ 직접 검증
결론

희소 캐리어 설계로 SFT 행동의 인과적 지역화 달성

  • SFT 유도 행동을 희소하고 인과적으로 필요한 캐리어에 의도적 압축이 실현 가능함을 최초로 입증
  • LCDD가 명시적 효용 예산 하에서 마스크-가중치 공동 최적화로 희소 캐리어 구성
  • SFT-Eraser가 가중치 고정 하에서 입력 트리거만으로 3가지 행동 유형 × 4개 모델 패밀리에서 강력한 역전 달성
  • 절제 연구로 희소 구조가 역전의 결정적 전제조건임을 확인 — 구조가 아닌 트리거 설계가 실질적 인자

향후 방향

배포 모델에서의 추론 시 행동 감사, 선택적 억제, 더 복잡한 행동(다단계 추론 등)으로의 확장, 그리고 SFT 이후 개입 없이 구조를 유지하는 방법 탐구
