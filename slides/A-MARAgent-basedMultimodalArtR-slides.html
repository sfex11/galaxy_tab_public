문제 정의

미술 작품 이해는 왜 어려운가?

미술 작품을 깊이 있게 이해하려면 단일 이미지 분석을 넘어 시각적 요소, 상징적 의미, 양식적 관례, 문화·역사적 맥락을 다단계 추론으로 연결해야 합니다.

"르네상스 회화를 해석하려면 시각적 모티프를 식별하고, 이를 종교적·철학적 상징과 연결하며, 더 넓은 역사적 서사 안에 위치시켜야 한다 — 이는 순서가 있는 다단계 추론 과정이다."

시각적 관찰
색채, 구도, 인물, 사물 등 표면 요소 식별
상징적 연결
모티프와 종교·철학·문화적 의미의 매핑
맥락적 해석
역사·양식·작가 맥락에서 종합적 의미 구성
기존 방법의 한계

정적 검색 vs 구조적 추론의 간극

최신 MLLM은 이미지 캡셔닝에 강하지만, 미술 분야에서는 암묵적 추론과 내재화된 지식에 의존하여 해석 가능성과 근거 기반이 부족합니다.

기존 정적 RAG
  • 사용자 질문을 그대로 검색 쿼리로 사용
  • 평면적·무순서 문서 집합 반환
  • 모든 근거가 동일한 역할 수행
  • 추론 단계별로 다른 유형의 증거 필요 무시
  • 검색이 추론 구조에 무관함
A-MAR (제안)
  • 명시적 추론 계획 기반 검색 조건화
  • 각 단계의 목표와 증거 요구사항 정의
  • 단계별 맞춤형 근거 검색
  • 시각·메타데이터·구조 지식 조합
  • 해석 가능하고 통제 가능한 추론
프레임워크 개요

A-MAR: 에이전트 기반 멀티모달 미술 검색

A-MAR은 추론, 검색, 생성을 명시적으로 분리한 에이전트 프레임워크입니다.

작품 I + 메타데이터 M
질문 Q
추론 계획 P
조건부 검색
근거 기반 생성 A
1단계: 추론 계획
질문을 순서가 있는 추론 단계로 분해, 각 단계의 증거 유형 명시
2단계: 조건부 검색
계획에서 도출된 검색 의도로 단계별 맞춤 증거 검색
3단계: 근거 기반 생성
계획을 따라 단계별로 검색된 증거에 기반한 설명 생성
핵심 모듈 1

에이전트 기반 추론 계획

VLM 기반 플래너가 작품 이미지, 메타데이터, 질문을 입력받아 구조화된 추론 계획을 생성합니다.

Pgen = { (rt, et) }Tt=1 rt: 추론 하위 목표  |  et ∈ ℰ: 필요 증거 유형 (시각/메타데이터/맥락 지식)
qret = ψ(Pgen) 질문의 표현이 아닌, 플래너의 증거 요구를 구조화한 검색 쿼리

핵심 차별점: qret는 질문 Q의 단순한 의역이 아닙니다. 플래너가 무엇을 왜 검색해야 하는지를 구조화하여, 추론 흐름과 검색 행동을 명시적으로 연결합니다.

핵심 모듈 2

추론 조건부 검색

추론 계획에서 파생된 검색 의도를 통해 멀티홉 검색을 수행하고, 검색된 증거를 재순위화하여 압축 컨텍스트를 구성합니다.

검색 의도 생성
계획 Pgen의 각 단계별 증거 요구를 집계하여 구조화된 검색 쿼리 qret 생성
멀티모달 멀티홉 검색
미술 지식 베이스(작가, 양식, 시대, 상징 의미 등)에 대해 단계별 타겟팅 검색 수행
증거 재순위화
추론 계획과의 관련성을 기준으로 검색 결과 재정렬, 중복 제거
컨텍스트 조립
재순위화된 증거를 단계별로 그룹화하여 압축된 구조적 컨텍스트 구성
핵심 모듈 3

근거 기반 단계별 설명 생성

최종 생성 단계에서는 멀티모달 LLM이 추론 계획을 따라 각 단계를 검색된 증거에 기반하여 순차적으로 설명합니다.

일반 MLLM
  • 단일 패스 합성
  • 내재화된 지식에 의존
  • 환각 가능성 높음
  • 추론 과정 불투명
A-MAR 생성
  • 계획 기반 단계별 생성
  • 외부 근거에 명시적 기반
  • 각 단계마다 근거 표기
  • 추론 과정 해석 가능

판프스키의 해석학에서 영감: 전통 미술사에서도 분석은 시각 관찰 → 맥락화 → 해석의 순서로 진행됩니다. A-MAR은 이 순서를 에이전트가 명시적으로 모델링합니다.

벤치마크

ArtCoT-QA: 다단계 추론 미술 QA

기존 미술 설명 데이터셋은 최종 설명만 제공합니다. ArtCoT-QA는 다단계 추론 체인과 단계별 근거 주석을 제공하여 추론 과정 수준의 평가를 가능하게 합니다.

다단계 CoT 주석
질문-답변 쌍에 명시적 사고 사슬(chain-of-thought) 주석
단계별 근거 grounding
각 추론 단계가 참조하는 구체적 증거를 명시
세분화 평가
최종 정답뿐 아니라 단계 완결성, 추론 충실성, 근거 적합성 평가
다양한 미술 질문
상징 해석, 양식 분석, 역사적 맥락 등 다양한 유형 포함
