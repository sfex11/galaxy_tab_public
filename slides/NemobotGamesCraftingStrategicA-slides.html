01
동기

자가 프로그래밍이라는 오래된 질문

"컴퓨터가 따르는 명령의 99%를 컴퓨터 스스로 작성하게 할 수 있는가?"

Claude Shannon, 1950

Shannon은 디지털 컴퓨터가 자율적으로 명령을 생성하는 자가 프로그래밍 시스템을 구상했습니다. Marvin Minsky는 프로그래밍의 구조화된 규칙이 제약이 아니라 창의성의 도구라고 주장했습니다. 오늘날 LLM은 이 비전을 현실로 만들고 있습니다.

지능 vs 프로그래밍

LLM에 게임을 시키는 것은 비결정적이고 불투명합니다. 프로그래밍 프레임워크는 재현 가능하고 교육적인 구조를 제공합니다.

에이전트 엔지니어링

학습된 표현을 프로그래밍적으로 활용하여 하이브리드 인간-AI 솔루션을 만드는 새로운 패러다임입니다.

06
유형 1

사전 기반 게임: 틱택토

모든 상태-행동 매핑을 사전에 저장하는 전통적 방식을 LLM이 효율적으로 압축합니다.

전통적 사전 방식

  • 모든 보드 상태와 최적 수를 미리 저장
  • 메모리 소모가 큼 (틱택토: 5,478개 위치)
  • 확장성 제한 — 복잡한 게임에 적용 불가

LLM 압축 모델

  • 상태-행동 매핑을 일반화된 모델로 압축
  • 신경화 메모이제이션으로 효율적 저장
  • 새로운 상태에 빠르게 적응 가능

게임판은 미시세계입니다 — 이산적이고 관찰 가능한 세계 상태로, 세계 모델의 표현 학습을 연구하기 위한 구조화된 시험대가 됩니다.

07
유형 2

수학적 엄밀 게임: 님(Nim)

완전히 해결 가능한 게임에서 LLM은 수학적 추론으로 최적 전략을 계산하고, 인간이 이해할 수 있는 설명을 생성합니다.

님 게임 규칙

여러 줄의 돌에서 번갈아 돌을 가져갑니다. 마지막 돌을 가져가는 사람이 집니다. XOR 분석으로 완벽한 해결이 가능합니다.

LLM의 역할

기호 추론을 통해 님 합(Nim-sum)을 계산하고, 각 수에 대한 인간 친화적 설명을 자연어로 생성합니다.

// 님 게임 예시 상태
줄 1: ●●●● (4개)   줄 2: ●● (2개)   줄 3: ● (1개)
Nim-sum = 4 XOR 2 XOR 1 = 7 ≠ 0 → 선공 승리 가능
LLM: "줄 1에서 3개를 가져가면 nim-sum이 0이 되어 상대를 패배 위치로 보냅니다."

완전 해결이 불가능한 복잡한 게임에서 LLM은 고전적 알고리즘과 크라우드소싱 데이터를 통합하여 전략을 합성합니다.

미니맥스 알고리즘

게임 트리 탐색으로 최적 수를 예측하지만, 만칼라의 복잡성으로 인해 제한된 깊이까지만 탐색 가능합니다.

크라우드소싱 통합

09
유형 4

학습 기반 게임: 강화학습

경험을 통해 적응하는 학습 기계에서 LLM은 인간 피드백 기반 강화학습(RLHF)과 자기 비판을 통해 전략을 반복적으로 정제합니다.

1
LLM이 초기 전략으로 게임 플레이
2
인간 플레이어와 대결 후 결과 수집
3
자기 비판(self-critique): LLM이 자신의 수를 분석하고 개선점 식별
4
모방 학습(imitation learning)으로 성공 패턴 내재화
5
정제된 전략으로 반복 — 자가 프로그래밍 루프 완성
10
방법론

협력적 프롬프트 엔지니어링

다수 사용자가 크라우드소싱 방식으로 게임 전략 프롬프트를 공동 설계하고, 집단 지성을 통해 AI 게임 개발의 창의성을 foster합니다.

프로그래밍 가능한 프롬프트

단순한 텍스트가 아닌, 구조화된 함수형 프롬프트로 게임 전략을 정의. 매개변수 조정, 조건 분기, 루프 등 프로그래밍 개념을 활용.

크라우드소싱 생태계

학생과 연구자가 자신의 전략 프롬프트를 공유하고, 다른 사용자의 전략을 학습하여 상호 향상하는 선순환 구조.

자기 강화 사이클

LLM이 최적화된 전략을 생성 → 인간 플레이어와 상호작용 → 게임 결과를 피드백으로 수집 → LLM 재학습 → 더 나은 전략 생성의 무한 루프

결과

평가 결과와 핵심 기여

원격 학습 환경 평가

게임화된 AI 교육에서 학생들의 기초 AI 개념 이해도가 유의미하게 향상되었습니다. 협력적 프롬프트 엔지니어링을 통해 상호작용적 학습 촉진.

4가지 게임 유형 검증

틱택토, 님, 만칼라에서 각각 LLM 강화 방식의 유효성을 입증. 강화학습 기반 게임에서는 반복적 자기 정제의 효과 확인.

세 가지 핵심 기여

  • Nemobot Games 프레임워크 — Shannon 분류 체계와 LLM의 통합. 추론, 의사결정, 신경화 메모이제이션 함수를 통한 시행착오 학습.
  • 크라우드소싱 기반 협력적 프롬프트 엔지니어링 — 프로그래밍 가능한 게임 플레이 전략과 인간 피드백을 통한 창의적 AI 게임 개발.
  • 게임화된 AI 교육 검증 — 원격 학습 환경에서 협력적 프롬프트 엔지니어링이 학생들의 AI 개념 습득을 효과적으로 지원함을 입증.
