문제 인식
왜 RadAgent가 필요한가?
🔒 기존 VLM의 한계
3D 비전-언어 모델(VLM)은 흉부 CT 보고서를 생성하지만, 중간 추론 과정을 전혀 보여주지 않음. 임상의는 최종 출력의 수동적 관찰자에 머무름.
⚠️ 신뢰성 위기
고위험 의료 환경에서 AI의 판단 과정을 검증·수정할 수 없는 블랙박스 구조는 임상 채택의 핵심 장벽.
💡 RadAgent의 해법
CT 해석을 도구 기반의 단계적 추론 트레이스로 구조화하여, 각 보고서에 검사 가능한 중간 결정 및 도구 상호작용의 완전한 기록을 제공합니다.
기존: CT → 블랙박스 → 보고서
vs
RadAgent: CT → 단계적 추론 → 보고서

커리큘럼 보상 전략

초기 단계 (1~90 스텝)
R_early = R_quality + 0.5·R_div + 0.5·R_coh + 0.1·R_succ
다양한 도구 시퀀스 탐색 장려 → 정책 붕괴 방지
후기 단계 (91~150 스텝)
R_late = R_quality + 0.2·R_div + 0.2·R_coh + 0.1·R_succ + 0.2·R_toolJudge
체크리스트 준수 강화 → 구조화된 진단 워크플로우 정착

보상 구성 요소

R_quality — 보고서 품질 (F1₁₈ + F1_abnorm) R_succ — 도구 성공률 R_div — 도구 다양성 R_coh — 도구 그래프 일관성 R_toolJudge — LLM 기반 체크리스트·시퀀스 판단

견고성 & 충실성
적대적 힌트 실험
잘못된 임상 힌트에 대한 AI의 저항력과 투명성 측정
🧪 실험 설계
CT-RATE 테스트셋에서 1,000건을 무작위 추출, 각 사례에 대해:
✅ 올바른 힌트
"이 스캔에서 [실제 존재하는 병리]가 보입니다"
❌ 잘못된 힌트
"이 스캔에서 [존재하지 않는 병리]가 보입니다"
견고성 (Robustness)
잘못된 힌트에도 올바른 예측을 유지하는 비율.
RadAgent의 도구 기반 증거 축적이 허위 힌트를 "근거 없음"으로 식별.
충실성 (Faithfulness)
힌트에 의해 판단이 바뀌었을 때, 보고서에 그 영향을 솔직히 언급하는 비율.
CT-Chat은 힌트 영향을 숨기고 그럴듯한 보고서를 생성.

시퀀스 보상 없음
R_quality + R_div + R_coh + R_succ만 사용.
체크리스트를 무시하는 정책으로 붕괴. 비일관적 도구 호출, 낮은 보고서 품질.
처음부터 시퀀스 판단
R_late를 처음부터 적용.
초기 탐색 억제 → 보고서 품질과 체크리스트 준수 사이의 트레이드오프.
핵심 인사이트

후기의 구조 강화가 안정적인 워크플로우 정착을 보장합니다.
핵심 통찰
RadAgent의 의의
🤖 RL을 통한 정책 자동 발견
수동 워크플로우 설계나 프롬프트 엔지니어링 대신, 강화학습이 효과적인 도구 사용 전략을 자동으로 발견. 훈련 없는 RadAgent도 CT-Chat을 능가하지만, RL이 특히 외부 일반화를 크게 향상.
👨‍⚕️ 인간-AI 협업의 새 패러다임
임상의가 중간 소견을 직접 질의·검증하는 Human-in-the-Loop 워크플로우 실현. 보고서 생성 → 임상의 검증 → 필요시 수정의 순환 구조.
🔌 전문화와 범용성의 융합
RadAgent가 유연한 프론트엔드로서 다양한 임상 요구를 수용하고, 전문 도구에 하위 작업을 동적 라우팅. 특히 어려운 병리에서 전문 도구 접근이 큰 성능 향상을 가져옴.
⚗️ 워크플로우 증류 가능성
학습된 정책을 고정 추론 워크플로우로 증류하여 계산 효율성 향상 및 규제 환경에서 요구하는 안정적·전향적 검증이 가능한 시스템 동작 제공.
한계 & 전망
한계점 및 향후 연구

한계점

💻 높은 계산 비용
2개 노드 8개 GPU 필요. 자원 제약 환경에서는 배포 어려움. 단, 학습 후 드물게 사용되는 도구 비활성화로 일부 완화 가능.
🔧 도구 세트 의존성
학습된 에이전트는 학습 시점의 도구에 최적화. 도구 세트가 크게 변경되면 RL 파이프라인 재실행 필요.
📉 충실성 개선 여지
37.0%의 충실성은 CT-Chat(0%)에 비해 크게 향상되었으나, 상당한 개선 여지가 남아 있음.

향후 연구 방향

충실성 37% → 더 높은 수준으로 도구 세트 확장 워크플로우 증류 최적화 흉부 CT 이외 영상 모달리티 실제 임상의와의 Human-in-the-Loop 평가 규제 준수 시스템 구축
