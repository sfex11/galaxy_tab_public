LLM을 임상 그래프 구조 정제기로 활용한
EEG 뇌전증 발작 진단의 표현 학습 향상

LLM as Clinical Graph Structure Refiner: Enhancing Representation Learning in EEG Seizure Diagnosis

Lincan Li, Zheng Chen, Yushun Dong

Florida State University / Osaka University

arXiv:2604.28178v1

배경

뇌전증 진단과 EEG 신호의 중요성

뇌전증의 임상적 중요성

  • 전 세계 수백만 명이 영향받는 만성 질환
  • 정확한 발작 감지가 timely 진단과 치료의 핵심
  • 환자 삶의 질 향상을 위한 자동화 필요

EEG 신호의 역할과 한계

  • 뇌 활동 모니터링의 1차 데이터원
  • 노이즈, 아티팩트, 환자 간 변이성에 취약
  • 발작 이벤트의 지속시간·주파수·공간적 양상이 개인마다 크게 상이
전통적 수동 분석은 시간 소모적이고 인간 오류에 취약하며, 기존 딥러닝 방법은 채널 간 의미론적 맥락과 고차 관계를 효과적으로 모델링하지 못한다.
문제 정의

EEG 기반 그래프 구성의 근본적 한계

상관관계 기반

Pearson, 상호상관 등 통계적 척도로 엣지를 정의하지만, 노이즈에 취약하여 불안정한 구조 생성

학습 기반

어텐션·마스크 학습으로 적응적 연결을 구성하나, 명시적 생리학적 근거가 부족

생성 기반

잠재 표현에서 그래프를 합성하지만, 생물학적 의미와 해석 가능성이 결여

공통 문제

EEG 신호의 노이즈로 인해 중복·무관·허위 엣지가 생성되며, 이는 GNN의 표현 학습을 오도하고 발작 감지 성능을 저하시킨다. 기존 방법은 엣지 정제를 위한 명시적이고 원칙적인 메커니즘이 부족하다.

핵심 아이디어

왜 LLM을 그래프 엣지 정제기로 사용하는가?

LLM의 고유 강점

  • 추론 능력: 통계적 증거와 의미론적 맥락을 종합하여 의미 있는 연결과 무관한 연결 구분
  • 맥락 이해: 다양한 정보 원본(텍스트, 통계)을 통합한 판단 가능
  • 해석 가능성: 엣지 유지/제거의 이유를 언어로 설명 가능
  • 임상 지식 활용: 발작 전파 패턴 등 생리학적 지식 반영

기존 vs 제안 접근

기존 방법 제안 방법
암묵적 구조 학습 명시적 엣지 정제
생리학적 근거 부족 임상 지식 통합
노이즈에 취약 추론 기반 강인성
해석 불가 정제 이유 설명 가능

핵심 설계 철학: LLM은 그래프를 처음부터 생성하는 것이 아니라, 신경망이 구성한 초기 엣지 집합을 검증하고 정제하는 역할을 수행한다.
1단계

Transformer 기반 초기 그래프 구성

엣지 예측 메커니즘

  • EEG 채널 쌍의 시계열을 입력으로 받음
  • Transformer 인코더로 채널 쌍 특징 추출
  • MLP를 통해 엣지 존재 확률 점수 산출
  • 임계값(τ) 적용으로 초기 엣지 집합 결정

설계 의도

Transformer는 채널 간 시계열 패턴을 포착할 수 있으나, 노이즈로 인해 여전히 불필요한 엣지가 포함될 수 있다. 이를 LLM이 2단계에서 해결한다.

엣지 존재 확률

P(eij) = MLP(Transformer(hi, hj))

eij ∈ Einit if P(eij) > τ

hi, hj: 채널 i, j의 특징 표현
τ: 엣지 존재 임계값
Einit: 초기 엣지 집합

2단계

LLM 기반 그래프 구조 정제

정제 프로세스

1
엣지 후보 추출: Einit의 각 엣지 (i,j)에 대해 노드 쌍 정보 수집
2
프롬프트 구성: 노드 쌍의 텍스트 특징(채널 위치, 기능 영역)과 통계 특징(상관계수, 확률 점수)을 결합
3
LLM 추론: 임상적 맥락과 통계적 증거를 기반으로 엣지 유지/제거 결정
4
정제된 그래프: Erefined ⊆ Einit으로 최종 그래프 구성

LLM 입력 정보

특징 유형 예시
텍스트 특징 채널 명칭, 해부학적 위치, 뇌 영역 기능
통계 특징 채널 간 상관계수, 엣지 확률 점수
구조 특징 양쪽 노드의 기존 연결 수, 이웃 정보

LLM은 통계 수치만이 아닌 의미론적 맥락까지 고려하여 판단

메커니즘

엣지 정제의 핵심 원리

유지되는 엣지

  • 높은 상관계수 + 생리학적으로 합리적인 연결
  • 발작 전파 경로와 일치하는 방향성
  • 인접 뇌 영역 간의 의미 있는 상호작용

제거되는 엣지

  • 통계적으로는 높으나 노이즈 기반의 허위 상관
  • 생리학적으로 설명 불가능한 원거리 연결
  • 중복된 정보를 전달하는 여분의 엣지

설계의 핵심 통찰

LLM은 그래프를 새로 구성하지 않고 기존 엣지를 "심사"하는 역할만 수행한다. 이는 (1) LLM의 그래프 생성 능력 부족 문제를 회피하고, (2) 신경망의 데이터 기반 패턴 인식과 LLM의 의미 기반 추론을 상호 보완적으로 결합하는 효과를 달성한다.

실험

데이터셋 및 실험 설정

TUSZ 데이터셋

  • TUH EEG Seizure Corpus: EEG 발작 감지의 표준 벤치마크
  • 다양한 환자의 실제 임상 EEG 기록 포함
  • 발작/비발작 이진 분류 태스크

백본 모델

GraphS4mer: State Space Model 기반 그래프 신경망으로, 시계열-그래프 결합 표현 학습 수행

평가 대상 LLM

다양한 SOTA 범용 LLM을 구조 판단자(structural judge)로 벤치마킹:

  • GPT 시리즈 (GPT-4 등)
  • Claude 시리즈
  • Llama 시리즈
  • 기타 오픈소스 LLM

평가 지표: 추론 능력, 일관성, 강인성

결과

발작 감지 성능 비교

TUSZ 벤치마크 주요 결과

방법 그래프 구성 LLM 정제 성능

핵심 발견

LLM 정제 단계만 추가해도 발작 감지 정확도가 유의미하게 향상되며, 그래프 구조의 질이 크게 개선됨

엣지 감소 효과

중복·무관 엣지가 대폭 제거되어 더 깨끗하고 희소한 그래프 구조 달성

벤치마크

LLM 임상 그래프 추론 벤치마크

본 논문은 LLM 기반 임상 그래프 구조 정제를 위한 최초의 종합 벤치마크를 제안한다.

01

추론 능력

통계적 증거와 임상 맥락을 종합하여 엣지의 타당성을 정확히 판단하는 능력 평가

02

일관성

동일하거나 유사한 엣지 쌍에 대해 일관된 판단을 내리는지 측정

03

강인성

노이즈가 추가된 통계 특징에도 안정적인 판단을 유지하는지 검증

벤치마크 결과: LLM 모델 간에 임상 그래프 추론 능력, 일관성, 강인성에서 상당한 차이가 존재하며, 더 큰 모델이 반드시 더 나은 성능을 보이지는 않는다.
분석

해석 가능성 및 일관성 분석

해석 가능성

  • LLM은 각 엣지 판단에 대해 자연어 설명을 제공
  • 제거된 엣지의 이유(노이즈, 중복, 생리학적 비합리성)를 명확히 기술
  • 유지된 엣지의 임상적 타당성을 해부학적 맥락과 연결
  • 정제된 그래프가 발작 전파 패턴과 더 잘 일치함을 확인

일관성 분석

  • 동일 입력에 대한 반복 추론 시 결과의 재현성 평가
  • 미세한 통계 변동에 대한 판단 안정성 검증
  • 프롬프트 순서 변경 시 결과 불변성 확인
  • 상이한 LLM 간 판단의 합의도 분석

핵심 가치: 단순히 성능 수치를 넘어, LLM 정제를 통해 얻은 그래프가 왜 더 나은지를 임상적·생리학적으로 설명할 수 있다는 점이 본 방법의 차별화된 장점이다.

결론

결론 및 주요 기여

세 가지 핵심 기여

1
새로운 패러다임: LLM을 임상 그래프 구조 정제 도구로 활용하는 최초의 프레임워크 제안. 기존 방법의 맥락 이해 부족 문제를 해결
2
종합 벤치마크: 다양한 SOTA LLM을 임상 그래프 구조 판단자로 체계적으로 평가한 벤치마크 구축. 모델 간 추론 능력·일관성·강인성 차이 규명
3
실증적 검증: TUSZ 벤치마크에서 발작 감지 성능과 그래프 표현 질의 동시 향상을 입증

향후 방향

  • 더 다양한 EEG 태스크(수면 단계, 뇌-컴퓨터 인터페이스)로 확장
  • 의료 도메인 특화 LLM 파인튜닝을 통한 정제 성능 향상
  • 실시간 엣지 정제를 위한 효율적 추론 파이프라인 개발
  • 멀티모달(EEG + MRI) 그래프 구조 학습으로 통합
