배경

지식 증류: Off-Policy vs On-Policy

Off-Policy 증류 (기존)

고정된 데이터셋으로 학생 모델 훈련

L = E(x,y)~P[D(πT ∥ πS)(y|x)]

문제: 훈련-추론 불일치
학생은 추론 시 자신의 출력을 기반으로 하지만, 훈련은 교사/정답 분포에서 수행

VS

On-Policy 증류 (OPD)

학생 자체 생성 데이터로 훈련

L = Ex,ŷ~πS[D(πS ∥ πT)(ŷ|x)]

장점: 훈련-추론 정렬
학생의 현재 정책에서 샘플링하여 실제 추론 환경과 동일한 조건에서 학습

문제 발견

OPD의 치명적 실패 모드

훈련이 진행되면서 갑작스러운 길이 팽창이 발생한다

초기 단계 (안정기)

모델이 안정적으로 학습. 절단율 ~0.5 (1.5B) / ~0.23 (7B), 반복율 ~0

위상 전이 (~30 스텝 내)

절단율이 ~1.0으로 급등 (거의 모든 생성이 최대 길이 도달)
반복율이 0.3~0.6으로 폭증

성능 붕괴

검증 정확도가 급격히 하락하며 회복되지 않음. 편향된 기울기가 훈련을 지배.

핵심 관찰: 이 현상은 학생 크기(1.5B, 7B)와 교사 모델에 관계없이 일관되게 발생한다.

Reverse-KL 어드밴티지

A(s,y) = log πT(y|s) - log πθ(y|s)

교사가 반복 토큰에 높은 확률을 부여 → 학생과의 차이가 크므로 어드밴티지가 비대칭적으로 커짐

붕괴 전후 비교

붕괴 전: 반복 토큰 ~5% → 큰 어드밴티지지만 전체 기여 제한적

붕괴 후: 반복 토큰 ~30% → 큰 어드밴티지 × 높은 빈도 = 기울기 지배

II

해결책: Stable-OPD

혼합 증류(Mixture Distillation) +
KL 정규화(KL Regularization)

해결책 1

혼합 증류 (Mixture Distillation)

On-Policy OPD와 Off-Policy SFT를 결합

Lmix(θ) = LOPD(θ) + λgold · E(x,y)~Dgold[LSFT(θ; x, y)]
On-Policy 롤아웃
학생 생성 데이터
+
Golden 솔루션
고품질 정답 데이터
→
동일 미니배치
함께 훈련

핵심 메커니즘

완전한(비절단, 비반복) 궤적의 일정 비율을 항상 유지하여 안정적인 앵커 역할

기존 방법과의 차이

On-policy 롤아웃의 교사 신호를 변경하지 않음 — Golden 데이터는 보조 SFT 항으로만 활용

해결책 2

KL 정규화 (KL Regularization)

참조 모델 기반 발산 제약으로 정책 드리프트 방지

LStable-OPD(θ) = Lmix(θ) + βKL · Est[DKL(πθ(·|st) ∥ πref(·|st))]

작동 원리

  • πref = 초기 학생 체크포인트 (훈련 전 상태)
  • 방문한 프리픽스 상태에서의 정책 이탈을 페널티
  • 토큰 수준에서 급격한 정책 변화를 제한
  • 반복적이고 긴 시퀀스 방향으로의 이동을 억제

두 전략의 상보적 효과

KL 정규화: 토큰 수준에서 급격한 정책 변화 제한
혼합 증류: 고품질 궤적의 안정적 비율 유지
→ 두 메커니즘이 서로 다른 수준에서 안정화를 달성

훈련 동역학

OPD vs Stable-OPD 훈련 과정

기존 OPD

  • 초기 낮은 절단율 (~0.3~0.5)
  • 스텝 150~180에서 위상 전이
  • 절단율 → 1.0 급등
  • 반복율 → 0.3~0.6 폭증
  • 검증 정확도 급격 하락
  • 회복 불가능한 붕괴
VS

Stable-OPD

  • 절단율 전 구간 평탄 (~0.2~0.4)
  • 위상 전이 없음
  • 반복율 전 구간 ~0 유지
  • 롤아웃 & 검증 모두 안정
  • 검증 정확도 꾸준히 상승
  • 처음부터 끝까지 안정적

다른 교사 모델(DeepSeek-R1-Distill-7B)에서도 동일: OPD는 급격한 팽창을 보이지만, Stable-OPD는 후반부에 경미한 드리프트만 관찰됨.

핵심 지표

측정 방법론

절단율 (Truncation Rate)

TruncRate = (1/N) Σ trunc(oi)

trunc(oi) = EOS 토큰 없이 최대 길이에 도달하여 생성이 중단된 경우 1

절단된 시퀀스가 많을수록 훈련 데이터 품질이 저하됨

반복율 (Repetition Rate)

CompRatio = |bytes(tail)| / |compress(bytes(tail))|

압축 기반 측정: 롤아웃 후반부의 압축률을 계산

꼬리 길이 > 10,000자 & 압축비 > 10이면 “반복”으로 판정 — 정보량이 매우 낮은 반복 패턴

핵심 통찰: 절단율과 반복율의 급등은 항상 동시에 발생하며, 검증 정확도 하락과 정확히 일치한다. 이는 두 현상이 동일한 근본 원인(자기강화 피드백 루프)에서 기인함을 보여준다.

관련 연구와의 차이

기존 길이 편향 연구와의 차별점

기존 연구 (Dr.GRPO, DAPO 등)

  • RLVR에서의 시퀀스 수준 길이 편향
  • 기울기 재가중(gradient reweighting)으로 해결
  • 보상 신호와 길이의 상관관계 문제
vs

본 연구 (OPD 특유)

  • 토큰 수준 reverse-KL 어드밴티지의 비대칭성
  • On-policy 샘플링이 이를 증폭하는 메커니즘
  • 교사 우도(likelihood) 신호와 학생 데이터 수집의 상호작용

질적으로 다른 실패 모드: RLVR의 길이 편향은 보상 해킹(reward hacking)에서 비롯되지만, OPD의 길이 팽창은 교사-학생 간 확률 차이와 on-policy 방문 빈도의 상호작용이라는 고유한 메커니즘에서 발생
