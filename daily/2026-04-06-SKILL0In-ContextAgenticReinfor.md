# 📚 오늘의 논문 (2026-04-06)

**제목**: SKILL0: In-Context Agentic Reinforcement Learning for Skill Internalization
**arXiv**: http://arxiv.org/abs/2604.02268v1

## 핵심 요약

SKILL0는 LLM 에이전트의 추론 시점 스킬 증강(inference-time skill augmentation)의 한계를 극복하기 위한 프레임워크다. 기존 방식은 검색 노이즈, 토큰 오버헤드, 지식의 비내재화 문제를 갖는다. SKILL0는 **인컨텍스트 강화학습(in-context agentic RL)**을 활용하여, 훈련 초기에는 전체 스킬 컨텍스트를 제공하고 점진적으로 철회하는 커리큘럼을 통해 스킬을 모델 파라미터에 내재화한다. 각 스킬의 유용성을 온폴리시로 평가하며 선형 감쇠 예산 내에서 필요한 스킬만 유지하다가, 최종적으로 완전한 제로샷 에이전트로 동작하게 한다.

## 💡 3가지 인사이트

1. **스킬을 "참조"에서 "체득"으로** — 추론 시 외부 스킬을 불러오는 대신 RL 훈련을 통해 모델 파라미터에 직접 내재화하여, 검색 노이즈와 토큰 비용을 근본적으로 제거한다.
2. **점진적 컨텍스트 철회 커리큘럼** — 처음엔 풍부한 스킬 정보를 제공하고 단계적으로 줄여가는 방식이, 모델이 스킬 지식을 자연스럽게 흡수하도록 유도하는 핵심 메커니즘이다.
3. **온폴리시 유용성 평가** — 각 스킬이 현재 정책에 여전히 도움이 되는지를 동적으로 판단하여, 이미 내재화된 스킬의 컨텍스트를 우선 제거함으로써 훈련 효율을 극대화한다.

## 🚀 2가지 응용 가능성

1. **경량 에이전트 배포** — 스킬이 파라미터에 내재화되면 추론 시 외부 검색/주입 없이 스텝당 0.5k 토큰 미만으로 동작하므로, 지연시간과 비용이 중요한 실시간 에이전트 서비스에 적합하다.
2. **도메인 특화 자율 에이전트 훈련** — 의료·법률·금융 등 특정 도메인의 절차적 지식(프로토콜, 워크플로우)을 스킬로 정의한 뒤 SKILL0로 내재화하면, 외부 가이드 없이 자율적으로 동작하는 전문 에이전트를 구축할 수 있다.
