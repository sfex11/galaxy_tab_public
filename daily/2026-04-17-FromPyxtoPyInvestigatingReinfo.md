# 📚 오늘의 논문 (2026-04-17)

**제목**: From $P(y|x)$ to $P(y)$: Investigating Reinforcement Learning in Pre-train Space
**arXiv**: http://arxiv.org/abs/2604.14142v1

## 핵심 요약

이 논문은 기존 RLVR(검증 가능한 보상 기반 강화학습)이 조건부 분포 P(y|x)만 최적화하여 베이스 모델의 출력 분포에 한계가 있음을 지적한다. 이를 해결하기 위해 사전학습 공간의 주변 분포 P(y)를 직접 최적화하는 **PreRL**을 제안한다. 핵심 발견으로, 부정 샘플 강화(NSR)가 잘못된 추론 경로를 빠르게 제거하고 반성적 사고를 유도함을 밝혔다. 최종적으로 NSR-PreRL로 추론 범위를 확장한 뒤 표준 RL로 정제하는 **DSRL(Dual Space RL)** 전략을 제시하며, 기존 강력한 베이스라인을 일관되게 능가하는 성능을 달성했다.

## 💡 3가지 인사이트

1. **log P(y)와 log P(y|x) 사이에 강한 그래디언트 정렬이 존재**하여, 사전학습 공간에서의 RL이 기존 조건부 RL의 유효한 대리(surrogate)로 작동할 수 있다.
2. **부정 샘플 강화(NSR)가 전환 사고를 14.89배, 반성 사고를 6.54배 증가**시키며, 잘못된 추론 경로를 가지치기하고 자기 반성 행동을 자극하는 데 핵심 역할을 한다.
3. P(y|x) 최적화는 베이스 모델의 기존 출력 분포에 근본적으로 제약되지만, **P(y) 최적화는 탐색 용량을 보존하면서 추론 능력 자체를 인코딩**할 수 있다.

## 🚀 2가지 응용 가능성

1. **LLM 추론 능력의 병목 해소**: 기존 RLHF/RLVR 파이프라인에 PreRL 단계를 추가하여, 베이스 모델의 출력 분포 한계를 넘어서는 더 강력한 추론 모델을 훈련할 수 있다.
2. **효율적 사전학습 데이터 활용**: 정적 코퍼스 기반 수동 학습 대신 보상 기반 온라인 업데이트로 P(y)를 최적화함으로써, 사전학습 단계에서부터 능동적으로 추론 능력을 강화하는 새로운 학습 패러다임에 적용할 수 있다.
