# Llm Agent: 합성 분석

**생성일**: 2026-05-04 (업데이트)
**관련 논문**: 117편
**분석 기간**: 2026-03-19 ~ 2026-04-14

## 1. 핵심 연구축

### (1) 평가의 현실화
GymAnything은 에이전트 환경 구축의 수동 병목을 메타 프레임워크로 자동화했고, ClawBench(3회 등장)는 144개 라이브 플랫폼에서 시뮬레이션-현실 격차를 정량화했다. TDAD는 기존 해결률 중심 평가에 **코드 회귀 방지 능력**이라는 새 차원을 추가하여, 에이전트가 "풀면서 망가뜨리지 않는지"를 AST 기반 그래프 분석으로 측정한다. Blindfolded LLMs는 티커 익명화로 암기 편향과 생존자 편향을 제거하여 금융 에이전트의 **진정한 시장 이해도**를 검증하는 방법론을 제시했다. Traffic Responsibility는 평가 대상을 해석 가능성(interpretability)까지 확장하여, 단순 정답 여부를 넘어 "어떤 근거로 판단했는가"를 평가 척도로 삼았다.

### (2) 구조·메타인지 설계
ActWisely는 도구 호출 여부를 스스로 판단하는 메타인지를 핵심 과제로 제기했다. 64-Token Creative Agent는 단 64토큰 템플릿이 복잡한 반복적 프롬프트 확장과 동등한 창의적 결과를 낳음을 보여주며, **"Less LLM, More Structure"**에 강력한 실증적 근거를 추가한다. Post-Training Local LLM은 4B 모델이 SFT+RL 2단계 후훈련만으로 Claude Opus 4.6급 보안 태스크 성능(95.8%)을 달성하여, 모델 크기가 아닌 **아키텍처적 설계와 훈련 전략**이 결정적임을 확인했다.

### (3) 보안·거버넌스의 계층 확장
TraceSafe는 안전성 취약점이 중간 실행 궤적에 존재함을 밝혔고, 사회심리학적 공격 벡터가 다중 에이전트를 훼손할 수 있음이 드러났다. Differential Privacy 논문은 기업 데이터 관점의 프라이버시 유출을 수학적으로 정립하여 **정보 이론적 프라이버시 보장과 출력 품질 간 최적 트레이드오프**를 제시하며, 이는 safety-diversity, resolution-precision 등 다른 트레이드오프 논의와도 수렴한다. Governed Memory는 멀티 에이전트 프로덕션에서 메모리 사일로·거버넌스 파편화·비구조화 메모리 등 5가지 구조적 과제를 식별하고, 공유 메모리의 구조화된 거버넌스가 시스템 신뢰성을 좌우함을 입증했다.

### (4) 자기 진화와 도메인 확장
AgentFactory는 성공 경험을 텍스트가 아닌 **실행 가능한 서브에이전트 코드**로 축적하는 자기 진화 패러다임을 제시한다. 이는 곧 에이전트가 자신만의 도구 라이브러리를 자동 구축하는 tool-use의 확장형으로 이해될 수 있다. 동시에 에이전트의 적용 도메인이 급격히 확장되었다: 체화된 내비게이션(AgentVLN), 법률 판단(Traffic Responsibility), 경제 시뮬레이션(MALLES), 창의적 이미지 생성(64-Token Agent), 보안 자동화(Post-Training Local LLM)까지. VLM 기반 에이전트가 2D 의미 이해를 넘어 3D 공간 추론까지 통합하는 흐름(AgentVLN, CADENCE)은 범용 체화 에이전트 구현의 이정표다.

### (5) 통합 아키텍처
PSI의 공유 상태 버스는 독립 모듈 간 고립을 해결하고, Paper Circle은 멀티에이전트 역할 분화를, Maestro는 GUI 동적 적응을 제시했다. VideoAtlas는 장시간 비디오를 계층적 그리드로 변환하여 로그 스케일 연산의 탐색 가능한 환경을 제공하고, **상태 공유·효율적 환경 설계·협조 프로토콜**이 핵심 설계 원리로 부상한다.

## 2. 트렌드

- **평가의 다차원화**: 시뮬레이션 → 라이브 플랫폼(ClawBench) → 회귀 방지(TDAD) → 편향 제거(Blindfolded) → 해석 가능성(Traffic Responsibility)으로 평가 축이 다각화된다.
- **"Less LLM, More Structure"의 실증화**: 64토큰 템플릿, 4B 모델의 대형 모델급 성능 등 구조적 효율성이 반복적으로 입증된다.
- **경험의 코드화**: 텍스트 기반 리플렉션에서 실행 가능 코드로 경험 저장 패러다임이 전환되며, 이는 tool-use 개념과 자연스럽게 수렴한다(AgentFactory).
- **트레이드오프 정량화**: 프라이버시-품질(DP), 안전성-다양성, 해상도-정밀도 등 에이전트 설계의 핵심 트레이드오프들이 수학적 프레임워크로 정립되는 추세다.
