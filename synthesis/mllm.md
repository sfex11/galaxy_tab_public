# Mllm: 합성 분석

## 1. 공간 추론의 규모 확장과 암묵적 프라이어의 발견

MLLM의 공간 이해는 단안 비디오(Loc3R-VLM)에서 도시 규모(3DCity-LLM)로 급격히 확장 중이다. 핵심은 **명시적 3D 감독 없이 공간 능력을 확보**하는 두 갈래 접근이다. Generation Models Know Space는 비디오 생성 모델이 시간적 일관성 학습에서 내재적으로 획득한 3D 프라이어를 장면 이해에 전이하는 패러다임 전환이며, UniDriveVLA는 2D 언어 지식과 3D 공간 정보의 융합 아키텍처로 자율주행에서 공간-의미 트레이드오프를 해결한다. 3DCity-LLM의 coarse-to-fine 병렬 브랜치 인코딩은 객체→관계→전역 장면의 계층적 표현 실증 사례다. 세 논문 모두 명시적 3D 주석 의존을 줄이면서 공간 추론 스케일을 환경 수준으로 밀어붙인다.

## 2. 에이전틱 MLLM과 메타인지의 수렴

MARCUS(의료), AMIGO(벤치마크), UI-Voyager(GUI), Stop Wandering(VLN), Chameleon(로봇)은 도메인을 넘어 **자기 모니터링과 적응적 행동**이라는 공통 요구를 드러낸다. Stop Wandering은 VLN의 비효율적 탐색을 메타인지 부재로 진단하고, UI-Voyager는 실패 궤적에서 credit assignment로 자기진화를 달성한다. Chameleon은 원시 지각 단서를 보존하는 에피소딕 메모리로 비마르코프 의사결정을 해결하며, AMIGO는 이 능동적 추론을 다중 이미지 갤러리에서 순차적 질문 전략으로 체계적으로 측정한다. 이들은 에이전트가 **추론 과정을 감시-조절하는 상위 인지 계층**의 필요성으로 수렴한다.

## 3. 추론 효율화의 직교적 전략

AdaptToken은 모델 자기 불확실성(엔트로피)을 글로벌 토큰 선택과 조기 종료의 제어 신호로 활용하는 training-free 접근이다. VISOR은 시각 토큰을 제거하지 않고 레이어별 cross-attention을 동적으로 희소화하여 정보 손실 없이 연산을 절감한다. SpecEyes는 토큰/레이어 수준을 넘어 에이전트의 인식-추론-도구호출 파이프라인 전체를 추측적으로 병렬화한다. 세 전략은 정보 이론적, 아키텍처적, 파이프라인적 관점에서 접근하므로 상호 호환되며 통합 적용 시 시너지가 기대된다.

## 4. 표현 조종과 학습 신뢰성의 긴장

Steerable Visual Representations은 ViT의 범용 특징을 텍스트 조건부로 유도하여 MLLM의 언어 중심성과 ViT의 비유도성 사이 간극을 메운다. 반면 Hallucination in RL Post-Training은 RL 후훈련이 시각적 추론을 향상시키는 것처럼 보이나, 실제로는 시각 정보 무시와 텍스트 패턴 기반 환각에 의존하는 추론 단축을 학습할 수 있음을 경고한다. **표현을 조종할수록 그 표현이 실제 지식을 반영하는지 검증하는 메커니즘이 동시에 필요하다**는 구조적 긴장이 드러난다. NavTrust가 내비게이션의 입력 손상 하 강건성을 평가한 것도 조종된 표현의 신뢰성을 외부 검증하는 방법론적 보완으로 읽힌다.

## 5. 핵심 논점: RL 후훈련의 이중성과 해결 방향

가장 예리한 긴장은 **RL 후훈련의 이중성**이다. 추론 강화를 주장하는 흐름과 이가 보상 해킹의 멀티모달 변형일 수 있다는 지적이 정면 대치한다. 이를 해결하기 위한 세 가지 방향이 논문들에서 싹튼다. 첫째, 메타인지적 자기 모니터링(Stop Wandering, AMIGO)으로 추론 무결성을 내부에서 검증하는 방향. 둘째, adversarial 강건성 평가(NavTrust)로 외부에서 신뢰성을 검증하는 방향. 셋째, 암묵적 프라이어(Generation Models Know Space)나 표현 조종(Steerable Visual Representations)처럼 RL에 의존하지 않는 능력 확보 방향이다. MLLM 연구는 "어떻게 더 잘하게 만들까"에서 "어떻게 진정으로 이해하게 만들까"로 질문이 변화하는 지점에 서 있다.
