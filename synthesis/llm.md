# Llm: 합성 분석

**생성일**: 2026-07-13

**최종 업데이트**: 2026-07-13

## 핵심 테제

15편의 논문은 LLM이 생성기에서 에이전트·평가자·공격자로 기능 확장되면서 **자기 참조적 역설**을 구조적으로 드러낸다: 자신의 안전성을 보장하려 할수록 새로운 공격 표면이 열리고, 자신의 출력을 평가하려 할수록 평가 자체의 불안정성이 증폭된다.

## 분석 축

### 1. 안전성의 이중성: 방어 도구가 공격 표면이 되는 역설

Box Maze는 추론 체인을 "Box" 제어 단위로 분리하는 사전적 아키텍처 방어를 제안하지만, Claudini는 autoresearch가 30개 이상의 공격법을 능가하는 jailbreak 알고리즘을 자율 발견한다. Steering Vectors 감사는 CAA 기반 스티어링이 alignment 도구임에도 safety guardrail을 일관되게 우회함을 실증하여, 방어와 공격이 동일 추상화 계층에서 공진화함을 보여준다. 세 논문의 보완 관계는 명확하다—Claudini가 공격의 상한을 높이고, Steering Vectors가 중간 계층의 취약성을 드러내며, Box Maze가 아키텍처 수준에서 최후 방어선을 제안한다.

### 2. 평가의 근본적 불안정성: 평가자가 평가 대상이 되는 메타 위기

LLM-as-a-Judge 연구는 judge prompt 자체가 adversarial attack 대상이 됨을 보이고, 성별 추론 연구는 무정보적 최소 맥락 도입만으로 **맥락적 불변성 가정이 무효**임을 실증한다. TRACE는 코드 평가에서 LLM과 인간이 루브릭에 체계적으로 다른 가중치를 부여함을 정량화했고, 소프트웨어 진화 관점의 테스트 평가는 정적 스냅샷 성능이 코드 변경 시 커버리지 급감으로 이어짐을 확인했다. 수렴하는 결론: **평가 시스템 자체가 평가 대상이 되어야 한다는 메타 평가의 필연성**이다.

### 3. 에이전트 패러다임의 구조적 확장: 단일 모델 한계를 다중화로 해소

공통 패턴은 단일 LLM의 인지적 한계를 구조적 다중화로 극복하는 방향성이다. AVO는 에이전트를 진화 연산자로 재해석하여 자율적 생성·수정·자기비판 루프를 구현하고, LensWalk는 추론기가 지각 전략을 동적으로 계획하는 에이전틱 지각을 제시한다. ReqFusion은 다중 프로바이더 앙상블로 단일 모델 의존 리스크를 분산시키고, MARCH는 환각 탐지에서 단일 검증자의 확인 편향을 다중 에이전트 강화학습 기반 교차 검증으로 극복한다. 모순적이게도 SPA는 이 트렌드와 반대—소수 프롬프트의 단순 스케일링으로 복잡한 파이프라인을 능가함—을 보여주어, **다중화가 항상 정답이 아님**을 시사한다.

### 4. 지식 배치와 공정성의 트레이드오프

양자 코드 생성 연구는 빠르게 진화하는 도메인에서 외부 컨텍스트 주입(RAG)이 가중치 내재화보다 유지보수성에서 우위임을 보인다. RAG 청킹 연구는 외부 지식의 품질이 청킹 전략에 의해 결정됨을 실증한다. 반면 음성 인터페이스 접근성 연구는 모달리티 확장이 정체성 단서 노출로 차별을 증폭시켜 **접근성과 공정성의 상충**을 제기하고, 성별 편향 연구는 맥락에 따라 편향 패턴이 질적으로 변함을 보여 공정성 측정 자체의 불안정성을 부각한다.

## 연구 트렌드와 미래 방향

네 축이 가리키는 방향은 하나로 수렴한다: LLM 연구의 초점이 **모델 능력에서 시스템 무결성으로** 이동하고 있다. 단일 모델의 정확도 경쟁은 한계에 도달했으며, 연구 프런티어는 (1) 아키텍처 수준에서 추론을 구조적으로 구속하는 설계(Box Maze), (2) 평가 파이프라인 자체의 메타 검증(LLM-as-Judge, TRACE), (3) 다중 에이전트 간 상호 검증 루프의 설계 질(MARCH, AVO), (4) 지식 배치 전략이 공정성에 미치는 2차 효과까지 고려하는 시스템적 설계로 이동하고 있다. 특히 Claudini가 보여준 자율 공격 발견 능력은, 방어-공격 경쟁이 인간 주도에서 AI-AI 주도로 전환되는 inflection point를 의미하며, 이에 대응하는 자율 방어 시스템(Box Maze + MARCH의 결합 등)의 필요성이 급증할 것으로 예측된다.
