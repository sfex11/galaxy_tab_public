# Transformer: 합성 분석

**생성일**: 2026-06-22
**관련 논문**: 59편
**최종 업데이트**: 2026-06-22

## 공통 주제: 고치지 않고 뛰어넘고, 경계 없이 확장하다

59편의 논문은 Transformer가 2026년 초 두 가지 명확한 진화 축을 따라 움직이고 있음을 보여준다. 첫째, **아키텍처 수정 없이 추론 효율의 극단적 추구**, 둘째, **비전·언어를 넘어 3D 공간과 물리 법칙으로의 모달리티 무한 확장**. 두 축은 시퀀스 길이에 대한 이차적 복잡도와 토큰 단위 순차 처리라는 Transformer의 본질적 한계를 공통의 적으로 삼되, 해법의 방향은 정반대다. 전자는 Transformer **내부**에서 압축하고, 후자는 Transformer **외부**로 입력을 확장한다.

## 토큰 수준 효율화: 프루닝, 프로빙, 계층화의 삼각편대

세 가지 독특한 패턴이 드러난다. **통합 시공간 토큰 스코어링**은 ViT와 LLM의 경계를 허물어 단일 프루닝 파이프라인으로 비디오 VLM의 시간적 중복을 제거하며, MMEmb-R1의 추론 강화 멀티모달 임베딩과 상보적 관계를 형성한다. **임베딩 공간 프로빙**은 더 파격적—별도 드래프트 모델 없이 mask token으로 speculative decoding을 구현하며, next-token prediction만으로 학습된 Transformer가 내재적으로 다중 토큰 예측 능력을 보유한다는 발견이다. **VideoAtlas**는 토큰 수를 줄이는 대신 비디오를 계층적 그리드로 재구성하여 로그 복잡도 탐색을 달성한다. 세 접근 모두 "Transformer를 고치지 않고 뛰어넘는다"는 철학을 공유하며, 산업적 배포에 가장 현실적인 경로다.

## 모달리티 확장: 3D 기하학, 물리 법칙, 시각 속성의 직접 모델링

Transformer는 이제 물리 공간의 구조를 직접 토큰 시퀀스로 변환한다. **GMT**는 3D 장면에서 6-DOF 궤적을 합성하고, **Loc3R-VLM**은 단안 비디오만으로 VLM에 3D 공간 추론을 부여한다. **LoST**는 3D 형상을 의미론적 계층으로 토큰화하여, 토큰화 전략이 모델이 학습할 **구조적 사전 지식**을 결정한다는 점을 명확히 한다—NLP에서 BPE가 언어 모델 성능을 좌우했던 역사의 3D 도메인 재현이다. **Egomotion-Aware** 연구는 IMU 데이터를 MLLM에 통합하여 무거운 3D 재구성 없이 공간 추론을 달성하고, **RHYME-XT**는 PDE 기반 물리 시스템을 신경 기저 함수로 근사한다. **텍스트 임베딩 보간** 연구는 시각적 속성 제어에서 Transformer의 임베딩 공간이 이미 충분히 구조화되어 있음을 확인한다.

## 절대적 지표로의 전환: 신뢰성과 하드웨어 효율

**Box Maze**는 추론 신뢰성을 사후 교정(RLHF)이 아닌 프로세스 아키텍처 수준에서 강제하는 패러다임 전환을 제시하며, **SOL-ExecBench**는 GPU 커널 최적화를 상대적 speedup이 아닌 하드웨어 이론 한계(Speed-of-Light) 대비 절대적 효율로 평가한다. 두 논문은 Transformer 생태계의 소프트웨어 신뢰성과 하드웨어 효율성에 대해 "상대적 지표"를 "절대적 지표"로 대체하려는 인식론적 전환을 공유한다.

## 일치·모순·보완 관계

임베딩 공간 프로빙이 Transformer의 내재적 능력을 긍정적으로 해석하는 반면, Box Maze는 동일 아키텍처의 구조적 한계(환각, 추론 불안정성)를 명시적 제어로 보완해야 한다고 본다. "모델을 믿는다" vs "모델의 과정을 통제한다"는 긴장은 Transformer 연구의 핵심 딜레마다. LoST의 의미론적 토큰화와 Unified Spatio-Temporal Scoring의 토큰 프루닝은 토큰이라는 단위에 대해 서로 다른 해법—전자는 토큰 **설계**를, 후자는 토큰 **선택**을 최적화한다—을 제시하여 보완적이다.

## 미래 방향

Transformer 아키텍처 자체의 대체가 아니라, 주변 인프라(토큰화, 프루닝, 프로세스 제어, 하드웨어 최적화)의 극단적 정교화가 당분간 주류를 이룰 것이다. 특히 3D·물리 모달리티 확장과 토큰 효율화가 만나는 지점—의미론적 3D 토큰화를 효율적 프루닝과 결합하는 통합 파이프라인—이 다음 돌파구다. 절대적 지표 기반 평가의 확산은 Transformer 생태계 전반의 성숙도를 한 단계 끌어올릴 잠재력이 있다.
