# Transformer: 합성 분석

**생성일**: 2026-07-06  
**관련 논문**: 63편  
**최종 업데이트**: 2026-07-06

## 공통 주제: 고치지 않고 뛰어넘고, 경계 없이 확장하다

63편의 논문(중복 출판 3편 포함)은 Transformer가 2026년 초 두 가지 명확한 진화 축을 따라 움직이고 있음을 보여준다. 첫째, **아키텍처 수정 없이 추론 효율의 극단적 추구**, 둘째, **비전·언어를 넘어 3D 공간과 물리 법칙으로의 모달리티 무한 확장**. 두 축은 시퀀스 길이에 대한 이차적 복잡도와 토큰 단위 순차 처리라는 Transformer의 본질적 한계를 공통의 적으로 삼되, 해법의 방향은 정반대다. 전자는 Transformer **내부**에서 압축하고, 후자는 Transformer **외부**로 입력을 확장한다.

## 토큰 수준 효율화: 프루닝, 프로빙, 계층화의 삼각편대

세 가지 독특한 패턴이 드러난다. **통합 시공간 토큰 스코어링**은 ViT와 LLM의 경계를 허물어 단일 프루닝 파이프라인으로 비디오 VLM의 시간적 중복을 제거한다. **임베딩 공간 프로빙**은 더 파격적—별도 드래프트 모델 없이 mask token으로 speculative decoding을 구현하며, next-token prediction만으로 학습된 Transformer가 내재적으로 다중 토큰 예측 능력을 보유한다는 발견이다. **VideoAtlas**는 토큰 수를 줄이는 대신 비디오를 계층적 그리드로 재구성하여 로그 복잡도 탐색을 달성한다. 세 접근 모두 "Transformer를 고치지 않고 뛰어넘는다"는 철학을 공유하며, 산업적 배포에 가장 현실적인 경로다.

## 모달리티 확장: 3D 기하학, 물리 법칙, 시각 속성의 직접 모델링

Transformer는 이제 물리 공간의 구조를 직접 토큰 시퀀스로 변환한다. **GMT**는 3D 장면에서 6-DOF 궤적을 합성하고, **Loc3R-VLM**은 단안 비디오만으로 VLM에 3D 공간 추론을 부여한다. **LoST**는 3D 형상을 의미론적 계층으로 토큰화하여, 토큰화 전략이 모델이 학습할 **구조적 사전 지식**을 결정한다는 점을 명확히 한다—NLP에서 BPE가 언어 모델 성능을 좌우했던 역사의 3D 도메인 재현이다. **Egomotion-Aware** 연구는 IMU 데이터를 MLLM에 통합하여 무거운 3D 재구성 없이 공간 추론을 달성하고, **RHYME-XT**는 PDE 기반 물리 시스템을 신경 기저 함수로 근사한다. **텍스트 임베딩 보간** 연구는 시각적 속성 제어에서 Transformer의 임베딩 공간이 이미 충분히 구조화되어 있음을 확인한다.

## 절대적 지표로의 전환: 신뢰성과 하드웨어 효율

**Box Maze**는 추론 신뢰성을 사후 교정(RLHF)이 아닌 프로세스 아키텍처 수준에서 강제하는 패러다임 전환을 제시하며, **SOL-ExecBench**는 GPU 커널 최적화를 상대적 speedup이 아닌 하드웨어 이론 한계(Speed-of-Light) 대비 절대적 효율로 평가한다. 두 논문은 Transformer 생태계의 소프트웨어 신뢰성과 하드웨어 효율성에 대해 "상대적 지표"를 "절대적 지표"로 대체하려는 인식론적 전환을 공유한다.

## 일치·모순·보완 관계

임베딩 공간 프로빙이 Transformer의 내재적 능력을 긍정적으로 해석하는 반면, Box Maze는 동일 아키텍처의 구조적 한계(환각, 추론 불안정성)를 명시적 제어로 보완해야 한다고 본다. "모델을 믿는다" vs "모델의 과정을 통제한다"는 긴장은 Transformer 연구의 핵심 딜레마다. LoST의 의미론적 토큰화와 통합 시공간 스코어링의 토큰 프루닝은 토큰이라는 단위에 대해 서로 다른 해법—전자는 토큰 **설계**를, 후자는 토큰 **선택**을 최적화하며, 이 둘의 결합이 다음 단계의 효율화를 이끌 가능성이 높다.

## 연구 트렌드

1. **Training-free 추론 가속의 정교화**: 드래프트 모델 없는 speculative decoding, 캡션 없는 비디오 탐색 등 학습 비용 제로의 가속 기법이 주류로 자리 잡고 있으며, 이는 오픈소스 모델의 실제 서빙 비용을 획기적으로 낮추는 방향이다.
2. **토큰화의 도메인 특화**: NLP의 BPE → 3D의 의미론적 계층(LoST) → 비디오의 계층적 그리드(VideoAtlas)로 이어지는 토큰화 혁신이 모달리티마다 독립적으로 진화 중이며, 통합 토큰화 이론의 부재가 다음 병목으로 예상된다.
3. **절대적 벤치마크의 등장**: 상대적 평가의 한계가 인식되면서, 하드웨어 한계(SOL-ExecBench)와 프로세스 무결성(Box Maze)이라는 절대적 기준이 동시에 부상하고 있다.
4. **3D-물리 모달리티의 Transformer 정복**: 로보틱스 궤적(GMT), 공간 추론(Loc3R-VLM), 물리 시뮬레이션(RHYME-XT)이 모두 Transformer 기반으로 전환되며, Transformer가 "언어 모델"에서 "범용 시퀀스 프로세서"로의 지위 변화를 가속한다.
