차원별 분석

🧠 심리적 에이전트가 스스로를 어떻게 인식하는가

MTI: 행동 ≠ 자기 보고
에이전트가 "나는 이런 성격이다"라고 말하는 것과 실제 행동 패턴은 다르다. 심리적 아이덴티티는 자기 선언이 아닌 행동적 관찰로 정의되어야 한다.
  • Act Wisely — 에이전트는 "무엇을 할 수 없는지 아는 것"이 아이덴티티의 일부. 메타인지적 결핍이 존재
  • How Much LLM — 능력 중 LLM에서 오는 것과 외부 구조에서 오는 것을 구분할 수 있는가?
  • Beyond Preset Identities — 사전 설정된 프롬프트를 넘어 담화적 상호작용으로 자율적 입장 형성. 아이덴티티는 고정 속성이 아닌 창발적 과정

👥 사회적 타자가 에이전트를 어떻게 인식하는가

Among Us: 전략적 기만의 자율적 수행
1,100회 게임에서 LLM 에이전트가 전략적 기만을 자율적으로 수행. 사회적 아이덴티티는 의도적으로 조작될 수 있다.
  • Social Dynamics — 사회적 정체성(지각된 전문성, 지배력)이 의사결정에 체계적 편향 생성
  • Dynamic Belief Graphs — 타 에이전트의 신념을 모델링하며 자기-타자 경계 형성
  • NavTrust — 신뢰는 이상적 조건이 아닌 역경 하에서의 강건성으로 평가

🔐 기술적 시스템이 에이전트를 어떻게 구분하는가

  • MIGT — 머신 아이덴티티가 인간 아이덴티티를 80:1로 압도하지만 통합 거버넌스 부재
  • SCRAT — 아이덴티티 = 제어 + 메모리 + 검증 삼중 구조. 무엇을 기억/잊는지가 "성격" 구성
  • TraceSafe — 최종 출력이 아닌 전체 행동 궤적에서 진정한 아이덴티티 평가

⚖️ 법적 책임은 누구에게 있는가

  • Creator/Editor — 창작자/편집자의 이중 역할 시 책임 귀속 복잡
  • DAO Belief Aggregation — 사후 평가 가능한 책임 체계
  • Triadic Architecture — 인지적 무중력 방지가 법적 귀속의 전제

🔄 이식 플랫폼 간 아이덴티티 이동

  • ZKP Survey — 속성 노출 없이 검증: DID, Soulbound NFT, SPIFFE
  • PSI — 이식성은 자격증명이 아닌 통합된 상태 컨텍스트의 연속성 보장

🆕 Wiki가 새로 발견한 것

1. 이중 불투명성 (Dual Opacity)

MTI(행동 ≠ 자기 보고) + Among Us(전략적 기만)

에이전트의 아이덴티티는 자기 선언으로도, 타자 관찰로도 완전히 포착할 수 없다.
→ 행동 궤적(TraceSafe)만이 가장 신뢰할 수 있는 근거.

2. 유령 에이전트 (Ghost Agent) 🆕

5차원 모두 결여된 에이전트

자기 인식 없음 → 사회적 책임 없음 → 기술적 추적 불가 → 법적 귀속 불가 → 이식 가치 없음

인지적 무중력 상태의 에이전트는 시스템 전체의 신뢰와 안전을 훼손하는 가장 위험한 형태.

3. 이식성의 존재론적 질문

PSI가 "공유 상태 = 아이덴티티 일관성"이라면:

에이전트를 다른 플랫폼으로 옮길 때 상태 전체를 이식해야 하는가, 아니면 자격증명만 이식하면 되는가?

이는 아이덴티티가 연속적 경험에 있는지, 검증 가능한 속성에 있는지에 대한 존재론적 질문이다.

4. 창발 vs 거버넌스 충돌

Beyond Preset Identities: 아이덴티티는 창발적으로 진화한다.
MIGT: 아이덴티티를 체계적으로 관리해야 한다.

→ 자율적 아이덴티티 진화를 허용하면서 거버넌스를 유지하는 것이 핵심 과제.
