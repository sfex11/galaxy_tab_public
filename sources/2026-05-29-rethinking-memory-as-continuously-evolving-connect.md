# Rethinking Memory as Continuously Evolving Connectivity

**타입**: 논문
**출처**: arXiv
**날짜**: 2026-05-29
**링크**: http://arxiv.org/abs/2605.28773v1

## 💡 핵심 인사이트

이 논문은 memory-as-model-paradigm이 외부 저장소에서 모델 파라미터 공간으로 메모리를 인코딩하는 방향으로 메모리의 내재화를 시도한다. FluxMem은 이 방향을 넘어, 그래프 구조 자체를 진화의 대상으로 삼아 메모리를 외부 인프라에서 내부 인지 모델의 일부로 내재화하는 패러다임의 구체적 경로를 제공한다.
