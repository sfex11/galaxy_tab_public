## 📖 분석

# Nemobot Games: LLM 기반 전략 게임 에이전트 엔지니어링 환경

Nemobot는 Claude Shannon의 게임 기계 분류법(1950)을 LLM 시대로 확장하는 인터랙티브 에이전트 엔지니어링 환경이다. 사용자가 LLM 기반 게임 에이전트를 생성·커스터마이즈·배포하며 AI 전략을 직접 체험할 수 있다. Shannon이 제시한 4가지 게임 기계 범주(완전 정보·불완전 정보·협력·경쟁) 각각에 대한 LLM 에이전트 구현을 실증한다.

기존 Wiki의 agent-environment-generation과 구조적 대비가 흥미롭다: 기존 연구들이 소프트웨어를 에이전트 환경으로 변환한다면, Nemobot는 에이전트 생성 자체를 환경으로 제공한다. transformational-game-design의 구체적 실현으로도 읽힌다—게임의 목표가 'AI 에이전트를 엔지니어링하는 것'이며, 학습은 플레이가 아닌 구성을 통해 발생한다. 이는 ai-literacy 연구에 새로운 경로를 제시하는데, AI 리터러시를 'AI 도구 사용 능력'이 아닌 'AI 에이전트 설계 능력'으로 재정의한다.

llm-agent 합성 분석의 '능력 확장' 축과도 연결된다. 기존 97편 논문이 에이전트의 적용 도메인 확장을 다루었다면, Nemobot은 메타 수준에서 사용자 자신이 에이전트를 구성하도록 권한을 이전한다는 점에서 차별적이다.
