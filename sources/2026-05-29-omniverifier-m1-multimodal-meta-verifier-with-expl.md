# OmniVerifier-M1: Multimodal Meta-Verifier with Explicit Structured Recalibration

**타입**: 논문
**출처**: arXiv
**날짜**: 2026-05-29
**링크**: http://arxiv.org/abs/2605.28805v1

## 💡 핵심 인사이트

판단(decision-only)과 검증(verification)을 분리하여 검증자의 기호 논리(logic chain, bounding box, execution trace)를 시각적 판턴에서 생성하고, 이를 모델 학습의 명시적 피드백으로 활용하는 구조적 재보정 메커니즘을 제안한다. 기존 output-epistemic-reliability가 시각적 판턴 평가에만 적용되던 한계를 시각 판턴 진단 도구 분석으로 확장하는 방향을 제시한다.

## 정의
시각적 판턴에 대한 기존 평가 체계(판단/거부만 확정)가 검증자의 기호 논리(logic chain, bounding box, execution trace 등)를 생성하여 시각적 판턴의 검증 가능성을 진단하는 다중 모달 메타 검증 프레임워크. 검증자의 기호 논리를 시각적 판턴 검증의 명시적 피드백으로 활용하여, 기존 output-epistemic-reliability가 시각적 판턴 평가에만 적용되던 한계를 시각 판턴 검증으로 확장하는 방향을 제시한다.
