# SAGAI-MID: A Generative AI-Driven Middleware for Dynamic Runtime Interoperability

**타입**: 논문  
**출처**: arXiv  
**날짜**: 2026-03-31  
**링크**: None

## 핵심 요약

**SAGAI-MID**는 이기종 분산 시스템(REST API, GraphQL, IoT 등)에서 발생하는 스키마 불일치를 LLM을 활용해 **런타임에 동적으로 감지·해결**하는 FastAPI 기반 미들웨어다. 구조적 diff와 LLM 의미 분석을 결합한 하이브리드 감지, 요청별 LLM 변환과 재사용 가능한 어댑터 코드 생성의 이중 해결 전략, 검증·앙상블 투표·규칙 기반 폴백의 3계층 안전장치로 구성된 5계층 파이프라인 아키텍처를 제안한다. 10개 시나리오에서 6개 LLM을 평가한 결과, 최고 구성에서 pass@1 정확도 0.90을 달성했으며 코드 생성(CODEGEN) 전략이 직접 변환보다 우수했다.

## 인사이트

1. 기존 설계 시점(design-time)의 상호운용성 전술을 **런타임 역량으로 전환**함으로써, 수작업 어댑터 코딩 없이 새로운 스키마 조합에 즉시 대응할 수 있다.
2. LLM이 직접 데이터를 변환하는 것보다 **재사용 가능한 어댑터 코드를 생성(CODEGEN)**하는 전략이 평균 정확도에서 더 우수하다 (0.83 vs 0.77).
3. 모델 간 비용이 **30배** 차이 나지만 정확도는 비례하지 않으며, 가장 정확한 모델이 오히려 가장 저렴했다 — 비싼 모델이 항상 좋은 것은 아니다.

## 응용 가능성

1. **마이크로서비스/IoT 플랫폼**: API 버전 업그레이드나 이기종 IoT 디바이스 통합 시 스키마 변환 미들웨어로 배포하여 운영 중단 없이 실시간 호환성을 확보할 수 있다.
2. **레거시 시스템 현대화**: 구형 시스템과 신규 클라우드 서비스 간 데이터 형식 차이를 런타임에 자동 브리징하여, 대규모 재작성 없이 점진적 마이그레이션을 지원할 수 있다.
