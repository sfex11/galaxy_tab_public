## 📖 분석

## Nemobot Games: Shannon 분류학의 LLM 시대적 재현

Nemobot은 Claude Shannon의 게임 플레이 기계 분류학을 LLM 기반 에이전트 환경으로 구현한 인터랙티브 엔지니어링 플랫폼이다. Shannon이 정보이론의 선구자로 제안했던 게임 기계 4분류를 LLM 에이전트 설계 공간으로 변환하여, 사용자가 전략적 게임 에이전트를 생성·커스터마이즈·배포할 수 있게 한다.

### 기존 Wiki와의 연결

이 논문은 Wiki의 여러 축에 걸쳐 위치한다. **agent-environment-generation** 축에서는 Gym-Anything이 '임의 소프트웨어를 에이전트 환경으로 변환'한다면, Nemobot은 '게임이라는 특정 도메인을 전략적 학습 환경으로 변환'하는 특화 경로를 제공한다. **ai-literacy**과 **transformational-game-design**의 교차점에서, 게임을 통한 AI 전략 사고 교육이라는 목표를 공유하되, 단순한 교육 도구가 아닌 에이전트 엔지니어링 환경이라는 점이 차별화된다.

### Shannon 분류학의 현대적 의미

Shannon의 원래 분류는 미니맥스·패턴 매칭·학습 기계 등을 구분했으나, LLM 시대에는 이 분류가 에이전트 설계 패턴(추론 기반·패턴 기반·학습 기반)으로 재해석될 수 있다. 이는 [[concepts/shannon-game-taxonomy|shannon-game-taxonomy]]가 단순한 역사적 분류가 아니라 현대 에이전트 아키텍처 선택의 기준框架으로 살아있음을 보여준다.

### 네 가지 게임 에이전트 클래스

논문은 네 가지 상이한 게임 플레이 기계 클래스에서 LLM 챗봇의 능력을 실증하며, 각 클래스가 요구하는 전략적 추론의 차이를 노출한다. 이는 [[entities/mathduels|MathDuels]]가 문제 출제-해결의 이중 역할으로 LLM 능력을 측정하는 것과 유사하게, 게임 전략 생성-대응을 통해 에이전트의 전략적 추론 깊이를 프로빙한다.
