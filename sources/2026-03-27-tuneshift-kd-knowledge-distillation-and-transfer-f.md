# TuneShift-KD: Knowledge Distillation and Transfer for Fine-tuned Models

**타입**: 논문  
**출처**: arXiv  
**날짜**: 2026-03-27  
**링크**: None

## 핵심 요약

**TuneShift-KD**는 파인튜닝된 모델의 전문 지식을 새로운 모델로 이전할 때, 원본 학습 데이터 없이도 지식 증류(Knowledge Distillation)를 수행하는 방법을 제안한다. 핵심 아이디어는 **퍼플렉시티(perplexity) 차이 분석**으로, 파인튜닝 모델은 자신 있게(낮은 퍼플렉시티) 응답하지만 베이스 모델은 어려워하는(높은 퍼플렉시티) 프롬프트를 찾아 전문 지식 영역을 자동 식별한다. 이를 통해 합성 데이터셋을 생성하고, 별도의 판별기 학습 없이 새로운 모델에 지식을 전이한다. 기존 접근법 대비 더 높은 정확도를 달성했다.

## 인사이트

1. **베이스 모델과 파인튜닝 모델 간의 퍼플렉시티 차이**가 "모델이 새로 학습한 전문 지식"을 식별하는 효과적인 신호로 활용될 수 있다.
2. 원본 데이터 없이도 소수의 대표 프롬프트만으로 합성 데이터를 반복 생성하여 지식 증류가 가능하므로, **데이터 프라이버시 제약을 우회**할 수 있다.
3. 판별기(discriminator) 학습 등 외부 모델 의존 없이 **완전 자동화된 파이프라인**으로 지식 전이가 가능하다는 점에서 실용성이 높다.

## 응용 가능성

1. **의료·법률·금융 등 민감 데이터 기반 파인튜닝 모델**의 전문 지식을 원본 데이터 공유 없이 최신 LLM 아키텍처로 마이그레이션하는 데 활용할 수 있다.
2. 새로운 세대의 사전학습 모델이 출시될 때마다 기존 **LoRA 어댑터의 전문 지식을 빠르게 이전**하여, 반복적인 파인튜닝 비용과 시간을 절감할 수 있다.
