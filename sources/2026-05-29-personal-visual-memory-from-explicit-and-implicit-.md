## 정의
사용자의 시각적 경험이 텍스트 기반 접근만으로는 달리 기억 시스템에서, 이미지 턴의 명시적/암묵적 증거(사용자 연관 엔티티, 로고/UI 상태, 시각적 습관 등)를 포착하는 기억 시스템. 항상상 텍스트 기반 접근이 이미지에 담긴 사용자 개인 정보를 회피할 수 없게 만드는 근본적 한계를 구조적으로 지적한다.
