# Appear2Meaning: A Cross-Cultural Benchmark for Structured Cultural Metadata Inference from Images

**타입**: 논문  
**출처**: arXiv  
**날짜**: 2026-04-10

## 핵심 요약

Recent advances in vision-language models (VLMs) have improved image captioning for cultural heritage. However, inferring structured cultural metadata (e.g., creator, origin, period) from visual input remains underexplored. We introduce a multi-category, cross-cultural benchmark for this task and evaluate VLMs using an LLM-as-Judge framework that measures semantic alignment with reference annotations.
