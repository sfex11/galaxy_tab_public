# Collaborative Task and Path Planning for Heterogeneous Robotic Teams using Multi-Agent PPO

**타입**: 논문  
**출처**: arXiv  
**날짜**: 2026-04-03  
**링크**: None

## 핵심 요약

이 논문은 이종(heterogeneous) 로봇 팀의 외계 행성 탐사를 위한 협업 작업 및 경로 계획 문제를 다룬다. 기존 고전적 계획 알고리즘은 문제 규모가 커질수록 계산 비용이 급증하는 한계가 있다. 이를 해결하기 위해 **다중 에이전트 근접 정책 최적화(MAPPO)** 기반의 분산 의사결정 프레임워크를 제안하며, 작업 할당과 경로 계획을 동시에 수행한다. 학습 기반 접근법으로 계산 부담을 훈련 단계로 이전시켜 실시간 추론 속도를 확보하고, 임무 중 온라인 재계획(replanning)도 가능하게 한다. 전수 탐색 기반 최적 해와 비교하여 성능을 검증하였다.
